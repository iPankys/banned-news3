 <p>
  2日，
  <span href="https://zh.wikipedia.org/wiki/%E6%9C%9D%E9%B2%9C%E6%B0%91%E4%B8%BB%E4%B8%BB%E4%B9%89%E4%BA%BA%E6%B0%91%E5%85%B1%E5%92%8C%E5%9B%BD">
   朝鲜
  </span>
  平安南道消息人士向DailyNK透露，最近在平安南道文德郡的部分农场，发生了难防治的谷枯病，导致水稻叶子变灰枯死了。由于该病会直接影响收成，使得农民们都十分担心。
  <br>
   <br>
    谷枯病主要是因低温或者频繁的降雨，而致使农作物抵抗力变弱，因而导致滋生谷枯病菌。据了解，频繁地使用氮肥或者水田太过于肥沃，也会引起谷枯病。
    <br>
     <br>
      <strong>
       目前朝鲜处于持续高温干旱状态
      </strong>
      <br>
       <br>
        7月26日，据朝鲜劳动新闻报导称，截至7月中旬为止，全国平均降雨量是21.2mm，其只有平年雨量的25.8%而已，这已经是1981年以来的第2个最少降雨量。如今大部分地区仍持续高温，农作物已经开始受到干旱影响了。
        <br>
         <br/>
         据报纸报导，咸镜北道全域的土壤湿度只有20至50%左右。自7月以来，包括平安南道在内的朝鲜全域都处于持续高温状态。亦即平安南道地区水稻发生谷枯病的主要原因，根本不可能是因为低温和持续降雨。
         <br/>
         <br/>
         消息人士说，在1970年代之时，因为过分使用氮肥，也发生过谷枯病。但今年氮肥使用的非常少，却也发生了谷枯病。所以，很可能是因为过分使用堆肥，而导致水田过度肥沃了。
         <br/>
         <br/>
         据分析认为，当土壤中的水分不足时，且农作物的生理状态并不良好的情形之下，如果过分使用堆肥，将会导致土壤中的氮含量相对地增多，而让水稻患上了谷枯病。
         <br/>
         <br/>
         消息人士称，其实，不仅是文德郡而已，在肃川郡、平原郡等著名粮仓地带，也都出现了谷枯病。病害已经快速扩散至西海岸的水稻生产基地了。
         <br/>
         <br/>
         因为水稻谷枯病必须避免使用氮肥等化肥，并使用药物来进行防治。但因为缺少谷枯病防治的药品，使得都处于束手无策的状态。
         <br/>
         <br/>
         消息人士指出，在水稻发穗之前，就应及时采取措施来防止扩散，可是因为没防治药品，因此才无从下手。在这种情况之下，若天气转阴，受灾情况将会更加严重了，所以都在密切关注事态之发展。因为非常担忧今年大米的收成会受到影响。
         <br/>
         <br/>
         7月29日，美国农业部于《2021/2022年朝鲜各季节收获量展望》的报告中估计，今年朝鲜大米产量为200万吨，其比近5年的大米平均产量，还少了10%左右。
         <br/>
         <br/>
         报告也指出，朝鲜在全球爆发COVID-19疫情之后，随即采行封锁中朝边境措施，导致化肥、除草剂、杀虫剂和农机械零部件等物品进口量少，因而导致了粮食产量降低。
