 <p>
  【看中国2021年10月27日讯】7月27日，
  <strong>
   朝鲜
  </strong>
  领导人
  <strong>
   金正恩
  </strong>
  在第7届全国参战老兵大会上说，
  <strong>
   COVID-19
  </strong>
  疫情所带来之威胁，让我国面临了这个史无前例的世界公卫危机，以及长期制裁带来的困难，好像是当年朝鲜战争的困境一样。据悉，在华朝鲜劳工遭剥削大部分工资。据消息人士披露，为了弥补不足的忠诚资金与去年因疫情发生之损失，竟从工人身上榨取金钱。
  <br>
   <br>
    26日，中国的对朝消息人士向DailyNK透露，朝鲜劳工一般工作至晚上10点到12点，其月薪为2800元至3200元人民币。其中有一家公司每月仅提供50元人民币的生活费，其他的皆作为党资金没收了。
    <br>
     <br>
      <span href="https://zh.wikipedia.org/wiki/%E6%9C%9D%E9%B2%9C%E6%B0%91%E4%B8%BB%E4%B8%BB%E4%B9%89%E4%BA%BA%E6%B0%91%E5%85%B1%E5%92%8C%E5%9B%BD">
       朝鲜
      </span>
      从过去开始自月薪中收取部分的费用，以作为忠诚资金，但
      <span href="https://zh.wikipedia.org/wiki/2019%E5%86%A0%E7%8A%B6%E7%97%85%E6%AF%92%E7%97%85">
       COVID-19
      </span>
      之后就连提供给工人之餐饮费，也被收走了。由于订货量的减少，导致无法完成忠诚资金目标额，而采取这种措施。
      <br>
       <br>
        有部分在华朝鲜工人，他们在凌晨4至6点就上班，也就是每天工作长达16小时以上，其劳动强度非常大，真正是名副其实的“血汗工厂”。
        <br>
         <br/>
         最近剥削着工人工资约高达98%。亦即在高强度与超长时间劳动之情况下，还拿不到工资的一半，而且生活在像奴隶一样的环境之中。
         <br/>
         <br/>
         在此种情形之下，滞留于中国国内的朝鲜工人们都非常大的不满，但哑巴吃黄连也只能上交金钱。
         <br/>
         <br/>
         一名朝鲜工人向DailyNK表示，管理者声称，能够以生命来报效祖国，而上交金钱只是小事一桩，甚至还对工人说，50元人民币在祖国已经是一大笔钱了。
         <br/>
         <br/>
         他还不满地说，因为祖国正处于艰难之中，要求忠诚第一位，虽然相当不愿意，但还是听从安排了。近一年没能够赚钱，在今年才稍微有了点活儿干，但（公司）却要求弥补之前的亏损。亦即举着“忠诚”和“祖国贫困”的名目来剥削工资，但也不能正面反对。
         <br/>
         <br/>
         据消息人士称，在艰难的情况之下，驻华朝鲜贸易公司为了筹备上交给政府之忠诚资金，已经焦头烂额了。为能筹备汇至国内的资金，也只能说服工人，或者迳行压榨。
         <br/>
         <br/>
         消息人士指出，有很多朝鲜社长可能将大部分月薪都上交到党了。其实他们的苦恼很多，需要考虑的，也很多。也就是说，在朝鲜当局的忠诚资金压力之下，驻华朝鲜贸易人员与劳工们都遭受着迫害，可说在相当艰难的环境中过活。
         <br/>
         <br/>
         2月，平壤制定了海外贸易机构之上交党资金，以及处罚的新规定。若连续4个月未完成党资金的任务，将会替换机构、饭店、团体、作业所与工厂基层的负责人，并调回国内。很可能是在此一规定之施压下，使得驻华贸易工作人员们不得不加大了针对工人的压榨力度。
         <br/>
         <br/>
         据了解，有一些工人因党资金，而遭夺走了一直攒下来的金钱。
         <br/>
         <br/>
         有一位朝鲜工人说，从COVID-19发生之后，迄今只给家里汇了一次1000元人民币而已。自2019年开始，攒起来的钱也都成了党资金。
         <br/>
         <br/>
         在此种情况下，大部分朝鲜工人还是尽量省钱来给家里汇款。因为疫情出不了门，没地方可以花钱了。
