 <p>
  <br>
   上星期，布查惨案的图片和视频不断曝光，虽然莫斯科矢口否认，但世界舆论纷纷谴责俄罗斯。在俄罗斯军队从基辅以北的一系列村庄被击退后，人们发现了平民的尸体倒在街上。
   <br>
    <br>
     布查镇长阿纳托利·费多鲁克(Anatoliy Fedoruk)对美国之音(VOA)俄语组说，目睹街头陈尸和城镇遭到的破坏让他感到震惊。
     <br/>
     <br/>
     他说：“做为历史学者，我以为即使是在战争期间有些规则也是会被遵守的。”他补充说，重建城镇的基础设施需要很长时间，那些战争开始时逃离俄军占领的人，现在就返乡还为时过早。
     <br/>
     <br/>
     美国和西方盟友对布查和其他乌克兰城镇的杀戮事件开始了战争罪调查。美国总统乔·拜登(Joe Biden)说，俄罗斯总统弗拉基米尔·普京(Vladimir Putin)应当作为战争罪犯接受审判。
     <br/>
     <br/>
     一名德国情报官员说，柏林掌握的卫星图像显示俄罗斯参与了布查的杀戮，不过尚未确定那段无线电通信与那处地点之间的关联。德国外国情报局(BND)也许能够把这些卫星图像与截获到的这段无线电通讯联系在一起。
     <br/>
     <br/>
     官员们说，无线电通讯似乎显示与普京及其盟友关系密切的私人军事组织瓦格纳集团(Wagner Group)或者另一个私家承包商有可能参与了对平民的袭击。
     <br/>
     <br/>
     随着俄罗斯过去六个星期来不断攻击乌克兰，西方防务和情报分析人士说，俄军依靠未加密的通讯设备，包括智能手机和对讲机，这让他们容易成为被打击的目标。
