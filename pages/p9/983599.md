 <p>
  【看中国2021年9月13日讯】据了解，
  <strong>
   <span href="https://www.secretchina.com/news/gb/tag/朝鲜" target="_blank">
    朝鲜
   </span>
  </strong>
  为了提高国家对贸易之管控力度，且扩大物流量，而计划于
  <strong>
   边境地区
  </strong>
  建立新的机构，但在
  <strong>
   中国
  </strong>
  的反对之下，该计划将面临取消的危机。所以，朝鲜非常难在以往的朝中贸易地区之外来建立新的贸易地区。
  <span id="hideid" name="hideid" style="color:red;display:none;">
   <span href="https://www.secretchina.com">
   </span>
  </span>
 </p>
 <p>
  <br>
   10日，根据
   <span href="https://zh.wikipedia.org/wiki/%E6%9C%9D%E9%B2%9C%E6%B0%91%E4%B8%BB%E4%B8%BB%E4%B9%89%E4%BA%BA%E6%B0%91%E5%85%B1%E5%92%8C%E5%9B%BD">
    朝鲜
   </span>
   内部消息人士透露，上个月末，由党中央、贸易机构负责人、国家保卫省，以及对外经济省等机构干部所组成的小组，已经结束了边境地区的现场考察。
   <br>
    <br>
     之前，DailyNK就曾经报导过，自7月末开始，以负责贸易工作的干部为中心所组成的小组来到边境地区，以寻找能够成为朝中贸易新的桥头堡之地区。
     <br>
      <br>
       该小组前往平安北道、慈江道、两江道和咸镜北道等边境地区，以查看是否适合朝中贸易的地区，还查看有没有打击走私贸易之死角。
       <br>
        <br>
         消息人士称，现今朝鲜的国家贸易，仅限于平安北道新义州与南浦特别市，非常难以扩大物流量，有鉴于此，朝鲜当局认为有必要新设贸易的据点。
         <br/>
         <br/>
         据悉，小组通过现场考察结果，选定了慈江道与平安道等几个地区。但由于中方对此并没有明确表态，以致暂时中断了计划。
         <br/>
         <br/>
         朝鲜表明了于边境地区新设与中国进行贸易地区，并建设海关、防疫设施等计划。然而中国却表示需要进一步探讨，似乎表明了否定的立场。
         <br/>
         <br/>
         据了解，此种事情必须由中朝双方进行事先磋商，但朝鲜却先展开考察之后，再单方面地通报给北京，中国对此相当不满。
         <br/>
         <br/>
         朝鲜为了能够根除走私，而选定了人员流动少的地区来作为新的贸易据点，可是与这些地区相邻的中国地区，亦属于人迹罕至之处，因此中国方面也需要建设新的海关与防疫设施等。
         <br/>
         <br/>
         如今该计划可说是面临破灭的危机，熟知朝鲜内部情况的官员们指称，在元首的上边有习近平主席。而劳动党的上边还有一个中国共产党，在此可以感觉到了北京的威力。
         <br/>
         <br/>
         在这种情况之下，另有传闻说称，在今年年末之前，仅允许国家批准的少数贸易机构来进行贸易，而在明年的春节或者光明星节（指2月16日，亦即金正日诞辰）之前所批准取得贸易许可的机构与个人进行贸易。
        </br>
       </br>
      </br>
     </br>
    </br>
   </br>
  </br>
 </p>
 <p>
  事实上，党中央亦制定了这种计划，很可能是由消息灵通的干部们将消息透露给了贸易机构。但尚不清楚允许有贸易许可的机构或者个人进行贸易活动是暂时的措施，或者是以光明星节为起点来继续允许搞贸易。
  <br/>
  <br/>
  在明年1月或者2月初扩大贸易许可的计划传开之后，各地贸易机构与个人已开始展开贿赂党干部，以查探党与政府需要的进口物品的种类。
  <br/>
  <br/>
  向党申请贸易许可之时，必须申报进口物品的种类，因此能够事先知道党需要的东西，就会有助于贸易许可申报。
  <br/>
  <br/>
  而贸易人员所谈论的物品，大都是建设和耕农资材，以及在明年的光明星节与太阳节（指4月15日，其为金日成诞辰）等节日时供应给干部的工业产品等。
  <br/>
  <br/>
  消息人士称，重要的并不是大家皆知道的必需品，而是要找到党需要，但没有太多机构表示要进口之物品。为了能够得到国家许可，所以竞争很激烈。
