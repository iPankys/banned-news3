 <p>
  【看中国2022年6月9日讯】
  <strong>
   <span href="https://www.secretchina.com/news/gb/tag/加拿大" target="_blank">
    加拿大
   </span>
   移民部长
  </strong>
  弗雷泽（Sean Fraser）6月7日公布，7月4日起实施
  <strong>
   父母祖父母探亲
  </strong>
  的
  <strong>
   超级签证
  </strong>
  新措施，签证持有人逗留加国时间一次可长达7年。
  <br>
   <br>
    移民部数据显示，加拿大每年签发大约17,000份超级签证。新政规定，超级签证持有者每次入境加拿大的停留时间从2年延长至5年，并且可选择在加拿大延长逗留最多2年。这意味着超级签证持有者可以在加拿大连续居住7年。
    <br>
     <br>
      移民部长弗雷泽说：“家庭是加拿大社会的核心。超级签证计划的改进使家庭成员可以在加拿大更长时间团聚，这有助于加拿大公民和永久居民取得成功并为社会做出贡献，同时为他们的父母和祖父母提供宝贵的机会，让他们在加拿大与家人共度时光。”
      <br>
       <br>
        新政还允许移民、难民和公民事务部长指定国际医疗保险公司在未来为超级签证申请人提供保险。
        <br>
         <br>
          超级签证是一种多次入境签证，有效期最长为10年。
          <br>
           <br>
            目前，只有加拿大保险提供商可以提供超级签证申请人必须拥有的必要医疗保险。有关位于加拿大境外的任何指定医疗保险公司的信息将稍后在移民部（IRCC）的网站上公布。
            <br/>
            <br/>
            这些改革最初由加拿大保守党国会议员塞贝克（Kyle Seeback）提出。
            <br/>
            <br/>
            过去的十几年，因PGP名额有限，无法满足移民希望父母祖父母来加探望和居住的需求，超级签证就充当了PGP的替代方案。
            <br/>
            <br/>
            父母和祖父母团聚移民计划（ Parents and Grandparents Program，简称PGP），是指计划加拿大公民和永久居民能够担保其父母和祖父母有资格且顺利移民加拿大。获得批准的父母和祖父母能够获得合法加拿大永久居留权，并有资格申请加拿大公民身份。
            <br/>
            <br/>
            根据2022年至2024年加拿大移民规模的计划，2022年PGP共有2万5千个名额，2023年2万8千5百个名额，2024年增加至3万2千。
            <br/>
            <br/>
            资助父母和祖父母来加的子女或孙子女必须满足最低收入要求。
            <br/>
            <br/>
            移民部迄今尚未发出有关2022年PGP计划的更新。
