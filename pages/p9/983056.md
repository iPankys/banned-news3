 <p>
  【看中国2021年9月7日讯】朝鲜迄今仍宣称该国仍无
  <strong>
   COVID-19
  </strong>
  确诊病例，但外界却持相当怀疑态度。据多数消息人士披露，
  <strong>
   朝鲜
  </strong>
  部队内的大批COVID-19疑似症患者，病房发生集体死亡情事，让人非常惊讶。 当该事件报告给到
  <strong>
   <span href="https://www.secretchina.com/news/gb/tag/金正恩" target="_blank">
    金正恩
   </span>
  </strong>
  国务委员长时，据说金委员长立刻大发雷霆。因为于第八届第三次政治局扩大会议上，决定加强COVID-19防疫力度。
  <span id="hideid" name="hideid" style="color:red;display:none;">
   <span href="https://www.secretchina.com">
   </span>
  </span>
 </p>
 <p>
  6日，江原道部队内部消息人士向DailyNK透露，7月与8月，第二军团指挥部之传染病临时病房中，发生了45名士兵死亡。这个数字反映于军医局的《2020至2021年作战和战斗政治训练期间死亡者统计》之中，目前病房已被封闭了，其内患者皆移送至其他地方。
  <br>
   <br>
    4日，南浦的部队消息人士也说，第三军团
    <span href="https://zh.wikipedia.org/wiki/2019%E5%86%A0%E7%8A%B6%E7%97%85%E6%AF%92%E7%97%85">
     COVID-19
    </span>
    病房将第一二期训练期间医院的死亡统计人数上报给军医局，而捅了这个马蜂窝。亦即死亡人数比预计的还多，而上头也不断斥责医院。
    <br>
     <br>
      更大的问题是，因为综合统计中全军的死亡人数，也比预计的还要多，而让部队当局非常震惊。
      <br>
       <br>
        其中死亡人数最多的，是第一与第二军团等驻扎于朝中边境地区之部队，其次是后方军团与训练所等。其中COVID-19疑似症临时病房的死亡人数，最多的为第二与第三军团。
        <br>
         <br/>
         消息人士称，虽然
         <span href="https://zh.wikipedia.org/wiki/%E6%9C%9D%E9%B2%9C%E6%B0%91%E4%B8%BB%E4%B8%BB%E4%B9%89%E4%BA%BA%E6%B0%91%E5%85%B1%E5%92%8C%E5%9B%BD">
          朝鲜
         </span>
         部队当局制定了强有力之防疫措施，但还是出现了许多死亡人员，因此大惊失色。对此，
         <span href="https://zh.wikipedia.org/wiki/%E9%87%91%E6%AD%A3%E6%81%A9">
          金正恩
         </span>
         委员长更是大发雷霆，所以不得不进一步再强调防疫了。
         <br/>
         <br/>
         COVID-19疑似症病房发生集体死亡悲剧之因，首先是就因为“安逸的对策”。
         <br/>
         <br/>
         去年为止原先以为COVID-19会很快结束了，各部队的临时措施，就是在部队内的一个仓库内摆床具来隔离疑似症患者。
         <br/>
         <br/>
         从去年12月1日起的冬季训练中，前沿部队内就发生了有人员发烧，为了进行集体隔离，而在离部队较远的山沟搭建了临时建筑，亦即COVID-19隔离措施，但并无按规定进行，且还无明确的诊断与治疗。
         <br/>
         <br/>
         消息人士透露，不论是患有肺结核、肝炎或者感冒，都未进行正确的诊断，就强制将病患关进了临时病房。而后也没有任何医学与病理学上的治疗，每日仅量一次体温，以及晒太阳1次，且也没有正规之营养供应。消息人士披露，给隔离病房的士兵，每日供应450g凉透的杂粮饭与白菜盐水汤。
         <br/>
         <br/>
         不能够公开临时病房的各部队，试图想隐瞒有隔离病房之事实，在7月初时，通知家属称“因事故死亡”。但家属要求应该颁发“战死者证书”，若部队不给，就上报给了国防省。
         <br/>
         <br/>
         消息人士表示，由于在国防省之的严厉追问下，各部队才上报真实的情况，有一些部队家属听说了消息之后，相当担心在集体兵营生活的情况下，无法保障不会再次发生集体死亡事件。
         <br/>
         <br/>
         根据内部的规定，各部队必须于结束一年训练的一个月之前，亦即至8月为止，通过军医局上报全军的人数、住院的人数和训练中的死亡人数，以及原因等。
