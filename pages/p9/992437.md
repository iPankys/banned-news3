 <p>
  根据DailyNK报导，
  <span href="https://zh.wikipedia.org/wiki/%E6%9C%9D%E9%B2%9C%E6%B0%91%E4%B8%BB%E4%B8%BB%E4%B9%89%E4%BA%BA%E6%B0%91%E5%85%B1%E5%92%8C%E5%9B%BD">
   朝鲜
  </span>
  从今年1月至12月8日为止，针对使用中国手机的情况进行打击，据悉遭两江道保卫局逮捕的百姓，就已经达几百名之多。其中大部分为20至40多岁女性，他们在两江道保卫局拘留所内接受调查之后，就送进了教化所（劳改所）或者政治犯收容所。
  <br>
   <br>
    据调查指出，道保卫局拘留所的调查官与狱警们，经常针对遭捕人员采行了倒挂、电击、暴晒和受冻等酷刑，甚至还随时进行性侵。
    <br>
     <br>
      13日，根据DailyNK朝鲜内部的消息人士透露，4月中旬，金贞淑郡的30多岁金某（女），她与中国的亲人通话之时，被电波探测仪发现了，郡保卫部立刻进行了入户逮捕。
      <br>
       <br>
        据了解，金某于金贞淑郡保卫部内接受了7天的调查，最后还遭扣上间谍嫌疑，而移送至道保卫局拘留所。但到达了道保卫局拘留所的第一天，所方就在她的两腿之间放了一根方木，竟然让她跪着接受调查。第二天更惨，还她倒挂起来，更把她的头放进装满水的桶里，强逼她承认了间谍之嫌疑。
        <br>
         <br>
          另外，惠山市20多岁的朴某（女），她在5月替定居于韩国的脱北民转交汇款之时，遭市保卫部逮捕了。
          <br>
           <br/>
           朴某接受了7天的调查，且在微信上发现了她与脱北民针对粮食价格、外汇兑换率等内部消息的聊天记录，而被扣上间谍之嫌疑，遭移送到道拘留所。
           <br/>
           <br/>
           在这里接受调查的过程之中，朴某一到晚上就遭到狱警性侵。
           <br/>
           <br/>
           据说，朴某在反复不断地遭到性侵，期间她一直大声喊叫，并进行了强烈反抗，因而被狱警打破了头及打断肋骨等，最后因为伤势太严重而失去了生命。
           <br/>
           <br/>
           如以上所述，遭关在两江道拘留所的百姓们，他们被调查官与狱警们的性侵及殴打，但谁也不会追究他们的责任。主要因为关进拘留所的人员，皆遭扣上了相当于叛国罪的间谍嫌疑。
           <br/>
           <br/>
           消息人士称，道保卫局拘留所内严守秘密的水准与政治犯收容所，可说是不相上下。因为在里边所发生的任何事，对外都是保密的，以致里边发生了惨不忍睹的暴行。
           <br/>
           <br/>
           报导指出，朝鲜民间曾流传着一段话，“一夜之间，右边的邻居遭人带走了；又一夜之间，左边邻居亦被人带走了”，由于朝鲜特务与间谍遍布，甚至就连家人之间的对话也都必须小心翼翼。此种情况在COVID-19疫情流行期间更加剧，人们遭抓走受尽虐待与折磨，在监狱和政治犯收容所内，几乎天天传出了严刑拷问与性虐待的恐怖事件。

         </br>
        </br>
       </br>
      </br>
     </br>

   </br>
