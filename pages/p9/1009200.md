  【看中国2022年6月15日讯】卢甘斯克州州长谢尔盖·盖达伊(Serhiy Haidai)星期二(6月14日)说，

   俄罗斯
  </strong>
  目前控制了
  <strong>
   乌克兰
  </strong>
  东部工业城市
  <strong>
   北顿涅茨克
  </strong>
  大约80%的地段，并摧毁了出城的所有三座桥梁。
  <br>
   <br>
    盖达伊承认，俄罗斯摧毁了桥梁之后，从
    <span href="https://zh.m.wikipedia.org/zh/%E5%8C%97%E9%A0%93%E6%B6%85%E8%8C%A8%E5%85%8B" target="_blank">
     北顿涅茨克
    </span>
    大规模转移平民如今“根本不可能了”，因为俄军在不停地无情炮击和攻打这座城市。
    <span id="hideid" name="hideid" style="color:red;display:none;">
     <span href="https://www.secretchina.com">
     </span>
    </span>
   </br>
  </br>
 </p>
 <p>
  他说，由于“
  <span href="https://www.secretchina.com/news/gb/tag/俄罗斯" target="_blank">
   俄罗斯
  </span>
  人https://zh.m.wikipedia.org/zh/%E5%8C%97%E9%A0%93%E6%B6%85%E8%8C%A8%E5%85%8B使用焦土方式和重炮轰击”，乌军已被逼退至城市郊外。
  <br>
   <br>
    但是盖达伊对美联社说，俄军还没有封死进入城市的渠道，使乌克兰有“机会转移伤员并与乌军和当地居民通讯”。
    <br>
     <br>
      这座城市战前有10万人口，目前仍有大约1.2万人留在城内，包括在亚速特化工厂栖身的500名平民。俄军正在炮击这座工厂。
      <br>
       <br/>
       俄罗斯大将米哈伊尔·米津采夫(Mikhail Mizintsev)说，星期三将开通一条人道走廊，以转移化工厂内的平民，但是他们将被带往俄军与分离主义武装控制的城镇斯瓦托。
       <br/>
       <br/>
       俄罗斯在争夺顿巴斯地区控制权的战斗中似乎正在缓慢然而却步步紧逼式地攻城略地。顿巴斯包括乌克兰的卢甘斯克和顿涅茨克两州，俄罗斯宣布承认两地为独立国家。
       <br/>
       <br/>
       “局势是困难的，“乌克兰总统弗拉基米尔·泽连斯基(Volodymyr Zelenskyy)星期二与丹麦媒体举行的记者会上说：“我们的任务是反击。”
       <br/>
       <br/>
       泽连斯基说，顿巴斯之战“肯定将成为欧洲军事史上最为暴力的战斗之一”。
       <br/>
       <br/>
       泽连斯基说：“对我们来说，这场战斗的代价非常高。实在是吓人。”
       <br/>
       <br/>
       他再次呼吁伙伴国家向乌克兰提供现代火炮，以抵抗俄军。
       <br/>
       <br/>
       泽连斯基的顾问米哈伊洛·波多里亚克(Mykhailo Podolyak)星期一说，乌克兰正在等待星期三在北约布鲁塞尔总部参加美国领导的联络组会议的国家做出加大军援的决定。
      </br>
     </br>
    </br>
   </br>
  </br>
 </p>
 <p>
  波多里亚克发推说，“要结束战争，我们必须有重武器均势”。他提到了几类武器，包括1千门榴弹炮、300套多管火箭发射系统、500辆坦克、2千辆装甲车和1千架无人机。
  <br/>
  <br/>
  俄罗斯声称已经控制了北顿涅茨克所在的卢甘斯克州97%的地区，但是攻占这座工业城市仍然是莫斯科实现其控制顿巴斯的更大目标的关键。
  <br/>
  <br/>
