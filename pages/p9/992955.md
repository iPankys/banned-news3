 <p>
  【看中国2021年12月21日讯】
  <strong>
   朝鲜
  </strong>
  在领导人
  <strong>
   金正恩
  </strong>
  统治之下，
  <strong>
   处决
  </strong>
  似乎已司空见惯。据悉，有3名
  <strong>
   金策工业综合大学
  </strong>
  的优秀男
  <strong>
   大学生
  </strong>
  ，由于批判政府的思想教育强化政策，而在本月初遭秘密处决。
  <br>
   <br>
    16日，据
    <span href="https://zh.wikipedia.org/wiki/%E6%9C%9D%E9%B2%9C%E6%B0%91%E4%B8%BB%E4%B8%BB%E4%B9%89%E4%BA%BA%E6%B0%91%E5%85%B1%E5%92%8C%E5%9B%BD">
     朝鲜
    </span>
    内部消息人士向DailyNK透露，
    <span href="https://zh.wikipedia.org/wiki/%E9%87%91%E7%AD%96%E5%B7%A5%E6%A5%AD%E7%B6%9C%E5%90%88%E5%A4%A7%E5%AD%B8">
     金策工业综合大学
    </span>
    秀才组的3位男大学生，因为批判了时常召开会议的党政策。以致被印上现代修正主义宗派分子之烙印，在12月5日已遭秘密执行枪决。
    <br>
     <br>
      据消息人士表示，11月初，有3个金策工业综合大学秀才组的好朋友，在参加完电影文献学习及观后感发表会之后，回到宿舍随即批判了政府政策。
      <br>
       <br>
        据了解，他们说，宣传说今日的我国是一个东方的核强国，但心里却想着我国并不是核强国，而是一个会议强国。
        <br>
         <br>
          他们批指，应该给予我们提供专心读书之条件，但只召开毫无用处的会议，甚至开会的时间还比上课的时间多，根本没法能够好好学习。
          <br>
           <br/>
           对政府而言，这是一种批判
           <span href="https://zh.wikipedia.org/wiki/%E9%87%91%E6%AD%A3%E6%81%A9">
            金正恩
           </span>
           国务委员长亲自所强调的加强青年思想教育的行为，可是大学生们并没太在意。
           <br/>
           <br/>
           当时现场有一名盖着棉被躺在床上的大学生，他听到了此种言论之后，告发到大学保卫部，使整件事情就变大了。
           <br/>
           <br/>
           每当召开政治活动之时，秀才组的这3位学生就时常对这名普通大学生冷嘲热讽地说，怎么会让秀才组与普通学生一起来参加会议啊？应该要区分开来才行啊。自尊心遭到伤害的这名同学一直都想找机会进行报复，正好听到了他们的这种言论。
           <br/>
           <br/>
           他们在大学保卫部接受调查的言论亦出现问题。
           <br/>
           <br/>
           保卫员说，你们知道会议强国的这种言论，有多么危险吗？但他们却毫无顾忌地回答，什么样的会议一天竟然要开三次啊？到外边的研究所进行研究未能接到通知，不就要被扣上了反动罪名吗？
           <br/>
           <br/>
           “如果说会议强国是我们错了，但秀才组仅是口头上的秀才组而已，作业还比别的同学多，也让我们与普通学生一样参加每次会议，这是你们做错了。”
           <br/>
           <br/>
           尚未等到大学党委员会插手，此件事就被上报至国家保卫省。11月中旬，这3名学生就在大学生们聚集的大会议室内，遭逮捕拉走了。
           <br/>
           <br/>
           当中央国家保卫省进行调查的时候却发现，虽然这3位学生出生地与生活经历各有不相同，不过有一个共同点。即是他们从小开始就特别喜欢外国的电影、歌曲和舞蹈等，甚至因为这个还遭到处罚。但因学习特别优秀，所以大学推荐书中并没有写进被处分之经历。
           <br/>
           <br/>
           对此，国家保卫省就下结论认为他们的基本思想有问题，虽然他们是人才，可是若默认下去，到了国外时这种思想就会发芽，非常可能就会流亡或者成为一个脱北民，因此决定斩草除根。
           <br/>
           <br/>
           在国家保卫省得到了中央的批准之后，在12月4日就通知召开了“全体教职员工与学生的公开斗争大会”。可是却又突然取消大会，并于5日迳行将他们秘密枪决。
           <br/>
           <br/>
           消息人士称，原先打算要召开思想斗争大会，以强调在国外留学时逃亡的学生，都是讨厌组织生活，且思想发生动摇之分子。不过可能是考虑到了金策工业综合大学是一所党最信任的大学，深怕影响大学之形象，所以才取消斗争大会。消息人士还说，当时正是金正恩父亲金正日逝世（12月17日）第10周年的哀悼期，所以党尽量不进行公开处决及审判的指示，所以进行了秘密处决。
           <br/>
           <br/>
           据了解，他们的家人皆因没有将孩子培养成一个信念坚定的思想强者之罪名，而被送进了社会安全省的第18号管理所（亦即平安南道北仓政治犯收容所），这就是朝鲜的连坐制。
           <br/>
           <br/>
           据消息人士称，在事件发生之后，金策工业综合大学教职员工与学生们都声称，学习再怎么好，若管不住三寸不烂之舌，也会致使全家遭受灭顶之灾。今年是金委员长领导第10周年，所以处罚更加严厉了。
