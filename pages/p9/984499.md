   大米

  <br>
   <br>
    17日，根据DailyNK的中国消息人士透露，最近
    <span href="https://zh.wikipedia.org/wiki/%E6%9C%9D%E9%B2%9C%E6%B0%91%E4%B8%BB%E4%B8%BB%E4%B9%89%E4%BA%BA%E6%B0%91%E5%85%B1%E5%92%8C%E5%9B%BD">
     朝鲜
    </span>

    <span href="https://zh.wikipedia.org/wiki/%E7%A8%BB">
     大米
    </span>
    ，但手续却没办好。
    <br>
     <br>
      据消息人士称，朝鲜贸易公司已经通过中国方面的交易伙伴确保了大米，但中国当局并未着手出口的行政手续。亦即中国政府已经严管了对于朝鲜出口大米的事情，没有经过当局的许不可以出口粮食。
      <br>
       <br>
        据悉，朝鲜贸易公司的人员已将船停靠于辽宁省大连港，并做好了装船准备。
        <br>
         <br/>
         中国的交易伙伴向当地的业内人士说，将大米运到港口必须支付10万元人民币，但因为北京政府加强了打击力度，因此没有人愿意给运输。
         <br/>
         <br/>
         目前于中国搞贸易的朝鲜机构，是在党中央批准之下，并利用党资金来开展活动，如果取消交易或者推延程式，就必须向上级进行报告与说明，很难可以逃避责任。

   朝鲜

  政府进行贸易的时候，朝鲜都会预先支付物品价格的30%～50%，以作为定金，当拿到全部物品之后，再支付余额。据了解，因为已通过中国交易伙伴来确保了大米，因此平壤方面肯定也支付了定金。
  <br/>
  <br/>
  在没有拿到物品的情况之下，就支出了党的资金，所以若进口大米的工作泡汤或者遭到推延，对于贸易工作人员的处罚亦会加重。
  <br/>
  <br/>
  一名在海外工作的朝鲜专家指出，最近中国政府采取了禁止与朝鲜进行非正式交易之措施。仅允许通过政府间的正式管道来达成协议的交易。
  <br/>
  <br/>
  专家也说，除了大米之外，北京政府屡次禁止了朝鲜贸易公司透过中国的交易伙伴非正式进口的物品。中国似乎正在向朝鲜传递应该通过正式程序来进行贸易之资讯。
  <br/>
  <br/>
  在此种情况下，朝鲜贸易工作人员们指称，在封锁边境之后，中国对于朝鲜的威力已经更大了，“中国好像是在驯服朝鲜”。
  <br/>
  <br/>
  但尚不清楚最近中国所采取的对朝出口相关措施，是一时的或者是长期的。有人担忧，若中国方面继续坚持这种原则，即使朝鲜当局解除封锁，而正常开展贸易，那么朝中之间的贸易，亦不会很顺畅。

 <h2>
  <strong>
   朝鲜出现异常气象
  </strong>
  <strong>
   致粮食更加雪上加霜
  </strong>
 </h2>
 <p>
  朝鲜自从去年COVID-19疫情爆发以来采行关闭边界，但却导致粮食短缺日益严重。尽管该国当局试图自给自足，可是收效甚微。据《日经亚洲评论》（Nikkei Asian Review）稍早的报导，朝鲜配给系统已经濒临崩溃，粮食短缺的情况已逼近人道危机了。
 </p>
 <p>
  继去年台风不断，以及梅雨期异常致使农获量大减的情形。然而，今年初秋将进入收获期之际，却出现异常的气象，降下了直径4公分的冰雹，又加上狂风暴雨肆虐，导致稻米收获量将可能因此而减少5至6成，甚至9成之多，导致粮食问题更加雪上加霜。
