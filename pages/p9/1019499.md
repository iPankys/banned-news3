  <strong>
   香港示威者
  </strong>
  被拉入中国驻曼彻斯特总领事馆遭
  <strong>
   殴打
  </strong>
  事件演变成外交事件，引发英国议会讨论，有多名议员要求英国政府驱逐
  <strong>
   中国
  </strong>
  驻曼彻斯特
  <strong>
   总领事
  </strong>
  。
  <br>
   <br>
    英国就上星期天香港示威者遭殴打事件，传召中国驻英国大使馆临时代办，并将调查事件，以决定会否有跟进行动。
    <br>
     <br>
      在上星期天，一个港人组织抗议中国共产党举行第二十次党代表大会，期间多名男子从领事馆中走出来，毁坏标语及带走讽刺
      <span href="https://www.secretchina.com/news/gb/tag/习近平" target="_blank">
       习近平
      </span>
      的画像，他们又尝试将两名示威者拉到领事馆内，其中一名示威者Bob被拉到领事馆围栏内遭拳打脚踢，全身多处受伤。
      <span id="hideid" name="hideid" style="color:red;display:none;">
       <span href="https://www.secretchina.com">
       </span>
      </span>
     </br>
    </br>
   </br>
  </br>
 </p>
 <p>
  在10月18日星期二，英国议会就这宗事件进行紧急质询。英国议会外交事务委员会新任主席、保守党议员凯恩斯（Alicia Kearns）指控其中一位有份袭击香港示威者的人是中国驻
  <span href="https://zh.wikipedia.org/wiki/%E6%9B%BC%E5%BD%BB%E6%96%AF%E7%89%B9" target="_blank">
   曼彻斯特
  </span>
  总领事郑曦原。凯恩斯和多位议会议员要求英国政府宣告郑曦原为不受欢迎人物（Persona non grata），并把郑曦原驱逐出境。
 </p>
 <p>
  凯恩斯说：“我们不能接受中国共产党引入他们攻击
  <span href="https://www.secretchina.com/news/gb/tag/示威者" target="_blank">
   示威者
  </span>
  、打压言论自由的做法。”
  <br>
   <br>
    英国外交事务部部长诺曼（Jesse Norman）指出，十分关注事件，大曼彻斯特警方（Greater Manchester Police）正在进行调查。
    <br>
     <br>
      他说：“外相已经传召中国驻伦敦大使馆的临时代办，表达英国政府就事件的深切关注，并要求就领事馆职员的行为的解释。”
      <br>
       <br/>
       诺曼说，在调查完结之前不宜评论事件细节，英国内政部将与大曼彻斯特警方合作，决定下一步行动。但诺曼强调，和平示威是英国社会及生活方式的“基本组成部份”。
       <br/>
       <br/>
       凯恩斯及多名议员继续要求英国政府驱逐涉案人士出境，包括前保守党党魁伊恩·邓肯·史密斯（Iain Duncan Smith，又译施志安）。史密斯引述事件中受伤的示威者Bob对警方作出的证供说，在闸口的保安扯了他进去，拉扯他的手和头发，拳打脚踢他，包括有四人最少踢了他一分钟，导致Bob的脸、头、手、身及背部受伤，现在坐下也觉得痛。
       <br/>
       <br/>
       伊恩·邓肯·史密斯说：“政府需要行动，回答我的同事一个简单的问题：外相会否不只是传召临时代办，而是直接传召大使？第二，如果有证据证明事件，外相是否就会驱逐总领事及任何有参与殴打及毁坏的人？”
       <br/>
       <br/>
       诺曼则重复指需要经调查后确认事实，才可能作出检控。
      </br>
     </br>
    </br>
   </br>
  </br>
 </p>
 <p>
  代表
  <span href="https://www.secretchina.com/news/gb/tag/中国" target="_blank">
   中国
  </span>
  驻曼彻斯特总领事馆所在区域的工党议员阿夫扎尔・汗（Afzal Khan）说，他对殴打事件感到“呕心”，他亦要求英国政府宣告总领事郑曦原为不受欢迎人物。
 </p>
 <p>
  他说：“英国支持自由、法治以及民主。在英国土地上打压和平示威是永远不能接受的。”
 </p>
 <p>
  中国外交部发言人汪文斌10月18日星期二在例行记者会上说，“滋扰分子非法进入中国驻曼彻斯特总领馆，危及中国外交官舍安全，任何国家的外交机构都有权采取必要措施，维护馆舍的安宁和尊严。”
  <br/>
  <br/>
  汪文斌又说：“我要强调的是，中国驻外使领馆的安宁和尊严不容侵犯，希望英方切实履行责任，采取有效措施，加强对中国驻英使领馆馆舍和人员的保护。”
  <br/>
  <br/>
  英国大曼彻斯特警方（Greater Manchester Police）证实，正在调查事件，目前未有人被捕。警方指，事发当日，有大约30至40人在中国领事馆外聚集，当时有警员在场巡逻。在接近下午四时，中国
  <span href="https://www.secretchina.com/news/gb/tag/总领事" target="_blank">
   总领事
  </span>
  馆内有一小群人走出来，一名年约30岁的男子被拖入总领事馆范围内，并遭到袭击。有警员担心该男子安危，上前介入并把他移离总领事馆范围。
  <br/>
  <br/>
  大曼彻斯特警方说，该名男子身体多处受伤，当晚需要留院接受治疗。警方呼吁在场目击者提供相关资料。
 </p>
