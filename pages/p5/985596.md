  <strong>

  <strong>
   经济
  </strong>
  复苏面临危险。
  <br>
   <br>
    据美国之音报导，他们警告说，中国的能源紧缩对全球经济造成的影响，将超过欧洲的
    <span href="https://zh.wikipedia.org/zh/能源" target="_blank">
     能源
    </span>
    紧缩。
    <br>
     <br>
      中国正面临十年来最严重的电力短缺，有些地区正在限制工厂的能源，三分之二的中国上周都经历了停电。

 <p>
  <span href="https://www.secretchina.com/news/gb/tag/能源" target="_blank">

 </p>
 <p>
  他们说，北京政府为了实现降低化石燃料能源的目标而采取的惩罚措施加剧了能源紧张，中国承诺在2030年前把碳排放减少65%。中共地区官员9月份被告知，如果碳减少目标没有完成，他们将被追究责任。
  <br>
   <br>
    发改委称，中国20个省没有实现2021年的减排目标，包括广东和江苏。数据显示，这两个省占中国
    <span href="https://zh.wikipedia.org/wiki/经济" target="_blank">
     经济
    </span>
    产出的三分之一还多。
   </br>
  </br>
 </p>
 <h2>
  <strong>
   工厂受到影响
  </strong>
 </h2>
 <p>
  美国之音报导称，中国工厂的电力需求暴增，煤矿产量低于预期，造成中国能源价格今年增长了一半以上。中国31个省级地区有20个正在限电，影响到铝、钢、水泥和化肥的生产。电力供应商要求用电大户降低白天的电力消耗，要求东北的工厂每隔几天就轮流停电。
  <br>
   <br>
    日本野村证券上周警告称，全球
    <span href="https://www.secretchina.com/news/gb/tag/经济" target="_blank">
     经济
    </span>
    可能遭遇“另一次重大供给侧冲击波”，因为风险“被低估或被忽视了”。野村和高盛集团都降低了中国的经济增长预估，表示中国经济这个季度将收缩0.2%，以后的增长也会低于预期。
    <br/>
    <br/>
    经济学家警告说，这个预估可能会减缓全球疫情之后的经济复苏，导致通胀加剧。新加坡华侨银行经济学家李浩（Howie Lee）说，中国能源紧缩“将对世界各地产生涟漪效应”。
    <br/>
    <br/>
    政府数据显示，中国工厂活动上个月降到2020年2月疫情封锁重创经济以来的最低点。
   </br>
  </br>
 </p>
 <p>
  彭博新闻社上周报导说，副总理韩正命令国有能源企业确保发电厂的天然气和煤炭供应。使用天然气和煤炭的火力发电厂提供中国一半以上的电力。官员们在上周的一次紧急会议中警告说，停电可能会持续到明年3月。
  <br/>
  <br/>
  报导称，尽管中国是世界上最大的煤炭生产国，但中国10月4日只有14天的煤炭储备，需要增加煤炭进口。北京当局去年禁止从世界最大煤炭输出国澳大利亚进口煤炭，这项禁令部分增加了煤炭进口的难度和成本。
  <br/>
  <br/>
  北京禁止进口澳大利亚的煤炭是报复堪培拉要求对COVID-19
  <span href="https://www.secretchina.com/news/gb/tag/病毒" target="_blank">
   病毒
  </span>
  起源进行国际调查。
  <br/>
  <br/>
  由于北京命令国有能源公司确保供应，全球能源市场可能会爆发煤炭和天然气供应的竞价战争，提高全球的能源价格。伦敦经济研究咨询公司凯投宏观（Capital Economics）上周对客户说：“电力短缺近期似乎不可能缓解。”
  <br/>
  <br/>
  凯投宏观资深中国经济事务专家朱利安·埃文斯·普里查德（Julian Evans-Pritchard）说，“中国电力短缺体现全球能源市场的紧张，不会一夜之间解决”，“限电将制约产能，直到需求足够减弱，让国内电力市场重归平衡”。
