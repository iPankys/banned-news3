  1 继马云遭罪，企业家孙大午及其家人被捕、集团被强制接管后，南京前首富杨宗义，本周二(17日)被指涉嫌非法吸收公众存款犯罪刑事拘留。上周，中共总书记习近平刚刚在毗邻南京的扬州考察并大赞民企。中国的私有经济体又一次惶惶不可终日。
  <br/>
  <br/>
  警方通告指，南京前首富杨宗义的福信集团，以高额利息为诱饵，向社会不特定公众吸收资金，涉嫌非法吸收公众存款犯罪。
  <br/>
  <br/>
  大陆自媒体消息，杨宗义被刑事拘留后，政府有关部门已经控制了他的南京福中集团总部，所有高管被要求不得离开公司。
  <br/>
  <br/>
  据报道，位于南京的福中集团，在上世纪90年代依靠电子消费起家，创始人杨宗义为辞职下海经商的前中共党政干部。2015-2018年胡润百富榜，杨宗义以40亿身家上榜。

  2 接连出现种种危机的除了民营企业， 大型国企也频传债务违约，包括中国最大晶片制造商清华紫光集团。
  <br/>
  <br/>
  近年中国积极推动半导体行业自力更生，紫光集团被视为实现“中国芯”的龙头。这家中国最大晶片制造商日前债券价格突然暴跌。集团周一表示，因公司流动资金紧张，令规模达13亿元人民币、票息5.6%的私募债不能按期足额支付。
  <br/>
  <br/>
  同日出现债务违约的还有华晨汽车集团。中国债券信息网的公告显示，华晨汽车集团因债务违约加上被降低评级，拖累还债和融资能力，债务违约金额达65亿元人民币，利息逾1.4亿元人民币。此外，光大银行与其他债券银行组成委员会，推动公司重组债务和筹集资金；河南大型国有煤企永城煤电上周则发公告，说由于公司流动资金紧张，两只即将到期的债券未能按期足额偿付本息...等等等等。
  <br/>
  <br/>
  国内媒体以耸人听闻的标题党称“信用债集体爆雷扑街”形容这最新的一波“狼牙山五壮士”。

 <p>
  3 据维权网11月16日发布的消息，四川重庆民营企业家、复员转业军人李怀庆在被当局以煽动颠覆国家政权、诈骗、敲诈勒索和非法拘禁等四项罪名关押近三年之后，案件将于本周五在重庆市第一中级法院一审宣判。
  <br/>
  <br/>
  据悉，2018年1月31日，李怀庆被重庆市警方以“涉黑” 罪名刑拘，妻儿及公司职员等受牵连被抓。同年2月，其名下公司、家人账户等上亿元资产被重庆警方以“打黑”名义冻结。
  <br/>
  <br/>
  2018年4月24日，重庆市检察院又将“涉黑” 罪名更换为涉嫌“煽动颠覆国家政权”，指控李怀庆在2017年10月至2018年1月间，在微信“以造谣、诽谤等方式煽动颠覆国家政权”，包括转发“红四方面军在四川杀人百万”文章，及涉及“道义抗争”、“暴力革命”等内容的录音。
  <br/>
  据介绍，李怀庆长期关注和资助弱势群体，曾多年资助大凉山地区贫困儿童及家境困难的昔日战友，并参与救助尘肺病工人的“大爱清尘”活动，先后捐资数十万元。
  <br/>
  <br/>
  羁押过程中，李怀庆的律师及其家属均未收到涉嫌“煽动颠覆国家政权罪”的逮捕通知书或文件说明。该案于今年6月8日在重庆市第一中级法院开庭审理，但没有当庭宣判。
  <br/>
  <br/>
  4 日前英国驻重庆总领事馆在官微上发布声明：“上周六（11月14日）上午，已61岁的英国驻重庆总领事馆候任总领事史云森在重庆中山古镇游玩时，突然听到人群惊呼声，发现一名年轻女孩落水。情况非常危急。史云森立即跳下水，将已失去知觉的女孩托出水面，在与岸上施救人员配合下，将女孩安全送上岸。由于救援及时，女孩很快恢复呼吸和知觉，已无大碍。”
  <br/>
  <br/>
  总领事忘我救人的感人故事引发中国百万网民点赞，称其“英雄”。中国外交部和驻英大使也点赞总领事的英勇善举。
  <br/>
  <br/>
  视频显示，出事地点周围有近百围观者看热闹，或窃窃私语，或用手机拍摄溺水场面，竟无一国人相救。
