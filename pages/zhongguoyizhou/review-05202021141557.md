<div id="storytext">
 <p>
  听众朋友，欢迎收听自由亚洲电台的“中国一周”节目。“中国一周”关注一周热点话题，邀请嘉宾深入分析。
 </p>
 <p>
  美国纽约城市大学政治学教授夏明和资深新闻工作者、香港《动向》杂志前主编张伟国先生，将为我们点评本周发生的与中国相关的重大新闻事件。
  <br/>
  <br/>
  1.据《纽约时报》报道，苹果公司将包括天安门事件、西藏独立和台湾独立等中国政府视为的禁忌词，列为不能在应用程式出现的议题。苹果中国版APP还自动让5万多款应用程式消失，但在其他国家却可使用。纽时指出，苹果还帮助中国向世界传播其观点。
  <br/>
  苹果公司表示，删除应用程序只是为了遵守中国法律。
  <br/>
  <br/>
  2. 一部抗衡来自中国越来越大的竞争压力的法案，在美国参议院朝着最终通过又迈近一步。参议员们5月17日以86票对11票的表决结果，终止了围绕《无尽边疆法》的冗长辩论程序，把法案向前推进。这部法案将授权五年内拨款1100多亿美元，用于基础和先进技术研究。
  <br/>
  参议院多数党领袖、民主党人查克·舒默(Chuck Schumer)说：“我们或者有一个由中共来决定5G、人工智能和量子计算的道路规则的世界---或者我们可以确保美国捷足先登。”他称这是“世代难逢的对美国科学和美国技术的投资”。
  <br/>
  <br/>
  3. 今年5月16日，是中国文化大革命爆发五十五周年。毛左派团体原先计划的纪念活动，在官方的压力下临时取消。官方媒体不见对文革的反思评论。
 </p>
 <p>
  在1981年彻底否定文革后，中共在习近平治下重新定调文革。4月20日，中宣部副部长、网信办主任庄荣文在党史教育会议上，对1949年至1976年文革结束的历史给予积极的评价，称其为“取得社会主义革命和建设伟大成就的27年。”
  <br/>
  <br/>
  4. 韩国总统文在寅5月19日从首尔启程，开启为期5天的正式对美工作访问，并将于本周五到访白宫。文在寅此行预计将着重强化美韩同盟关系，并在涉及新冠疫苗、气候变化、半导体产业合作，以及对华、对日关系和朝鲜问题上与美国政界交换意见。这也是继日本首相菅义伟到访白宫之后，拜登总统第二次与亚洲盟国首脑举行面对面会晤。
  <br/>
  <br/>
  5. 美国国会举行听证会，直面新疆种族灭绝和北京冬奥会的关联，表明华盛顿对这场赛事可能会按计划举行越发感到忧虑。
 </p>
 <p>
  汤姆·兰托斯人权委员会共同主席、民主党众议员麦戈文(Jim McGovern)在会上指出，抵制奥运会并非是在将体育运动政治化。
 </p>
 <p>
  不过，拜登政府目前未明确表态。
 </p>
 <p>
  一个代表维吾尔人、藏人、香港居民和其他人的联盟5月17日发表声明，呼吁全面抵制奥运会，而不要采取像“外交抵制”这样的半抵制或与国际奥委会或中国进行进一步谈判这类措施。
 </p>
