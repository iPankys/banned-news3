 <p>
  1 中国决定取消举办原定于本月举行的第七届世界互联网大会。官方未解释原因，民间却表示关注。
  <br/>
  <br/>
  中国国家网信办副主任赵泽良本周一（2日）在新闻发布会上说，将取消第七届世界互联网大会，改为本月23日至24日在浙江省桐乡市乌镇举办“世界互联网大会·互联网发展论坛”。
  <br/>
  <br/>
  2 在美国大选投票日，美国国务院正式批准售台四架海上卫士无人机，总金额约6亿美元，并正式通知国会。这是美国国务院今年度第五次对台军售，也是特朗普政府任内的第十次。更是特朗普政府放宽无人机技术出口以来，首度对外出售“海上卫士”。
  <br/>
  <br/>
  台湾国防部长严德发对此回应指出，感谢美方在台湾关系法与六项保证下，支持台湾在整个区域稳定、安全所做的贡献。
  <br/>
  <br/>
  美国国务院在10月21日批准售台三项武器，包括增程距外陆攻飞弹、远程精准火力打击系统与F-16新式侦照荚舱，总金额18亿1130万美元；不到一个星期，美国国防安全合作署又在10月26日再度公布，卖给台湾一百套岸置鱼叉反舰飞弹系统，包含四百枚鱼叉飞弹（战弹）及四枚遥测弹，总价约23亿7千万美元。一个多星期之后，再度宣布售台四架海上卫士无人机。
  <br/>
  <br/>
  3 据证监会网站消息，11月2日，中国人民银行、中国银保监会、中国证监会、国家外汇管理局对蚂蚁集团实际控制人马云、董事长井贤栋、总裁胡晓明进行了监管约谈。
 </p>

  据中国媒体报道，关于中国金融创新及监管问题，马云日前发表的一系列针对中国金融体制的言论引发关注。他在10月24日外滩金融峰会上称“中国金融没有系统性风险，因为没有系统”，“真正的创新一定没有人带路，把风险控制为零就是最大的风险”，“好的创新不怕监管，但是怕昨天的方式去监管”，以及“中国的银行还是当铺思想，害了很多企业家”等。
  <br/>
  <br/>
  马云的揭露性发言人，正值以科技起家的中国移动支付市场霸主，阿里巴巴公司分拆旗下金融业务，以支付宝第三方支付为主要金融业务的蚂蚁金服上市之际。几天后，中国国务院金融稳定发展委员会召开专题会议，强调“要加强监管，依法将金融活动全面纳入监管，有效防范风险。” 银保监会和央行抢在蚂蚁集团股票最终发行前发布了《网络小额贷款业务管理暂行办法（征求意见稿）》，以应对蚂蚁上市定价估值超高。
  <br/>
  <br/>
  11月3日晚间，上交所发出公告，暂缓蚂蚁科技集团股份有限公司科创板上市。
  <br/>
  <br/>
  同日，蚂蚁集团在港交所公告，公司今天接获中国内地相关监管机构的通知，由于公司实际控制人及执行董事长、总经理被监管约谈以及金融科技监管环境发生变化等重大事项，可能会使得公司不符合相关发行上市条件，或者信息披露要求，因此决定暂缓公司的A股于科创版上市。
  <br/>
  <br/>
  4 距新冠疫情被曝在武汉出现已有近一年，世界卫生组织召集的国外专家与中国专家近日通过一场网络会议，正式启动了新冠病毒溯源调查。而《纽约时报》周一发布报道披露，世卫组织高层已与北京当局私下里达成协议，放弃了国际专家组在关键问题上的调查控制权，为了获得进入中国开展调查的机会，对北京当局作出了一系列让步。报道引用一些从未公开的文件说，中国政府近期批准了一份外部调查人员的名单。作为对中方的回报，世卫组织同意让中方科学家主持关键部分的调查工作，包括国内首批患者和华南海鲜市场在疫情传播中扮演的角色。文件还显示，世卫国际专家组将审查并“扩充，而不是重复”中方开展的研究。
  <br/>
  <br/>
  上周五，世卫组织国际专家组与中国专家举行了新冠病毒溯源的首次视频交流会，标志着中外联合专家组正式启动了对新冠病毒的溯源调查。
  <br/>
  <br/>
  5 习近平最新指示；“组织起来的中国人民不好办。”
  <br/>
  <br/>
  维权网周五报道，因在QQ群里传播政治内容于一年半前被捕的13位陕西咸阳网民日前被当地法院以“寻衅滋事罪”判处有期徒刑一年半到三年不等。判决书指控他们通过QQ群有组织的传播“涉及政治的谣言”，并攻击中国政府、中共和国家领导人。判决书还说，这个组织的核心人员雷某已经死亡，外界尚不清楚其死亡是否与本案有关。
  <br/>
  <br/>
  另据综合外媒10月31日报道，中国湖南省纪委省监委10月29日发布通报称，长沙市委原常委、市政府原常务副市长陈泽珲 “违反政治纪律，私自从国（境）外购买、携带有严重政治问题的书刊入境并长期阅看、私藏，被开除党籍，取消其享受的待遇，收缴其违纪违法所得，将其涉嫌犯罪问题移送检察机关依法审查起诉，所涉财物随案移送。据悉，在陈泽珲之前已有多名牵涉到“问题境外书刊”的中共党员领导干部被通报处理。
