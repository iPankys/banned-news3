 <p>
  1 历时四天的中共“五中全会”周四闭幕。
  <br/>
  <br/>
  中共官方当晚发表会议公报，通过有关“十四五”规划和2035年远景目标的建议。全会公报提出，到2035年基本实现社会主义现代化远景目标，经济总量和城乡居民人均收入，将再迈上新的大台阶，关键核心技术实现重大突破，进入创新型国家前列。
  <br/>
  <br/>
  全会将新冠肺炎疫情作为中共今年面临的“严重冲击”，却未提中美紧张关系。
  <br/>
  <br/>
  2 最新一期的美国《新闻周刊》（Newsweek）封面故事为《习近平密谋颠覆美国——六百个担心中国的理由》。杂志封面附上一张穿着连帽衫的习近平，坐在计算机前露出阴森森的微笑。
  <br/>
  <br/>
  这篇调查报道的作者是德国外交政策协会（German Council on Foreign Relations）资深研究员狄雨霏（Didi Kirsten Tatlow），她在为期四个月、针对超过二十四位分析师、政府官员和中美专家的采访及调查中发现， 在美国大选将近之日，中共网军正在企图制造内部混乱、并影响选举。
  <br/>
  <br/>
  报道还进一步指出，美国有至少六百个团体是与中国共产党有联系的统战分支，定期接受中国官方指导。
  <br/>
  <br/>
  与此新闻报道相呼应的，白宫国家安全副顾问博明（Matthew Pottinger）上周五用流利中文发表了他的第二篇满溢着学术色彩的中国问题演讲，他谈到："统一战线是一个在开放社会找不到同类的概念。中共领导人把它称为'法宝'，只要统战需要，九千万党员都必须支持。中共统战部门干部的数量是美国国务院外交人员的四倍。中国外交部负责与各国政府进行外事活动，统战部则在搜集他国情报，对海外居民施加影响。他们专门关注外国的精英阶层和机构。实际上， 统战人员是集情报、宣传和心理诱导于一身的通才。”
  <br/>
  <br/>
  3 美国司法部与联邦调查局28日早上举行联合新闻发布会，宣布起诉8名涉及“猎狐行动”的人，其中，5人今天分别在纽约、新泽西州与加州被捕，另有3人相信仍然在中国，他们被指控作为中国非法代理人在美国行动，其中六人面临共谋跨州和国际跟踪的指控。
 </p>

  <br/>
  美国司法部副部长德默斯（John Demers）在发布会上，指责中国的“猎狐行动”表面上以反贪腐为名，对在逃境外的经济犯罪嫌疑人进行缉捕，实际上则是中国最高领导人习近平抓捕海外异议人士的手段。
  <br/>
  <br/>
  “在许多情况下，这些被捕的人都是中国共产党总书记习近平的政治对手、异议人士和批评者，无论如何，这种行动显然违反了法治和国际准则…我们的讯息很明确：离开（stay out）。这种行为在这里（美国）不受欢迎。”德默斯说在发布会上说。
  <br/>
  <br/>
  美国当局指控这些嫌疑人在中共官员的指示下，对某些美国居民进行监视、骚扰、跟踪，胁迫其返回中国。共同出席发布会的联邦调查局长雷（Christopher Wray）强调，此案件并非个案，在全美境内甚至全球都有发生，指责其所作所为像是有组织性的犯罪集团。
  <br/>
  <br/>
  “他们（中国政府）告诉目标人士有两种选择：立即返回中国或自杀。当目标人士拒绝返回中国，他们在美国和中国的家人会遭到威胁，在中国的家人甚至会被逮捕作为筹码。”雷说。
  <br/>
  <br/>
  美国司法部门公开法律文件显示该8人的背景资料：现年64岁的朱勇为中国公民，持有美国"绿卡"拥有合法永久居民身份，他于纽约法拉盛区居住。朱勇聘请美国私家侦探麦克马凯（Michael McMahon）调查和跟踪一名在美的中国公民，即是案件中的受害人1号（John Doe-1）。洪如进（音译，Hongru Jin）今年30岁、已归化美籍的洪如进，居住在纽约皇后区，根据资料显示，他从事导游工作，专门接待中国公民在纽约市或其他地方观光。洪于2017年4月开始参与跟踪受害人1号的行动，并协助另一被告朱峰及共党官员到美国时组织旅游和物流事宜，同时协助监视受害人2号（John Doe-2 ）。荣京（音译，Rong Jing）现年38岁的荣京为中国公民，同时拥有美国"绿卡"，居住在南加州库卡蒙格，他出面聘用私家侦探调查中共"猎狐行动"中在美国逗留的目标人物，他也因应中共政府的指示寻找特定目标人物的背景资料，包括受害人1号及其家人。郑聪颖（音译，Zheng Congying）现年24 岁郑聪颖也是一名中国公民，拥有美国"绿卡"，在进行监视目标人物行动时居住在纽约布鲁克林区，现在已迁往加州。郑与同谋向受害人1号传送威胁讯息，2018年9月到其居所进行滋扰，在大门贴上"如果你愿意回坐十年牢你的老婆子女没事，这事到此为止！"朱峰（音译，Zhu Feng，又名Johnny Fung）现年33岁的朱峰为中国公民，也持有美国"绿卡"，直至2017年4月居住在纽约皇后区。胡吉（音译，Hu Ji）45的胡吉为中国公民，申请美国签证时报称"警察"（policeman），在武汉公安部门任职。李敏君 （音译，Li Minjun）今年64岁的李敏君申请美国签证时填写"退休"，曾是湖北襄阳市中心医院医生。李敏君曾参与跟踪受害人1号行动，于2017年4月曾与受害人2号一同由中国飞往美国。
  <br/>
  <br/>
  4 据中共党媒报导，10月24日，2020外滩金融峰会在上海开幕，久未露面的中共国家副主席王岐山在开幕式上亮相并视频致辞，现任中共上海市委书记李强、中共央行行长易纲和中共原国家开发银行行长，同时也是本次会议组委会的主席陈元也分别讲话。王岐山在致词中发出“中国金融不能走投机赌博的歪路，不能走‘金融泡沫自我循环’的歧路，不能走庞氏骗局的邪路”三大警告讯号。
  <br/>
  <br/>
  耐人寻味之处的是，王岐山的“歪路、歧路、邪路”讲话，中共喉舌新华社在第一时间的报导中直接删除，似乎侧面验证了中共最高层的内斗正酣。
  <br/>
  <br/>
  5 香港《南华早报》报道，四名香港社运人士周二(27日)下午试图进入美国驻港澳总领事馆被拒。与此同时，三名独派组织“学生动源”前成员相继被捕，其中包括前召集人钟翰林。但是，有两名被捕人士稍后获释。有分析认为，港人已如图北韩的“脱北者”，寻找一切方式投奔自由。
  <br/>
  <br/>
  此前，十二名香港人上月底乘快艇偷渡到台湾未遂，被广东省海警截获，目前仍被扣押在深圳盐田看守所。律师引述公安表示，他们可能涉及更严重的组织偷越国边境罪，最高可被判无期徒刑。稍后这些大陆律师所在的事务所接到司法部门通知，禁止代理“深圳12港人案”。
  <br/>
  <br/>
  6 在10月27日美印第三届“2+2”部长级对话中，美国国务卿蓬佩奥（Mike Pompeo）和国防部长埃斯珀（Mark Esper) 与印度外交部长苏杰生（S. Jaishankar）和印度国防部长辛格（Rajnath Singh）举行了对话，就二十一世纪国防伙伴关系和即将举行的马拉巴海军事演习进行讨论，并正式签署《地理空间基本交流与合作协定》。该协定将使印度实时从美国军事卫星得到精确的数据和地形图，提高自动化武器、无人机和导弹的精确度，同时通过访问卫星和传感器数据从而密切关注中国军舰在印度洋的航行。
  <br/>
  <br/>
  在印度和中国边境冲突持续升级，美中关系恶化之际，该协定终于尘埃落定。美国国防部长马克·埃斯珀（Mark Esper）公开表示，随着中国试图扩大在印度太平洋地区的经济和军事影响力，印度和美国之间日益增长的国防安全伙伴关系将继续稳定该地区的局势。
  <br/>
  <br/>
  在美国总统特朗普今年二月访问印度之际，美印两国达成了三十亿美元的军备交易。同时印度也与俄罗斯等国积极进行军购合作，以实现军事技术现代化。今年六月，印度与澳大利亚签署《后勤相互保障协议》，共享军事基地；九月，印日签署《互相提供物资与劳务协定》，建立美日印澳四国防务合作网络。同时印度积极寻求与俄罗斯、英国、越南等国在后勤保障方面的合作，以共同提高在印度太平洋地区的军事打击能力。
  <br/>
  <br/>
  7 美国第九巡回上诉法院10月26日驳回了美国司法部有关禁止中国社媒软件微信在苹果手机和谷歌应用界面下载的禁令。
  <br/>
  <br/>
  上诉法院的法官在裁决中指出，美国政府没能有效证明如果不禁止微信，将会立即导致无法挽回的伤害。
  <br/>
  <br/>
  美国司法部随后在10月3日向巡回法院提起了上诉。
  <br/>
  <br/>
  本次巡回法院再次驳回微信禁令，意味着美国用户可以继续使用微信。
 </p>
