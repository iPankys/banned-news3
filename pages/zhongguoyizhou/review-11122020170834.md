 <p>
  1 美国2020总统大选落幕，点票接近尾声，拜登发表胜选演说，特朗普阵营尚未接受败选结果，大多数西方国家领导人随后发来贺电，但出人意料的，拜登多年的老朋友习近平却至今按兵不动。
  <br/>
  <br/>
  德媒专文分析中方迟迟不肯表态背后的多重原因：概由拜登在选战中怒斥中国共产党，甚至将中国国家主席习近平称作“匪徒”。
  <br/>
  <br/>
  拜登宣布当选下任美国总统后，面对野心勃勃的中国，新政府的对华政策令外界好奇。有专家指出，拜登团队开始认识到过去奥巴马时代的对华政策已经不再有效。前美国国防部负责政策规划的次长艾得曼（Eric Edelman）认为，拜登的幕僚团队将会针对新的国防战略有激烈辩论，但将中国视为挑战已经是两党共识。
  <br/>
  <br/>
  《华盛顿邮报》7日报道，拜登在明年上任后将会发布一连串行政命令，推翻现任总统特朗普的多项政策。拜登政府虽然不认同特朗普的部分政策，但可将这些制裁行动或强硬手段视为“筹码”，借此逼中国做出改变。
  <br/>
  <br/>
  2 北京周三(11日)公布，宣扬“港独”或有其他危害国家安全行为者，可即时丧失香港立法会议员资格。港府随即剥夺4名民主派立法会议员的资格，民主派议员则宣布集体总辞。北京这次绕过一切法律程序，直接取消民选议员资格，被舆论炮轰“无法无天”，更被形容是处死了香港的政治制度。
  <br/>
  <br/>
  全体19名民主派议员同日傍晚按照早前共识，宣布集体总辞，并在周四向立法会正式交上辞职信，议会工作到此为止。
  <br/>
  <br/>
  在港府公布决定前12分钟，中国全国人大常委会会议在北京闭幕，并通过关于香港立法会议员资格问题的决定，订明香港立法会议员，因宣扬或支持“港独"主张、拒绝承认国家对香港拥有并行使主权、寻求外国或境外势力干预香港特别行政区事务，或具有其他危害国家安全等行为，不符合拥护香港基本法、效忠香港特区的法定要求和条件，一经依法认定，即时丧失立法会议员的资格。
  <br/>
  <br/>
  3 美国国务卿蓬佩奥星期二（11月10日）在美国国务院的新闻简报会上宣布，美台经济繁荣伙伴对话(Economic Prosperity Partnership Dialogue)将在20日展开。
  <br/>
  <br/>
  当日稍早，美国国务卿蓬佩奥在华盛顿“里根总统基金会”（Ronald Reagan Presidential Foundation and Institute）自由与民主中心的揭牌仪式中回顾特朗普政府四年来美国对华战略的转变时，把中国共产党称作是“马克思列宁主义的怪兽”，是自由世界的首要威胁。
  <br/>
  <br/>
  4 周六11月7日，马英九基金会与中国文化大学社会科学院举办马习会5周年研讨会，前总统马英九在会上发言，期待两岸都能回到原汁原味的“九二共识”，为两岸避战与谋和，甚至能够促成“蔡习会”，重启被关闭的两岸和平大桥。
  <br/>
  <br/>
  台湾民进党对此批评说，5年前的马习会是前总统马英九单方面追求个人历史地位，单边押宝中国的作为，却未解开两岸历史的枷锁；尤其在习五条中，早已确立九二共识就是一国两制的立场，毫无中华民国存在的空间。马英九的坚持和诉求早已完全落空。民进党奉劝马英九正视现实，不要再沉溺于“九二共识等同两岸和平”的幻觉里。
  <br/>
  <br/>
  5 视频分享应用TikTok的母公司字节跳动，周二向美国一家上诉法院提出上诉，挑战特朗普政府一项将于本周四生效的行政令。
  <br/>
  <br/>
  据抖音在声明中表示。“由于不断收到新的要求，而且我们提出的解决方案能否被接受还不明朗，因此我们要求将行政令延期三十天，这一点在8月14日的行政令中也得到明确允许，”该公司在声明中表示。该公司称，由于最后期限逼近“而且尚未延期，我们别无选择，只能上诉以捍卫我们的权利”。
  <br/>
  <br/>
  路透社说，白宫和美国财政部未予置评。美国司法部也未立即置评。特朗普政府声称，TikTok存在国家安全隐患，因为中国政府可能会通过TikTok获得美国用户的个人数据。拥有超过一亿美国用户的TikTok否认了这些指控。
  <br/>
  <br/>
