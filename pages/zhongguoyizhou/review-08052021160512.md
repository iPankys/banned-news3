<div id="storytext">
 <p>
  听众朋友，欢迎收听自由亚洲电台的“中国一周”节目。“中国一周”关注一周热点话题，邀请嘉宾深入分析。
 </p>
 <p>
  资深新闻工作者、香港《动向》杂志前主编张伟国先生和美国纽约城市大学政治学教授夏明，将为我们点评本周发生的与中国相关的重大新闻事件。
  <br/>
  <br/>
  1.美国印太司令部3日宣布，即日起至本月27日，将举行“2021大规模全球演习”，英国、澳大利亚和日本将参与。印度国防部同日宣布将派四艘军舰进入南海，举行联合军演。有美国传媒形容，这是美军40年来最大规模联合训练，旨在向中俄展示，美军有能力同时在黑海、东地中海、南海和东海应对挑战，也为了证明美国海军和海军陆战队能阻止对手控制海洋，这在西太平洋尤其重要，因美国希望阻止中国占领台湾，及从日本手上取回钓鱼岛控制权。
 </p>
 <p>
  美国印太司令部的声明指出，美国承诺在复杂的作战环境中与盟友和伙伴合作，确保区域稳定，强化规则为基础的国际秩序、捍卫沟通管道安全，并维护有利于所有国家的自由、开放的印太地区。
  <br/>
  <br/>
  2. 本周一晚间，湖北武汉、荆州两市分别通报，确诊7人以及2人为新冠病毒核酸检测阳性。这是武汉去年6月新冠病例清零后，首次出现的本土病例，该市十多个居民小区目前实行封闭式管理，居民不得外出。武汉市政府周二宣布，全城一千多万居民须接受核酸检测。
  <br/>
  南京市、张家界疫情陆续蔓延各地，北京全面升级疫情防控政策。据报道，海淀区一处大型社区，因为新发现1名确诊者，万人社区立刻封闭。这也是北京今年以来“最大规模部分区域”封锁防疫。德尔塔毒株考验中国引以为傲的严防死守。
  <br/>
  <br/>
  3. 中国官方以防疫为由，暂停办理非紧急的出入境签证安排。过去几个月，大陆网上已有不少网民表示，无法办理新护照和出境签证的审批，并且有网民披露，护照当场被出入境管理人员撕毁。直至几天前，这些传闻终于得到中国官方证实。
  <br/>
  有学者相信，此次做法，防疫考虑比政治因素大。但也有评论认为，这是为配合实现内循环和跟美国脱钩做准备。
  <br/>
  <br/>
  4.红城寺（又名御令宝塔寺）位于甘肃临夏回族自治州永靖县境内，属藏传佛教萨迦派寺院，具有悠久的历史。近期遭到政府人员强行关闭，僧尼被暴力驱赶。僧人们在寺院内拉起写有“逼僧还俗，国法不容”等横幅，强力抗争。当地民宗局人员承认此事，但拒绝回应。
  <br/>
  <br/>
  5. 在东京奥运场地单车女子团体争先赛中，夺得金牌的中国选手鲍珊菊和钟天使站上颁奖台领奖时，外衣上均佩戴了中共已故领导人毛泽东的像章。据路透社报道，该做法涉嫌违反《奥林匹克宪章》第五十条，即不得以任何形式在奥运场馆示威或作政治、宗教或种族宣传，包括不得在个人、运动服或饰物上展示，违者可被取消资格。国际奥委会表示，已要求中国代表团解释事件。
 </p>
