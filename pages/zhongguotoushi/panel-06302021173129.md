 <p>
  <strong>
   众议院七一决议起诉中共 愿见其灰飞烟灭
  </strong>
 </p>
 <p>
  1）《谴责中国共产党百年来严重侵犯人权并与为自由而斗争的中国人民站在一起》决议
  <br/>
  <br/>
  中共百年党庆之际美两党众议员推出决议谴责中共践踏人权史：期盼中共不再存在这一天。
  <br/>
  <br/>
  换言之：这是美国众议院明确宣布将促成中共退出历史舞台。
  <br/>
  <br/>
  来自威斯康星州的共和党众议员麦克·加拉格尔(Mike Gallagher)星期五(6月25日)联袂一些两党议员推出了《谴责中国共产党百年来严重侵犯人权并与为自由而斗争的中国人民站在一起》的决议。
  <br/>
  <br/>
  决议列举了中共在历史上的一系列恶行：
  <br/>
  <br/>
  1940年代有2百万到3百万人被中共杀死的“大规模的土地改革运动”；
  <br/>
  <br/>
  解放军1949年在苏联帮助下吞并新疆；
  <br/>
  <br/>
  解放军1951年吞并西藏并在1959年迫使达赖喇嘛逃亡、1951年和1952年针对“资本家”和被推翻的中华民国的效忠者所发动的并导致成千上万人被处死或自杀的“三反五反”和“镇反肃反”运动；
  <br/>
  <br/>
  毛泽东在1950年代末到1960年代初发动的“大跃进”，导致有史以来最恶劣的人为灾难-大饥荒，致使多达2千万到4千万公民饿死”；
  <br/>
  <br/>
  毛泽东1966年发起并持续到1976年的十年“文化大革命”，期间“任意逮捕、酷刑折磨和处决千百万公民，人数据估计在1百万到3千5百万之间”；
  <br/>
  <br/>
  1979年到2015年实行的禁止千百万家庭多生育的“独生子女政策”；
  <br/>
  <br/>
  解放军在1989年6月4日对天安门广场民主抗议运动的镇压和屠杀；
  <br/>
  <br/>
  六四镇压后的侵权事件，包括对法轮功的镇压、对基督教的打压和迫害以及海外“猎狐”行动；
  <br/>
  <br/>
  中共最近在香港展开打压行动；
  <br/>
  <br/>
  在新疆严重践踏维吾尔和其他少数民族的人权，针对维吾尔人犯下反人类罪和种族灭绝罪；
  <br/>
  <br/>
  律师出身的公民记者张展因为独立报道与官方宣传叙事不符的武汉新冠疫情而被判处四年徒刑。
  <br/>
  <br/>
  决议草案支持中国人民独立于一党统治的自决和自由政治表达的固有权利；呼吁美国政府和志同道合的盟友与伙伴支持中华人民共和国境内的人权，包括通过使用技术来支持并促成表达与信息自由；并期盼中国共产党不再存在的那一天——这一点是抓到了根本。
  <br/>
  <br/>
  （ 联署决议案的有众议院外交关系委员会首席共和党成员、共和党中国工作组组长麦克·麦考尔(Michael McCaul)、众议院共和党会议主席爱丽丝·斯特凡尼克(Elise Stefanik)和共和党众议员布莱恩·菲茨帕特里克(Brian Fitzpatrick)以及民主党众议员鲁本·加莱戈(Ruben Gallego)、乔什·格特海默(Josh Gottheimer)和贾里德·戈尔登(Jared Golden)。）
  <br/>
  <br/>
  2）《决议案》带有不加掩饰的意识形态特点，显示美国及其盟邦对中共的对抗已经由地缘政治领域、经贸领域、科技领域、外交领域、军事领域扩展到了意识形态领域。这在拜登访欧后所架构的工业七国（加上澳、韩、印、南非）、北约30国、欧盟27国形成的民主盟国对中共包围圈相辉映，深入并固化了上述新的世界格局。
  <br/>
  <br/>
  3）历史意义：《决议案》是美国国会首次对一个国家的执政党的历史进行的全面谴责和宣判，并声明将促成它退出历史舞台。这在历史上尚无先例。冷战时期对前苏联美国国会也未曾发布过类似决议案。
  <br/>
  <br/>
  4）虽然中国知识阶层对中共上述滔天罪行耳熟能详，但鉴于过去世界各国因为诸种原因，对此知之甚少。现在由美国国会公告天下，以人类的名义，为中国人的普世权利伸张，把中共骇人听闻的罪行放在阳光之下，诉诸国际公论和国际审判，将大大缩减中共的“软实力”和话语权，使其进一步处于被孤立的基本态势。
  <br/>
  <br/>
  <strong>
   贸易版北约 纳入台湾
  </strong>
  <br/>
  <br/>
  正在酝酿的《贸易版北约》——“民主国家联盟条约组织” （Democracies’ Alliance Treaty Organisation，简称Dato）
  <br/>
  <br/>
  主旨：抗拒中共的分化瓦解、各个击破的新架构：当中共当局对任何成员国采取威吓胁迫性措施时，也要对中共采取同样的报复行动。
  <br/>
  <br/>
  对抗北京已成为国际主流，为防范部分国家因中共胁迫，被迫低头，英美抗中鹰派最新呼吁西方盟友成立贸易版北约组织，并邀请台湾等民主国家加入，以对抗中共战狼外交。
  <br/>
  <br/>
  《南华早报》28日报道，资讯技术和创新基金会是与美国政府和大型科技公司有关联的科技政策智囊团，该基金会与英国保守党议员团体近期合作拟议一份抗中提案。该提案与北约规则类似，就是一旦成员国遭到攻击，就会出面保护遭到威胁的国家，形同贸易版北约组织。这项倡议名为“民主国家联盟条约组织” （Democracies’ Alliance Treaty Organisation，简称Dato），其支持者主张，当中共当局对任何成员国采取威吓胁迫性措施时，也要对中共采取同样的报复行动。
  <br/>
  <br/>
  该报道还援引获得的文件显示，贸易版北约倡议欢迎任何民主国家加入，台湾也获纳入，并称如果任何国家在“民主国家联盟条约组织”做出决定后，不采取必要措施做出回应，将失去成员国的权利。报道举例，如果北京威胁要从一个依赖中国留学生收入的国家，撤出学生，那么成员国将以禁止中国学生予以报复；倘若北京威胁要将一个国家的企业列入黑名单，该联盟组织将同意限制从中国公司进口商品。此提案还强调，这份报告是为了回击中共不断升级的侵略，以及西方迟迟未能拿出有效应对中共侵略的政策。
  <br/>
  <br/>
  中国人民大学国际关系学教授、国务院顾问时殷弘坦言，新的共识原则对中共非常不利，尤其北约国家已普遍认为不能过分依赖中国市场，否则可能会陷入澳洲及加拿大的状况。
  <br/>
  <br/>
  上述贸易版北约若能顺利运作，将逐步取代WTO而成为一个新的国际贸易组织。关键在于它能否如愿成功组成并迅速开始运作。
  <br/>
  <br/>
  <strong>
   疫情溯源  临门一脚
  </strong>
  <br/>
  <br/>
  拜登访欧达成的民主国家联盟的轮廓，之前印太地区美、日、印、澳四国论坛的（盟约）架构， 美台关系的深化及美台贸易协议的谈判成功，美国国会起诉中共罪行的决议广布世界，以及纳入台湾的新的贸易版北约，将使北京遭受极大的压力，处于危机之中。
  <br/>
  <br/>
  最有可能引发剧烈变化的是新冠疫情的溯源有了突破性进展，那将是世界当下潮流的临门一脚，世界或将面临类似柏林墙倒塌的历史辉煌瞬间。
 </p>
