<div id="storytext">
 <p>
  主持人：陈奎德
  <br/>
  座谈人：夏业良教授，经济学家與政治评论家
 </p>
 <p>
  <strong>
   一、 中共近日屡提八国联军
  </strong>
 </p>
 <p>
  哪壶不开提哪壶。
 </p>
 <p>
  中国外交部发言人华春莹：“美国及其“五眼联盟”盟友是进行了协调，摆出了一副要打群架的样子。他们的嘴脸不禁让人想起了当年的八国联军。” 当然，还有她没有说出口的，她也想起了慈禧太后和义和团。
  <br/>
  <br/>
  这是一种自然而然，设身处地的联想。她的慈禧和义和团已呼之欲出了。
  <br/>
  <br/>
  北京不自觉地把中国当代类比于满清晚年，把中共统治者类比于当年慈禧了。至于谁为义和团，五毛与小粉红们谅是心知肚明的。
  <br/>
  <br/>
  难怪资中筠先生指出，“（一百多年了，）上面还是慈禧，下面还是义和团。”
  <br/>
  <br/>
  <br/>
  <strong>
   二、 慈禧、义和团、八国联军 演义
  </strong>
 </p>
 <p>
  <figure class="image-richtext image-inline captioned" style="width:1160px;">
   <img alt="八国联军。（维基百科）" src="https://www.rfa.org/mandarin/zhuanlan/zhongguotoushi/panel-04072021123803.html/boxertroops.jpg/@@images/eec6de65-6cc7-41ed-a227-321d7adf84d0.jpeg" title="3"/>
   <figcaption class="image-caption">
    八国联军。（维基百科）
   </figcaption>
   <small>
   </small>
  </figure>
  <br/>
  <br/>
  1898年戊戌政变后，慈禧太后于1900年1月24日以光绪帝名义颁诏，称其不能诞育子嗣，乃收继端郡王载漪之子，15岁的溥儁为义子，称大阿哥，史称己亥建储。各国公使反对慈禧太后废除光绪帝的皇位，引发并强化了慈禧太后与守旧派大臣对外国人的仇视，后来决定利用义和团运动。
 </p>
 <p>
  于是原在山东、河北的义和拳（后称义和团）打起“扶清灭洋”的旗号，在1900庚子年大举进京勤王，到处杀害外交官、传教士等外国人及中国的基督徒，以使馆、教堂为武力攻击目标，烧教堂、拆电线、毁铁路，捣毁一切外国来的东西，对任何使用“洋物”的中国人都冠以“二鬼子”的称呼，抓住就杀。其中，德使克林德代表各国前往总理衙门要求保护，途中被清军伏击杀死为其主要导火线。
  <br/>
  <br/>
  6月21日，慈禧向十一國宣战，在其《对万国宣战诏书》中慷慨陈词：“朕今涕泣以告先庙，抗慨以示师徒，与其苟且图存，贻羞万古，孰若大张挞伐，一决雌雄。”
  <br/>
  <br/>
  清军与义和团围攻各国驻北京大使馆，于是，英、法、德、美、日、俄、意、奥派遣联合远征军，以保护驻京使馆为名，与排外的义和团作战，并与支持义和团的清朝军队作战。
  <br/>
  <br/>
  人数、军人及其武器均远多于八国联军的清军和义和团大败，慈禧仓皇辞庙西狩，逃至西安。
  <br/>
  <br/>
  但是在今天，中共官员和官媒带风向将所谓“八国联军”舆论战炒得火热，大肆鼓吹他们政权是惹不得的，如果惹翻了，是不好办的。其意图何在？中共“面子至上”外交将点燃义和团式的民族情绪，国家尊严凌驾国家利益，恐重蹈清末悲剧。
  <br/>
  <br/>
  <strong>
   三、 从八十年代对排外主义的批判倒退到毛话语
  </strong>
  <br/>
  <img alt="1" class="image-richtext image-inline captioned" src="https://www.rfa.org/mandarin/zhuanlan/zhongguotoushi/panel-04072021123803.html/bebdd178-05b5-4f35-bd60-c388aa407ccc.png" title="1"/>
  <br/>
  国际背景：
  <br/>
  <br/>
  民主国家合流抗共：跨大西洋盟国与印太地区盟国联手抗共
  <br/>
  <br/>
  A. 3月22日，西方主要国家欧盟、美国、英国、加拿大以及澳大利亚、新西兰30多个国家及国家联盟罕见地统一行动，对中共压制人权和新疆的种族灭绝严厉斥责并实施制裁。
 </p>
 <p>
  B. 3月12日，在阿拉斯加美中会谈 之前，有美日澳印四方首脑会谈。美国总统拜登与日本首相菅义伟、澳大利亚总有莫里森 （Scott Morrison）与印度总理莫迪（Narendra Modi）举行视讯会议，先在盟国间取得共识，应对日益崛起并扩张的中共。有其重要的地缘政治及意识形态涵义。
 </p>
 <p>
  C. 欧洲议会临时取消审议中欧全面投资协定。
 </p>
 <p>
  D. 此外，德国国防部长卡伦鲍威尔4月4日重申，德国将在8月份派一艘军舰前往南海宣示航行自由，并称中国想要按自己的蓝图打造世界秩序，西方世界不应受中国逼迫。日本媒体援引消息人士的话说，日德将在4月份举行视频“2+2”会晤，另外法国与美日印澳在孟加拉湾的联合军演已于5日开始，为期三天。
  <br/>
  <br/>
  这多桩主要外交与军事行动，表明跨大西洋盟国与印太地区盟国携起手来，围堵和制裁中共。这30多个民主国家的联手，凸显了中共陷于四面楚歌的战略窘境。中国官媒与网络舆论称为“新八国联军”。
  <br/>
  <br/>
  中共反应：
  <br/>
  <br/>
  近日，中共辽宁号航母突破第一岛链。北京从南至北模拟封锁台湾。解放军突布南北夹击包围圈，中美军舰家门口擦肩，军情骤然升级。美国罗斯福号航母穿行南海……。
  <br/>
  <br/>
  在南海，北京派出200多艘船身份不明的中国船只占据南沙群岛牛轭礁海域，引发危机。
  <br/>
  <br/>
  中国要形成“共管”的局面，在台海，中美共管；在东海，中日共管；南海中国要主导，排除其他国家干涉权利。
  <br/>
  <br/>
  对西方国家的制裁与抗议，最近，中共摆出一副“中国人不吃这一套”的“大張撻伐，一決雌雄”与天下为敌的姿态，厉兵秣马，高调排外。似乎一步也不退缩，一个热点也不退缩，其原因及其心态何在？
  <br/>
  <br/>
  1） 习以毛为师，拉高谈判价码（如毛在1945年赴重庆与蒋谈判时的故伎，对延安发电称，你们打得越凶，我越安全。）
 </p>
 <p>
  2） 吓阻拜登政府
 </p>
 <p>
  3） 对《更长电报》(该电报建议西方集中火力于习近平) 的反应
 </p>
 <p>
  4）在党内抬高强硬派的声势
  <br/>
  <br/>
  <br/>
  <strong>
   四、 中共欲重蹈慈禧庚子年的覆辙？
  </strong>
 </p>
 <p>
  <figure class="image-richtext image-inline captioned" style="width:1500px;">
   <img alt="2021年3月7日，大批中国船只在菲律宾外海的牛轭礁附近停泊。（法新社）" src="https://www.rfa.org/mandarin/zhuanlan/zhongguotoushi/panel-04072021123803.html/312c8e19-f597-4803-bc9a-eac954413765.jpeg/@@images/182ceb99-cf69-414c-a07f-1353a6d22040.jpeg" title="2"/>
   <figcaption class="image-caption">
    2021年3月7日，大批中国船只在菲律宾外海的牛轭礁附近停泊。（法新社）
   </figcaption>
   <small>
   </small>
  </figure>
  <br/>
  <br/>
  除了上述的地缘政治与军事的基本态势，在经济与疫情方面，从中国武汉爆发的新冠大瘟疫，因当局严密掩盖并打压曝露真相者而流溢全球，万千生灵消亡，其祸远大于120年前的巨灾。北京在前一阶段因严厉封堵造成的优势也开始逐渐丧失。中共没能算到美国以及西方疫苗后发制人的科技优势，比较接种率较高的美英以色列三国和接种中国疫苗较高的土耳其和智利等国， 前三国疫情受到显著控制， 而后二者新病率却节节上升，几乎完全符合此前我们的预估。所以北京很快将面临巨大的形势逆转——疫情的和经济的。它面对的，甚至高于当年庚子拳祸。
 </p>
 <p>
  目前习中共的思路，跟慈禧以及大清帝国末期朝廷官员的思维逻辑一致。它笃信时来运转，竟敢八面树敌，招惹了国际主流社会，现在几乎成同仇敌忾之势。它如何应对，很难想象它能逃过重蹈覆辙之命，让我们拭目以待。
 </p>
