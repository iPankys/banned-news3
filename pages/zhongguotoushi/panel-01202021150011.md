 <p>
  主持人：陈奎德
  <br/>
  座谈人：冯崇义, 澳大利亚悉尼科技大学中国研究教授
 </p>
 <p>
  <strong>
   紫阳先生16周年忌日—世界面临的危机
  </strong>
  <br/>
  <br/>
  1月17日，是赵紫阳先生去世16周年纪念日。而当下的世界，聚讼纷纭，混乱不堪，似乎失却了方向： 十来天前，1月6日，美国的民主殿堂国会山遭受了两百年来未曾见过的成百上千民众的冲击和涌入，共计五人丧生。而一年前自中国武汉爆发的新冠病毒肆虐全球，至今未见止息。中共政权正利用当下的世界乱局，一改去年因遭遇民主国家围堵和孤立而呈现的内缩姿态，趁乱出击，声称“时与势“都在他们那边，大有天下将由中共收拾的狂态。
  <br/>
  <br/>
  在此时，紫阳先生的忌日给了我们反省和思考的机会：自由民主制度是否已经失效？中国是否离民主更远了？紫阳先生晚年大彻大悟所推崇的理念和政治憧憬已经不可企及了吗？……
  <br/>
  <br/>
  所有这些，都悬浮在中国人乃至全世界人们的精神上空，期待历史的答案。
  <br/>
  <br/>
  <strong>
   1.6事件与6.4屠杀有可比性吗？
  </strong>
 </p>
 <p>
  1） 北京对1月6日事件的幸灾乐祸能为其6.4滔天之罪辩护吗？
  <br/>
  <br/>
  众所周知，赵先生是因为反对中共的六四屠杀而被罢黜的。世界目睹，6. 4屠城是一桩反人类罪行，政府动用几十万野战军，坦克、装甲车，镇压百多万和平游行表达自由意愿的国人，屠杀了数以千计和平抗议的徒手民众！滔滔罪行，没有任何借口能够为之辩护。
  <br/>
  <br/>
  而1.6事件是一批人违法强行闯入国家立法机构，中断了参众两院正在进行中的选举认证议程，触犯了法律，并非和平表达意见，并无国家军队碾压和扫射人群，只有一人因破窗而入很危险地接近了主持会议官员而被特勤保安射杀，其余4 人皆因身体意外发生状况不幸遇难。整个事件历时几个小时，情势已经得到控制。总之，这是一起破坏美国法治的违法事件，是一桩美国悲剧。违法者将受到独立司法体系的公正审判，秩序已得到恢复，体现了美国民主制度的复原性能力。
  <br/>
  <br/>
  1.6事件与6.4事件是在两个不同制度的国家发生的完全不同性质的两桩事件，不具有可比性。
  <br/>
  <br/>
  赵紫阳的精神与政治遗产将为中共提供出路，为世界提供参考
  <br/>
  <br/>
  北京宣传机构针对目前乱局，称中共的极权政治模式优于美国式的自由民主体制。似乎赵紫阳先生所推崇的“民主与法治的轨道“业已过时，而习氏的特色”强国梦“正大行其道，风靡全球。
  <br/>
  <br/>
  果真如此吗？关于1.6事件与6.4事件我们已谈过，另外方面呢？
  <br/>
  <br/>
  <strong>
   关于防疫的比较：
  </strong>
  <br/>
  <br/>
  目前，鉴于新冠疫情仍然在世界泛滥，而中国由于首先爆发，后来又采取了极其严酷的封城、封区、封门的措施和由全球最严密的监控网络，故前段时间中国比其他国家较早控制住了疫情，经济复苏也提前出现。（但现在中国国内，据有限的管道透露，一些省区疫情有开始猛烈复发，呈现出类似一年前武汉市的恐怖景象。）
  <br/>
  <br/>
  但是，我们不要忘记，正是中共这样一种不透明不开放并且惩罚传播真相者的制度，使得一个武汉区域性的流行病被隐瞒了（人传人等）真相三周，造成了全球性蔓延，这个不透明的封闭的社会主义是大瘟疫的罪魁祸首。而前几日美国国务院发布声明指出，武汉病毒研究所是最可能的病毒泄漏源头。不要忘记，2002-2003年的萨斯流行也是首先从中国爆发的，中共也封锁信息，好在蒋彦永医生投书《时代》杂志揭露真相，才没有发展为新冠疫情这样的全球大流行。两次致命疫情大流行都爆发于中共统治下的大陆，是偶然的吗？
  <br/>
  <br/>
  台湾中华民国才是防范新冠疫情最优秀，它并未采取武汉式严厉封城措施。中共用世界第一的严密监控国民的措施，在封堵疫情方面可能是比自由国家有效果，但它的次生灾害也不容低估。
  <br/>
  <br/>
  疫苗的研制是防疫的关键指标。很明显，在开发疫苗的竞争中，美、英等工业民主国家走到了前面。
  <br/>
  <br/>
  因此，再过几个月，当疫苗接种普遍完成后，中国与西方各国的疫情可能出现与目前不同的情势。有鉴于此，习近平所说的“时与势“恐怕会悄悄用脚投票，跑到对方那里去了。
 </p>
 <p>
 </p>

 <p>
  <strong>
   关于其他领域：
  </strong>
  <br/>
  <br/>
  自由民主制度与共产中国的极权制度还有众多可比较的领域。实际上，从去年的“去中共化“、”脱钩潮“即可看出，从各国对中共的民意测验亦可管中窥豹。
  <br/>
  <br/>
  赵的反思并未过时，也是中共未来的唯一出路。
  <br/>
  <br/>
  在中共内部，毛的政治遗产和邓的政治遗产，已经成了其政治走向的两个基本参考系、两个基本的极点。中共各派的视野已经完全被这中共的左、右传统的两极的框架所框限住了。
  <br/>
  <br/>
  但是，六四之后，赵的遗产凸现了出来。这是区别于毛，不同于邓的新的政治遗产。正如1988年8月赵紫阳所说：“十年来，我们的基本经验是什么呢？我看是两条，一个市场经济，一个民主政治，缺一不可。”在新出版的遗著中，赵更清楚地指出：“一个国家要实现现代化，不仅要实行市场经济，发展现代的文明，还必须实行议会民主制这种政治制度。不然的话，这个国家就不可能使它的市场经济成为健康的、现代化的市场经济；也不可能实现现代的法治社会。就会像许多发展中国家，包括中国出现权力市场化，社会腐败成风，社会两极分化严重的情况。”
  <br/>
  <br/>
  这是完全不同于毛、邓的崭新的政治坐标，具有普世性的政治遗产。在今日混乱的世界大局中，仍然有其稳定的永恒的意义。
  <br/>
  <br/>
  <strong>
   赵紫阳的精神遗产在中共党内还有多大影响力？
  </strong>
  <br/>
  <br/>
  <strong>
   赵紫阳的政治（包括人脉）遗产在中共党内尚存吗？
  </strong>
  <br/>
  <br/>
  晚年赵紫阳成为了天安门亡灵的守护者，成为在中国实现“民主与法治”的人格象征，分享了自由的价值，成为全球悼念的对象。赵紫阳遗著《代后记》的作者、中国问题专家罗德里克·麦克法夸尔（RoderickMacFarquhar）指出：“如今的中国，赵紫阳是不允许存在的。到了不再偏执的未来，也许他将进入华夏先烈之列——他们为国效力，功勋卓著，但不见容于最高统治者。当他们腐坏的政敌早已灰飞烟灭的时候，他们的英名将久远长存，受人景仰。”
