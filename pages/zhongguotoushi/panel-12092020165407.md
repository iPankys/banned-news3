 <div class="sidebar">
 </div>
 <p>
  主持人：陈奎德
  <br/>
  座谈人：胡平先生： 《北京之春》荣誉主编
 </p>
 <p>
  <strong>
   一、美国对中共加紧制裁，加速美中之间人员、金融、科技脱钩
  </strong>
  <br/>
  <br/>
  <strong>
   美国制裁14名中共人大副委员长
  </strong>
  <br/>
  <br/>
  美方指他们涉及参与取消4名香港民主派立法会议员的资格。
 </p>
 <p>
  这些被制裁的14名中国官员包括：蔡达峰、曹建明、陈竺、白玛赤林、丁仲礼、郝明金、艾力更·依明巴海、吉炳轩、沈跃跃、万鄂湘、王晨、王东明、武维华、张春贤。
 </p>
 <p>
  上述14人都是本届全国人民代表大会常务委员会副委员长。美国这项制裁将禁止他们本人与家属入境美国。这14人在美国的资产也会遭到冻结。被制裁的是副国级，中国这种级别的官员不超过100人。这些人都是促使港版国安法实施的关键人物，所以对他们的制裁很到位，会产生直接影响。我们知道这14个人大常委会的人都是中国最顶级的权贵，他们亲属在海外的资产难于估量。
  <br/>
  <br/>
  <strong>
   美众院通过《香港人民自由和选择法》
  </strong>
  <br/>
  <br/>
  若生效，在美港人可依此法申请临时保护，受政治迫害港人也可以向美国申请难民身份。
  <br/>
  <br/>
  国务院发布新规，限制中共党员及其直系亲属赴美旅行。中共党员签证有效期从10年被缩减至1个月。
  <br/>
  <br/>
  国务院禁止中共统战官员或参与中共统战活动的人士入境美国。
 </p>
 <p>
  美国国务卿认为：“中共统一战线部门资助和支持海外组织散布宣传，胁迫和欺凌反对北京政策的人。”
 </p>
 <p>
  中国的游说努力包括试图阻止其对立的政策，例如对台军售，以及推广有利于北京的美国政策。中国过去曾利用类似的交流来招募来访的美国核科学家作为间谍。
 </p>
 <p>
  美国国家情报总监约翰·拉特克里夫（John Ratcliffe）宣布，中国政府一直在开展针对数十名国会议员和国会助手的“大规模影响力运动”。他没有详细说明。
  <br/>
  <br/>
  美国务院取消了五项文化交流计划的批准，这些计划由全国人民代表大会外事委员会（中国立法机关）和中国外交部资助。
 </p>
 <p>
  它们是：决策者中国教育访问计划，中美友好计划，中美领导力交流计划，美中跨太平洋交流计划和香港教育和文化计划。
 </p>
 <p>
  这些访问提供给中共官员，而不是不享有言论和集会自由的中国人民。
 </p>
 <p>
  美国官员说，实际上，这些计划是针对国会的秘密共产党统一战线的一部分。
  <br/>
  <br/>
  <strong>
   金融脱钩中国公司
  </strong>
 </p>
 <p>
  美国众议院12月2日全票通过《外国公司问责法案》（The Holding Foreign Companies Accountable Act），该法案在今年5月已在美国参议院获得通过。
 </p>
 <p>
  该法案要求在美国交易所挂牌上市的外国公司，必须遵守美国的审计标准，否则将面临退市。
 </p>
 <p>
  目前美国监管部门未能有效获取审计文件的上市公司中，中国公司占比超九成。基本上都会被退市。这是美中金融脱钩的重要一步。
 </p>
 <p>
  数百万美国家庭依靠适度的投资来安排退休、送子女读大学、应对财务紧急状况。但是，许多人在投资看似合法的中国公司后钱被骗走了，因为这些中国公司秉持跟其他上市公司不同的标准。这项法案纠正了这一错误，以确保在美国交易的所有公司遵守相同规则。
  <br/>
  <br/>
  <strong>
   科技脱钩美国对于中芯国际等的断供
  </strong>
  <br/>
  <br/>
  美国国防部周四（12月3日）将中国最大的芯片制造商中芯国际（SMIC) 列入“中国军方企业”（Chinese military companies）名单。此举意味着美国投资者被禁止持有或交易中芯国际股票。
 </p>
 <p>
  除了中芯国际外，这次美国还将中国建设科技集团（CCTC）、中国国际工程咨询公司（CIECC）、中海油（CNOOC）等其余三家中国企业列入此名单。
  <br/>
  <br/>
  这既是科技脱钩，也是金融脱钩。因为这意味着美国企业在向这四家企业供应相关技术或产品时，需要获得政府许可，同时将无法获得来自美国的资金支持。
  <br/>
  <br/>
  <strong>
   二、美国对中共加紧制裁以及人员、金融、科技脱钩的同时，加强了与台湾在军事和外交上的挂钩，也加强了与香港人权的挂钩
  </strong>
 </p>
 <p>
  <figure class="image-richtext image-inline captioned" style="width:620px;">

   </figcaption>
   <small>

  </figure>
  <br/>
  <br/>
  近几个月来美国卫生部长、国务次卿、太平洋舰队最高阶情报官…相继访台。
  <br/>
  <br/>
  日益升级和频密的对台外交访问以及日益升级的大规模高端武器销售。“高机动性多管火箭系统”丶波音公司生产的“增程型距外陆飞弹”，以及装于F-16战机外部的传感器吊舱，最后一种装备能将影像和数据自战机即时回传到地面。另外4个武器系统包含大型无人驾驶飞机丶陆上鱼叉反舰飞弹，以及水下地雷也即将售往台湾。
  <br/>
  <br/>
  对香港市民的保护如前所述。
  <br/>
  <br/>
  与中共的脱钩同与台湾香港人民的挂钩同时加速发生。这是地缘政治经济军事格局的急速改观。在下届政府上任前，还可能有重要改观。
  <br/>
  <br/>
  <strong>
   三、上述地缘政治经济军事格局近几个月来的急速改变，哪些可能保留，哪些可能消失？
  </strong>
  <br/>
  <br/>
  拜登先生宣称，不会有第三届奥巴马政府了，因为世界的格局 ( Landskape )已经改变。最近的急剧变化，会铸成一个可持续的美中关系态势吗？
 </p>
