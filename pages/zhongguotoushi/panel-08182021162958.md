 <p>
  主持人：陈奎德

  座谈人：冯崇义先生, 悉尼科技大学中国研究教授
 </p>
 <p>
  <strong>
   一、20年后塔利班重夺政权
  </strong>
  <br/>
  <br/>
  8月6日，塔利班占领阿富汗第一个省会；8月15日，就围住了首都喀布尔，开始和政府进行权力交接谈判；8月16日，塔利班士兵进入阿富汗首都喀布尔并占领了总统府。这意味着阿富汗政府的垮台，而塔利班即将重掌该国的控制权。
  <br/>
  <br/>
  没有任何抵抗，总统逃了，首都被塔利班占领。65.23万平方公里的阿富汗，不到十天全部沦陷。
  <br/>
  <br/>
  <br/>
  <strong>
   二、阿富汗何以是“帝国坟场”？
  </strong>
  <br/>
  <br/>
  阿富汗被称为“帝国坟场”，主要是来自其1978年之后的历史。
  <br/>
  <br/>
  在20世纪，1978年之前，阿富汗大体平静，连一战、二战都没有介入。但是1978年之后，一切都急转直下。40年来，阿富汗人民再也没有过和平安宁的生活，成为一个典型的失败国家。
  <br/>
  <br/>
  1978年，在阿富汗发生了一场被称为“沙尔革命”的政变，极左的人民民主党推翻了达伍德政府，建立了一个激进左翼政权，从此开启了阿富汗的噩梦。达伍德1973年也是通过政变上台的，当时他认为王室过于保守，而他希望推动更加左翼的社会变革。结果5年之后，他本人又被更左倾的人民民主党推翻。
  <br/>
  <br/>
  1978年沙尔革命后，人民民主党上台，开始推行更激进的改革，遭遇激烈的反抗。新政权很快挺不住了，于是向苏联求援。1979年，苏军开着坦克就进来了，战争由此开始：一边是政府及其后台苏军，一边则是武装民众。
  <br/>
  <br/>
  当时是冷战高峰，阿富汗的武装民众背后有美国、巴基斯坦、中共以及沙特，致使苏军陷入泥潭。此时，本拉登抛弃了他在沙特的富豪生活并来到了阿富汗。在他成为美国的头号敌人之前，他确实曾是美国的战友。
  <br/>
  <br/>
  1989年苏东巨变，苏联撤军。1992到1996年，当初共同抵抗苏联的武装组织开始相互厮杀。混乱的内战最后的确产生了一个胜利者，只不过，这不是一个普通的胜利者，而是极端的组织塔利班。
  <br/>
  <br/>
  本来，凶残如塔利班即使不能构建一个美好的国家，也能构建一个有效的国家，但是因为塔利班太奇葩了，各种倒行逆施，导致原先相互厮杀的小伙伴们纷纷团结起来与它对抗，这就是著名的“北方联军”。因此，即使塔利班上台，阿富汗内战还在继续，直到2001年塔利班被美军推翻。1978年到2001年，阿富汗大约有100-200万人战死，有400万人逃亡到巴基斯坦和伊朗，还有数百万人在国内流离失所。
  <br/>
  <br/>
  2001年911事变发生，阿富汗塔利班政权因为庇护本拉登而导致美军入侵，这曾经被视为阿富汗的转机。然而，正如20年前的苏联，美军也逐渐发现自己踏入了一个难以醒来的噩梦。2001年，全世界都认为美军已经赶跑了塔利班，但是从2006年起塔利班卷土重来。到2019年，政府只控制着35%的领土，塔利班占领了阿富汗13%的土地。而另外一半的领土上，双方展开拉锯战。
  <br/>
  <br/>
  实际上，在阿富汗，任何政府、任何政体都难以建构一个正常国家。过去40余年，阿富汗尝试过君主立宪、威权共和、共产政权、神权政治、美式民主以及无政府，但是在所有这些实验中，国家建构全都失败了。阿富汗被叫做“帝国坟场”。
  <br/>
  <br/>
  <br/>
  <strong>
   三、何以失败？
  </strong>
  <br/>
  <br/>
  阿富汗的国家建构进程为何如此困难重重？至少有四重因素：第一，多山的地形使得中央集权式政府难以形成，造就了部落主义的政治传统；第二，大国对冲的地缘位置则使得内战往往成为大国代理战争，战争难以打出胜负，毕其功于一役是不可能的，暴力垄断的格局难以形成；第三，苏联入侵激起的宗教狂热主义，毒化了阿富汗的文化土壤，使得世俗政治力量难以建立；第四，多民族国家的结构又使得各方政治力量难以就权力分配的方案达成共识。
  <br/>
  <br/>
  阿富汗把上述因素合而为一，也因此沦落为当今世界上最顽固的“失败国家”之一。
  <br/>
  <br/>
  <br/>
  <strong>
   四、美国何以撤出？
  </strong>
 </p>

  <br/>
  <br/>
  1）2001年美国出兵的缘由
 </p>
 <p>
  9.11袭击的元凶本.拉登受到阿富汗塔利班政府庇护，美国因而出兵推翻了塔利班政权。
  <br/>
  <br/>
  2）撤军的基本缘由：美国的国家利益
 </p>
 <p>
  在2011年美军击毙本.拉登后，美国国仇已报，没有了在阿富汗存在的理由，就开始筹划撤出，希望它扶持的政府能自我保护，并保护国民。这一希望落空，新政权没有保护国家的能力和意愿。美国不能永远填阿富汗这个无底洞，必须撤军，以避免无休止地消耗国民的生命和纳税人的钱财。简单说，止损，越快越好。
  <br/>
  <br/>
  3）当前的世界格局，使美国必须收缩国家力量，实施战略大转移，以便集中全力应对中共对全球秩序的威胁。
  <br/>
  <br/>
  4）撤军有损美国对其承诺的国际义务的信誉吗？台湾有担忧的理由吗？
  <br/>
  <br/>
  A. 阿富汗与台湾不能类比。简言之，保障台湾是美国货真价实的国家利益，也是西方的地缘政治利益，现仍驻军阿富汗是损害美国国家利益。
 </p>
 <p>
  B .没有任何国家能长期保卫和支撑缺乏保卫自己政治意志及能力的政府。
 </p>
 <p>
  C . 对美国而言，长痛不如短痛。
  <br/>
  <br/>
  <br/>
  <strong>
   五、中共能填补美国留下的真空吗？
  </strong>
  <br/>
  <br/>
  1） 北京眼下态度与几个月前的差异
 </p>
 <p>
  先前谴责美国撤军为“不负责任，制造了地区不稳定”；目前又推赞塔利班为“阿富汗重要的政治力量和军事力量”，“希望同中方发展良好关系，期待中方参与阿富汗重建与发展，绝不允许任何势力利用阿富汗领土做危害中国的事情。”
  <br/>
  <br/>
  2） 目前中共是塔利班最重要的国际支持者
 </p>
 <p>
  中共不仅愿意为塔利班提供经济支持，还在为塔利班的政权合法性背书。
 </p>
 <p>
  北京企图以机会主义战术，获取地缘政治利益。
  <br/>
  <br/>
  3） 北京在陷进民主世界的“跨大西洋联盟”及“美日澳印”印太北约的双面夹击围堵后，日益孤立。为摆脱其孤立困境，饥不择食，“捡进篮子都是菜“，不惜求助过去曾诅咒的塔利班。
  <br/>
  <br/>
  4） 北京首鼠两端，既想获得来自塔利班的经济好处（一带一路）和地缘政治好处，又想避免塔利班伊斯兰原教旨极端主义对中国西北部的影响。我的感觉是，到头来也许效果恰恰相反：既难获得一带一路好处（美国撤军后，该地区将更混乱），又难避免中国西北部伊斯兰极端势力的膨胀。让我们仔细关注历史演化的结果吧！
 </p>
