  <strong>
   一、“四方安全对话”（QUAD ）紧接“三国联盟”（AUKUS）出台的意涵
  </strong>
  <br/>
  <br/>
  背景1：
  <br/>
  <br/>
  正当拜登总统在联合国声言美国不寻求一场新冷战之时，美国、中共、加拿大之间富于冷战色彩的（加拿大两位迈克尔与中方孟晚舟）人质交换谈判，正在密锣紧鼓地进行。互相对照的这一幕，颇有黑色幽默感。
  <br/>
  <br/>
  紧接着，另一场亦有冷战意味的峰会在美国开场。9月24日，美国总统拜登、印度总理莫迪、日本首相菅义伟和澳大利亚总理莫里森在白宫举行“四方安全对话”（Quad）首次面对面领导人峰会。之所以称其有冷战意味，是因为这个为印太地区的未来制定新战略、新规则的“四方安全峰会”，恰恰撇开了该地区的共产大国——中共。
  <br/>
  <br/>
  峰会发表“四方安全对话”（Quad）联合声明强调，四国今后将更为紧密地合作，在印太地区扩大影响力。同时，声明也特别提到，东道主拜登总统和其他三名到访的国家领导人认可东盟的中心角色，以避免得罪东南亚其他盟友。
  <br/>
  <br/>
  此前，四方首脑曾在3月12日视频会晤，之后美国又与日本、澳大利亚和印度进行了外长和防长2+2会谈，日本首相还访问了白宫；仅过了半年，四国首脑再次面对面会晤。
  <br/>
  <br/>
  背景2：
  <br/>
  <br/>
  几天前，一个新的军事盟约 “AUKUS”（澳英美）军事联盟刚刚宣布成立。
  <br/>
  上述各相关事件发生的节奏，非同寻常，显示印太地区正在加紧塑造新格局。
  <br/>
  <br/>
  <strong>
   二、“四方安全对话”（Quad ）与“三国联盟”（AUKUS）的异同及二者的互补功能
  </strong>

  1、二者的异同
  <br/>
  <br/>
  "四方安全对话"和美英澳三国联盟不同："四方对话“ 聚焦新冠疫情、气候变化、新科技、基础设施。而三国联盟是军事同盟，赋有强制性的军事条约义务。
  <br/>
  <br/>
  二者之共同处是排除了中共政权的参与，甚而言之，是剑指中共。
  <br/>
  <br/>
  中国是西太平洋最大的国家，但中共不自量力地扩张、争霸，迫使美国与日本、澳大利亚、印度加速合作，正在打造一个没有中国参与的新印太体系。
  <br/>
  <br/>
  四国最初从安全角度考虑，认为有必要联合应对中共的军事扩张和挑衅。中共隐瞒疫情，致病毒泛滥全球，又不断战狼外交、频频挑衅，四国加速结成了全面对抗中共的联盟，实际是中共一手造成的。如今，四国正在共同塑造一个新的印太体系，中共孤悬于外，构成了21世纪一幅新的印太地缘政治版图。
  <br/>
  <br/>
  美日澳印聚首白宫 四方对话剑指中国
  <br/>
  <br/>
  本次“四方安全对话”峰会的正式议题是“科研合作与交流”。但实际上，美国总统、日本首相、印度总理、澳大利亚总理9月24日在白宫的会晤，主要还是谈论如何抗衡中国。
  <br/>
  <br/>
  这场峰会的气氛多少有些怪诞：大家都知道真正的议题是中国，但是与会各方却都避免对中国"点名"。东道主美国总统拜登面对到访的其他三国领导人，高调谈论四国共同项目所取得的成果："我们四个民主国家拥有丰富的合作经验，能够共同应对挑战。"
  <br/>
  <br/>
  "四方安全对话"(Quad)此次在华盛顿的峰会，旨在对外界释放一个信号：在对抗专制国家的斗争中，华盛顿寄希望于外交手段。澳大利亚总理莫里森说："我们是四个民主国家，相信一个重视自由的国际秩序。我们还相信一个自由开放的印太地区，只有这样才能实现该地区的强大、稳定和繁荣。"
  <br/>
  <br/>
  日本首相菅义伟在峰会后对媒体表示，四国今后将每年举行一次峰会。
  <br/>
  <br/>
  根据美国的构想，"四方安全对话"应当构成抗衡中国"一带一路"的同盟。四国在会晤中谈及了不少交流和投资项目，还谈及了半导体供应链、摆脱对中国5G技术的依赖等问题。
  <br/>
  <br/>
  此外，印度总理莫迪在会议上宣布，今年10月底前将批准800万剂新冠疫苗出口。四国达成了共识：主要依靠印度的产能，力争在2022年底前制造10亿剂新冠疫苗，优先供应亚洲地区。
  <br/>
  <br/>
  Quad此次在华盛顿的峰会谈及了不少交流和投资项目，还提到了半导体供应链、摆脱对中国5G技术的依赖等问题。
  <br/>
  <br/>
  印度和澳大利亚也利用此次白宫会晤的机会，达成了一项双边协议：两国将在氢能源技术开发、廉价太阳能等方面展开合作，推动印度的能源转型。
  <br/>
  <br/>
  印度盼Quad金援 对抗中国疫苗外交
  <br/>
  <br/>
  美国方面还在峰会上表示，将为四国留学生设立一个奖学金项目，资助他们前往理工科领域的顶尖高校深造。奖学金将主要由私营企业出资赞助。
  <br/>
  <br/>
  尽管美澳日印四国定期会举行海上联合军演，但是"四方安全对话"并非北约那样的军事同盟组织。
  <br/>
  <br/>
  澳大利亚前总理陆克文也在接受最新一期德国《明镜周刊》采访时指出，"四方安全对话"无法和北约相提并论，"我不认为该组织会在可以预见的未来演变成北约那样的结构。但是，北京依然严肃看待'四方安全对话'，因为它会成为酝酿跨地区对华战略的工具。"
  <br/>
  <br/>
  美国政府一名高官此前也曾承认，本次在白宫举行的峰会会谈及"太平洋地区的非法捕鱼以及主权归属问题"。这明显是在影射中国。
  <br/>
  <br/>
  四国从军事、安全合作，拓展到抗疫合作，共同向印太地区提供疫苗；四国还将一起打造半导体供应链，联合建设安全5G，并准备共同参与建设地区内的高标准基础设施。此外，四国还加强网络、太空合作，并肩反恐、解决朝鲜半岛危机，强调共同维护自由、开放和稳定的印太地区。
  <br/>
  <br/>
  印太地区正在面临关键的重新规划，中共被淘汰出局。
  <br/>
  <br/>
  2、 习为何借助“联合国”？ 他的那点小算盘
  <br/>
  <br/>
  习近平在联合国大会上公开不承认美国的领导地位，称“世界只有一个体系，就是以联合国为核心的国际体系”，还称“只有一个秩序”，“只有一套规则”，也是以联合国为基础。
  <br/>
  <br/>
  因为安理会常务理事国的地位、否决权；多年来北京撒币贿赂的亚非拉小兄弟国家政府的联合国票源；因为多年来北京用金钱和好处收买拉拢了相当部分联合国下属机构及其人员，如世界卫生组织、联合国人权理事会……，但是，中共真正尊重联合国吗？
  <br/>
  <br/>
  中共军队是侵略军，是联合国军的敌人——联合国大会第498号决议。
  <br/>
  <br/>
  这是联合国大会于1951年2月1日通过的一项决议。决议谴责中国人民志愿军介入韩战，认定中华人民共和国介入朝鲜是“侵略行为”。这是联合国大会第一次决议判定一个国家为武装侵略者。现在498号决议仍然存在。
  <br/>
  <br/>
  中共国违反联合国的《世界人权宣言》
  <br/>
  <br/>
  《世界人权宣言》正是以联合国为核心的国际体系，以国际法为基础的国际秩序最主要的内容。《世界人权宣言》与《经济、社会及文化权利国际公约》、《公民权利和政治权利国际公约》一起合称国际人权法案，是人类文明的一项重大成就。
  <br/>
  <br/>
  中华人民共和国违反世界人权宣言（第十九条），是没有言论自由的国家
  <br/>
  <br/>
  人人有权享有主张和发表意见的自由；此项权利包括持有主张而不受干涉的自由，和通过任何媒介和不论国界，寻求、接受和传递消息和思想的自由。
  <br/>
  <br/>
  中共签署的《世界人权宣言》第十九条非常清楚地表明，所有国家的人民都有寻求、接受和传递消息的权利，而且不受国界、国籍、地域和媒介的限制。所以，中共政权无权封锁网络信息，它的臭名昭著的长城防火墙是非法的，是反联合国的。
  <br/>
  <br/>
  同时，中共政权也签署了联合国的《公民及政治权利国际公约》，该公约的第十九条第二款载明： “人人有自由发表意见的权利；此项权利包括寻求、接受和传递各种消息和思想的自由，而不论国界，也不论口头的、书写的、印刷的、采取艺术形式的或通过他所选择的任何其他媒介。”
  <br/>
  <br/>
  按照《世界人权宣言》，任何政府都没有“堵住国人耳朵、蒙住国人眼睛，捂住国人的嘴巴“的所谓“主权”。中华人民共和国违反世界人权宣言（第二十一条），是反民主的专制国家。杨洁篪在阿拉斯加说：“美国有美国式的民主，中国有中国式的民主。”这是偷换民主概念。
  <br/>
  <br/>
  《世界人权宣言》第21条三款写明：人民的意志是政府权力的基础；这一意志应以定期的和真正的选举予以表现，而选举应依据普遍和平等的投票权，并以不记名投票或相当的自由投票程序进行。
  <br/>
  <br/>
  在中华人民共和国，72年来，从来没有举行过定期的和真正的选举，从来没有过依据普遍和平等的投票权，并以不记名投票或相当的自由投票程序进行的选举。有鉴于此，不存在所谓“中国式的民主”，只有中国式的专制。
  <br/>
  <br/>
  中共违反《联合国海洋法公约》
  <br/>
  <br/>
  所以，中共根本不在乎什么联合国的规则和秩序。
  <br/>
  <br/>
  <br/>
  3、QUAD 和 AUKUS的互补关系
  <br/>
  <br/>
  中共企图称霸印太，令日本、印度、澳大利亚感到了安全威胁，很快与美国形成四方对话QUAD。中共祸乱了香港，自以为耍弄了英国，也令英国更加清醒，近日与美国、澳大利亚组成了冷战后的第一个军事同盟AUKUS。
  <br/>
  <br/>
  QUAD主要着眼于经济、政治以及安全方面联合，构成抗衡中国"一带一路"，以对抗、孤立中共；AUKUS 是军事同盟，有强制性的条约义务联合在军事上抗击中共。二者之间有互补的关系，在全方位的领域围堵打击中共。

  <strong>
   三、台湾在印太格局中的地位
  </strong>
  <br/>
  <br/>
  台湾是第一岛链的枢纽，在印太格局中具有不可替代的战略地位。
  <br/>
  <br/>
  四方安全对话 Quad 往下的延伸计划是延揽韩国与台湾加入，从而完成一个亚太区的经济、政治、安全的环状结构。
  <br/>
  <br/>
  3月在东京举行的美日两国“2+2”外交与防长级别的会谈，首次在联合声明中强调台海和平稳定的重要性； 随后，5月举行的美韩峰会、欧日峰会，以及6月的G7峰会等重要领导人峰会都强调“台湾海峡和平与稳定的重要性”。
  <br/>
  <br/>
  据各方研判，本次四国领导人聚首一定提到了台湾问题。根据日经亚洲（Nikki Asia）18日的报道，四国领导人还将在本次Quad峰会上就建构安全的半导体供应链达成一致，再次凸显与台湾方面的合作。“台湾在半导体产业链方面具有优势，四国将利用这一优势牵制中国。”
  <br/>
  <br/>
  目前，日本力挺台湾加入CPTPP, 则是在更大战略范围孤立中共。
  <br/>
  <br/>
  <strong>
   四、新印太格局的雏形
  </strong>
  <br/>
  <br/>
  印太格局正在加速演变，“四国机制”可能还会扩大，毕竟更多国家渴望美国的资金、技术和市场，还想得到美军的保护；美国讲规则，中共却没规则或经常潜规则。中共当然也想组建类似的联盟，无奈实在缺乏盟友。俄罗斯没有展示参与印太事务的意愿；朝鲜只能搅局，却不能成事；伊朗正试图争取与美国谈判的最佳结果，自己都顾不过来，更无力染指印太，在阿富汗恐怕与中共还有一争。
  <br/>
  <br/>
  中国周边的更多国家或许还打算再观望一下，不愿意太早、太明显地站队，但应该也拖不了太久。每个国家应该都看到了大势所趋，只是在等待时机罢了，哪个国家会眼看着被新格局排除在外？谁又真愿意接受中共的统治呢？
  <br/>
  <br/>
  美、日、澳、印正在共同实施印太新战略，还欢迎欧洲的印太战略，世界的新格局也在形成。撇开中共政权，不仅成了印太新规则，很可能也将成为世界的新规则。
