<div id="storytext">
 <p>
  主持人：陈奎德
  <br/>
  座谈人：王丹博士，1989年中国学运领袖，《对话中国》所長
 </p>
 <p>
  美国常务副国务卿舍曼（Wendy Sherman）7月26日在天津会晤了中国外长王毅和副外长谢峰，时间长达4小时，坦率直接但有时艰难。
  <br/>
  <br/>
  <strong>
   美中天津会谈背景
  </strong>
  <br/>
  <br/>
  美中阿拉斯加会谈不欢而散之后的首次双方高级官员会晤
  <br/>
  拜登总统访欧后，美国已经基本达成以美英新大西洋宪章、G7+4、 北约，欧盟为四大主轴的跨大西洋联盟，和以美、日、印、澳“四方安全对话”（Quad）为轴心的印太联盟。此二“联盟”，明言指向北京，对中共形成了隐形的东西夹击。在此态势下，可与北京谈了。
  <br/>
  会前，曾出现曲折波澜。关于出席会谈的人选，中方曾欲派出层级不对称的官员出席企图占外交便宜，美方遂取消会谈；之后，北京只得派出王毅出席，才达成会谈意向。
  <br/>
  <br/>
  <strong>
   美中天津会谈概况
  </strong>
  <br/>
  <br/>
  美中双方都用“坦率”来描述这些会谈，意指双方讨论了在诸多问题上的分歧。
  <br/>
  <br/>
  <strong>
   双方报道会议各说各话：
  </strong>
  实际上，会前双方在人选问题上的争端仍然遗留在会后双方的会议报道中。中方顾及面子，居然在开初报道中不提舍曼与王毅的会谈，只提舍曼与谢峰的会谈。而美方文告则强调舍曼与王毅的会谈，对谢峰只是后来一笔带过。
  <br/>
  舍曼到天津，美国国务院的表述是与王毅等官员会谈，不提谢锋，因为谢锋是随从官员，因为王是外交部长，王如果不去天津，舍曼就不会去天津，美国国务院到了最后才宣布舍曼去中国，这是因为中方通报王毅到场之故。
  <br/>
  <br/>
  <strong>
   美方基本观点：
  </strong>
  <br/>
  美国国务院说，舍曼副国务卿在私下里对中国采取的一系列有悖于美国及其盟友和伙伴的价值观和利益，损害基于规则的国际秩序的行动表示了关切。
  <br/>
  <br/>
  声明说：“特别是，她提出我们对人权的关切，包括北京在香港的反民主镇压、在新疆的种族灭绝和危害人类罪、在西藏的虐待，以及对媒体采访和新闻自由的剥夺。她也谈到我们对北京在网络空间的黑客攻击、台湾海峡两岸，以及东中国海和南中国海行为的关切。”
  <br/>
  <br/>
  声明说，舍曼还向中方提到了美国和加拿大公民在中国被拘留或被禁止出境的案件，并提醒中国官员，人不是谈判的筹码。她重申了对中国不愿与世界卫生组织合作以及不允许在中国对新冠病毒起源进行第二阶段调查的关切。
  <br/>
  <br/>
  舍曼“在北京不喜欢听的问题上对中共进行了非常诚实的忠告。”
  <br/>
  <br/>
  谢尔曼在会谈后不久接受美联社的电话采访时呼吁中国超越这些分歧，担负起大国的责任，“有一些事情超越具体的分歧，是大国全球性的责任，”她说，“美国欢迎与中国进行强劲的经济竞争，但不希望这种竞争转为冲突。” “需要有一个公平的竞争场所和护栏，防范竞争演变成冲突。”
  <br/>
  <br/>
  美国国务院发言人办公室普莱斯在一份声明中说，副国务卿舍曼在会谈中重申了中美两国在全球利益领域合作的重要性，如气候危机、打击毒品、防扩散以及包括朝鲜、伊朗、阿富汗和缅甸在内的地区关切。
  <br/>
  <br/>
  针对北京有关华盛顿干涉中国内政的指责，舍曼在接受美联社的采访时说，人权问题并不只是一国的内政。
  <br/>
  <br/>
  “我们确实希望他们明白，人权不仅仅是一个内部事务，而是他们在签署联合国的《世界人权宣言》时所做出的一项全球承诺，”她说。
  <br/>
  <br/>
  今年4月上任的舍曼表示，她在天津举行的会晤是试图与中国解决关键问题过程中的又一步。
  <br/>
  <br/>
  “事实上，我们要看是否会有后续行动，我们是否能够迈出另外一步，”她说。“在建立这种关系的早期阶段，无法知道我们是否会到达所有我们希望到达的地方。”
  <br/>
  <br/>
  <strong>
   中方基本观点：
  </strong>
  <br/>
  <br/>
  王毅会前接受采访时严厉指责美国认为自己“高人一等”，中国“不会接受任何国家自诩高人一等”，他还说，如果美国到现在还没有学会与其他国家平等相处的话，“中国就有责任和国际社会一道，好好给美国补上一课”。
  <br/>
  <br/>
  王毅在与舍曼会谈时就如何有效管控分歧，防止中美关系失控明确提出了中方的三条底线。
  <br/>
  <br/>
  根据中方新闻稿，谢锋对舍曼说出美国“既要坏事做绝，还想好处占尽”的攻击语言，在与舍曼举行的四个多小时的会谈期间，针对美方对中方的做法提出的批评逐条做出了回应，并对美国提交了2条清单。
  <br/>
  <br/>
  中国外交部发言人赵立坚说：“总体看，这次会谈深入、坦率，增进了对彼此立场的了解，对争取下一阶段中美关系健康发展是有益的，”他说。
  <br/>
  <br/>
  <strong>
   习拜会未上台面
  </strong>
  <br/>
  美中双方未能达成具体协议，也未讨论两国领导人会晤的可能性。
  <br/>
  白宫发言人莎琪在记者会上被问到美中双方在天津会谈后，两国领导人之间是否可能举行某种会谈的可能性时说，这次会谈没有讨论拜登与习近平举行会晤的问题。
  <br/>
  <br/>
  “总统继续对面对面的外交有信心。这是他长期以来一直倡导的。我们预计在某个时候会有一些进行接触的机会。但这些会议里并没有涉及这个话题，这也不是这些会议的目的，”莎琪说。
  <br/>
  <br/>
  一般预计今年10月在罗马举行的20国集团峰会是拜登与习近平举行面对面会晤的一个可能的机会。
  <br/>
  <br/>
  <strong>
   从王毅和谢峰的要求看中共的核心焦虑
  </strong>
  <br/>
  <br/>
  <strong>
   评王毅的“三条底线”
  </strong>
  <br/>
  <br/>
  美国不得挑战、诋毁甚至试图颠覆中国特色社会主义道路和制度。中国的道路和制度是历史的选择，也是人民的选择，事关14亿中国人民的长远福祉，事关中华民族的前途命运，是中方必须坚守的核心利益。
  <br/>
  <br/>
  评：中国制度是如何自己选择、如何历史选择的？72年来，中国人看见过一张自己的选票吗？他们怎么选择？他们能自己办媒体自由发言选择吗？统统没有，谈何选择？事关中国人的利益还是中共的利益，还是习近平的利益？
  <br/>
  <br/>
  美国不得试图阻挠甚至打断中国的发展进程。中国人民当然也有过上更美好生活的权利，中国也有实现现代化的权利，现代化不是美国的专权，这涉及人类的基本良知和国际公义。中方敦促美方尽快取消对华实施的所有单边制裁、高额关税、长臂管辖以及科技封锁。
  <br/>
  <br/>
  评：正是中共，通过盗窃美国知识产权、补贴中国国有企业和拒绝兑现加入WTO时的诸多承诺等不公平的手段，扩大了对美的贸易顺差，阻挠了美国的发展进程。美国对华实施的制裁、高额关税、长臂管辖以及科技封锁，是对中方不公平贸易手段的还击和再平衡，也是对中共试图挑战和破坏世界主流秩序的捍卫。
  <br/>
  <br/>
  美国不得侵犯中国国家主权，更不能破坏中国领土完整。涉疆、涉藏、涉港等问题从来不是什么人权、民主问题，而是反“疆独”、反“藏独”、反“港独”的大是大非问题，任何国家都不会允许国家主权安全受到损害。至于台湾问题，更是重中之重。两岸虽尚未统一，但大陆和台湾同属一个中国、台湾是中国领土一部分这一基本事实从来没有改变，也不会改变。如果“台独”胆敢挑衅，中国有权利采取任何需要的手段予以制止。我们奉劝美方在台湾问题上务必恪守承诺，务必慎重行事。
  <br/>
  <br/>
  评：美国、欧盟、英国、加拿大以及澳大利亚和新西兰先后发表声明，谴责在中国新疆发生的酷刑、强迫劳动、性虐待以及种族灭绝政策，并对中共进行了制裁或谴责。这三十多个国家都严肃声明和谴责了中共的反人权行径，证据确凿，跟随谴责和制裁的国家越来越多。中共这个被告政权试图把人权侵犯问题转移为国家主权问题，化为所谓独立或分裂主义问题，这里有多大可信度呢？究竟是什么问题，中共你敞开大门让各方人士去调查去探索好了，你开放网络，解除报禁，解除新闻审查好了，中共、王毅，你有这个胆量吗？
  <br/>
  <br/>
  <strong>
   评谢峰的2份“清单”
  </strong>
  <br/>
  <br/>
  中共外交部副部长谢锋在“吹风会”上称，北京在会中向美国提交2份清单，一份是要美国“纠正”所谓“错误的中共政策和言行”，另一份则是中共关切的重点个案清单。
  <br/>
  <br/>
  第一份中方的“纠错清单”包括：撤销对中共党员及家属的签证限制，撤销对中方领导人、官员、政府部门的制裁，取消对中国留学生的签证限制，停止对中国企业的“打压”与对中国留学生的“滋扰”，取消对孔子学院的限制，撤销将中国媒体登记为“外国代理人”或“外国使团”，撤销对华为集团首席财务官孟晚舟的引渡等。
  <br/>
  <br/>
  第二份中国关切的个案清单则包括中国部分留学生赴美签证遭拒，中国公民在美遭受不公正待遇，中国驻美使领馆在美遭受滋扰与冲撞，美国国内仇亚、反华情绪滋长，中国公民遭暴力袭击等。
  <br/>
  <br/>
  网民称：“中共怕的不是经济制裁，怕的更不是灾难死多少中国人，中共只在乎资产，只有共产党会把家人往敌人家送，把钱存在敌人家。”“这是中共的第一个要求，也就是他们官员的财产大于国家……”“没有一项是针对老百姓提出的。这是什么样的纠错清单？侮辱智商。”“原来他们最在乎的是自己及其家属去不了美国！
