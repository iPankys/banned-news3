 <p>
  主持人：陈奎德
  <br/>
  座谈人：夏业良教授，经济学家與政治评论家
 </p>
 <p>
  <strong>
   一，双面夹击：大型民企的命运
  </strong>
  <br/>
  <br/>
  1）中国重拳打击滴滴、教培等科技公司 ，全球投资者不寒而栗
  <br/>
  <br/>
  中国政府对大型科技公司的打击力度不断增大，引发了本周此类公司的股票风雨飘摇。代表在美国上市的近百家中国公司股市行情的纳斯达克金龙中国指数（Nasdaq Golden Dragon China Index），在过去的两个交易日里，下跌了近15%。自2月份创下历史新高以来，这个指数现在已经大幅下挫了45%以上。
 </p>
 <p>
  从今年2月至今的五个月里，在美国上市的中国公司的股票价值就被抹去了约7700亿美元。
  <br/>
  <br/>
  近日，在美国上市的中国公司股票出现了自2008年金融危机以来的最大跌幅。
  <br/>
  <br/>
  本月早些时候，滴滴出行在纽约首次公开募股（IPO），仅仅几天之后就被要求从中国的应用商店下架，理由是担心用户数据可能落入外国人手中。滴滴出行的股票的市值一周内蒸发了170亿美元。食品外卖配送平台美团网本周被中国政府勒令遵守新的劳动保护规定，引发了其股票狂跌30%。
  <br/>
  <br/>
  最新的打击政策针对广受欢迎的校外教育培训公司，命令他们停止从数百万中国学生的教育投资中获利。在美国上市的三家最大的从事课外培训的公司好未来、新东方和高途公司的股票一周内暴跌了65-70%。
  <br/>
  <br/>
  8月2日，中共要求网络平台增强社会责任意识，不单纯追逐利益，完善未成年人防沉迷系统，完善平台内容审核机制。港股网游股今日全线暴跌，腾讯跌超9%，网易跌近15%，心动公司跌超14%。
  <br/>
  阿里巴巴的马云是近期中共打击私企的第一个目标。现在，滴滴出行、美团外卖、新东方、好未来、高途、腾讯、网易等等企业一一遭到重击。全球投资者也对此感到不安。
  <br/>
  <br/>
  习中共高层对资本大鳄越来越感到不安全，全方位打击身价亿万的私人企业家，约束他们的权力。
  <br/>
  <br/>
  这些限制措施引起了人们的猜疑，中国经济才刚刚从新冠重击中复苏，为什么要辣手摧毁大企业？
  <br/>
  <br/>
  显然，为了政治安全，习近平越来越愿意承受重大经济损失。 自去年以来，中共就一直在宣称"防止资本的无序扩张"，因为他们对中国企业家快速积累巨量财富表示担心。
  <br/>
  <br/>
  2）美国监管机构向滴滴等中国公司问责，中国海外上市私企两面不是人
  <br/>
  <br/>
  针对中国政府监管在美上市公司的措施，美国证券交易委员会(SEC)委员艾丽森·利（Allison Lee）7月26日表示，作为定期报告义务的一部分，在美国证券交易所上市的中国公司“必须披露中国政府干预其业务的风险”。
  <br/>
  <br/>
  过去是中国公司不合美方审计标准而问责，即根据2020年12月美国前总统特朗普签署生效的《外国公司问责法》，如果外国公司连续3年未能通过美国公众公司会计监督委员会(PCAOB)的审计，将禁止这些外企的证券在美国的任何交易所上市。
  <br/>
  而现在，必须加上中国政府干预该公司业务导致风险的信息。
  <br/>
  <br/>
  中国海外上市公司面临雪上加霜的严重困境，面临两面夹击，生存空间丧失的巨大困境。
  <br/>
  <br/>
  国内民企的生态环境也日益逼仄，著名企业家孙大午及其家人、同事遭遇中共残酷镇压，孙以“莫须有”之“罪”竟然被判监禁18年，骇人听闻，开创了中共压迫民营企业家的历史记录。
  <br/>
  <br/>
  3）中国大型民营公司如何选择？
  <br/>
  <br/>
  事实上，如果中国公司坚持在境外证券交易所上市，可能获得部分免受中国政府打压的庇护。
  <br/>
  <br/>
  "他们基本上都被要求遵守上市规定和国际规范。他们可以利用这些规则和规范作为挡箭牌，应对来自中国政府的压力。"如果中国政府试图干涉，他们可以说'嘿，我们在纽约证交所公开上市呢，我们不能这样做。'"
  <br/>
  （ 在目前形势下，赴美上市没有什么优势，劣势却进一步凸显。比如，中概股主要市场大多集中在中国，美股投资者不如本土投资者了解企业的发展潜力，倾向给出更低估值，以及不断恶化的中美关系和不断收紧的监管措施。这些因素共同推动中概股公司的回流。
  <br/>
  总体观察，中概股公司采取选项为：仍保留在美国上市，同时开启在香港二次上市，包括阿里巴巴、京东、网易等已成功在港二次上市。此外，具备在港二次上市资格的中概股公司还有34家，其中不少已启动流程。这些公司在香港二次上市后，有可能逐步提高在香港的交易规模，最终从美股退市。
  <br/>
  还有公司则直接选择私有化，从美股退市，然后择地再次上市。比如今年7月，新浪和携程两家公司就有此计划。
  <br/>
  最后，对于更小的上市公司，既不符合在港二次上市的资格，又没有资金进行私有化，也可能选择“改换门庭”，选择与在美国注册的审计事务所合作，接受美国公众公司会计监督委员会的监管。）
  <br/>
  <br/>
  对国内企业家，满清末年中国企业家（张謇等人）的选择也许可资参考。
  <br/>
  <br/>
  <br/>
  4）习近平用自己私人的政治利益绑架了中国经济
  <br/>
  <br/>
  过去国际投资者曾经以为，是美国而不是中国想要通过发动贸易战来阻碍中国高科技公司的崛起。这个预期提升了人们对中国上市企业未来收益的信心，故前几月国际投资家对中概股投资并未下跌。
  <br/>
  <br/>
  但他们全错了。
  <br/>
  <br/>
  投资家不懂中国政治，特别是不懂习近平唯保权主义的政治。
  <br/>
  <br/>
  习近平第一优先是确保他自己的权位，第二优先是确保中共在中国的一切重大事务中拥有强大的话事权。
  <br/>
  <br/>
  为此，他不惜毁掉中国经济的基本利益。显然他对中国大型民营企业的严厉打压远未到达终点，他要杀鸡儆猴，并且杀鸡取卵，并不管中国经济的长远发展，只需度过眼下的紧迫危机。
  <br/>
  <br/>
  因此，经济上的脱钩是符合习近平个人的政治利益的。因此可懂得他何以一定要扼杀民企，何以绝不放弃战狼外交的心理了。
  <br/>
  <br/>
  于是，国际投资者落荒而逃 。
  <br/>
  <br/>
  习近平用一己之私利绑架了中国经济的前途。
  <br/>
  <br/>
  <br/>
  <strong>
   二，双方都说不脱钩，但双方都在脱钩
  </strong>
 </p>
 <p>

  <br/>
  <br/>
  疫情再起：中外隔绝
  <br/>
  <br/>
  最近，从南京到15个省市，中国疫情再次爆发，加剧了中国 经济与中国和外部世界的脱离。
  <br/>
  <br/>
  中国官方以防疫为由，暂停办理非紧急的出入境签证安排。有滞留在美国的中国人表示，朋友半年前回中国办新护照时，已不获审批。有政治学者认为，今次的做法，防疫考虑比政治因素大。但有评论员相信，今次的做法是要配合实现内循环和与美国脱钩做准备。
  <br/>
  <br/>
  过去几个月，大陆网上已有不少网民表示，无法办理新护照和出国的签证审批，并且有网民披露，护照当场被出入境管理人员撕毁。直至几天前，这些传闻终于得到中国官方证实。
  <br/>
  <br/>
  中国移民管理局在刚过去的周五(7月30日)举办记者会，新闻发言人陈杰表示，因防疫需要，暂停办理非紧急的出境签证安排。
  <br/>
  <br/>
  陈杰：”坚持非紧急、非必要不出境，严格审批签发内地(中国)居民出入境证件，确保防疫安全。”
  <br/>
  <br/>
  他表示，上半年中国共签发超过33万本普通护照，主要与出国留学、就业和商务有关，强调新措施不会一刀切，只要证实有出境留学、就业、商务等需要，经审核资料属实，当局亦会为申请人办理证件。
  <br/>
  <br/>
  从脱钩到闭关
  <br/>
  <br/>
  绝大部分因私护照停发，中国退回毛时代
  <br/>
  <br/>
  移民局拒续办新护照，在美工作中国人与家人重聚无期
  <br/>
  <br/>
  在美居留的中国商人张胜其对本台表示，在美国工作的中国人，以往护照快要到期时，可以选择回中国，或向中国驻当地的总领事馆办理新护照。但他在美国大学工作的朋友，半年前回中国续办护照，一直不获审批，至今仍滞留中国，未知何时能一家团聚。
  <br/>
  <br/>
  张胜其：”我们这个朋友护照到期，需要更换新的护照，妻子刚生了孩子，还未满月他就回中国，但回去以后就发现办不了。他已经回去大半年，一直办不了，不批，朋友在当地已经找了关系都办不了，都不给办，人在国内呆着，也出不了来，对我朋友这个家庭来说，老婆带着孩子快要半岁，爸爸因要办护照回不了来，家人不能团聚呀！这个太没有人情味了。”
  <br/>
  <br/>
  滞美中国人计划在美国起诉中共驻洛衫矶总领馆
  <br/>
  <br/>
  张胜其表示，有朋友曾要求美国洛杉矶总领事馆协助办理新护照，但都被拒绝，没有有效护照虽然并不影响他们在美居留，但对需要跨境工作的人来说，有很大影响。
  <br/>
  <br/>
  张胜其：”他(官方)现在说非紧急的，是他(官方)说的，不是我们提供什么资料证明你有特别需要。我们现在有几个人护照都到期，联络洛衫矶总领馆都拒绝办理，困难很大，工作出差谈东西，面对面的沟通都做不到，带来很多不便。现在再等一等，与他们协商，如果都不能办，准备在美国请律师，起诉洛衫矶总领馆，我们已经有这样的想法。”
  <br/>
  <br/>
  在美国与中国关系持续全面紧张之际，北京连下重手“治理”在美国上市的中国高科技和教育公司，而那些在美国上市的中国股近来让人“跌破眼镜”的表现，也开始引起外界更多关于未来中美关系走向的揣测与联想。
  <br/>
  <br/>
  <strong>
   三，新冷战信号？
  </strong>
  <br/>
  <br/>
  1）正在各个领域开展的脱钩现状，与现时美中大环境相关
  <br/>
  <br/>
  中共防疫只是表面的理由，官方想借疫情，加强对民间的控制，防止西方意识形态输入，影响中国民众。
  <br/>
  <br/>
  都说不脱钩，实际上都在脱钩。
  <br/>
  <br/>
  就美国而言：日韩首脑访美以及拜登访欧所奠定的印太四国联盟以及跨大西洋联盟，已经构成从东西两面围堵中国的态势。加上病毒溯源，以及美国对中国科技公司的制裁，对中港官员的制裁、警告企业不要与中国公司打交道等等，事实上就是新冷战的信号。
  <br/>
  <br/>
  就中共而言，暂停发出非紧急出入境证件（与疫情有一定关系，但非主要原因），根本上是要集中资源力量，对美国进行贸易和金融战。两次中美会谈的战狼外交姿态，对国内外中企的打压，对香港的毁约改制及压迫，对台湾的恫吓，……同样是新冷战信号。
  <br/>
  <br/>
  习政权是在应对极其不利的外交内政紧急形势，看来是不见棺材不落泪了。
 </p>
