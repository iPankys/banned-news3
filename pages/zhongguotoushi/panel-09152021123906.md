 <p>
  <strong>
   一、 拜习第二次通话的来龙去脉
  </strong>
  <br/>
  <br/>
  它发生于第一次通话七个月之后。
  <br/>
  <br/>
  拜登发起通话，是为了试图确认美中首脑之间通话相比低层级官员的更有效。前几次美中对话时，中方官员只是读稿机，只为了大内宣。
  <br/>
  <br/>
  美国一名资深官员表示，“我们听到的都是为宣传目的设计的谈话要点，加上习主席的多方面集权，因此要推动双边关系前进可能需要领导阶层的往来。”也就是说，较低层官员接触，中方屡屡战狼咆哮，扮演不负责任角色，所作所为都是对内宣传。
  <br/>
  <br/>
  拜习通话想通过高层对话建立预防机制，强化危机管控，避免冲突。与其说是向北京示弱，毋宁说是试图给北京提醒，要北京勿错估形势，自认东升西降、时与势站在中国一边(习近平语)。正如白宫资深官员所说，在与普世民主价值相关的香港、西藏、新疆及台湾议题上，即使中国认为美国“干涉内政”，拜登政府也不会退让。
  <br/>
  <br/>
  拜登打电话的另一目的是建立预防机制，强化管控。使得“负责任地处理”双边关系，“竞争而不寻求冲突”，“使得美中之间达致真正稳定的处境。” 这与拜登上台以来表达的对华战略并无二致。
  <br/>
  <br/>
  基本背景：
  <br/>
  <br/>
  拜登的欧洲之行，与欧盟，与北约，与日本、澳大利亚，与北约组织，有形无形结下一种不同程度的对抗专制的联盟。从“四方安全对话机制”（QUAD）到与日、韩的“二加二外交国防高层对话”，甚至在“七大工业国集团”（G7）及“北约”（NATO）等各种全球性平台及组织，拜登政府无时无刻都高举“围堵中国”的旗帜。就连以前在对华关系上表述谨慎的日本政府，也公开对南海问题、东海问题，台湾问题，香港及新疆维吾尔人权表态。北京愈显孤立。
  <br/>
  <br/>
  就此，美国已经准备好了同盟关系，以逸待劳，可以正面应对北京了。
  <br/>
  <br/>
  这个背景绝对不要忘记。
  <br/>
  <br/>
  另一背景是，美国终结20年阿富汗战争，开始腾出手全力应对中俄，也意识到美中军事摩擦冲突机会升高。
  <br/>
  <br/>
  另外有人指出还有一个内幕，为取悦美国，北京用钱说通塔利班不在9.11搞登基大典，让塔利班另择吉日。因此拜登需要还人情。但这一内情的真伪尚待考证。
  <br/>
  <br/>
  <strong>
   二、 双方对会谈的描述迥然不同
  </strong>
 </p>
 <p>
  <img alt="1" class="image-richtext image-inline captioned" src="https://www.rfa.org/mandarin/zhuanlan/zhongguotoushi/panel-09152021123906.html/4be81cc9-aa15-4274-afe7-090450406bad.png" title="1"/>
  <br/>
  <br/>
  拜登、习近平二度通话“确保美中竞争不会演变成冲突”。这是双方唯一的共同点。
  <br/>
  <br/>
  1) 中国外交部长王毅的反应 -- “这次通话还是有它特殊的意义”来看，表面上中方似乎比较满意。主要是拜登主动打来电话，让中方有一种面子上的荣耀。
  <br/>
  <br/>
  北京喉舌多维新闻说：拜登上台后第一回合较量，北京胜。多维说，此次通话是半年多来，中美政要间进行的唯一一次看不出多少火药味的对话。
  <br/>
  <br/>
  当“拜习二度通话”结束后，中方就有人过分解读，认为“美中关系将有机会柳暗花明，甚至会春暖花开”，甚至认为双边将进入“元首外交”的里程碑。不过，这些类似的观点其实多是来自习近平的“东升西降”、“中国即将取代美国”的前期心理。实际上，现在习本人是否这样看恐怕都未必了。你看他引陆游的诗句就可知道，他在西方联盟已经大体形成的大态势下，已经怀疑无路可走了，“山重水复疑无路”也好，“山穷水尽疑无路“，总之是遭遇围堵，没有路了。因此，才把拜登这通电话视为“柳暗花明又一村”，感激涕零之情，溢于言表。
  <br/>
  <br/>
  <br/>
  2) 美方对这次通话的解读，很简单。白宫表示，两人就避免美中竞争演变成为冲突进行了讨论。白宫资深官员说，美国愿意与中国进行开诚布公的沟通，但中国总是以宣传式的官样文章应付。在与普世民主价值相关的香港、西藏、新疆，以及台湾议题上，虽然中国认为美国在干涉其内政，但拜登政府不会退让。拜登总统与习近平进行了“广泛且策略性”的讨论，拜登总统清楚表明，美国持续努力且负责任地处理美中之间的竞争，并强调美国对印太地区与世界和平、稳定与繁荣的长期关注。
  <br/>
  <br/>
  3) “拜习二度通话”真的是代表美中关系将突破僵局吗？
  <br/>
  <br/>
  拜登打电话的目的是建立预防机制，强化管控。使得“负责任地处理”双边关系，“竞争而不寻求冲突”，“使得美中之间达致真正稳定的处境。”其实，这与拜登上台以来表达的对华战略并无二致。
  <br/>
  <br/>
  当美国将合作作为权宜之计时，中方则将合作当成了弱化对抗的筹码。中方频频提条件，将合作置于中美关系之前，告诉美国——没有无条件的合作。美国不可能随心所欲将竞争、合作、对抗区分开来。
  <br/>
  <br/>
  中共非常清楚中美关系未来会面临何种困境。如何扩大合作的占比，如何缩小对抗的成分才是关键。中方的用意是用合作来抵消，甚至取消对抗。
  <br/>
  <br/>
  其实，拜登政府不必过于看重北京的合作，不必受到北京的“不合作”要挟。你中共不愿合作，那就请便吧。你不合作最终将损害到你自己。
  <br/>
  <br/>
  <strong>
   三、 拜习通话的后续发展
  </strong>
 </p>

  <br/>
  <br/>
  拜习通话后，美方几个后续动作都在踩北京痛脚：
  <br/>
  <br/>
  1）白宫发言人莎奇透露，拜登与习近平短短90分钟通话，曾讨论新冠病毒溯源调查，显示美国不放弃追责。
  <br/>
  <br/>
  2）台湾驻美机构将更名为“驻美台湾代表处”：
  <br/>
  <br/>
  据英国《金融时报》9月10日披露，拜登政府正朝着允许台湾当局更改其“驻美国代表处”名称，将“台湾”一词纳入其中的方向发展。多名了解内幕的人士透露，美方正认真考虑台湾方面的要求，将“驻美国台北经济文化代表处”（Taipei Economic and Cultural Representative Office，TECRO）更名为“台湾代表处”（Taiwan Representative Office，TRO）。这将是美国与中华民国1979年断交以来，双方关系最具象征意义的进展。
  <br/>
  <br/>
  9月10日，美国和民进党当局的高级官员在马里兰州的安纳波利斯举行了一轮名为所谓“特殊通道”（Special Channel）的会谈，民进党当局的成员包括了国安会秘书长顾立雄和外交部长吴钊燮。
  <br/>
  <br/>
  3）美日澳印将于9月24日在华盛顿举办四方安全对话峰会。中共最担心的战略发展，终于降临。
  <br/>
  <br/>
  4）美将对中国启动新贸易战：美方祭出301条款。
  <br/>
  <br/>
  拜登政府正考虑依美国贸易法第301条款，对中国的贸易补贴展开调查，结果可能导致对中国进口商品实施新制裁。美方主动披露上述信息意在告诫北京，美国手中有很多牌可打，期待一次通话就让两国关系柳暗花明，未免一厢情愿。
