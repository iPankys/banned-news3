  <strong>
   一、 中共“十大涉党史辟谣榜单”
  </strong>
 </p>
 <p>
  7月15日，中共中央网信办再在湖南长沙召开“2021中国网络诚信大会“，全会发布了一个“十大涉党史辟谣榜单”，目的是要遵照习近平“反对历史虚无主义”指示，“向抹黑、造谣、污蔑党史行为打响‘第一枪’”。
  <br/>
  榜单如下：
  <br/>
  <br/>
  1、《沁园春·雪》作者是胡乔木？谣言！
  <br/>
  <br/>
  2、中共启封“邓颖超日记”供党史研究？谣言！
  <br/>
  <br/>
  3、狼牙山五壮士中两名战士是“溜崖”？谣言！
  <br/>
  <br/>
  4、毛岸英牺牲是因为做蛋炒饭暴露了目标？谣言！
  <br/>
  <br/>
  5、雷锋的事迹是假的？谣言！
  <br/>
  <br/>
  6、红军长征没有“二万五千里”？谣言！
  <br/>
  <br/>
  7、“飞夺泸定桥”战斗不存在？谣言！
  <br/>
  <br/>
  8、共产党抗战“游而不击”，不是“中流砥柱”？谣言！
  <br/>
  <br/>
  9、周扒皮、黄世仁是好地主，土地改革错了？谣言！
  <br/>
  <br/>
  10、美国没打算进攻中国，抗美援朝不是保家卫国？谣言！
  <br/>
  <br/>
  <br/>
  <strong>
   二、 如此隆重颁布“谣言榜单”，中共意在何为？
  </strong>
 </p>
 <p>
  <strong>
   <figure class="image-richtext image-inline captioned" style="width:633px;">
    <img alt="中共“十大涉党史辟谣榜单”。（网站截图）" src="https://www.rfa.org/mandarin/zhuanlan/zhongguotoushi/panel-07212021150031.html/11.png/@@images/61492282-7aa4-4712-ae91-b8f7dfcffe92.png" title="2"/>

  </strong>
 </p>
 <p>
  1、 避重就轻，
  <br/>
  <br/>
  首先，中共企图以几个边缘性的具争议性的问题来抹杀自毛泽东死后，中国知识界以及党内外改革派清算毛时代罪恶，拨乱反正，展开新启蒙所获得的重大历史成就；企图以此来洗白中共恶行累累、罪恶滔天的黑历史。
  <br/>
  <br/>
  其次，北京也想借此把四十多年来，在拨乱反正的新启蒙运动中成长起来一两代人重新洗脑，蜕变为毛时代的愚民。
  <br/>
  <br/>
  但是，启蒙是可逆的吗？
  <br/>
  <br/>
  2、 中共的重大罪行：
  <br/>
  <br/>
  大饥荒、文革、六四屠杀是人类史无前例的滔天罪行，中共是否敢公开辟谣呢？
  <br/>
  <br/>
  中共百年，很多国人提到的，以及美国众院两党议员声讨中共的《谴责中国共产党百年来严重侵犯人权并与为自由而斗争的中国人民站在一起》的决议案提及的中共罪行，如：
  <br/>
  <br/>
  1940年代，有2百万到3百万人被中共杀死的“大规模的土地改革运动”；
 </p>
 <p>
  解放军1949年，在苏联帮助下吞并新疆；
 </p>
 <p>
  解放军1951年吞并西藏，并在1959年迫使达赖喇嘛逃亡；
 </p>
 <p>
  1951年和1952年，针对“资本家”的三反五反运动；
 </p>
 <p>
  针对被推翻的中华民国的效忠者所发动的，并导致成千上万人被处死或自杀的“镇反肃反”运动；
 </p>
 <p>
  毛泽东在1950年代末到1960年代初发动的“大跃进、人们公社”运动，导致有史以来骇人听闻的人为大饥荒，致多达3千7百56万国民饿死；
 </p>
 <p>
  毛泽东1966年发起并持续到1976年的十年“文化大革命”，期间“任意逮捕、酷刑折磨和处决公民，人数据估计在1百万到3千5百万之间”；十年间，中国青年被剥夺了受教育的权利；
 </p>
 <p>
  1979年到2015年实行的禁止千百万家庭多生育的“独生子女政策”，
 </p>
 <p>
  解放军在1989年6月4日对天安门，广场民主抗议运动的镇压和屠杀；
 </p>
 <p>
  六四镇压后的侵权事件，包括对法轮功的镇压、对基督教的打压和迫害以及海外“猎狐”行动；
 </p>
 <p>
  中共最近在香港展开打压行动；
 </p>
 <p>
  在新疆严重践踏维吾尔和其他少数民族的人权，针对维吾尔人犯下反人类罪和种族灭绝罪，………
  <br/>
  <br/>
  中共是否敢于出来公开讨论、辩驳呢？
 </p>
 <p>
  谅他们是没有这胆量的。
  <br/>
  <br/>
  3、 话语权是北京的命根子
  <br/>
  <br/>
  北京拿这些鸡毛蒜皮的所谓“谣言”，什么《沁园春·雪》作者究竟是毛泽东还是胡乔木？“邓颖超日记”是真是假？…………由中共来郑重其事地“辟谣”，岂不是侮辱公众的智商吗？
  <br/>
  <br/>
  谁在乎《沁园春·雪》作者何人？谁在乎 邓颖超日记”真假? ……….中国人关心的是自己的父母兄姐一辈同胞被饿死达几千万人之多，谁之罪？是文革十年中国青少年没有学上，是千百万中国国民被任意逮捕、酷刑折磨和处决，是坦克开上北京大街屠杀成百上千手无寸铁的民众和学生……，谁应负责？
  <br/>
  <br/>
  中共是否应拆除信息封锁墙，公开上述“谣言”，来一番历史真相大辩论呢？究竟谁是“历史虚无主义”呢？
  <br/>
  显然，中共极其惊恐的是历史真相的披露，其黑暗与血腥的历史罪行一旦大白于天下，为所有中国人所知，中共政权将不可抗拒地走向末日。话语权（在中共，就是言论封锁）是它的命根子。
  <br/>
  这就是北京何以如此焦虑于它的话语权命运，如此惊恐于其合法性因其历史罪行而丧失。
  <br/>
  <br/>
  4、 甚至就连鸡毛蒜皮的 “十大谣言“ ，大多也经不起公开自由的辩驳
  <br/>
  <br/>
  看中国墙内网络的反应，十大谣言里面至少有八个在国人眼中答案是不确定的。
  <br/>
  <br/>
  如：
  <br/>
  <br/>
  关于毛泽东长子毛岸英之死：据原任志愿军总部分管美军情报参谋，后任军事科学院军事历史研究部军史研究员王天成2004年回忆，毛泽东的大儿子毛岸英是在彭德怀作战办公室围着火炉热饭时，被美军飞机炸死的。 “炒蛋”与“热饭”有多大距离？
 </p>
 <p>
  红军长征25000里：毛泽东这样说过，斯诺依据毛的说法在书中引用过，就这样传下来了。2003年，两名英国研究人员用了384天重走长征路线，估计长征实际距离大约6000公里(12000里)；
 </p>
 <p>
  “飞夺泸定桥”这件事，倒是邓小平比较实在，他当年会见美国国家安全顾问布热津斯基时说：“那是一个非常简单的军事行动，没什么大不了的……，但是我们觉得必须要将此戏剧化”。
 </p>
 <p>
  十条里面，有三个是重大问题，然其结论都是不利于中共的：一是抗日谁是主力，国还是共？二是朝鲜战争，三是土改。
 </p>
 <p>
  共产党抗日时，中共不是“中流砥柱”，这是历史学家普遍承认的，因为当时在正面战场作战的是国军。
 </p>
 <p>
  朝鲜人民军与中共志愿军对抗联合国决议，对抗联合国军。中朝是侵略者，是铁板钉钉在案的。
 </p>
 <p>
  周扒皮，黄世仁不是真人，是虚拟人物。中共编造故事隐含着中共为发动土改，杀死几百万中国农村精英—地主寻找借口。中共把地主渲染成恶霸，为血腥的土改开路。
 </p>
 <p>
  关于所谓“抗美援朝”：
 </p>
 <p>
  联合国几个决议谴责北韩人民军与中共志愿军为侵略者，白纸黑字，至今仍在。
 </p>
 <p>
  1950年6月25日，联合国安全理事会通过第82号决议，“断定”“北朝鲜部队对大韩民国施行武装攻击”，“构成对和平之破坏”，“要求立即停止敌对行动”，“促请北朝鲜当局即将军队撤至北纬三十八度”。
 </p>
 <p>
  7月7日，联合国安理会通过关于组织统一司令部之决议，美国驻远东部队总司令麦克阿瑟被美国总统任命为“联合国军总司令“。联合国安理会在苏联缺席下，通过成立联合国军司令部。安理会通过第84号决议，“建议………会员国将此项部队及其他援助置于美国主持之联合司令部指挥之下”，“请美国指派此项部队之司令”，“授权联合司令部斟酌情形，于对北朝鲜人民军作战时，将联合国旗帜与各参战国旗帜同时使用。”
 </p>
 <p>
  有18个国家军队组成联合国军，抗击北朝鲜和中共的军队。
 </p>
 <p>
  1951年5月18日，为了制止朝鲜侵略，联合国大会通过美国提出的联合国大会第500号决议，要求联合国成员国对中国大陆与朝鲜实行禁运。
 </p>
 <p>
  因此，就连中共的十大“辟谣”，基本上也是站不住脚的。
  <br/>
  <br/>
  <strong>
   三、 2012年诺贝尔文学奖得主莫言被踢出“中国百年名作家”之列
  </strong>
 </p>
 <p>
  <strong>
   <figure class="image-richtext image-inline captioned" style="width:622px;">

 </p>
 <p>
  1、莫言陷入困境
 </p>
 <p>
  近期，中共作协、《光明日报》等官方权威机构和央媒，正式将作家、2012年诺贝尔文学奖得主莫言踢出“中国百年名作家”之列。莫言有不少作品暴露大陆社会的黑暗，一般认为，这或许是他被踢出“中国百年名作家”之列的原因之一。
 </p>
 <p>
  在北京看来，莫言的作品没有一部是歌颂社会主义新生活的，几乎都在揭露所谓的“黑暗面”。在他的作品里，干部都是粗俗不堪的，老百姓都是愚昧无知的。他之所以能够获得诺文奖，完全是因为他彻底地否定了49年之后30年的历史，抹黑了中共党所取得的重大成就。
 </p>
 <p>
  莫言曾经写道 :“只有正视人类之恶，只有认识到自我之丑，只有描写了人类不可克服的弱点和病态人格导致的悲惨命运”，才能真正产生“大悲悯”（莫言《生死疲劳》）。他在一次采访中表示：文学艺术绝不是唱赞歌的工具。今年6月22日，《光明日报》发表“中国作协”党组成员秘书处书记吴义勤的《中国文学的红色基因》。文中列出了中共百年来，上百具有红色基因的作家和所谓经典作品，1949年中共建政后作家的作品有数十部，而莫言和他的“魔幻大作”却不见踪影。
 </p>
 <p>
  《文艺报》总编梁鸿鹰的《让人们重回百年文学现场一一写在“红色经典初版影印文库”出版之际》，也列出的大量红色作家和作品，莫言亦榜上无名。
 </p>
 <p>
  中共央媒几乎同时发表重磅文章，强调文艺为中共服务，称凡是抹黑、诋毁“中国”的作品都缺少红色基因，都被唾弃等。
 </p>
 <p>
  近日，莫言被踢出“中国百年名作家”之列的信息开始发酵。社交媒体上流传一段莫言讲话的视频，他说：讲真话是一个作家宝贵的素质。如果一个作家不敢讲真话，那么他势必要讲假话，这样他对社会无意义，对老百姓无意义……。“我觉得文学艺术永远不是唱赞歌的工具，文学艺术就是应该暴露黑暗，揭示社会的黑暗，揭示社会的不公正，也包括揭示人类心灵深处的阴暗面。”
  <br/>
  有网民指，官方此举意味莫言和他的作品被官方彻底否定。
 </p>
 <p>
  莫言曾被官方标举为作家的榜样，官方的面子。如今，连他也逃脱不了被打击抹黑的命运，中国的文化空间被习当局挤压的几乎归零。当前，凡中共当局认为有敏感性的东西，全部都要封杀。这也是民族主义高涨的后果。
 </p>
 <p>
  中国当前进入文化领域最黑暗的时代。
  <br/>
  <br/>
  2、恭喜莫言
  <br/>
  从稍长历史时段看，其实，应当恭贺莫言没有进那个臭气烘烘的百人榜！
 </p>
 <p>
  北京此举必将促销莫言的小说，莫言作品将成为畅销书。
 </p>
 <p>
  莫言在文学史上的名声，或许因北京此举而由负转正，获得翻身。
 </p>
