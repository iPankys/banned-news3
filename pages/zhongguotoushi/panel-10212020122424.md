 <p>
  <b>
   一．民调显示的美国对外义战的风险承受度
   <br/>
  </b>
  <br/>
  1）多数美国人愿承担风险 保卫台湾安全
  <br/>
  <br/>
  华盛顿智库战略与国际研究中心(CSIS)14日发布“测绘未来美国对华政策”(Mapping the Future of US China Policy)民调结果，针对美国一般民众及对亚太事务有影响力的意见领袖(thought leaders)进行问卷调查。
  <br/>
  <br/>
  此份民调发现多数人愿意承担风险，保护盟友不受中国军事威胁，包含日本、台湾、韩国。民调以1到10为刻度来评估受访者认为值得承担风险的程度，数值越大表示越值得承担重大风险。 更多美国人支持防御台湾。
  <br/>
  <br/>
  在针对一般民众的调查中，台湾获得6.69分，其他依序为韩国6.92，日本6.88，澳大利亚6.38，而南中国海不知名盟国或伙伴是6.97。
  <br/>
  <br/>
  针对400多名美国意见领袖的调查，认为美国应该为防卫台湾承担风险的分数是7.93，较一般民众更高，其他依序是日本的8.86，澳大利亚8.71，韩国8.6，南中国海不知名盟国或伙伴7.12。
  <br/>
  <br/>
  该民调显示，美国和盟友普遍同意对中国采取更强硬的立场。在台湾问题上，多数（约三分之二）美国人民也愿意承担风险，保护台湾不受中国军事威胁。
  <br/>
  <br/>
  意见领袖、专家的观点更强硬，近80% 认为应当为台湾承担战争风险。
  <br/>
  <br/>
  特别要注意：在调查所提及的国家中，台湾是美国唯一没有签署共同防御条约（defense treaty）的国家或地区，这个结果格外引人注目。
  <br/>
  <br/>
  2）何以美国人比过去更愿意承担风险 保卫台湾？
  <br/>
  <br/>
  过去并非如此。
  <br/>
  <br/>
  战略与国际研究中心中国实力专案主任葛来仪（Bonnie Glaser）说：“过去的民调显示如果台湾遭到中国的攻击，仅有少数美国人会支持台湾的防御，我们这次的民调表明情况已不再如此，中国不能、也太不可能排除一旦中国对台使用武力，美国会干预的可能性。”
  <br/>
  芝加哥外交事务委员会（Chicago Council of Foreign Affairs）去年的民调显示，若中国武力犯台，只有不到四成的美国民众支持出兵协防台湾。
  <br/>
  <br/>
  社会心理转变的原因何在？
  <br/>
  <br/>
  1、 美国以及西方国家对抗中共的大气候
  <br/>
  <br/>
  2、 台湾的民主价值及其卓越抗疫成就及其在印太战略中的地缘政治地位
  <br/>
  <br/>
  3、 中共近期文攻武吓台湾
  <br/>
  <br/>
  4、 新冠疫情武汉爆发初期中共隐瞒及撒谎导致疫情大流行于全球
  <br/>
  <br/>
  5、 中共香港国安法毁约一国两制和断送国际金融中心
  <br/>
  <br/>
  6、 中共战狼外交和愈益高压的内政
  <br/>
  <br/>
  以上是美国人民态度转向更加愿意为台湾等国家承担风险的原因。
  <br/>
  <br/>
  关于中、俄两国对美威胁的大小，该民调结果发现有54％的民众认为中国是对美国构成最大挑战的国家，其次是俄罗斯，占22％。
  <br/>
  <br/>
  民调显示，绝大部分的意见领袖认为，美国对华的国家安全战略重点，应该放在优先考虑与盟国和伙伴的合作上，即使这会损害与中国的关系。此外，无论意见领袖或是一般美国民众，都高度支持美国促进中国的人权问题。
  <br/>
  <br/>
  <b>
   二、 应对中共近期在台海的挑衅，美国的警告日趋强硬而明晰
  </b>

  <br/>
  1）从模糊逐渐走向清晰
  <br/>
  <br/>
  解放军在台湾海峡频繁挑衅之际，美国白宫国安顾问奥布莱恩（Robert O'Brien)10月16日警告，美国对中国各种虎视眈眈的侵略行动了若指掌，中国胆敢武力犯台，一旦美国介入台海冲突中，中国的处境将变得十分危险，美国还会联合全世界组成反中大联盟，中国必将孤立无援，成为“国际弃儿”。
  <br/>
  <br/>
  话说得很重，也说得很白，实际上把美国传统在台海问题上的模糊政策向清晰化的方向推进了一步。如果说，过去是五五开的话，现在可以说是三七开了。无论是有意还是无意，这恰好是呼应了最新的美国民意的。
  <br/>
  <br/>
  同时，这一表态提到了“反中大联盟”，特别可圈可点。实际上，它不仅是话语的警告，同时是付诸实施的重大政策行动。联系到前一段时间蓬佩奥国务卿穿梭外交所积极构建的“美、日、澳、印”亚洲小北约“，已经渐趋成型，联系到台湾在这一地缘战略构图中的枢纽地位，这一亚洲北约机制，势将演变为日后的民主国家同盟的核心。在这一态势下，来审视中共这一个“国际弃儿”的地缘政治地位，中共将比当年的“德-意-日“轴心国或后来的”以苏联为首的社会主义大家庭“的地位还要差得多。它基本上已没有了盟国，”茕茕孑立，形影相吊“是其最形象的写照。
  <br/>
  <br/>
  2）若中共蠢动，将成“国际弃儿”
  <br/>
  <br/>
  就在奥布莱恩讲话前，中国解放军运八反潜机当地时间10月16日上午再次扰台。奥布莱恩重提历史警告中国。他说，在1950年代前后的金门登陆战（台湾称：古宁头大捷）与八二三炮战，解放军都吃了败仗，尤其是金门登陆战，解放军全军覆没。
  <br/>
  <br/>
  “我们一向不评论美军的具体军事计划，包括在这种情况下，我们将如何应对，但是，我们工具箱里资源不少。如果我们美军卷入，中国就得要面临'非常危险'的处境……我还想提醒，届时，全世界、特别是大部分的印太区域国家都会群起抵制，中国胆敢对台湾采取这种行动，他们会更陷孤立，我们甚至会组成反中大联盟。”奥布莱恩说。
  <br/>
  <br/>
  奥布莱恩还预估，解放军不会以两栖登陆作战的方式入侵台湾，因为台湾易守难攻，可登陆的地点仅四到五个；况且，解放军有大量导弹瞄准台湾，中国若真用导弹轰炸，是有可能摧毁台湾，但“拿下一个毁掉的台湾，我不知道他们（中国）能得到什么。我觉得他们会因此失去想要的一切，成为国际弃儿。”
  <br/>
  <br/>
  他也再次提醒台湾的朋友必须发展“不对称战力”，他说台湾的国防开支已经接近国民生产毛额（GDP）2%，这差不多是北约成员国的水平，但有鉴于中国的军力建设，台湾可能还需要增加投资。

  3）美国的民意将转换为巨大的战力
  <br/>
  <br/>
  1.民意不可违
  <br/>
  <br/>
  民意看起来只是几个数字，似乎无足轻重。但它背后蕴含着难于估量的核力量。二战中，珍珠港事件后，美国被唤起的巨大民意，转化成了无坚不摧的军事力量，从而赢得了二战的辉煌胜利。北京如果低估了民主国家民意的神秘力量，冒险犯难，挑战国际秩序及其文明国家的基本共识，将重蹈当年德意日和前苏联的覆辙，陷入万劫不复之境。
  <br/>
  <br/>
  2.再提韩战，意欲何为？
  <br/>
  <br/>
  最近，中共最高当局借纪念70年前的朝鲜战争来鼓动民族主义对抗西方，甚至否定了改革开放以来中共自己对韩战的痛苦反思，将一场非正义的受到联合国决议谴责的对抗联合国军队、而埋葬了几十万中国年轻人生命的失败的侵略战争，歪曲为胜利战争，这是异常危险的举动。
  <br/>
  <br/>
  蠢蠢欲动的中共当局，不要误判了当下的情势。否则，将死无葬身之地。
  <br/>
  <br/>
  3. 前瞻
  <br/>
  <br/>
  我们期待的中国最好前景，不是与文明世界陷入一场热战，而是在国内外巨大的压力下，中国人摆脱习氏中共对全中国人的绑架，实现和平转型，走向宪政民主。
