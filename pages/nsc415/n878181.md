 【大纪元4月5日讯】由于拆息与最优惠利率息差收窄，香港银行周一(4月4日)纷纷宣布提升最优惠利率，业内人士认为，美国5月议息后，本地银行还有可能会追随加息。
 <br/>
 <br/>
 港基国际银行、永隆银行、美国银行(亚洲)、中国工商银行、华比银行等宣布，周三 (4月6日)开始，调高港元存款利率0.25厘。
 <br/>
 <br/>
 星展银行财资市场董事总经理冯孝忠认为，本港银行在不到三周内两度升息，反映拆息与最优惠利率息差收窄。冯孝忠说，银行可按本身的贷款组合及资金成本, 而大银行有较多存款户,资金成本较低，可以解释此次有银行不加入加息行列。
 <br/>
 <br/>
 冯估计，银行之间最优惠利率水平不同的情况会持续数月。@
