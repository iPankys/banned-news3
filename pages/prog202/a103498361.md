<div><div class="post_content" itemprop="articleBody">
 <p>
  【新唐人北京时间2022年08月09日讯】周三（8月10日），日本首相兼自民党主席
  <ok href="https://www.ntdtv.com/gb/岸田文雄.htm">
   岸田文雄
  </ok>
  将启动内阁及党务主管人事改组，预计当天上午汇整临时内阁辞呈，下午进行改组，经过日本王族德仁亲自任命和认证后，新内阁将在10日晚正式上路。
 </p>
 <p>
  据NHK报导，岸田明天将进行自民党党务主管人事调整，并且改组内阁。岸田周二（8月9日）在长崎市记者会被问及改组时表示，为了突破难局，政府与执政党必须比以往更加团结。
 </p>
 <p>
  朝日新闻报导，这波人事调动最受注目的是任用
  <ok href="https://www.ntdtv.com/gb/萩生田.htm">
   萩生田
  </ok>
  担任政调会长。萩生田是已故首相
  <ok href="https://www.ntdtv.com/gb/安倍.htm">
   安倍
  </ok>
  晋三的
  <ok href="https://www.ntdtv.com/gb/心腹.htm">
   心腹
  </ok>
  ，在自民党内最大派阀安倍派中具有影响力。据分析，让
  <ok href="https://www.ntdtv.com/gb/萩生田.htm">
   萩生田
  </ok>
  负责协调加强国防能力及财政支出等政策，可避免党内分歧。
 </p>
 <p>
  报导说，萩生田被视为
  <ok href="https://www.ntdtv.com/gb/安倍.htm">
   安倍
  </ok>
  派领袖呼声高者之一，与岸田交情好，可与保守派做好沟通，岸田任用萩生田担任政调会长，也借此展现关照安倍派。
 </p>
 <p>
  以下为内定阁员名单：
 </p>
 <p>
  内阁官房长官：松野博一（细田派），留任。
  <br/>
  财务大臣：铃木俊一（麻生派），留任。
  <br/>
  经济产业大臣：西村康稔（安倍派），曾任经济再生大臣。
  <br/>
  防卫大臣：滨田靖一（无派阀），曾任防卫大臣。
  <br/>
  外务大臣：林芳正（岸田派），留任。
  <br/>
  厚生劳动大臣：加藤胜信（茂木派），先前两度担任厚生劳动大臣。
  <br/>
  经济再生大臣：山际大志郎（麻生派），留任。
  <br/>
  经济安全保障大臣：高市早苗（无派阀），现自民党政调会长。
  <br/>
  总务大臣：寺田稔（岸田派），首度入阁，现首相辅佐官。
  <br/>
  少子化大臣：小仓将信（二阶派），首度入阁，自民党青年局局长。
  <br/>
  法务大臣：叶梨康弘（岸田派），首度入阁，曾任农林水产副大臣。
  <br/>
  文部科学大臣：永冈桂子（麻生派），首度入阁。
  <br/>
  农林水产大臣：野村哲郎（茂木派），参议员。
  <br/>
  环境大臣：西村明宏（安倍派），首度入阁，曾任官房副长官。
  <br/>
  复兴大臣兼任冲绳及北方大臣：秋叶贤也（茂木派），首度入阁，曾任首相辅佐官。
  <br/>
  国家公安委员长：谷公一（二阶派），首度入阁。
  <br/>
  数位大臣：河野太郎（麻生派），自民党宣传主任（广报本部长）。
  <br/>
  地方创生大臣：冈田直树（安倍派），参议院国会对策委员长。
  <br/>
  国土交通大臣：齐藤铁夫（公明党副党主席），留任。
 </p>
 <p>
  在自民党党务主管的人事方面：
 </p>
 <p>
  自民党副总裁麻生太郎留任。（麻生派）
  <br/>
  干事长（相当于秘书长）茂木敏充留任。（茂木派）
  <br/>
  政调会长内定由经济产业大臣萩生田光一担任。（安倍派）
  <br/>
  总务会长内定由选举对策委员长远藤利明担任。（谷垣团体）
  <br/>
  选举对策委员长内定由前国会对策委员长森山裕担任。
 </p>
 <p>
  岸田于6日宣布将进行党务主管人事及
  <ok href="https://www.ntdtv.com/gb/内阁改组.htm">
   内阁改组
  </ok>
  ，由于人事布局时间短促，被说成是“电光石火”的改组。他6日在广岛市出席和平纪念仪式后表示，Covid-19的防疫对策、物价飙涨、乌克兰局势、台湾局势、经济对策、前首相安倍晋三的国丧等都是须早日处理的课题，需要尽快启动新体制。
 </p>
 <p>
  （记者陈北晨综合报导/责任编辑：林清）
 </p>
