  <ok href="https://www.ntdtv.com/gb/女儿.htm">
   女儿
  </ok>
  ，他经常在推特上分享他与她们的日常
  <ok href="https://www.ntdtv.com/gb/对话.htm">
   对话
  </ok>
  。这些女孩的童言童语往往会让人发笑，这也难怪他的粉丝已经超过100万名。以下摘译他最近分享的一些有趣推文，以飨读者：
 </p>
 <p>
  （1）
  <br/>
  7岁
  <ok href="https://www.ntdtv.com/gb/女儿.htm">
   女儿
  </ok>
  ：“这是我的朋友。她从幼儿园开始就一直和我同班。”
  <br/>
  我：“那只有3年。”
  <br/>
  7岁女儿：“那就像我的大半辈子。”
 </p>
 <p>
  （2）
  <br/>
  7岁女儿：“我们应该储存更多的熏肉。”
  <br/>
  我：“为什么？”
  <br/>
  7岁女儿：“以防世界末日。”
  <br/>
  （我不知道我这个为人父母的是成功还是失败。）
 </p>
 <p>
  （3）
  <br/>
  我：“我跟你说过，只能吃2颗糖果。”
  <br/>
  9岁女儿：“我知道。”
  <br/>
  我：“（这里）有6张包装纸。”
  <br/>
  9岁女儿：“但我的嘴巴一次只能吃2颗。”
 </p>
 <p>
  （4）
  <br/>
  9岁女儿：“我今天没事做。”
  <br/>
  我：“你可以帮我打扫。”
  <br/>
  9岁女儿：“我不行。我很忙。”
 </p>
 <p>
  （5）
  <br/>
  7岁女儿：“我找到1美分。”
  <br/>
  我：“做得好。”
  <br/>
  7岁女儿：“我还需要多少才足够上大学？”
 </p>
 <p>
  （6）
  <br/>
  我：“你午餐想吃什么？”
  <br/>
  9岁女儿：“糖果！”
  <br/>
  我：“我们现在已经不在奶奶家了。”
  <br/>
  9岁女儿：“别哪壶不开提哪壶。”
 </p>
 <p>
  （7）
  <br/>
  我：“你今天起得早。”
  <br/>
  7岁女儿：“是的。今天是周末。”
  <br/>
  我：“不，今天是要上课的日子。”
  <br/>
  7岁女儿：（又回床上睡觉了）
 </p>
 <p>
  （8）
  <br/>
  我：“去睡觉。”
  <br/>
  5岁女儿：“我要先做一件事。”
  <br/>
  我：“什么事？”
  <br/>
  5岁女儿：“永远保持清醒。”
 </p>
 <p>
  （9）
  <br/>
  5岁女儿：“看看我能做什么。”
  <br/>
  我：“你现在什么事也没做。”
  <br/>
  5岁女儿：“我知道。我可以静静地坐着。”
  <br/>
  （这真是奇迹。）
 </p>
 <p>
  （10）
  <br/>
  我：（在网上查看房屋）
  <br/>
  9岁女儿：“我们要搬家吗？”
  <br/>
  我：“没有，我只是看看，梦想一下。”
  <br/>
  9岁女儿：“至少看一下城堡嘛。”
 </p>
 <p>
  （11）
  <br/>
  7岁女儿：“我的老师今天给我糖果。”
  <br/>
  我：“她真好。”
  <br/>
  7岁女儿：“上学终于有了代价。”
 </p>
 <p>
  （12）
  <br/>
  11岁女儿：“我的数学作业有问题。”
