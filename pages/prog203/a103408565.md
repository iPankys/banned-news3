     <br/>

     <br/>
     五彩的光芒 照彻碧空万里
     <br/>
     微风吹拂着 修炼者祥和的面庞
     <br/>
     他们来自四方，胸中怀着崇高坚定的信仰
     <br/>
     他们静静伫立，无声中散发着“真、善、忍”的力量
     <br/>
     他们义无反顾，为了澄清欲加之罪的弥天大谎
     <br/>
     他们冷静从容，穿行的路人投来赞许的目光
    </p>

     古老的城墙
     <br/>
     宁静的窄巷

     没有权利的 争斗

     只为了世人知晓“法轮大法”的美好
     <br/>
     只为了生命有一个选择从善的自由
    </p>

     <br/>
     乾坤颠倒，神州大地阴霾笼罩
