 <p>
  今天，2020年11月13日，本周【不同的声音】截稿日，也是中国大陆最虔诚的基督徒张展“因撑伞游行被拘留65天后获释一周年的日子”。
  <br/>
  <br/>
  意外获释一个月后的2019年圣诞节，【不同的声音】电话专访了张展，赐名《伞下的天空：平安夜张展访谈》。
  <br/>
  <br/>
  又一个月后，江城瘟疫爆发，她再一次以苦行者殉道士般的义无反顾，带着一部《圣经》，踏上了危域武汉的不归路。
  <br/>
  <br/>
  张展因此被捕已经半年了。罪名是：在英雄之城“寻衅滋事”，日前传出的最新罗织是：
  <br/>
  <br/>
  <b>
   被指在武汉期间“编造谎言”，被以“寻衅滋事罪”起诉。公诉方建议法官对被告人张展量刑五年。张展以绝食表达抗议，六个月来，在三名室友喂食下得以生存。——本台11月12日报道
  </b>
  <br/>
  <br/>
  张展案最早本月下旬开庭审判。
  <br/>
  <br/>
  海外民间组织“人道中国”理事王剑虹发起的“自由张展”联署签名活动，至今已有千人参与。
  <br/>
  <br/>
  流亡英国的王剑虹女士在推特上发表了一系列营救张展的推文。不想，其中的几篇，勾起了【不同的声音】伤心的回忆，也是成功回放这档节目的一个“小小的冲动”，渴望这些流逝的音容能够帮到庭上不屈的张展姐妹。
  <br/>
  <br/>
  【不同的声音】2019圣诞节造访张展。万暗中，光华射，听牧羊人在平安夜展现她那“失望的美丽”……
  <br/>
  <br/>
  <br/>
  以下是推特上王剑虹女士摘录的一部分访谈文字：
  <br/>
  <br/>
  <b>
   去年圣诞节，张展接受RFA专访。当时她撑伞游行被拘，获释才一个多月。记者问，你为什么一而再再而三，明知其不可为而为之呢？ 张展说：”国内政治高压太让人窒息，需要一种破裂。” “我希望去做一点点突破，因为太多人基于恐惧不敢说话…...
  </b>
  <br/>
  <br/>
  <b>
   <br/>
   张展: 我不是英雄, 我肯定不是英雄。
   <br/>
   <br/>
   主持人：问题就在这里。我知道你有很多群很多维权的朋友，昨天你的朋友还给我传来了一大帮人把你围在中间的一张照片，那些群友在饭桌上肯定都是慷慨激昂，说不定说的话比你激烈得多，我觉得你还是很儒雅的，但到了行动的时候，只有你一个人在做，残酷的现实往往就在这里。
   <br/>
   <br/>
   张展: 我觉得也不是什么残酷的现实吧。
   <br/>
   <br/>
   问: 您很理解他们，是吧？
   <br/>
   <br/>
   张展: 我理解啊，因为每个人都害怕呀。
   <br/>
   <br/>
   问: 为什么他们在饭桌上就有这么大的勇气呢？
   <br/>
   <br/>
   张展: 这是人性的弱点，每个人都是这样。
   <br/>
   <br/>
   问: 别人为什么不说，就让你来说？
   <br/>
   <br/>
   张展 : 就是恐惧吧。
   <br/>
   <br/>
   问: 但你也恐惧啊。
   <br/>
   <br/>
   张展: 恐惧每个人都有，但每个人做的事情不一样。国家带来改变，我看到一个资料，其实大部分情况下都是少数人在做，绝大部分人没有做，这是一个正常的情况。
   <br/>
   <br/>
   问: 这两年的高压下连贺卫方这样的人都不说话了。作为一个在中国大陆罕见的勇于公开自由表达不为政府容忍的个人思想和敏感政治观点的举牌示威者, 您觉得自己孤独吗？
   <br/>
   <br/>
   张展 : 我不觉得孤独啊，可能因为我觉得上帝与我同在，或者说其实国内情况不乐观，我只是把话说出来而已。
   <br/>
   <br/>
  </b>
