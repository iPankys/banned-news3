 <h2>
  今年国家税务安全宣传周的亮点包括：
 </h2>
 <h3>
  第 1 天 - 网络星期一：保护在线个人和财务信息
 </h3>
 <ul>
  <li>
   在电脑和手机上使用安全软件、并保持更新。
  </li>
  <li>
   避免网络钓鱼诈骗，尤其是与退税和新冠疫情、经济影响付款和其它税法变更相关的诈骗。
  </li>
  <li>
   为所有帐户使用强大且唯一的密码。
  </li>
  <li>
   尽可能使用多重身份验证。
  </li>
  <li>
   只在安全网站购物；寻找网址中的"https"和挂锁图标；避免在咖啡店、商场或餐馆等地方使用不安全的公共 Wi-Fi 购物。
  </li>
 </ul>
 <h3>
  第 2 天 – 慈善星期二：提防骗子使用假慈善机构
 </h3>
 <ul>
  <li>
   不要接受强迫和压力下的马上捐款。 捐赠前请务必花时间做好研究和调查。
  </li>
  <li>
   询问受捐赠对象的确切名称、网站和邮寄地址并稍后确认，确认该慈善机构是真实的。
  </li>
  <li>
   注意捐赠的方式。在研究了慈善机构后，用信用卡或支票支付，而不是用礼品卡或电汇。
  </li>
 </ul>
 <h3>
  第 3 天 - 获取身份保护个人识别号码
 </h3>
 <p>
  纳税人如果可以在线验证身份，则请选择加入国税局个人身份识别号码（IP PIN） 计划。该计划是前几年的一项重大扩展。这是纳税人可以用来保护自己和退税的另一种工具。以下是纳税人需要知道的：
 </p>
 <ul>
  <li>
   身份保护 PIN 或 IP PIN 是一个只有个人和国税局知道的六位数代码。它为纳税人在纳税申报表上的社会安全号码提供了另一层保护。
  </li>
  <li>
   使用 IRS.gov/ippin 上的 获取身份保护 PIN (IP PIN) 工具立即获取 IP PIN。
  </li>
  <li>
   切勿与任何人共享 IP PIN，除非是可信赖的代报税人。
  </li>
 </ul>
 <h3>
  第 4 天-税务专业人员应审查他们的安全协议
 </h3>
 <ul>
  <li>
   采取基本的安全措施。
  </li>
  <li>
   使用多重身份验证来保护税务软件帐户。
  </li>
  <li>
   如果远程工作，则创建一个虚拟专用网络。
  </li>
  <li>
   根据联邦法律的要求制定书面数据安全计划。
  </li>
  <li>
   了解网络钓鱼和电话诈骗，尤其是与电子归档识别号 (EFIN)、新冠疫情相关税法变更（包括经济影响补助金的付款）相关的诈骗。
  </li>
  <li>
   创建数据安全和数据盗窃恢复计划。
  </li>
 </ul>
 <h3>
  第 5 天--企业应实施保障措施；提防与税务相关的骗局
 </h3>
 <p>
  大多数网络攻击都针对员工少于 100 人的小型企业。以下是该部分的一些详细信息：
 </p>
 <ul>
  <li>
   了解小型企业的最佳安全实践。
  </li>
  <li>
   国税局会一直保护性屏蔽业务记录中的敏感信息。
  </li>
  <li>
   企业身份盗窃宣誓书《14039-B 表》可供所有企业向国税局报告盗窃行为。
  </li>
  <li>
   谨防各种骗局，尤其是试图窃取员工收入信息的 W-2 骗局。
  </li>
  <li>
   在 IRS.gov/identitytheft 上查看国税局身份盗窃中心的"企业"部分。
  </li>
 </ul>
