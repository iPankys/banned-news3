  据了解，带货主播们的收入主要分为三部分：一是坑位费；二是销售返点提成；三是直播平台受众的打赏收入。此外，主播所拿的收入还要与机构进行30%-50%的分成。
  <br/>
  <br/>
  面对如此高的销售额，在中共看来
  <span href="https://zh.wikipedia.org/wiki/%E8%96%87%E5%A8%85" target="_blank">
   薇娅
  </span>
  必然是一只肥的流油的羔羊。同样参考曾经娱乐圈一姐范冰冰被罚8.8亿元人民币的结局看，薇娅如果已经被盯上，肯定难逃一劫。
  <br/>
  <br/>
  其实让网红直播主补税首个被祭旗的并非薇娅，本月初，河南一位红就被税务部门追缴了662万元人民币。早些时候，9月18日，中共税务总局发布通知，要求进一步加强文娱领域从业人员税收管理，其中也包括了网红主播。通知明确表示，对存在涉税风险的明星艺人、网络主播进行一对一风险提示和督促整改。可以说税务的大棒此次挥向薇娅并不令人感到意外。
  <br/>
  <br/>
  就在封杀加补税传闻满天飞时，薇娅的直播活动尚未暂停而是如期举行，薇娅团队也想借此证明所谓的打压传闻并不属实，并且薇娅团队还计划提告封杀新闻的爆料媒体。
  <br/>
  <br/>
  不过，大陆的各种事件有让子弹飞一会的习惯，即便是逃得了一时也逃不了一世。中共眼下的套路就是掀起新一轮的阶级仇视，培养平民仇富心里，进而在搜刮富裕阶层财富时能牢牢把握道德制高点。薇娅能够直播并不能代表其完全着陆，还需继续更近。
  <br/>
  <br/>
  眼下有几种猜测，首先可能是薇娅已经向中共表达出愿意配合的态度，所以官方才没有赶尽杀绝，因为中共的屠刀是要挥向整个行业，如果整的太狠会引起不必要的反弹，毕竟直播行业还是一个朝阳产业。
  <br/>
  <br/>
  其次，马上就是淘宝天猫的双十一购物节，这是大陆电商行业一年中最为重要的时刻，此时拿下行业头牌遭殃的不单单是以天猫淘宝为首的阿里集团，背后数十万的商家可能会受到波及，即便要收拾网红主播先让他们完成最后的“疯狂”。
  <br/>
  <br/>
  总之，整个直播带货领域已经成为中共碗里的肉，共产党只会考虑何时下筷子捞肉吃，不会让肉烂在锅里的，这个道理必须明白。薇娅不要天真的以为帮过农民带货就是帮中共排忧解难，随着你水涨船高的财富积累，中共是不会错过也舍不得放过的小肥羊。
  <br/>
  <br/>
  中共今天肯定直播行业，第二天也可以用为富不仁、偷税漏税的由头将薇娅们拉下神探，可以看看自称为平民天后的范冰冰，她有不嫁豪门，因为自己就是豪门的霸气傍身，但挨过中共铁拳后再也没能爬起来。前车之鉴就在眼前，希望薇娅和直播圈子的小伙伴们能提高警惕，未雨绸缪，别等下雨了才记起要收衣服。
