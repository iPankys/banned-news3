 <p>
  各位听众朋友：
  <br/>
  <br/>
  在这次的“王丹热邮”中，我要向大家推荐一本认识中国的好书《低端中国》。
  <br/>
  <br/>
  这本书的作者罗谷（Dexter Roberts）是派驻北京的西方资深记者，1995年以后，在中国生活长达23年，曾经担任《彭博商业周刊》驻中国分社的社长。在中国期间，他的足迹遍及全中国，尤其是一些偏远地区。与许多热心关注和报道中共高层权力斗争的西方媒体记者不同，罗谷更加关心中国改革开放之后的底层民众，尤其是农民工的命运。这本书是他深入到贵州炳华村，长期追踪报道那里的农民工，并在此个案的基础上，深入讨论中国所谓“低端人群”的生活状况的成果，给外界展示了“另一个中国”。
  <br/>
  <br/>
  当中共吹嘘自己已经完成脱贫任务并自称“大国崛起”的今天，在很多都市出生长大的年轻世代为中国经济繁荣的假象掩盖耳目的今天，罗谷通过他的严谨调查和敏锐分析，告诉了外界一个事实真相，那就是：“四十年改革开放的成功，是建立在官僚和企业联手压榨农民之上的”。而在习近平宣称“全面脱贫”的背后，则是半数中国人口其实仍然处于贫弱状态的真相。他指出：中国农村人口的收入只有城市居民的三分之一，只有13%的农村男性高中毕业，有6100万的工人子女被迫与父母分居。在现行的户籍制度下，高达十分之一的中国公民在自己的国家，居然是“非法居民”。
  <br/>
  <br/>
  罗谷的这本《低端中国》最关注的，就是农民工在中国社会发展中的现状。他告诉我们，占了中国人口近一半的农民工，处于社会的低端位置。为了从他们身上榨取最大的利益，从政府到工厂老板，从城市精英到地方势力，联手打造了一种独特的身份制度，剥夺了他们在城市合法就业、就学、就医的权利。他们忍受着极低的工资，恶劣的工作环境，警察的肆意拘禁，成为中国的“二等公民”。从这本书的描写我们可以看到，中国才是歧视最严重的国家。在中国，歧视主要表现在这种对于低端人群的压迫上。我认为，这本书最大的贡献之一，就是让我们看到了中国经济腾飞的另一面，那就是为经济腾飞付出的代价。而农民工，就是这样的代价之一。不认识到这点，对中国的认识就是盲目的。
  <br/>
  <br/>
  最后我也要感谢罗谷先生。作为一个外国记者，他关切的目光能够投射到底层民众，这要比那些眼睛只看着上层，只关心本阶层命运，对弱势群体的命运漠不关心的中国人，更让我们感到温暖。这也是我要向大家推荐这本书的重要原因之一。
 </p>
