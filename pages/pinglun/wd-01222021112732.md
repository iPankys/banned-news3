 <p>
  1月23日，就是新冠病毒疫情在中国爆发后武汉封城的一周年。这场最早起源于中国武汉的疫情，一年来已经导致全球9400万人染病，200万人死亡，对人类社会造成巨大的损失和威胁，而疫情至今仍旧没有大幅缓解。我们在这里要指出的是：对于这场疫情造成的灾难，至今真正的责任者还没有被追究责任。而那些一年前的吹哨着，却几乎已经被遗忘。这是对历史的极大嘲讽，也是对疫情受害者，包括死难者的不尊重。值此武汉封城一周年之际，我认为，人类社会需要历史记忆，需要从历史记忆中吸取教训。而今天，最好的历史记忆，就是追究一年前疫情扩散的真正责任者。而这个真正的责任者，就是中国政府，就是中国政府的领导人习近平。这才是真实的历史，历史的真实。
  <br/>
  <br/>
  然而，我们所有人都应当反思的是：武汉封城一周年了，我们是否还记得当年那种非人道的、暴力手段下的封城措施，让武汉人民付出了多么惨重的代价。而如今，武汉人民的代价得到赔偿了吗？当年风行一时的《方方日记》，现在还有多少人提起？我们是否还在关心：那些以公民身份记录疫情真相的人，陈秋实，方斌等，他们现在到底在哪里？我们是否还怀念李文亮医生，是否还记得他因为吹哨而被政府警告的事实？我们也应当追问，张展到底报道了什么，要被判四年的重刑？我们都知道，中国政府从来就有掩盖历史的习惯，然而，我们作为国人，是不是也要逐渐淡忘一年前的历史了呢？
  <br/>
  <br/>
  同时，我们都应当警惕的是：中共当局现在不仅是掩盖历史，而且已经在肆无忌惮地改写历史叙事，肆无忌惮地扭曲历史。今天，当你走进武汉的所谓抗击疫情纪念馆的时候，迎面而来最突出的画面，不是李文亮医生的画像，而居然是习近平的画像。当局正在欺骗人民，宣传说，是习近平的“英明领导”带领中国迅速控制住了疫情，而完全回避了这个所谓“亲自指挥，亲自部署”的习近平，早在2019年12月就知道疫情严重，但在接近一个月的时间里，下令向全社会和全世界隐瞒真相的事实。中国政府至今也没有给外界一个信服的解释，为何在疫情爆发之后，武汉病毒研究室附近发生过一次神秘的大爆炸？为什么中国的疫情在去年夏天据说就已经被控制，但时过一年之后，中国政府才允许世界卫生组织的专家到武汉调查？而为什么在这些专家到来之前，上百万页的相关研究资料被当局从网络上删除？中共及其政府，到底在隐瞒什么？
  <br/>
  <br/>
  最荒谬的，还是中共当局对历史的改写。今天，全世界似乎都在赞扬中国的防疫成就，但却忘记了取得这个成就的这个政府才是疫情的始作俑者；而中共当局已经安排写手，在微博上散布“疫情不是起源于武汉”的消息，最应当承担责任的人，俨然成为了被歌颂的英雄。按照中共官方的腔调，现在不是中国要给世界一个解释，反倒是世界欠中国一个道歉了，这不是天大的荒谬吗？要知道，短短一年，历史竟然被如此改写，这不仅是中共宣传的责任，也是我们这些普通人容易淡忘过去的责任。
  <br/>
  <br/>
  类似新冠病毒这样的全球流行疫情在未来很可能还会出现，人类社会必须做好思想准备，面对一波波来自病毒的挑战。厘清事实，追究掩盖真相者的责任，就是我们能做的最好的准备。因此，值此武汉封城一周年之际，我们不能放弃追究责任，这才是对封城一周年最好的纪念。
  <br/>
  <br/>
  以上是自由亚洲电台特约评论员王丹为您做的评论。
