 <p>
  世界卫生组织的专家小组于1月14日到中国武汉调查武汉肺炎的源头，事前敲锣打鼓，似乎已经取得中国的配合，可以进行客观独立的调查。期间也爆出他们的行动受到限制，显示中国在进行干扰。不知道中国怕什么？中国不是有一句谚语，“平时不做亏心事，半夜敲门心不惊”吗？已经受到中共控制的世卫组织，中共还怕什么？是不是丧尽天良的亏心事做得太多，到了难以遮掩的程度？
  <br/>
  <br/>
  2月12日，世卫总干事谭德塞宣布调查已经结束。他还指出，团队在“非常艰难”的情况下，进行了相当重要的科学任务，并对源头的所有假设持开放态度，需要进一步的分析和研究。此后调查报告数度难产，原因是要所有成员国同意。中国是成员国，又是发源地，当然要它同意。这样的结果也是在大家设想之中，对这个结果并没有任何期望。
  <br/>
  <br/>
  但是到正式宣布调查结果时，还是让人拍案惊奇。世卫组织3月30日发布国际联合专家组对新冠病毒溯源研究报告后，世卫总干事谭德塞说，关于病毒“实验室泄漏论”需要进一步调查。联合专家组中方组长梁万年却重申世卫报告的结论及中方一向说法，即新冠病毒“极不可能”源于实验室事故。
  <br/>
  <br/>
  新华社3月31日发布中方记者会的报道，会场背景照片写的是《中国——世界卫生组织新冠病毒溯源联合研究中方专家新闻发布会》。这里出现几个令人难解的问题：
  <br/>
  <br/>
  一，世卫专家小组什么时候变成“联合专家组”？而且是“中国——世界卫生组织”的联合？中国摆在世界的前头，难道中国不是世卫组织的一个成员国吗？
  <br/>
  <br/>
  二，这个联合专家组竟是中方专家17人，外国专家17人，这是何时决定的？很符合中共“一帮一，一对红”的“人盯人”战术，中方可以轻而易举控制住最少一个外国专家，就立即成为多数，可以决定调查报告内容。
  <br/>
  <br/>
  三，中方有另一个记者会，是你说你的，我说我的？何来“联合”？
  <br/>
  <br/>
  上述情况所呈现的是，中国已经在世卫组织内搞分裂活动，是世卫组织变成中国与非中国两个部分？还是世卫组织让中国来决定一切，改变了世卫组织的性质？或者双方有默契，根据需要分分合合？
  <br/>
  <br/>
  这以前的3月24日，美国国务卿布林肯在接受CNN采访时就表示：“世卫组织不久将发表报告，我们对这份报告的方法和过程有着切实的担忧。例如，北京当局显然帮助撰写了这份报告。”布林肯的担忧已成为事实。美国已经恢复对世卫组织的资助，自然不会听任中共操弄这个组织，而自甘充当冤大头。
