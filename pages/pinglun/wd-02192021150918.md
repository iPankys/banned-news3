 <p>
  长久以来，西方社会，包括欧洲国家，面对中共在全球的扩张都是以美国为马首是瞻。与此同时，欧洲国家一向扮演的是在美国和中国之间调和的角色。英国作为欧洲大国，曾经长期被批评对华政策软弱，重视贸易利益大于对于人权和国家安全等问题的关注。而现在，国际形势发生了一些变化，传统上的领军国家美国由于政权轮替，拜登政府是否能继续扮演领导西方社会、在全球捍卫民主价值的角色，尚有待观察；而与此同时，同样值得观察的是，英国在对华政策上的立场却日趋强硬，似乎有与澳洲、印度等国成为对抗中共主力的态势。
  <br/>
  <br/>
  除了在是否采纳中国5G技术的问题上，英国展现了强硬的姿态，现在英国政府对于政治问题的立场也开始发生变化。最突出的就是在2月初，英国负责监督媒体的“通讯管理局”宣布，对总部设立在伦敦的中国大外宣机构“中国环球电视网”撤销其广播执照。我们知道，“中国环球电视网”的前身是中国中央电视台的外语国际频道，其伦敦总部负责全欧洲的广播业务。他们花大钱聘请CNN、BBC等西方主流媒体的记者与主播，政治目的十分明确，就是为中国政府的政策进行粉饰性质的宣传，可以说是名副其实的意识形态扩张工具。现在英国宣布撤照，对中共在整个欧洲宣传活动的布局都是一个沉重的打击。
  <br/>
  <br/>
  英国近年来对于中国的态度，从亲密友善逐渐变为警惕甚至对抗，不是没有特殊原因的。这首先应当说是香港问题的连锁效应。英国作为香港原来的管治者，对于香港有特殊的情感和关注，对于把香港交还给一个一党专制的国家一直有一种内心的不安。反送中运动之后，香港警方在街头强力镇压的画面，对很多关心香港的英国人来说极为震撼。之后中共颁布的《国安法》彻底撕毁“一国两制”，对英国来说等于是一种违约行为，让英国朝野上下对中国的不满大幅增长。这是中英关系中非常特殊和敏感的一部分，现在已经被中共触动。
  <br/>
  <br/>
  英国外交政策集团2月16日公布的一项调查显示，41%的受访者将中国视为严重威胁，高于去年的30%。调查还发现，只有22%的人支持英国政府与中国达成任何形式的经济协议，只有13%的人支持中国参与英国的基础设施建设；在人权问题上，40%的人支持英国就中国的人权记录发起挑战。该调查报告写道：“在过去的18个月里，英国与中国的关系发生了戏剧性的转变，部分原因是人们对安全漏洞的认识不断提高，但同时也是由于对中国国内的人权记录，及其作为全球参与者的行为越来越感到震惊。”另有报道称，在召开的七大工业国（G7）会议上，主办国英国的首相约翰逊已经计划着手制定应对中国的联合战略。
  <br/>
  <br/>
  种种迹象显示，在美国的中国政策还在举棋不定，欧洲本土的两个大国 -- 法国和德国已经表明态度，不愿意与美国结盟共同对付中国之后，英国的立场将有可能成为国际社会警惕中共势力扩张力量的新的希望。鉴于英国与美国传统上的紧密盟友关系，英国立场的强硬反过来也会对美国造成一定的影响。未来在全球政治格局中，面对中国的崛起，在澳洲、日本和印度之后，英国是否会在西方国家中起到领军作用，将十分令人期待。
