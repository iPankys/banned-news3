 <p>
  最近一个月，中国当局有组织地掀起了新一波民族主义浪潮。这波浪潮最早显示出D的迹象是3月初，在北京召开的全国人大、政协“两会”上，习近平的一番“中国已经可以平视世界”的言论。随后，美中外交高层在阿拉斯加会谈中，杨洁篪等中国外交官主动发动“战狼外交”攻势，中国国内官方媒体第一时间进行了大规模宣传；在欧盟加入对中共新疆官员的制裁行列后，中国也少见地迅速对欧盟国家官员进行反制裁。这一波民族主义浪潮的最新发展就是，围绕新疆棉问题展开的抵制Nike等西方大公司的全民运动。种种迹象表明，这是有预谋、有组织地安排，由中共当局在背后主导的一场民族主义动员。值得分析的问题就是：为什么现在要发动这样一场大规模的、全方位的民族主义动员？我认为有以下几个动机：
  <br/>
  <br/>
  首先是跟中共建党百年有关。习近平执政的一个重点，就是重新建立党对整个国家和社会的全面控制，恢复和确保中共的领导权威。今年的建党百年纪念，就是一个中共不会放过的机会。而中共长期以来把自己的合法性建立在经济发展和民族强大上，现在经济发展充满危机，唯一剩下可以动员民众支持的，就是中共可以带领中国走向强大这样的一个论述，民族主义动员有助于巩固中共一党专制的合法性。
  <br/>
  <br/>
  其次跟习近平本人有关。众所周知，明年中共二十大领导层换届，习近平对于连任有势在必得的决心，但党内也存在不少杂音。反对习近平连任的声音虽然无法公开发表，但是对习近平来说，仍然是一个隐患。此时如果中共作为全球围剿的对象，习近平不仅可以要求全党放下分歧，团结一致以对抗围剿，而且还可以通过塑造自己强硬立场的形象，来进一步说服党内不同派系认可他永久执政的必要性。
  <br/>
  <br/>
  第三个动机是长远打算。在全球化受到疫情冲击，中国经济发展的全球化红利也消耗殆尽的前提下，中国依靠与世界贸易关系来拉动经济增长的前景日益暗淡，这就是当局提出“内循环”的根本原因。中国不一定希望回到闭关锁国的状态，但是被迫要重新“自力更生”是中共必须面对的挑战。而民族主义动员，可以把经济上困境的焦点，用政治上的民族主义热情来稀释淡化，这也可以说是一种未雨绸缪。
  <br/>
  <br/>
  第四个动机是经济因素。随着“内循环”政策的提出，如何扶植和壮大中国的本土品牌，是中共面对经济危机必须考虑的事情。发起大规模抵制西方品牌，其中不乏借机壮大本土经济的用意。不难预料，接下来我们会看到中国官方对于“使用国产货”的大力宣传。
  <br/>
  <br/>
  最后一个可能的动机跟台湾问题有关。不管是对于中共还是习近平本身来说，解决台湾问题都关系到他们的统治基础，是他们在未来十年要做的事情中的重点。一旦要用武力解决台湾问题，民族主义动员就是必不可少的。这一波民族主义动员可以看作是一次测试，测试中国社会内部可以调动的民族主义资源的力量，为未来解决台湾问题开始进行评估。
  <br/>
  <br/>
  这一波新的民族主义动员背后的政府操作痕迹太过明显，一波接着一波的动员，也不是单一事件可以解释的。习近平和中共进行这样的操作显然有复杂和多因素的政治盘算，只是这样的如意算盘是否能够成功，其实还需要观察。
  <br/>
  <br/>
  以上是自由亚洲电台特约评论员王丹为您做的评论。
