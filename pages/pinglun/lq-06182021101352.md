  习近平要重走毛泽东文革老路，并且为追求个人崇拜再掀尊毛，已成今日大陆政治官场风气。在毛泽东腥风血雨文革半个多世纪，成为僵尸挺在天安门已奔五十年之时尤能如此，真可谓毛泽东阴魂不散。或许习近平当今对毛泽东所表现出的大力推崇，其利用成分远高于内心的真实崇拜，更多的是要实现自己的独裁和个人崇拜，毛泽东这尊共党魔头恰是能够善加利用的。不过，毛泽东对于上辈的中共元老们则绝非只是可以利用的一尊泥胎，而是皆败跪毛之膝下，想到便会半夜惊魂的邪党至尊。
  <br/>
  <br/>
  例如横蛮霸道、至死不认错的邓小平，他所最终坚持不改正的右派也不过七、八个人而已，但就是这七、八个人让他强硬坚持当年整肃各类右派二、三百万人，是必要而且正确的，仅是有点扩大化而已。但是对于僵尸一样挺在中南海里的毛泽东，邓小平却闻之丧胆，不惜颠倒黑白的丑化自己。
  <br/>
  <br/>
  再如将一般人玩弄于股掌之上的周恩来，在毛泽东面前总是一副唯唯诺诺的太监总管模样，凡事都竭尽心力曲意逢迎毛泽东所好。毛泽东遭逢困局时，周为其百般说项解困。如毛泽东因随意妄为而祸害大陆民众，并导致诸多不满甚至危及权势宝座时，周恩来便违背众议而竭力为毛泽东护航。周恩来可谓被毛泽东整得吓破了胆，临对生死难卜的手术台还嘶喊着输诚表忠，可见这个惯于揣度玩弄人的太监式党棍虽然曾有很长时间是毛泽东的上级或直接领导，却被毛泽东收拾到至死也不敢显露二心。

 <p>
  就是中共高层的康生、柯庆施等党棍，即使在惯于内斗撕咬的中共内部也是绝大多数人闻之色变的魔头，但是皆拜服毛泽东膝下，充当肚中蛔虫类打手。更不要说，被毛泽东斗死的数以百万、千万计的党徒，那些尸体铺满了自中共红区到老毛挺尸天安门的漫漫岁月，其中广为人知的如瘐死狱中的彭德怀、刘少奇、贺龙，以及因为年轻时曾与江青同居，被毛泽东吓成疯子而死的天津市委书记黄敬。
  <br/>
  <br/>
  虽然中共团伙的运行就是奉行炼蛊场法则，凡是进入炼蛊场撕咬的没有一个善茬，但是毛泽东能够在此中咬杀成众所败跪的蛊王，自然有其超越众多党棍的过恶本领。首先毛泽东从娘胎里就带有一项本能，就是极善识别他人的人性弱点并专对软肋下手。毛泽东自己夸耀说，幼年时与其父争执，在其父的专横高压下愤而要投水自尽，吓得双亲立即服软认输从而拿住他们软肋，此后其父再难以专横霸道占据他的上风。毛泽东这项做势吓人的手段在中共内也没少用，例如动辄以重上井冈山再闹革命，或是手捧宪法、党章说别人剥夺了他的权利，迫使中共唯唯诺诺地退让任其胡为。
  <br/>
  <br/>
  当然，毛泽东后天的学习和汲取经验也是让其成为一代炉火纯青权术家所必不可少的。毛泽东爱读书，可谓手不释卷。但是他从中国数千年书籍文化中汲取的，唯有如何权斗驾驭他人的霸术帝王术。例如，毛泽东在起用王洪文的过程中，就要王洪文看汉朝《刘盆子传》。可见毛泽东是从这些古书历史中汲取政治运作之术。
  <br/>
  <br/>
  不过，毛泽东之所以完胜中共炼蛊场，更重要的应该是整人、斗人对他来说其乐无穷。毛泽东自己说，“与天斗其乐无穷，与地斗其乐无穷，与人斗其乐无穷”。其实，毛泽东的三斗归总只有一斗，就是与人斗其乐无穷。与人斗是毛泽东兴趣所在的愉悦源泉，而其他中共党棍虽然也极其重视整人邪术，但大抵只视为手段，上升不到毛的境界。这二者之间的微妙差异，或许可以比喻为艺术家与匠人之别。因而中共尽管邪恶之人俯拾皆是，但遇上乐在其中的毛泽东，也只有败跪膝下了。今天习近平一心仿效毛泽东，但他天赋、见识、兴趣三无，怕是只有画虎不成反类犬的尴尬下场了。
