 <p class="p1">
  <span class="s1">
   今年是中国共产党建党百年。从年初开始，习近平和中共就号召开展全国“学党史”运动。为此，中共近日推出了
  </span>
  <span class="s2">
   2021
  </span>
  <span class="s1">
   年版的《中国共产党简史》。但是，中共真的有资格来写自己的历史吗？这部官方的新版党史就给出了否定的答案。
  </span>
 </p>
 <p class="p2">
  <span class="s2">
  </span>
 </p>
 <p class="p1">
  <span class="s2">
  </span>
  <span class="s1">
   新版《中国共产党简史》全书十章
  </span>
  <span class="s2">
   531
  </span>
  <span class="s1">
   页，约
  </span>
  <span class="s2">
   28
  </span>
  <span class="s1">
   万字。但其中，中共“十八大”之后的历史，也就是习近平执政之后的历史，就占了
  </span>
  <span class="s2">
   146
  </span>
  <span class="s1">
   页，字数占了全书的四分之一。百年的历史，其中十年的篇幅占去四分之一，这样的党史很明显是为现任的最高领导人习近平歌功颂德的党史；而中共其余
  </span>
  <span class="s2">
   90
  </span>
  <span class="s1">
   年的历史被大幅度压缩，这样的党史怎么可能是一部合格的历史书写呢？如果一个历史的书写者目的很明确，就是要用历史为政治来服务，甚至是涂抹历史，为某个个人打造光环，他是没有资格来写历史的。
  </span>
 </p>
 <p class="p2">
  <span class="s2">
  </span>
 </p>
 <p class="p1">
  <span class="s2">
  </span>
  <span class="s1">
   此外，这部中共写的党史充满了各种谎言。仅以台湾问题为例，新版党史在写到“太阳花学运”的时候，将其定位为“台独及外部势力在背后煽动、支持的一次反中事件”。我们都知道，“太阳花学运”是台湾新生代针对两岸服贸协议可能对台湾带来的负面冲击，自发举行的抗议示威活动，不仅没有提出台独诉求，更没有受到任何外部势力的煽动。当时我就在现场，亲眼看到连支持学生的民进党立法委员都无法顺利进入会场，更不可能参与运动的决策。这样的描述，可以说是彻头彻尾的编造，百分之一的真实性都没有。我相信，书中类似的问题一定比比皆是。这样一个满口谎言的党，有什么资格来写历史呢？
  </span>
 </p>
 <p class="p2">
  <span class="s2">
  </span>
 </p>
 <p class="p1">
  <span class="s2">
  </span>
  <span class="s1">
   更不用说，在新版的中共党史中，回避或者歪曲了中共一百年来从建党走向执政，从建国走到今天的大量历史事实。中共为了争夺政权发动内战，在长春围城，不让老百姓出城导致
  </span>
  <span class="s2">
   30
  </span>
  <span class="s1">
   万无辜民众饿死，中共党史会说吗？中共在
  </span>
  <span class="s2">
   1949
  </span>
  <span class="s1">
   年前后开展的土地改革运动，实质上就是用武力抢劫地主和富农，导致几百万农村的士绅阶层人士被肉体消灭，这样的事中共会写进党史吗？
  </span>
  <span class="s2">
   1959
  </span>
  <span class="s1">
   年开始的大饥荒饿死三、四千万中国人，创下了人类历史的记录，中共党史会把真实的死亡数字公布出来吗？文化大革命，连中共自己在“文革”结束的时候都承认是一场浩劫，现在习近平却说“不要用后三十年历史否定前三十年历史”，变相肯定这场浩劫，只说是“发展过程中的曲折”，这样的历史有一丝一毫的可信度吗？即使是新版党史最强调的习近平执政后的十年中，刘晓波仅仅因为言论问题被关押至死，中共会写吗？“
  </span>
  <span class="s2">
   709
  </span>
  <span class="s1">
   ”律师群体遭到全面的迫害，中共会写吗？汶川地震中的豆腐渣工程、毒奶粉事件、中国环境的全面破坏，中共会写吗？拿出官方版本的党史看看就知道，所有这些重大的事件都是发生在中共执政期间，都应当是中共党史中的一部分。但是，中共都不会写进历史，即使写进去也是极尽歪曲之能事。这样的中共，有什么资格来写自己的历史呢？
  </span>
 </p>
 <p class="p2">
  <span class="s2">
  </span>
 </p>
 <p class="p1">
  <span class="s2">
  </span>
  <span class="s1">
   中共作为一个百年老党，总结它的历史对于我们确实非常重要。只有看到真实的历史，我们才能对当今中国问题和中共问题的本质有更清晰的认知。但是，总结中共百年党史应当由人民来写，应当由客观的学者来写，应该由勇敢的史家来写，唯独不应该由中共自己来写。
