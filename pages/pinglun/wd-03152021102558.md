 <p>
  2022年2月4日即将在北京举行第24届冬季奥运会，在距离冬奥会开幕还有一年的时间，国际社会尤其是人权团体关于抵制北京主办冬奥会的呼声越来越高。2月3日，180个人权团体组成了抵制北京冬奥会联盟，抗议香港国安法、西藏和内蒙的宗教和文化政策，以及新疆的再教育营等中共暴政。美国和加拿大的议员也发出了抵制的呼吁。值得注意的是，3月4日，188名海外各界华人也发起联署，呼应国际公民社会的诉求。目前，联署还是继续进行中，人数已经接近300人。
  <br/>
  <br/>
  对于来自国际社会，包括国人自己的抵制北京冬奥会的呼吁，有些中国人不理解、不认同，认为举办冬奥会是国际社会给中国的荣誉，抵制奥运会就是反中辱华行为，就是把体育和政治绑在一起，而签署抵制呼吁的中国人就是“汉奸”等等。这样的观点是完全错误的，其核心问题还是在于没有把中国和中共分开。
  <br/>
  <br/>
  众所周知，最积极致力于把体育和政治捆绑在一起、借助体育运动进行所谓的“爱国主义宣传”的就是共产党政权。从八十年代推崇国家女排，引导出“振兴中华”的口号；到2008年奥运会的举办被完全解读为“洗刷百年国耻”，中共争取奥运会背后的主要目的，从来都不是为了给中华民族和中国人争光，而是为自己的统治涂脂抹粉，是为了巩固其专制制度的统治基础。
  <br/>
  <br/>
  我理解和支持中国民众希望中国强大并得到国际社会认可的民族主义心情，但我也认为，中国和中国人要被世界尊重，应当建立在对于普世价值的维护的基础上。而今天习近平和中共的所作所为完全与普世价值背道而驰，这样的一个政权，其实是在通过它们的政策给中国抹黑。这样的一个政权，是不配得到主办奥运会的机会的。
  <br/>
  <br/>
  更何况，全球疫情虽然有所缓解，但是对于中国造成的经济和社会损失惨重。在中国社会两极分化的今天，举办奥运会的巨大开支如果能够用于保障人民的民生建设，扶持中小企业的恢复生产，才是真正地把中国人民的福祉放在心上。奥运会就是中共的又一个面子工程，得到最大好处的是中共，而不是中国，更不是中国人民。因此，我们抵制2022年北京冬奥会，针对的并不是中国和中国人民，而是中国共产党。不明白这一点，就是不自觉地“把自己当成了赵家人”，分不清人民和政党以及政府的区别。
  <br/>
  <br/>
  更应当指出的是，2022年北京冬奥会一旦举办也必定是习近平的一场个人秀。习近平希望借助冬奥会的成功举办，拉升自己的声望，为自己能够在中共“二十大”上连任，为自己走向终身制壮大声势，制造舆论。而我们都知道，习近平大步伐向“文革”倒退，他扩大个人政治权力和影响力才是对中国人民和中华民族利益有害的事情。当年，西方社会纵容纳粹德国在希特勒的统治下举办了柏林奥运会，结果极大地鼓动了希特勒和德国极右翼势力的自信和野心。没过几年，他们就开始发动了第二次世界大战，最终德国自己也付出了惨重的代价。
  <br/>
  <br/>
  殷鉴不远，我们当然不希望看到这样的事情发生。因此，抵制北京冬奥会实际上也就是抵制习近平，抵制中共。这就是许多海外的华人，虽然热爱原本的母国，但是也主张抵制奥运会的原因。国人没有必要把自己跟中共和习近平绑在一起。
  <br/>
  <br/>
  以上是自由亚洲电台特约评论员王丹为您做的评论。
