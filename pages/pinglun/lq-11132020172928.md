  <br/>
  <br/>
  特朗普总统及其政府的领导班子，终于清醒意识到中共的伎俩和对民主世界的危险。中共所谓的弯道超车的发展手段，不过是坑蒙拐骗偷和无视世界市场规则。而中共意识形态上不仅不融入普世价值，而且是排斥敌视普世价值，并显露出要以中共所谓的成功手段，即中共意识形态和统治模式引领今后的世界。
  <br/>
  <br/>
  特朗普政府执掌政权之初，已经十分清醒的认识到，中共靠坑蒙拐骗偷的手段，正在侵害美国和其他市场经济国家，并正在摧毁和扭偏国际市场规则。于是从经济修整下手逐渐增加关税，并发展成经贸上全面紧压的贸易战。又因为中共无端关押数百万新疆维吾尔人，政治上宗教上残酷迫害和极端独裁，甚至公然废除关于香港的联合声明，最严重的是闹出中共病毒。至今不准国际调查毒源，让世界在染疫中饱受煎熬，终于认识并提升到中共不是中国，中国人民是友好的，尚未喊出的是中共则是美国之敌。而且特朗普政府并不是仅仅说说而已，一系列相关措施包括在世界上组建针对中共的联盟，全在紧锣密鼓的筹划进行中。
  <br/>
  <br/>
  历史的经验已经反复证明，与中共没有同生共荣的可能。国民党北伐前曾经与中共合作，结果中共仅只数百人的一个小团伙，不单短短二三年暴涨成数十万人，而且煽动民众迫交投名状，搞打土豪分田地的所谓土地革命，公然与国民政府另搞一套枪权分治。在国民政府军事围剿中共濒临灭亡之际，又利用了日本进占中国的机会，制造蛊惑言论煽动社会舆情，向国民政府虚承归顺之词，觅得死里逃生第二次国共合作。实质上中共用诸多伎俩，将国民政府军队迫上前线，与军力远超国军的日军拼耗，中共则实施七分发展二分应付一份抗日，借八年国军残酷血战时间，终在养精蓄锐迅猛发展的战后，打败国军夺去了大陆政权。就是世界在近几十年与中共相处，不也是有如第一次国共合作的翻版吗？中共一如当年不择手段的损害世界，并在极短的时间里发展得尾大不掉，而且已经显露出要引领世界的血盆大口。

  <br/>

  <br/>
