 <p>
  日前在新兴的社交媒体Clubhouse上与大家聊天时，主持人问：“上来的新朋友怎么用的都是动物的名字？”有朋友立即回答说：“因为害怕中共对我们进行打击报复所以不用真名，就连所用的头像也都是假的。我们担心，自己有一天会变成李明哲。”
  <br/>
  <br/>
  听到这里，我立刻想起了台湾的李明哲先生至今还在中共的牢狱之中；还有，中共把魔爪伸到沦陷区之外，绑架香港的出版商，跨境对桂明海进行绑架，并用酷刑折磨逼迫他放弃瑞典国籍，成为待宰的羔羊；还有通过构陷，抓捕、关押加拿大等其他国家的公民进行“人质外交”的一系列案件，都一一浮现于脑海。
  <br/>
  <br/>
  联合国《人权宪章》中所说的“公民有免于恐惧的自由……，免于被任意羁押的基本人权保障”等等规定，不是已经被中国的共产专制政权给彻底破坏了吗？这样的结果充分说明，中共政权哪有资格在联合国里继续代表中国人民！
  <br/>
  <br/>
  无独有偶，上周五，我收到朋友转来的一篇有关“全球受中共病毒打击最严重的10个国家中，有一半以上使用中国疫苗”的媒体报道。数据来源于一个叫Our World in Data 的机构，它是由英国牛津大学创建的在线数据库，提供每百万人每天的感染数据。该机构通过对塞舌尔、乌拉圭、蒙古、科威特、智利等国家的感染数据进行分析后，得出上述结论。该报道通过对全球范围内的统计结果，告诉我们一个惊人的事实是：哪个国家的政府与中共暴政走得近，哪个国家的人民就会遭殃、受害。
  <br/>
  <br/>
  这样的事实必须引起还在试图接近中共暴政、与中共搞好关系的西方国家提高警惕。在中国传统文化被誉为“群经之首”的《易经》中，有“积善之家必有馀庆，积不善之家必有馀殃”的说法。当今世界，与一个对内迫害自己的人民，对外破坏文明秩序的反人类政权同流合污，几乎等同于助纣为虐，实属大恶之举。
  <br/>
  <br/>
  谁靠近中共谁倒霉的事实说明：中共者，不祥之物也。要想使自己的国家国泰民安，那就离中共越远越好。一个对自己的国民都能大开杀戒的政权，怎么可能会善待他国人民？！
 </p>
 <p>
  期待世人对此，能有所警醒。
