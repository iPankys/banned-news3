  5月1日至5日，是中国五一劳动节长达5天的假期。由于旅游禁令的放松，也由于不少人今年初没有回家过年，因此中国全境1日起，出现大批出外旅游及返乡人潮，铁路、公路、车站及机场都挤满旅客，高速公路也出现拥塞。但河北地质大学以考虑疫情防控的需要为由，校方宣布不允许学生五一劳动节5天假期自行离校，必须经学校审批才能离开。这一禁止学生在假期离校的作法，引起学生普遍的不满。
  <br/>
  <br/>
  在网络上流传的一段视屏显示，5月1日，有约100多名学生聚集在校门口，一度与保安人员对峙。视屏画面显示，保安人员也仅有寥寥数人，且未见到其他校方人员。根据画面，学生中有人趁保安人员不注意推开闸门，率先从闸门缝隙冲出校门，其他学生见状纷纷接着涌出，还回头鼓励校内其他同学大胆出校。一些学生更高兴地举起大姆指叫好。“寡不敌众”的保安人员，最终也只能眼睁睁看着学生们离开。
  <br/>
  <br/>
  众所周知，在中国近代史上，通常总是莘莘学子比普通大众更多一点社会意识和反抗意识，无论是康梁变法，还是五四运动等等都是如此。震惊世界的六四事件，其发起者和主体力量也是在校的大学生。
  <br/>
  <br/>
  但很可悲也很奇特的是，六四以后，在中共的严厉控制之下，中共的暴力威慑和洗脑措施似乎大见成效。大学生似乎都丧失了独立思考的能力和反抗邪恶的意识，甚至还有不少人发自内心对中共顶礼膜拜！
  <br/>
  <br/>
  虽然在惹大的中国，河北一百多名大学生突破封锁冲出校园，这似乎是不显眼的小事件，但却有很强的象征意义。
  <br/>
  <br/>
  目前校方的表态是，冲出校园的学生受到了批评。但在国际反共联盟正在形成，中共感受压力日益巨大的时刻，中共对国内的稳定及其担忧，对这一事件是否有后续动作有待观察。
  <br/>
  <br/>
  可是，不管官方最后怎么处理这一事件，都不能抹杀这一事件的意义：这表明在过去三十多年变得唯唯诺诺的大学生，对官方的决定不仅有了对抗意识，同时也有了采取对抗行动的勇气。
  <br/>
  <br/>
