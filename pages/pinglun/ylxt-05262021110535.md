 <p>
  忘不了2009年7·5之后，寻找被警察于2009年7月9日自家里抓走儿子的维吾尔母亲帕提古丽·古拉姆，于2014年在自己被中国政府失踪前，接受自由亚洲电台采访时的哭诉：“羡慕巴勒斯坦和叙利亚的母亲，他们至少可以亲手埋葬自己被打死的儿女；而我的孩子从家里被抓走，至今活不见人、死不见尸。我只是想最后再看一眼我的儿子呀，为他洗去灰尘，亲手埋葬他，难道这还过分吗？”
  <br/>
  <br/>
  上周，下班回家路上，听美国国家广播电台深度报道巴以冲突。记者连线在加沙的一位女大学生，采访这位21岁女大学生对巴以冲突的看法和感受。对我触动最深的是，女大学生告诉记者，每次听到以色列空袭警报，她们全家都要跑到客厅里，大家抱在一起等待。当记者问她，为什么全家都要跑到客厅躲避时，大学生回答：要死大家一起死！
  <br/>
  <br/>
  巴勒斯坦人可以选择一家人死在一起，或者活在一起；他们在战火纷飞、冲突不断的巴勒斯坦有生死选择权！
  <br/>
  <br/>
  当记者问巴勒斯坦女大学生，她每天做什么时，她回答：每天上推特等社交媒体，查看每天新闻，查看是否有同学、邻里突然消失，突然消失意味着可能死在空袭中。也就是说，她们能很快搞清楚亲人邻里死活、伤亡情况。
  <br/>
  <br/>
  当然，我无意在此文章里讨论巴以冲突的是非曲直，早已有很多中东问题的国际专家、学者将巴以问题讨论的淋漓尽致，无需我班门弄斧。
  <br/>
  <br/>
  巴以冲突频繁、血腥，毋庸讳言。战争中，双方无论多么克制，多么照顾平民，战争仍然无法摆脱其野蛮性。但以色列作为强者，在能够斩断加沙与外界联系的条件下，仍然允许平民通讯联络，允许加沙巴勒斯坦平民与外界联系；在战争之残酷中，凸显一点人类恻隐之心和文明的花火。
  <br/>
  <br/>
  巴勒斯坦人，巴勒斯坦母亲，不仅可以亲手埋葬他们死亡的亲人，还可以在腥风血雨的战火中，始终不断地使用现代科技成果 -- 互联网，使用各类社交媒体，向外界，向世界通报正在发生的战争之残暴，表达各自对立的观点；而且，还可以互通信息，亲朋邻里保持联络，随时发现谁家被炸了，什么人死了。
  <br/>
  <br/>
  和冲突持续不断的巴以相比，新疆，没有战争也没有空袭警报，中国殖民政府拥有一切、垄断一切；而维吾尔人则手无寸铁被严重边缘化，因而维吾尔人的不满反抗都是和平示威。然而，新疆却是世界上军队最多、警察最多、监狱最多、集中营最多、强制失踪者最多的一个地方，而且还是最黑暗的信息黑洞。
 </p>

  <br/>
  自中国殖民占领后，新疆一直就有反抗者失踪。尤其是90年代之后的几次大的维吾尔人抗议示威，如1990年的4·5巴仁乡农民抗议示威，维吾尔人死亡人数、失踪人数一直就是个迷；1997年的伊犁2·5示威抗议，也有大批维吾尔年青人失踪；2009年的7·5之后，失踪的维吾尔年轻人更是成千上万。7·5失踪的维吾尔年轻人，他们绝大多数是从家里、在光天化日之下被警察抓走的，但也都就此人间蒸发，活不见人、死不见尸。
  <br/>
  <br/>
  2016 年起更甚，有成千上万维吾尔大学校长、专家学者、教授、教师、诗人、作家、艺术家、歌唱家、企业家、慈善家、球星、模特、农民、牧民，甚至于一生都为共产党效忠的维吾尔干部也都失踪了，杳无音讯、踪迹全无。
  <br/>
  <br/>
  在新疆的维吾尔人，他们不敢寻找失踪亲人，寻找的结果就如寻找儿子的维吾尔母亲帕提古丽·古拉姆，最后自己也会被吞噬于中国殖民政府的暴政黑暗中！
  <br/>
  <br/>
  因而，在海外，在中国驻各国使领馆前，自2017年起，出现了寻找亲人维吾尔、哈萨克人的身影。每天，在伊斯坦布尔的中国领事馆前，在哈萨克斯坦阿拉木图的中国领事馆前，在荷兰阿姆斯特丹的中国大使馆前，在澳大利亚中国使领馆前……，几乎每天都有无数个维吾尔人、哈萨克人，手举着他们失踪亲人的照片：父亲、母亲、儿女、丈夫、妻子，寻找被中国殖民政府强制失踪的亲人！
  <br/>
  <br/>
  中国自称拥有五千年文明，自称崛起大国，习近平更是号称要和美国为首的民主国家平起平坐，然而却没有自信打开互联网，更没有自信允许其统治下的民众拥有基本人权；更遑论允许维吾尔人自由迁徙、自由表达观点和保留自己的文化、传统和信仰！
  <br/>
  <br/>
  很多中国人觉得巴勒斯坦人不幸、叙利亚人不幸、阿富汗人不幸，总以为他们的国家战乱不断、生活在水深火热之中。然而，中国人不知道是，巴勒斯坦人、叙利亚人、阿富汗人有迁徙的自由、有表达的自由，有互联网、有畅通的信息、联络渠道，远在天涯海角还能知道家乡亲人安危，能够选择自己的生活方式。
  <br/>
  <br/>
  就如寻找失踪儿子而失踪的维吾尔母亲帕提古丽﹒古拉姆哭诉的，巴勒斯坦、叙利亚的母亲至少可以亲手洗去死亡儿女尸体上的灰尘，最后再看一眼儿女，和他们亲吻告别，亲手埋葬他们，每年还可以上坟为亡灵祈祷和平安详。而新疆的维吾尔人、哈萨克人，却被殖民中国政府残酷剥夺了这一在战乱中挣扎的巴勒斯坦人所拥有的最基本人权 -- 亲情知情权。
  <br/>
  <br/>
  我们不知道自己的亲人是否活着，不知道他们的孩子在哪儿？已经四年多了，我不知道自己的母亲、三个妹妹在哪儿？也不知道她们是否还活着？更不知道三个妹妹的丈夫、儿女在哪儿？是否还活着？
  <br/>
  <br/>
  美国维吾尔人孜芭·阿巴斯在寻找失踪母亲古丽仙医生，森塔什在寻找失踪的、已退休文学家父亲，阿尔法特在寻找失踪电视台记者父亲，还有很多美国维吾尔人在寻找失踪的哥哥、弟弟、父亲、母亲、同学、朋友……。
  <br/>
  <br/>
  文明的核心是人的尊严，文明的基础是对国际法的尊重。文明必须体现在对《世界人权宣言》所规定的基本人权的落实，文明必须体现在对亲情知情权的尊重！文明不应该成为强权的遮羞布，不应该成为只顾经济利益无视人权之借口，更不应该成为种族屠杀的工具！
  <br/>
  <br/>
  中国政府，还我们的亲人！
