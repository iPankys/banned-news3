  2020年10月13日，联合国人权理事会（UNHRC）进行席次改选，投票结果出炉，中国、俄罗斯、古巴、巴基斯坦、乌兹别克等人权状况相当恶劣的国家高票当选。在亚太区，中国的得票虽然落后于巴基斯坦、乌兹别克和尼泊尔，却仍获得139票的高票。

  <br/>
  <br/>
  今天的国际社会并非“得道多助，失道寡助”，而是“有钱能使鬼推磨”。中国的金钱外交、撒币外交，收买了不少嗷嗷待哺的亚非拉小国、穷国以及跟中国臭味相投的独裁国家，很多经济疲软的欧洲国家也争先恐后投入中国的怀抱。联合国丧失了建立的初衷，其宪章被中共及其一帮小兄弟任意践踏。
  <br/>
  <br/>
  与世界卫生组织一样，联合国以及若干国际机构，不但未能匡扶国际正义，反倒成为独裁政权的华丽橱窗及权力延伸。据《华尔街日报》报导，代表中国政府及联合国的两位中国籍官员签署意向书，联合国大数据研究中心将落脚中国杭州——这一场景宛如左手与右手亲密握手，完全不在乎旁人怎么看。习近平在9月即宣布，中国将设立“联合国全球地理信息知识与创新中心和可持续发展大数据国际研究中心”，该中心与联合国的数据库实现接轨具有重大意义：联合国的合法性招牌将使北京获得联合国各成员国的数据，进而将中共高科技暴政投射到世界各地。
  <br/>
  <br/>
  然而，桃李不言，下自成蹊。独立NGO机构“联合国观察”主席诺伊尔（Hillel Neuer）讽刺人权理事会让中国、俄罗斯等“最恶劣的人权侵犯者”当选堪称“让纵火犯加入消防队！”2020年4月，中国被任命加入联合国人权理事会协商小组、获得遴选人权调查专员的权力，诺伊尔即公开质疑这一任命“非常荒谬而且毫无道德可言”。
  <br/>
  <br/>
  对联合国观察这样的NGO的观点，财大气粗的中国置之不理。但美国国务卿蓬佩奥的声明，则让中国的胜利蒙上一层阴影。蓬佩奥直指此次改选是“拥抱独裁者”，他指出：“中国和俄罗斯、古巴都赢得联合国人权理事会席位，这是暴君的胜利，联合国的难堪。这是一个例证，一个指标，说明为什么我们离开这个机构是正确的。当机构变得无可救药，特朗普总统领导下的美国决不会参与。”
  <br/>
  <br/>
  退出绝不是美国最后的杀手锏，美国正在计划抛开联合国人权理事会、世界卫生组织、联合国教科文组织乃至联合国本身，建立一个新的“民主国家联盟”，以处理新形态之下的国际事务。在二战之后，美国领衔建立联合国等国际组织时，未能将独裁专制国家排除在外，留下无穷后患。如今，在中国的天下帝国主义肆虐全球的时刻，美国需要坚定地定义善恶、区分敌友、重建同盟、孤立中国。
