 <p>
  六四过去三十二年了！六四是中共实行独裁统治后，青年人第一次带有相对独立意识的奋起，在短短的几十天时间里，以大学生为主体的年轻人，以集体的力量宣示了对社会的责任和对未来的追求。然而，中共用暴力对年轻人进行了屠杀和镇压！
  <br/>
  <br/>
  六四之后，中共以暴力威慑和强制洗脑，更强化了对社会尤其是对年轻人的控制。在涂抹浪漫色彩的理想主义教育和充满物质引诱的心灵鸡汤双重浇灌下，真正为社会进步呐喊、为社会的公平和正义奔走，成了被嘲笑的愚不可及的言行。六四之后，中国社会再也没有发生过年轻人整体性为社会目标发声、呐喊和行动的事件！
  <br/>
  <br/>
  80后、90后、00后一拨一拨年轻人确实在努力和奋斗，不过他们是在虚幻化的社会理想幌子下，实打实地为物质生活上的成功努力打拼。可是在权贵资本体制之下，阶层固化的壁垒难以突破。除了个别例外，绝大多数年轻人的努力和奋斗都成了可怜的笑话。即便以“996”的节奏拼命，也看不到美好的前景。越来越多的年轻人体认到：无论怎样努力，结果都没有多大区别。花费如此巨大的生活成本是否值得？就成了一个巨大的问号！
  <br/>
  <br/>
  对这个问号的回答，今年五月有了一个突破性的答案：就是躺平！与其跟随社会期望坚持奋斗，不如选择“躺平”的处事态度。“躺平”一词最初出现在一篇网络短文：“两年多没有工作了，都在玩，没觉得哪里不对。压力主要来自身边人互相对比后，寻找的定位和长辈的传统观念......，人大可不必如此。既然这片土地从没有真实存在、高举人主体性的思潮，那我可以自己制造给自己，躺平就是我的智者运动。只有躺平，人才是万物的尺度。”
  <br/>
  <br/>
  接着“躺平”一词迅速在网上发酵，演变成了：不结婚、不生子、不买房、不买车、不创业的躺平主义！
  <br/>
  <br/>
  六四站起来的青年人被中共推倒，三十多年后，感悟到自己根本站不起来的年轻人干脆选择了彻底躺倒、躺平！不曾想，这看似无奈之下的懦弱反应竟引起了中共当局的恐惧。因为这是年轻一代在认命后的抗议，成了一种“宁静”的示威！
  <br/>
  <br/>
  5月27日，共青团中央发微博呼吁年轻人“不负使命，不负家国”，不要选择“躺平”。结果，这条微博下方的评论区大翻车，嘲讽的评论汹涌澎湃，迫使官方微博关闭了评论区。躺平主义正在势如破竹的涌动！有评论推测：民间的躺平主义和习大帝的加速主义正交相辉映，这两股力量的交互作用将使中共控制了七十多年的破车分崩离析！
