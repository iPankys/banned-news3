 <p>
  每当国际上发生一件大事，中共的宣传机器都会炮制出它自己的一套叙事版本。美军撤离阿富汗、塔利班卷土重来，中共宣传机器不会放过这个“天赐良机”。
  <br/>
  <br/>
  这一回，针对阿富汗局势的中共叙事成为大内宣的又一个典型：反美，并为塔利班洗地。综合北京党媒、党报精心策划的报道、评述，包括标题设计，其中心思想、主要调子，以及要对中国民众达成的洗脑效果就是：美国在阿富汗失败了，美军因为战败而撤离阿富汗；塔利班赢得了民族解放；塔利班卷土重来、迅速夺取政权，是阿富汗民心所向；塔利班重新当政，是阿富汗人民的选择；美国向阿富汗推广西方民主制度却失败了，因为水土不服；一个国家选择什么样的制度只能由这个国家的国情决定，外部力量不能强加；西方民主制度并没有优越性；台湾注意了：美国既然能放弃阿富汗，也能放弃台湾……。
  <br/>
  <br/>
  透过中共的宣传，对应的事实却恰恰相反：美军远征阿富汗，是因为九一一恐怖攻击（2001年）。这一事件迫使美国发动反恐战争，消灭基地组织，推翻庇护基地组织的塔利班（2001年），并最终击杀了策动九一一恐怖攻击的基地组织首脑本拉登（2011年）。自那以后，美国国内就响起了撤离阿富汗的声音，并从2014年开始逐步减少驻军，逐渐撤离。
  <br/>
  <br/>
  同期，美国帮助重建阿富汗，先后援助阿富汗高达1500亿美元。阿富汗国民生产总值从2002年的44.55亿美元，达到2013年的205.61亿美元，十二年间就翻了五倍，年增长率高达9%。美国也帮助建设阿富汗民主，正因为有了民主选举、言论自由和人权保障，尤其女性的人权保障，阿富汗社会呈现前所未有的生机勃勃、欣欣向荣。
  <br/>
  <br/>
  阿富汗并非不适合民主，而是民主遭颠覆，遭塔利班暴力颠覆；美军宣布完全撤离后，塔利班卷土重来，阿富汗赢得的不是民族解放，而是民族奴役；塔利班迅速占领首都喀布尔不是因为民心所向，而是因为民心所恐；民众不怕民选政府，就恐惧杀人如麻的塔利班；塔利班宣布自行当政并非阿富汗人民的选择，恰恰相反，塔利班进城之日就是阿富汗人民逃难之日。喀布尔国际机场人山人海，阿富汗人民恨不得全都跟美军一起离开就是明证。
  <br/>
  <br/>
  至于中共拿美军撤离阿富汗来威胁台湾乃是中共的心理战，虽有台湾个别统派人士配合演出这种心理战，说出一些似是而非、长他人志气、灭自己威风的糊涂话，但绝大多数台湾人民并未因此被吓倒，反而增强了保家卫国的危机意识。台湾外交部长吴钊燮如此回应：“中共想要仿效塔利班，但台湾有决心和能力保护自己。”言下之意：中共可以自比塔利班，但台湾不是阿富汗。
  <br/>
  <br/>
  中共这场对台心理战的结果还换得了美国总统拜登的一番意外回应：“阿富汗问题和台湾、韩国，以及北约之间存在根本区别。如果有人入侵或对北约盟友采取行动，美国将做出回应，对日本、韩国和台湾也一样。”言下之意：台湾与美国的盟友地位等同于北约、日本和韩国。这是美国总统的首次表态，把美国是否协防台湾的议题从战略模糊变为战略清晰。
  <br/>
  <br/>
  中共的宣传分为大内宣和大外宣：针对国际社会的大外宣，中共从来就失败，无论砸下多少钱；但针对国内民众的大内宣，中共却相对“成功”，即洗脑相对“成功”。中共的功夫其实很简单：对国内民众展开单方面的信息轰炸，同时屏蔽其他消息来源；对事件进行单方面评述，铺天盖地，同时封杀所有不同观点。其潜台词是：“不管你信不信，反正我是信了。”或者，“信不信由不得你，由我。”语带威胁，甚而以“煽颠罪”和“妄议罪”恐吓民间和党内的不同声音。这便是中共大内宣的“成功之道”。
