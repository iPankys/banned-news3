 <p>
  习近平在外交上主张战狼精神，声称“要解决挨骂的问题”。于是，中共战狼外交正式走上国际舞台，且不断升级。上到政治局委员杨洁篪、外交部长王毅，下到外交部发言人、中共驻各国大使，纷纷展示战狼嘴脸，展开战狼比赛。
  <br/>
  <br/>
  完成任务，挣表现，是中共特有的党文化。在这波战狼表演中，这种党文化展现得淋漓尽致。一个名叫卢沙野的人，从中共驻加拿大大使到驻法国大使，充分发挥战狼外交，对加拿大和法国竭尽撒野、造谣、辱骂、人生攻击之能事，以至于屡受法国外交部召见。其中，他辱骂一位法国学者是“小混混”、“小流氓”、“疯狗”、“喷子”，震动法国。
  <br/>
  <br/>
  前不久，中共驻巴西里约热内卢总领事李杨发推文攻击加拿大总理特鲁多是“美国的走狗”、“败家子”。而去年，中共驻巴西大使杨万明攻击巴西总统“从美国带回思想病毒”，遭到巴西外交部召见和抗议。
  <br/>
  <br/>
  近日，中共驻瑞典大使桂从友攻击一名瑞典驻台湾记者，说后者“散布错误信息”、“与台独者密谋”、“道德败坏”等，激起瑞典几乎所有在野党的愤怒和谴责，纷纷要求瑞典政府将这名中共战狼大使驱逐出境。因为，瑞典外交部召见这名中共大使已经不知多少次，召见对他已经不起作用。
  <br/>
  <br/>
  几乎同期，在土耳其，因为两名在野党领袖批评中共的新疆政策，遭到中共驻土耳其大使馆恶言相向。土耳其外交部召见中共大使，提出交涉。随后传出中共驻土耳其大使馆遭断水的消息，虽然难以证实断水的虚实，但中、土关系由此受损已是显而易见。
  <br/>
  <br/>
  回溯今年3月18日，美中高官在阿拉斯加举行会谈，这个会谈原本出自中方的请求，美方勉强安排。不料，中共政治局委员杨洁篪却利用这个机会，在开场白中放肆地攻击美国，声言：“美国没有资格居高临下同中国说话”、“中国人不吃这一套”，云云。这种高级别的战狼姿态，表现给美国人看，更多地表现给中国人看，被国际舆论评为“演戏”、“夸张的演出”。
  <br/>
  <br/>
  国内外中国人不禁纳闷：中共展开战狼外交，其外交官展开战狼比赛，一人凶过一人，一浪高过一浪，究竟捞到了什么？
  <br/>
  <br/>
  如前所述，中共外交官的每一次战狼表演都付出了代价，甚至是沉重代价：轻则被所在国外交部召见，重则面临被驱逐风险，更重则导致谈判破裂、或双边关系严重受损。既然没有捞到任何东西，战狼外交为何还要持续？
  <br/>
  <br/>
  最高领导人的自负、狂妄、无知和低水准是始作俑者，把外界的批评，多半是善意的批评视作“挨骂”；把中共自己宣传喉舌长期的对外攻击性语言视作“自卫”；居然犹嫌不够，还要求其外交人员抛开基本的外交礼节、外交辞令，直接开骂，货真价实地开骂，甚至不惜用上人身攻击、污言秽语。
  <br/>
  <br/>
  中共外交官在最高权势者的压力下，或者为了立功受赏、加官进爵、飞黄腾达，或者为了规避风险、保全自身、平安退休，纷纷挣表现。于是出现了令人匪夷所思的战狼外交、战狼比赛，搅得这个世界不得安宁，以至于烽烟四起，让共产中国八面受敌、四面楚歌。
  <br/>
  <br/>
  一些理智的中国网民评说：外交部变成“断交部”；一直不准中国人批评，现在还不准外国人批评；在中国限制言论自由，现在又到外国去限制言论自由。的确，那些受到召见的中共外交官中，他们或许明白或许不明白所在国召见和抗议，简单的一条，就是他们攻击、干涉、限制所在国政要、学者、记者和公民的言论自由，其实就是干涉了他国内政。因为，在这些民主国家、正常国家，言论自由是最基本的人权，是所在国的立国基石。
  <br/>
  <br/>
  “外交是内政的延伸”，这是苏联创始人列宁的名言。果然，今日共产中国对内不讲理，必然变成对外不讲理；对内专制，必然衍生对外专制；对内镇压，必然带来对外威胁。还是笔者那句老话：没有什么中国威胁，只有中共威胁，那是中国人民和世界人民面临的共同威胁。

  <br/>
  （2021年4月13日）
