 <p>
  《香港基本法》和《澳门基本法》是从宪法的高度，确定了完全不同于中国内地的“港人治港、高度自治”等法律原则和地位。《香港基本法》是特别法，其地位高于除宪法以外的其它任何一般法律。而中共置《香港基本法》于不顾，强行制定所谓的“港版国安法”，显然破坏了《基本法》的法律地位，中共以此彻底控制香港。
  <br/>
  <br/>
  港版国安恶法仅实施几个月，继香港的司法独立几乎彻底被中共利用“港版国安法”破坏殆尽之后，中共紧接着又对香港的言论自由开刀了。在壹传媒之下运营了26年的《苹果日报》近日经董事会决定，为了员工们的安全，宣布于2021年6月24日停止出版。
  <br/>
  <br/>
  消息一出令港人心痛不已，对中共打压破坏香港的言论自由，逼迫《苹果日报》停刊义愤填膺。24日最后一期的一百万份《苹果日报》上市后，很快就被市民冒雨赶来抢购一空。不仅如此，一些香港市民还在街头把自己多买的报纸转给因为迟来一步，没能买到报纸的其他市民。
  <br/>
  <br/>
  港人依依不捨挥泪抢别《苹果日报》的场面非常感人，人心所向一目了然。甚至很多港人在网上讨论分享，怎样把黎明前的最后一期《苹果日报》裱起来保存，以防止变黄。这不仅是留作纪念，更是一次期待天亮后，《苹果日报》必然重新在香港复刊的伟大日子早日到来。
  <br/>
  <br/>
  共产邪恶专制体制下，正义难存。6月17日，香港警务处国家安全处以“危害国家安全”为名，派出500名警员搜查《苹果日报》大楼，并拘捕了壹传媒和《苹果日报》的5名高层主管后，又于23日抓捕《苹果日报》社论的主笔李平（原名杨清奇）。
  <br/>
  <br/>
  迫于中共的迫害，为了员工的安全，壹传媒董事会决定《苹果日报》停刊，旗下的《壹週刊》也危在旦夕。而且在此之前，中共就冻结了壹传媒的上市资金。由此足以见到，中共反人类政权对司法独立与新闻、言论自由是多么的恨之入骨。只是世人完全没想到，在香港也出现了如同2008年，很多北京市民带着鲜花到Google大楼前献花的场景，今年也在香港《苹果日报》大楼前发生在港人身上。
  <br/>
  <br/>
  我们不禁要问：为什么在国际社会、民主国家为推行民主、保护人权与言论自由努力了这么多年之后，如今却出现共产专制侵蚀香港？在众目睽睽之下，香港的言论自由与司法独立全都遭破坏，沦陷于专制的魔爪，港人洒泪抢别《苹果日报》的事情？正常情况下，照理应该是民主体质在媒体自由与司法独立已经成熟的香港顺利扎根成长才对。
  <br/>
  <br/>
  面对这种反常的现象，自由世界应该作何反思？是不是说明民主国家过去的方法不对，或者有问题？究竟出了什么问题？应该采取怎样的措施立即纠正？这些问题都亟待自由世界认真反省。
  <br/>
  <br/>
  试想，一个共产邪魔横行世界的人世间，谁又能独善其身？现在我们已经看到并亲身体会到，越来越多的各方面代价正在付出之中。面对邪恶的魔鬼只有奋起铲除，任何明哲保身掩盖之下的懦弱逃避，都会使包括我们每个人在内的人类社会，于现在和将来付出更大、更惨重的代价。
