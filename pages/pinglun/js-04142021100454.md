<div id="storytext">
 <p>
  近日，中国校园空气紧张，包括大、中、小学在内，得到中共教育部的内部通知，要求清理“校园流毒”，即清除“问题读物”，包括排除西方读物，只留马列著作。为此，还开列了“12条负面清单”。据知，这个内部通知和负面清单由最高领导人习近平亲自批示。
  <br/>
  <br/>
  这是习近平的又一波折腾，在极左路线上的加速折腾。习式折腾包括：强化一党专政，堵死宪政萌芽；灭绝民族文化，强制汉化少数民族；国进民退，内循环；战狼外交，反西方；重编党史，用真正的历史虚无主义去代替他所指控的“历史虚无主义”……。
  <br/>
  <br/>
  习近平有一个幻想，以为现在是清除西方文化影响、重建中国传统文化的时候了，而习近平所理解的中国传统文化，就是东方专制文化。习近平和中共把最近几十年中国经济上的恢复、物质上的暴富、国力和军事硬实力的膨胀视为本钱，不得了的本钱、唯一本钱，因而发出“豪迈”之语：“时与势在我们这一边”。
  <br/>
  <br/>
  习近平的幻想，不知所从何来？或许是毛左派的极端思维影响了他；或许是被一批极端民族主义的学者包围，构成他身边的智囊群，进而塑造了他的强硬思维；或许就是他本人的偏激固执思维所致。或许，是上述多重因素的综合。
  <br/>
  <br/>
  习近平的幻想，实际上是习近平们的幻想。因为，不止他一个人陷入这种幻想，而且是一批人、一部分人、甚至数亿受他们蛊惑的中国人陷入这种幻想。
  <br/>
  <br/>
  习近平的这种幻想，又分为三层含义：其一，以为能够彻底清除西方思潮和文化；其二，以为只要清除了西方思潮和文化，就能够恢复中国传统文化（东方专制文化）；其三，以为只要清除了西方思潮和文化，就能够在国际上自立门户，联合反美国家构建以北京为中心的另一个世界，与以美国为首的民主阵营分庭抗礼，甚至取而代之。
  <br/>
  <br/>
  在国际交往和融合的近几个世纪里，不同文化的交流彼此影响和彼此借鉴，优胜劣汰，本来就是自然而然的事，既无须强力促成也无须强力抗拒。如今的习近平们，竟然要去强力清除，其注定的徒劳无功，犹如希腊神话中往山上滚石头的西西弗斯。
  <br/>
  <br/>
  西方文化以个人主义和人权理念为中心，中国文化以集体主义和专制王朝为中心。随着人类的进化、个性的张扬、文明的演绎，哪一种价值观更得人心、更具生命力、更有前途？明白人一望便知。
  <br/>
  <br/>
  近代西方思潮与文化，包括马列主义，在清扫西方文化与思潮时，能否一并清扫马列主义？这对习近平们而言，是一个无法破解的难题。因为马列主义正是中共的立党之本、立国之本，王沪宁已经给习近平加冕：当代最伟大的马克思主义者。
  <br/>
  <br/>
  以习近平们的短视，根本无法全面诠释中华文化。即便习近平动用举国体制，费尽全力清除西方文化与思潮，能否就此恢复东方专制文化？且不说民众不可能接受王朝与皇权回归，在中共党内就难以通行。如果习近平一意孤行、恣意妄为，恐反而促成党内反习势力暗中坐大，一旦时机成熟，习近平和习家军必全线崩溃。
  <br/>
  <br/>
  当下中国的制度、土壤和气候似乎都具备复辟文革的条件，故而习近平、王沪宁等人放胆重编文革史，放肆地尊江抑赵（开放江青墓地而严封赵紫阳墓地），平反臭名昭著的极左派，而封杀深得民心的自由派。然而，这个历史倒车能开多远？历史的经验昭告世人：倒行逆施者不得人心！翻车，是迟早的事。
  <br/>
  <br/>
  政治局委员杨洁篪奉旨宣布：“不承认美国说的就是国际舆论，不承认少数国家制定的规则就是国际规则。”这里所说的美国或少数国家，只是中共糊弄国内民众的托辞。其实，杨洁篪想要表达的意思是：不承认主流的国际舆论，不承认当今的国际规则。
  <br/>
  <br/>
  习近平自恃当下国力雄厚、军力强大，以为金钱能使鬼推磨，万邦来朝，自立为世界领袖，号令天下。其实，当下的现实却是，中共越强大，在世界上越孤立。何也？只有经济实力，而没有政治实力；只有硬实力，却缺乏软实力。经济崛起，军备庞大，而政治独裁，思想禁锢，如此形象只能令各国望而生畏。
  <br/>
  <br/>
  习近平的幻想，无法实现的幻想，折腾着中国，折腾着这个民族，也折腾着他自己。
  <br/>
  <br/>
  （2021年4月14日）
