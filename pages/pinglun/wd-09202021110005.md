 <p>
  在台海危机引发国际关注之际，中国对台湾再次进行所谓的“经济制裁”：中国海关总署继今年2月25日宣布禁止台湾凤梨进口中国后，如今又再度于9月19日无预警宣布，由于检出介壳虫，所以暂时禁止释迦、莲雾销往中国。这个政策一宣布，从台湾内部到国际媒体一致认为，背后是出于政治原因，是中共对于最近一段时间国际社会支持台湾的一种报复行为。
  <br/>
  <br/>
  这样的报复行为，从实际效果上讲，是损人不利已的行为，且损人的程度低于不利己的程度。先说“损人”的部分，台湾去年莲雾外销中国占比达97%、释迦95%；突如其来的禁令，表面上看势必将对产业经济造成严重冲击。但如果仔细分析，这样的禁令对台湾的经济打击其实是很有限的。台湾外销市场虽然以中国为大头，但从总产量上看，台湾释迦全年产量5.7万公吨，外销中国1.3万公吨，外销占总产量的比例并不大。换句话说，除了还有泰国等外销产地之外，台湾本土的内销产量占总产量的三分之二左右。只要内销市场打开，中国停止进口造成的损失并不是那么严重。实际上，2月份中国停止凤梨进口，台湾1-8月的凤梨外销量扣除中国后，还比去年同期大涨了566%。事实证明，对台湾凤梨禁止进口，并未对台湾凤梨产业造成很大影响。现在台湾政府已经采取一系列因应措施，除了农委会增加10亿元新台币的补助，地方政府号召企业扩大购买，同时开拓更多的外销管道外，还将在WTO机制下向中国提出诉讼。
  <br/>
  <br/>
  围绕禁止台湾农产品进口的政策，中国可以说是因小失大，这是不利己的部分。首先，这样的政策不仅无法拉近两岸人民之间的距离，反而激发台湾社会对于中国霸凌做法的方案。台湾著名网红“馆长”陈之汉PO出吃着释迦、莲雾的照片，表示要吃爆，迅速获得3万网友按赞。另一名网红“凤梨”也在脸书发文说：“我们自己吃吧。”按赞人数迅速达到一万八千人。这些网红对台湾年轻世代影响巨大，中共的做法只能更加激起台湾民众，尤其是年轻民众的反感。
  <br/>
  <br/>
  其次，此时正值国民党进行主席改选，四名候选人面对中共的打压虽然表达力挺台湾农民的态度，但没有一个敢于公开谴责中共；而执政的民进党自然会抓住这个机会表达“台湾价值”。蔡英文总统9月20日表示：“我们一直在做、一直在思考，都是如何地不卑不亢、不受别人的支配，为台湾撑出空间、为台湾争取最大的利益跟支持。”两相对比，台湾民众对于国民党的立场必然质疑，这不利于国民党在2022年和2024年的两场重要的选举中的选情。从这个角度看，中共真是国民党的“猪队友”。第三，正如台湾民进党立委王定宇所说，在WTO内，台湾以会员国的身份对另一个会员国中国提出控诉，对台湾来讲也有国际意义，等于向140个WTO的会员国彰显台湾、中国是互不隶属的独立关贸区域。第四，中国前不久刚刚正式提出申请，希望加入跨太平洋伙伴全面进步协定(CPTPP)，现在与台湾爆发贸易争端，势必给加入国际贸易组织增添阻力。
  <br/>
  <br/>
  中共一方面说要争取台湾民意，一方面又宁愿不利己也要打击台湾人民的福祉，这种粗燥、粗暴的政治手段，只能让台湾人民越来越反感。所以我才始终认为，中共，才是推动台独的最大势力。
