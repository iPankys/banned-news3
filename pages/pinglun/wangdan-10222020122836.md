 <p>
  1980年10月20日，中共中央召开的书记处会议，曾经有明文规定：任何地方，一律不得挂现任中央领导人的画像，不搞个人崇拜。这本来是吸取“文革”起源于对毛泽东的个人崇拜的教训，所做出的正确决定。从江泽民到胡锦涛，虽然也不断努力，试图树立自己的权威，但还算有所顾忌，没有大搞个人崇拜。
  <br/>
  <br/>
  但是，自从2012年习近平上台之后，中共内部的个人崇拜之风逐渐重新风行，在2016年左右达到一个高峰，各种对习近平的效忠和阿谀奉承在官场上盛行。新冠病毒爆发后，习近平的威信受到一定程度打击，对他的个人崇拜似乎也多少有一些收敛。然而，最近一段时间，中共的宣传机器似乎又开始了新一波的对习近平的造神运动，个人崇拜的风潮似乎有再起之势。典型的例子，就是网路上热传一封请愿书，是由国务院参事室党组书记王仲伟，香港中文大学教授郑永年和南洋理工大学校友会会长沈根林三人联名发起，内容竟然是要求将清华大学改名为习近平大学，只是因为习近平曾经在清华大学读过书。虽然这封请愿书的真实性还有待确认，但真名真姓的三个发起人至今都还没有出面否认。有人解读说，这是中国知识分子堕落的典型表现。但在我看来，以王仲伟和郑永年与中共高层的关系以及他们的身份，做出这样一定会引起外界非议的事情，如果说完全是他们自发的行为，可能性并不大。很有可能， 他们是得到了来自中共相关机构的私下授意，才有这样的投石问路之举。
  <br/>
  <br/>
  与此同时，根据外电报道，中共为了巩固对各领域的控制，准备提升所谓的“习近平思想”的位阶，将之与“毛泽东思想”并驾齐驱，并纳入正在讨论的第十四个五年计划的提纲中。根据香港大学新闻与传媒研究中心中国传媒研究计划主任钱钢的统计，从2018年以来，官方媒体上出现了的有关“习近平思想”在经济管理，军事改革，控制媒体和艺术等领域的应用的提法，已经不下20次。据说，推动“习近平思想”成为中国各领域的指导原则，将成为即将于10月26日举行的中共十九届五中全会讨论的主要内容。如果此事成真，不难预料，对于习近平的个人崇拜，必将形成新一波中共宣传的重点。
  <br/>
  <br/>
  习近平在此时推动新一波对于他自己的个人崇拜，应当有两个背景：一个是面对他口中所谓的“百年未有之变局”，他的一系列应对措施，从中美关系到香港问题，都遭遇到挫折，这使得他必须通过制造个人崇拜，重新巩固自己在党内的领导权威；尤其是在十九届五中全会举行之前营造全党支持的景象，有助于他抵抗有可能会出现的来自党内的对于他的质疑甚至挑战。另一方面，习近平修改宪法，取消领导人的职务终身制，在2022年继续执政的态势十分清楚。十九届五中全会将要讨论的中共执政规划，其时间线拉长到了2035年。显然，不管用什么样的形式，在政治上以最高权威面目出现，长期维护和保证自己领导权，是习近平为自己打造的政治目标。为了达到这个目标，必须以各种手段摆平党内的非议，奠定自己的权威性，而个人崇拜就是这样的手段之一。
  <br/>
  <br/>
  尽管习近平有这样的狂妄规划，尽管他的亲信努力发动新一波个人崇拜，但在我看来，他本人的能力与官方营造的形象实在相距太远，这一波个人崇拜风潮，只能制造更多的嘲讽他的口实而已。
