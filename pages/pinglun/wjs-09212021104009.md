<div id="storytext">
 <p>
  最近最热的新闻，第一是阿富汗，第二是澳洲潜艇，第三就是中共的内斗了。因为中共内斗时来已久，热度有些高不起来，其实这是和咱们中国人关系最大，对国际格局也影响最深远的事件。
  <br/>
  <br/>
  为什么这么说呢？因为美国和西方对中国的认识很浅薄，至今仍然在犹豫是竞争还是对抗，可是人家中共多年来都对内明确宣传：中美必有一战，先灭日本再进军华盛顿。当然了，对外的收买贿赂加韬光养晦，还是很有效，买得或者骗得西方的策略家们拿不定主意。多年来，西方特别是美国的政策摇摆不定，甚至被批评是没有政策。
  <br/>
  <br/>
  这些所谓的战略家们，由于被骗或者被收买或者根本就不懂装懂，都认为习近平和共产党的统治坚如磐石。没有人思考习近平之后和共产党之后的问题，就像当年对苏联一样。其实中共的问题比当年苏联的问题严重得多，这主要表现于在中共的黑箱操作下，内斗的激烈程度远超过当年的苏联。中共政权有随时崩溃的可能。
  <br/>
  <br/>
  内斗是个古老而又普遍的话题，比猴子的时代还早，哺乳类群居动物都有内斗。人类几万年来发展出了比动物高明的内斗规则，所以人类越来越发达，综合起来看有两大类模式：
  <br/>
  <br/>
  一种是从猴子那儿继承来的原始模式，弱肉强食、丛林法则，拳头硬的统治压迫所有人。它的后续模式就是奴隶制和蒙古人、苏联人的农奴制。这种模式有利于解决争议，统一意见办大事，短期优势明显。所以曾经击败古代民主制，形成一些大帝国。
  <br/>
  <br/>
  另一种就是协商妥协为主的民主制。照顾所有人的权利和利益，人们生活得更愉快，经济发展得更好，能够获得所有国民的认同和保卫。在经营整体利益的同时，最大限度保障个人自由，比较适合人性追求幸福和自由的倾向。从个人和长远来看，是最好的、最先进的制度。
  <br/>
  <br/>
  当人类的市场经济发达以后，民主制显然发挥出更大的优势。它得到全人类大多数人的向往，有逐渐取代原始的专制模式的趋势，成为所谓的世界大潮流。市场经济越发达，对民主政治的渴求越强烈，所以近一百多年来民主制的国家越来越多，而专制国家越来越少，并且在做着垂死的挣扎。
  <br/>
  <br/>
  说到这儿大家都可以看出来：中共的所谓集中力量办大事的体制，是标准的专制农奴制。在实行苏联农奴制三十年后，形势所迫，中共不得不改革体制。但是出于人性都贪恋权势的本性，搞了个只改经济不改政治的半吊子改革。实质就是企图恢复中国古代传统的，专制政治管理市场经济的不成功模式。这只是为了共产党和一帮大佬及其子孙们独霸权力。
  <br/>
  <br/>
  中国人从两千多年前进入市场经济开始，就在不断探索怎样约束权力，才能管好市场经济。但中国传统模式就是不能管好市场经济，就是不能管好权力寻租，腐败和混乱不负责，最终导致兴衰周期循环。一百多年前中国人民终于发现，西方人发明的现代民主制，是管理市场经济并且有利于国家和人民的最好模式。于是开始了轰轰烈烈的民主革命，建成了亚洲第一个民主共和国。
  <br/>
  <br/>
  邓小平设计的传统政体，甚至不符合马克思主义的原理：经济基础决定上层建筑。所以内斗越演越烈，你死我活，快速走向崩溃。接下来大家要考虑的就是中共垮台后，中国会走向哪里，而不是我想要中国走向哪里。书生们大多比较懒惰，喜欢说我认为应该怎么怎么样。这就是中国知识精英懒得思考，引导中国一百多年来走上邪路的原因。
 </p>
