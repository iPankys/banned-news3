 <p>
  中国新型政党制度白皮书表达的核心内容是：中共与其领导下的八个党派的合作。这八个党派，就是民革、民盟、民建、民进、农工党、致公党、九三学社和台盟。
  <br/>
  <br/>
  一九四九年九月，中共和各民主党派举行的政治协商会议通过了《共同纲领》，确定建立工人阶级领导的、以工农联盟为基础的政治制度，没有中共领导的说法。各党派的政治地位是平等的，唯一的区别是中共拥有庞大的军队。
  <br/>
  <br/>
  纲领第四条：中华人民共和国人民依法有选举权和被选举权；第五条：中华人民共和国人民有思想、言论、出版、集会、结社、通讯、人身、居住、迁徙、宗教信仰及示威游行的自由权；第二十条：中华人民共和国建立统一的军队，即人民解放军和人民公安部队，受中央人民政府人民革命军事委员会统率。
  <br/>
  <br/>
  仅仅是过了五年，依靠军队巩固了权力的中共就在“宪法”里规定了中共的领导地位，《共同纲领》被搁置，选举权和被选举权名存实亡。第五条规定的人民各项权利，都被明确剥夺或徒具虚名；属于政府统率的军队，由中央军委，实际由中央军委主席牢牢掌控。通过整风和反右等各项运动，各所谓民主党派的核心人物都遭到整肃和整治，或踢出政界，或变成郭沫若之类阿谀奉承之流，或变得胆小如鼠、沉默自保之辈。文革期间，干脆完全禁止了所谓民主党派的组织活动。
  <br/>
  <br/>
  所以，中共的领导地位与其说是“历史的选择、人民的选择”，不如说是人民被愚弄和恐吓下，无从选择的选择！
  <br/>
  <br/>
  到如今，那九千多万中共党员有几个敢讲是为了共产主义入党的？事实上，入党是升官的敲门砖，名利的优惠券。八个党派只是粉饰一党专制的花瓶，依附于中共的既得利益群体。中共称，与八个党派的关系是“共产党领导、多党派合作，共产党执政、多党派参政”，并且是“长期共存、互相监督、肝胆相照、荣辱与共”。但实际上，领导和执政是骗人的词汇，独裁和专制是事实的本相；合作和参政是哄人的高帽，帮凶和附庸是真实的写照。“长期共存、荣辱与共”是真的，但互相监督应该用互相利用来代替，而肝胆相照却只能用沆瀣一气来形容。
  <br/>
  <br/>
  中国目前的政党制度是新型的，但却是用多党合作掩饰的一党专制制度，极具欺骗性。但专制就是专制，本质就是这种权力是人民不能自由参与选择的，是强加给人民的，因此再多的形容词都是无稽之谈。
 </p>
