<div id="storytext">
 <p>
  再过两周，就是2009年在乌鲁木齐发生7.5惨案的十二周年。
  <br/>
  <br/>
  十二年前的7.5之夜，彻底改变了维吾尔人的命运。说十二年前的7.5，是现在正在发生维吾尔种族灭绝的始点并不夸张。就如三十二年前的6.4天安门大屠杀，7.5有很多人死亡，有很多人失踪。死亡者、失踪者大多数是维吾尔人，他们有的被军警打死，有的被暴徒打死，更多的维吾尔人是自家里被军警抓走失踪，自此杳无音讯。
  <br/>
  <br/>
  十二年前的乌鲁木齐7.5和三十二年前的6.4屠杀一样，都成了中国政府的敏感日，都成了中国政府的禁忌。7.5真相被政府恶意扭曲掩盖，没有真相也就没有公平正义，而且中国政府官员为推卸责任，以编造的7.5真相扭曲和掩盖了引发7.5维吾尔学生抗议的、长期存在的民族压迫之根本原因。
  <br/>
  <br/>
  被扭曲和掩盖的7.5，不仅使中国政府成功将维吾尔人妖魔化，而且还将维吾尔人全民族捆绑为恐怖主义，当作中国政府的国家敌人，为其实施种族灭绝打下了伏笔。
  <br/>
  <br/>
  7.5的爆发直接原因，是6月26日发生在广东韶关旭日玩具厂，当地几百名汉人暴徒对在玩具厂打工的维吾尔人的围攻，造成维吾尔人死亡。韶关玩具厂围攻维吾尔人，起因是有汉人造谣说是维吾尔打工者强奸了汉人女工，煽动当地暴徒围攻维吾尔人。暴徒追打维吾尔人的场面极其惨烈，还被一些汉人拍了视频，加上煽动性对话发到了网络上。
  <br/>
  <br/>
  围攻、殴打玩具厂维吾尔人的视频在互联网上很快大规模传播，激起了几千里外家乡维吾尔人的愤怒和不满。很快，新疆大学的学生们决定上街游行示威，要求维吾尔自治区政府作为以剩余劳动力为名，强行将维吾尔年轻人送到中国各地的政府承担其相应责任，调查围攻维吾尔人事件真相，依法惩治肇事者，给予死难者公平正义。
  <br/>
  <br/>
  为了避免中国政府以“分裂暴乱”罪名镇压和平示威学生，天真的维吾尔学生举着中国的五星红旗走上了乌鲁木齐街头。结果不言而喻，殖民政府出动了军警，进行了血腥镇压，使一场有维吾尔年轻学子发动的和平示威演变为了震惊世界的民族冲突。
  <br/>
  <br/>
  冰冻三尺非一日之寒。韶关旭日玩具厂围攻维吾尔人，是7.5的直接导火索。但中国政府自2001年，美国发生9.11恐怖袭击之后，借助国际社会反恐，持续妖魔化维吾尔人信仰，边缘化维吾尔人，对维吾尔人的任何不满表达，以“三股势力”之名血腥镇压等长期赤裸裸的歧视性民族政策，应该说是7.5最终演变为一场举世震惊的民族冲突的真正催化剂。
  <br/>
  <br/>
  自2001年起，中国殖民政府早已经开始了对维吾尔人信仰的各种限制。以党团员、公务员、学生等，不得进入清真寺；斋月不得封斋，不能作礼拜；除了中国政府办的伊斯兰学校、学院之外，任何人不得传授伊斯兰知识等，开始釜底抽薪维吾尔人的信仰。
  <br/>
  <br/>
  在教育领域，以“强化汉语”、“双语教育”和“内地初高中班”之名，中国政府开始完全排斥维吾尔语教学；大中专院校完全取消了维吾尔语授课；维吾尔语中小学则以民汉合并之名，被汉校吞并。维吾尔母语教育名存实亡，一周只有几节课，这使得大中专院校大量兢兢业业几十年的维吾尔教师被迫提前退休或失业，使维吾尔文化面临灭顶之灾。
  <br/>
  <br/>
  经济上，则以“西部开发”之名，将维吾尔自治区自然资源、厂矿实业、土地，大规模转让中央企业、自治区汉人领导的利益关系户，进行掠夺性开发。这些企业不仅不帮助当地人，还公然歧视维吾尔人，招聘启示居然可以打出“不招收维吾尔人”。维吾尔人失业率居高不下，尤其是年轻人，大学一毕业即失业。
  <br/>

  <br/>
  维吾尔人被排除在外的“西部大开发”，使作为维吾尔人主体、维吾尔文化载体的很多维吾尔农民不仅失去了祖辈相传的土地，也失去了赖以生存的家园，成为自己家园的无家可归者。这不仅使维吾尔人在经济上被严重贫穷化、边缘化，更制造了民族之间的严重不平等，使民族矛盾日趋激化。
  <br/>
  <br/>
  同时，伴随“西部大开发”，在维吾尔自治区南部，中国政府则开始了另一个釜底抽薪维吾尔人的长期战略阴谋：就是以“转移剩余劳动力”为名，开始将适婚维吾尔年轻男女以低工资，向中国各省劳动密集型企业转移。这一做法的目的非常明确：首先是以中国各省的强势汉文化环境，快速同化远离维吾尔文化氛围的维吾尔年轻人；同时，向维吾尔自治区南部大量移入汉移民，尽快改变那里的人口结构，彻底摧毁维吾尔文化堡垒——维吾尔社区。
  <br/>
  <br/>
  当时，已经成为反恐牺牲品的维吾尔人除了少数勇士之外，大多数普通维吾尔人敢怒不敢言。但愤怒和不满在发酵酝酿，爆发只是早晚的问题。
  <br/>
  <br/>
  2009年的7月5日，是以“西部大开发”为名的中国政府肆无忌惮掠夺维吾尔人家园以来，维吾尔人愤怒和不满的一次总爆发。但这一由维吾尔学生发起的，以和平示威开始的不满表达，在中国政府一以贯之的血腥镇压下，转变为史无前例的民族冲突；最终，不仅撕下了中国政府虚假自治的假面目，也彻底改变了维吾尔人的命运。
  <br/>
  <br/>
  维吾尔人的失踪、被射杀、种族灭绝，并不是自2017年开始的；大规模的抓捕、集中营、监禁也不是自2016年陈全国开始的；抓捕维吾尔知识精英、实业家、歌星、音乐家，也不是从2014年习近平上台开始的。
  <br/>
  <br/>
  2009年的7.5改变了一切，在习近平称霸世界的极端野心促使下，在中国极端民族主义御用学者、专家的鼓噪煽动下，在很多浑浑噩噩中国普通民众落井下石的漠视下，针对维吾尔人的强制失踪、射杀和种族灭绝，在7.5之后，成为了中国政府冒天下之大不韪，最终解决维吾尔人问题的政策。
  <br/>
  <br/>
  2009年的乌鲁木齐7.5之夜，是维吾尔人历史的分水岭！
