 <p>
  <br/>
  2021新的一年，世界许多国家为武汉肺炎所困，忙于自救；美国则陷入总统选举的纷争。对中共战狼来说，这是难得的出击机会。按照中共一贯的“革命两手”，总书记习近平在新年讲话中主要吹嘘他在内政的成绩，但国务委员兼外交部长王毅则是接受新华社和中央广播电视总台联合采访，继续充当战狼，矛头直指美国。
  <br/>
  <br/>
  王毅认为，“中美关系已经来到新的十字路口”，“其根本原因在于美国当政者对中国的认知出现了严重偏差，把中国视为所谓最大威胁，并由此采取了完全错误的对华政策。”显然，中共在指责美国的特朗普政府，并且警告未来的新政府在十字路口上要按照北京指点的方向走。
  <br/>
  <br/>
  美国总统是民选的总统，特朗普的中国政策得到国会两党的认同。未来不论谁出任总统，也得尊重国会的意见及其通过的法案。北京现在发出如此威胁，针对的并不只是未来的总统，更是国会与美国人民。说美国的政策“完全错误”，那就应该向中国投降，唯命是从？中共未免太嚣张了吧！
  <br/>
  <br/>
  王毅在责骂美国的同时，又称赞欧盟，认为“双方是全面战略伙伴，不是制度性对手”。王毅不但抹杀欧盟与中国的制度与意识形态的对立，更把欧盟拉到自己一边来对抗美国。中国与欧盟刚完成投资协议谈判，欧盟内部还有不少杂音，也还没有得到欧盟议会通过，北京就迫不及待把欧盟当作自己伙伴来威胁美国，这不但让欧盟情何以堪，更让美国看透中国称霸全球的野心。
  <br/>
  <br/>
  中共这匹战狼是愚蠢还是饥不择食？即使特朗普开始对德国不是那么友善，但是后来国务卿蓬佩奥多次飞欧洲进行解释工作，让他们明白中共的威胁，彼此有了很多共识。拜登也强调，要团结欧洲来应对中国。北京挑拨欧美关系显然是要分化西方国家，孤立美国，达到中国统战策略的“分化瓦解，各个击破”的目的。历史将会回答，欧洲是同中共站在一边，还是同美国站在一边。
  <br/>
  <br/>
  王毅对美国的攻击，也延续了习近平讲话中推卸对武汉肺炎传播全球的责任。他说：“有越来越多研究证明疫情是全球多地多点爆发”，又指中国是“最早向世界报告疫情”，并且列举中国协助全球抗疫；但是只举出口防疫用品，却完全不提早先从全球的搜刮，导致其他国家紧缺这些用品。而在同时，中共间谍头子熊向晖的女儿熊蕾则发表文章，指责武汉肺炎源头与美国有关，连2003年的SARS也是美国搞出来的。
  <br/>
  <br/>
  很明显，战狼的凶恶也在掩盖内心的虚弱，那就是全球对习近平的究责。马上就要进入中国的辛丑年，120年前的辛丑条约是习近平的噩梦，习近平逃不掉历史的审判，就如当年的慈禧太后一样。
