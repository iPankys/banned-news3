 <p>
  各位听众你们好，我是王丹。在全世界的关注焦点都集中在美国大选争议的时候，中共对香港抗争运动的秋后算账正在严重升级。多达53名香港社会运动人士，甚至包括一名美籍人士，在一天之内通过统一行动被逮捕。这不仅使得香港的政治环境更加险恶，而且明显是在向西方社会发出挑战。
  <br/>
  <br/>
  中共这一波逮捕行动，选择在中欧投资协议刚刚签署之后的时间点，使得这种挑衅的性质更加明显。对欧盟国家来说，现在处于相当尴尬的处境。如果制裁北京，耗时七年才签订的投资协议势必被搁置甚至废除。这对于急于打开中国市场大门的欧洲国家来说，是不可承受之重。但是同时，中共这一波大逮捕竟然以参加初选这样的正常民主活动为理由，指控53人意图颠覆政府，如此荒谬和严重的指控，欧洲国家的政府如果装作没有看见，或者仅仅谴责了事，恐怕会遭到民意强大的反弹，也会使得一直把“人权”挂在嘴上的欧洲国家颜面丢尽。要如何在贸易和人权之间做出合适的权衡与选择，北京给欧洲国家出了一个巨大的难题。
  <br/>
  <br/>
  对美国来说，这同样是一个挑战。还有十几天，拜登政府就会开始执政。在这样的关键时刻，北京在香港问题上发难，很明显是在试探未来美国新政府的底线。如果美国作出强烈反应，我们不排除北京会适当收敛其嚣张行为的可能性，并以此作为未来与美国新政府谈判的一张牌；而一旦美国不能或者不愿做出强烈反应，北京就会知道美国无意在未来四年与中国进行对抗，这对于他们“重新设置”中美关系的基调，是一个重要的参考标准。专门选择在新旧政府交替的时刻，在中美关系中较为敏感的香港问题上主动挑衅，北京当然是经过精心算计的。
  <br/>
  <br/>
  我认为，目前的时机非常关键。面对中共在香港问题上的挑衅，西方国家必须做出强硬的反应，这包括新的制裁措施，包括在其他领域的报复性行为。最近美国宣布驻联合国大使即将访问台湾，就是值得赞赏的反击。强硬反击的必要性，不仅仅在于北京这种侵犯人权的行为本来就应当受到国际社会的强烈谴责，更重要的意义在于，今天西方国家的反应，实际上可以为未来几年的对华政策划出一个边界和底线。
 </p>
 <p>
  从美国到欧盟国家，应当通过针对香港问题的强烈反应，向中共传达明晰的信号。那就是，北京不可以试图挑战西方社会的普世价值体系，否则双方只能走向对抗。这样的信号越清晰，西方国家未来处理与中国的关系会越简单，付出的代价越小。如果西方国家在这样的问题上保持战略模糊，只会向北京传达错误讯息，这样的讯息会误导北京当局，使得他们更加肆无忌惮。北京的肆无忌惮一旦扩展开来，西方国家未来要面对的麻烦就会更多。
  <br/>
  <br/>
  因此，当机立断，在第一时间对于北京的挑衅做出清晰、明确的强硬反应，或许短期内会导致与中国关系的紧张，但是从长远看，其实可以使西方国家在未来不必付出更大的代价。美国和欧盟国家应当知道，中共的扩张野心是贪得无厌的，一次让步，一次软弱，都会导致后患无穷的恶果。对此，西方国家一定要有清晰的认知，绝不能掉以轻心。以上是自由亚洲电台特约评论员王丹为您做的评论。
 </p>
