 <p>
  <br/>
  中欧投资协定历经7年的漫长谈判，原定2020年12月22日达成，但因欧盟部分成员国反对，尤其法国的表态，几乎泡汤；但却在12月的最后一天，戏剧性地峰回路转，宣布达成。据知，转折的关键在于德国总理默克尔和中共总书记习近平在最后阶段的强势介入。
  <br/>
  <br/>
  在中国方面，大概由习近平拍板，决定口头上答应法国提出的两项条件：中国更加开放市场；中共暂停在新疆的强迫劳动及其出口棉织品。口头上的答应，对中共而言并不难，而习近平大权在握，由他出面应允也不在话下。
  <br/>
  <br/>
  在德国方面，除了德国是欧盟最大国家、最大经济体，影响力和发言权最大之外，当时德国正值欧盟轮值主席国。默克尔利用其有利地位，赶在德国作为轮值主席国的最后一天，把事情敲定。
  <br/>
  <br/>
  北京顾面子，不愿直接表述新疆问题，而使用了这样的文字：“在劳工和环境领域，中国承诺不降低保护标准以吸引投资，不将劳工和环境标准用于保护主义目的，尊重在有关条约中承担的国际义务。中国将支持企业承担公司社会责任。中国还承诺致力于批准剩余的国际劳工组织基本公约，并对尚未批准的国际劳工组织关于强迫劳动的两项基本公约作出具体承诺。”
  <br/>
  <br/>
  “承诺致力于批准”，这种离奇造句充分暴露了中方的消极、勉强、应付，而且随时准备耍滑头、钻空子、大打拖延战，在实际过程中瞒天过海，绝无诚意执行有关劳工权益保障和废除强迫劳动的条款。这固然是中共本性，也是过去四十余年中共欺骗美国、欺骗西方的惯技。只不过时至今日，中共还想对欧洲故技重施。
  <br/>
  <br/>
  过去数十年，（共产）中国崛起、（民主）美国衰落，原因即在于此。特朗普任内提升中国商品关税，与中国开打贸易战，起因也在于此。此情此景，难道欧盟和德国领导人心中无数？其实，问题就在这里，德国或欧盟并非心中无数，问题主要出在默克尔身上。此人即将卸任她担任了四届、共计16年的德国总理，预定今年、即2021年退休。默克尔不仅利用德国作为欧盟轮值主席国的最后时间，也利用她即将卸任德国总理的最后机会，冒险与中共敲定中欧投资协定，图谋为她个人留下所谓的政治遗产。
  <br/>
  <br/>
  然而，这是怎样的政治遗产？又是怎样的时代背景？
  <br/>
  <br/>
  对内，中共在新疆大建集中营，把数百万维吾尔人、哈萨克人投入集中营，复辟上世纪纳粹德国的暴行；中共悍然撕毁中英联合声明、砸烂“一国两制”、砸烂“东方之珠”，并对香港抗争者大搞秋后算账，迅速在香港推进国家恐怖主义。对外，中共威胁台湾、偷袭印度、霸凌南海周边小国；从不遵守世界贸易组织和其他国际组织的规则，并大规模盗窃美欧等发达国家的知识产权；2020年以来，还以人为的隐瞒、封锁和欺诈手段，向全世界传播大瘟疫，重创191个国家……。
  <br/>
  <br/>
  习近平和中共的所作所为，既是向中国人民挑战，也是向世界人民挑战，以“专制无敌”、“一党专政无敌”的骄横姿态，猖狂挑战人类文明的底线。但，卸任前的德国总理默克尔竟然把与这样的人类公敌达成“投资协定”视为头等大事，视作自己的政绩和“政治遗产”，其目光短浅和急功近利令人惊诧！
  <br/>
  <br/>
  中欧投资协定，当代绥靖主义的极致版本，即使上世纪的英国首相张伯伦比之也犹恐不及。法国政府承认：面对当今共产中国，美国首先觉醒，而欧洲的觉醒晚于美国，但也正在觉醒。相比之下，德国总理默克尔似乎是少有的例外。
  <br/>
  <br/>
  默克尔在卸任前的临门一脚，以为是自己的“大手笔”，让欧洲门户洞开，开门揖盗，引狼入室。默克尔的自私自利、鼠目寸光、姑息养奸，超过张伯伦不知多少倍！而默克尔带给欧洲未来的隐患和重创，更不知是当年张伯伦带给欧洲隐患和重创的多少倍！
