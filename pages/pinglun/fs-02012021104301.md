 <p>
  习大帝登基后当局就展开了声势浩大的反腐败运动，迄今已有超过150万官员受到各种不同的处分。确实，习当局的反腐败一度获得普通民众的认同，也广为国际社会赞扬。然而，随着时间的推移，越来越多的人们看清楚：习当局的反腐败是政策性、选择性的反腐败。
  <br/>
  <br/>
  所谓政策性，那就是习当局随时可以发布指令，时而可以放宽，时而必须从严，法律条文可以按政策灵活调整。
  <br/>
  <br/>
  所谓选择性，那就是习当局和各级掌控案件的官员可以按政治形势或派别斗争的需要，以及方方面面的关系、权衡得失等等，选择从宽或从严处置，法律不过是掌控者随叫随到的婢女。因此威胁习大帝权位的薄熙来被判无期，抨击习大帝穿着皇帝新衣的任志强被判十八年有期徒刑。
  <br/>
  <br/>
  早在2015年，御用文人就汇总了习近平关于发起反腐败的60多段犀利论述。
  <br/>
  <br/>
  官方也曾宣称：反腐败斗争取得压倒性胜利！
  <br/>
  <br/>
  令人啼笑皆非的是，习当局取得决定性胜利的反腐败居然创下了1949年以来历史记录的一颗硕果，就是1月29号被执行死刑的赖小民，其被指控在2008-2018年期间受贿金额折合人民币达17.88亿元。
  <br/>
  <br/>
  据说，他一次就敢收受贿赂上亿元，家里现金有三吨重。更为奇葩的是，《财新网》曾报道，赖小民案创下了“三个一百”的奇观，即一百多套房子、一百多个涉案人、一百多个情妇！来源标明“华兴评论”的文章说，这一百多个情人加上前妻在一个小区里和平共处，所有孩子都叫他爸爸。
  <br/>
  <br/>
  赖小民在所谓“一体推进不敢腐、不能腐、不想腐”的大环境下，居然可以在那么长的时间段里，那么公然地贪污腐败，只能再次说明：
  <br/>
  <br/>
  一、习当局政策性、选择性反腐败的失败！从赖小民被判决到处决仅仅只有二十四天的特快速度来看，似乎有选择性警告一些人，掩护一些人的嫌疑，特别是确认他有重大立功表现的情况下。向他行贿的都是哪些人？怎么处理的？习当局对此不置一词!
  <br/>
  <br/>
  二、中国的腐败是制度性的腐败！在权力可以创造利益、交换利益的制度下，腐败是必然的。制度性的腐败只能靠制度的改变才能遏制。
  <br/>
  <br/>
  三、中共统治集团从上到下已经是无官不贪的利益集团，反腐往往是铲除异己的手段，不能、不敢，也不愿改变制度。
  <br/>
  <br/>
  因此整个统治集团的腐烂不可逆转！由于中共借助高科技手段严密管控社会，无论在国内还是海外，反对力量都难以集结，外力摧毁中共统治非常困难，但中共必将因腐烂而崩溃！
