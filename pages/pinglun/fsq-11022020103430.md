 <p>
  目前习惯上讲的中国是大陆中国即中华人民共和国，涵盖十三亿人口，而中共是其中九千万人组成的严密组织。中共等级森严，最高层在集体领导和个人独裁之间晃动，而在绝大多数时间段里是个人独裁或隐性的个人独裁。最高层把几乎所有的社会资讯都当作党的秘密，要求党员宣誓保守党的秘密。全体党员和普通民众都只能了解最高层允许了解的资讯，必须相信最高层所说的一切。
  <br/>
  <br/>
  因此，所有以中国或中国人民的名义所发的言论，所做的事情，都是中共的言论，中共的作为。当前，自由民主世界的反共浪潮正在兴起，其领军人物很清楚：中共不等于中国，不代表中国人民。他们要反对的是中共，而不是中国更不是中国人民。
  <br/>
  <br/>
  有朋友质疑说：中共和中国人民组成了中国，无法分开，是这样的人民造就了中共。四九年是中国人民选择了中共。有人拿出来自哈佛大学的一份调查报告说：百分之九十五的中国人满意现状，感到幸福。
  <br/>
  <br/>
  事实是，四九年中共是在苏联支持下用武力打败国民政府占据大陆全境的。中共政权是用暴力强加给中国人的，没有选择一说。至于哈佛的报告，那是真的，但被调查人的表达基本上是假的。可以说，中共统治下的所有民意调查都是假的。因为没有讲真话的可能。
  <br/>
  <br/>
  当然，在中共统治的七十多年里，质疑、挑战和反对的声音及其努力从未间断，甚至形成过很大的规模和影响，但都遭到无情镇压。尼克松访华破冰之后，自由民主世界对中共抱有不切实际的幻想，期待在国际社会的影响下，在市场经济的带动下，中国会改变，中共也会改变。也因此大陆民众的反抗所能得到的国际支持非常有限。
  <br/>
  <br/>
  如今自由民主世界已经放弃了幻想，即将从被动的“反击中共的侵略行为”，转向主动打击并消灭侵略者！灭共的世界大潮已经奔涌！在国际社会的围堵之下中国国内经济陷入了死循环，由此将触发经济、金融政治、社会等方方面面的危机，中共的崩溃不可避免！中共垮台后民众才有选择的可能。没有了中共，中国还将存在。转型和过渡会伴有阵痛，由于中共在少数民族地区犯下的罪行，中国甚至会面临分裂的危险。依我之见，如果民众选择回归以46宪法为蓝本的中华民国的道统和法统，将减少转型的代价，并最大限度地缓解分裂的危险。
  <br/>
  <br/>
  苏共垮台后，苏维埃联邦共和国不复存在。中共垮台后中华人民共和国也将不复存在，但不管以后的国号是什么或区域有什么变动，中国还将存在。中共灭亡后的中国和中国人民将会有美好的未来！
