 <p>
  距离四月底由中共特权家族发动、中央和地方政府及喉舌密切配合，以“刹车失灵”为借口诋毁、抵制特斯拉的批斗行动不足百日，在那位奉旨“维权”的女车主余情未了之时，特斯拉的中国官网无预警地放出了特斯拉全新Model Y标准续航版，起售价仅为27.6万元人民币（包含补贴）的消息。
  <br/>
  <br/>
  这则消息真可谓一石激起千层浪，不足30万人民币就能买到相当于国内其它车企生产、要花50万左右才能买到的同款中档越野车，而且续航里程（国标工况法）达525公里，远超其它品牌。这样的现实利益，瞬间打破了所有通过洗脑打造出的所谓“观念认同”与束缚。于是订购特斯拉的官网再次被抢订特斯拉的车主挤爆，热度比春节前的那一波来得更加猛烈。
  <br/>
  <br/>
  到此为止，中共权贵家族精心策划的诋毁特斯拉的阴谋彻底崩溃。若一定要说有什么意义的话，那就是中共官僚利益集团帮着从来不做广告的特斯拉公司在中共沦陷区免费做了一次广告，使得特斯拉的品牌家喻户晓。
  <br/>
  <br/>
  之前我在《谁叫你特斯拉不加价》一文中曾经写道：“中央、地方一起联手，如此规模的批判行动没有中共的直接操纵、统一指挥是不可能实现的”。那么，如今从人们纷纷再次抢订特斯拉的热潮来看，中共真的不代表中国人民的意志 -- 彻底的洗脑在现实利益面前都会瞬间崩塌，连招架之力都没有。因此，根据经过中共审查的……，或者是虽未经过审查但是长期宣传洗脑后条件反射出的某些网络言论，就得出的所谓“结论”是十分表面的。那些习惯性的条件反射之言，或根本就是糊弄当局的阳奉阴违之语，远不能代替潜意识中本能的自然反应。
