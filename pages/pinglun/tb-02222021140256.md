<div id="storytext">
 <p>
  <br/>
  1989年6月4日凌晨，在天安门广场附近，一辆坦克碾碎了方政的双腿。后来，这位活 "坦克人 "成为中国残运会铁饼和标枪的全国冠军，但他后来被政府禁止参加比赛，因为他的截肢会让人们想起天安门大屠杀。这个故事出自我与胡佳在2008年北京奥运前的一年，共同撰写的《奥运前的中国真相》。我们记录了中国政府恶劣的侵犯人权行径，希望国际社会借奥运之机关注中国人权状况。我们告诉世人: “你们看到的鲜花、微笑与繁华，正是建立在冤屈、泪水、囚禁、酷刑和鲜血的基础之上的。”
  <br/>
  <br/>
  1933年，美国的"体育公平竞赛委员会"质疑纳粹德国举办1936年夏季奥运会的目的，称其是 "为纳粹政权的需要和利益服务，而不是为奥林匹克理想而服务"。美国奥委会的答复是 "政治不应该与体育混为一谈"，并继续支持柏林奥运会。选择纳粹德国举办奥运，受到了广泛的批评，认为它成为了希特勒的 "宣传契机"。由于未能抵制1936 年奥运，当时一些观察家就断言：美国及其他一些西方民主国家可能失去了一次制止希特勒和抵制纳粹暴行的机会。奥运会结束后，德国加速扩张，加剧对犹太人以及其他“国家敌人”的迫害，最终导致二战和大屠杀的发生。纳粹奥运，是奥运史上的一个污点。
  <br/>
  <br/>
  北京准备举办08奥运时，也出现了类似的质疑。 为了建设奥运基础设施和 "清理 "城市，中共政权强拆房屋、任意抓人；说真话的记者、作家和异议人士被投入监狱。 遭遇侵权而长年上访的杨春林因为发起“要人权不要奥运”的联名呼吁，获刑6年并受到残忍的酷刑。因奥运场馆建设，叶国柱的房屋被强行拆毁，之后因为上访维权被判刑并在狱中遭受虐待；又是因为奥运，叶国柱刑满而不释放。我和胡佳的文章发表后，胡佳被控 "煽动颠覆国家政权罪"，被判处3年半监禁。我的律师执业资格被吊销，护照被没收，还被秘密警察绑架、拘留并受到酷刑。
  <br/>
  <br/>
  然而，炫目的奥运表演依然如期进行。八十位国家元首出席，大型跨国公司纷纷投入商业广告。即使国际人权组织不断发出呼吁，也没有一个国家进行抵制。开幕式离我家不远，但我被警察软禁在家，我的很多活动家朋友也被软禁或者“被旅游”。奥运的一项自行车比赛路过我所住的八达岭高速公路附近，当地社区居民可以用身份证申请观看比赛。我的亲友们都通过了，但我被揪了出来。当局认为，我观看比赛会影响“政治稳定”。
  <br/>
  <br/>
  国际奥委会一再向世界保证，2008年奥运会将给中国带来更多的开放。北京政府承诺，将在新闻自由、法治和人权方面有所改善。中国宪法有了很多保障基本的人权和自由的条款，中国政府也签署了二十几个国际人权文件，但这些都被中国当局当做废纸。那些宪法条文、国际条约和美妙的承诺，不过是欺骗国民和国际社会的花样而已。
  <br/>
  <br/>
  举个例子。北京承诺在奥运会期间设立三个特别示范区。但有两位年近80的老太太吴殿元和王秀英，仅仅因为提交了抗议申请就被判处 "劳教"。 奥运刚刚结束，刘晓波就被逮捕并获刑11年，后来他成为历史上唯一死于监禁中的诺贝尔奖得主。08奥运给中国带来的不是自由和民主，而是噩梦。国际奥委会知道1936年的柏林和2008年的北京，知道让独裁者举办奥运的后果。北京在2015年，竟然又赢得了2022年奥运会的主办权。同年，它在中国各地镇压人权律师和活动家，并急剧加强了对媒体、互联网、大学和出版的控制：非政府组织被关闭；基督教教堂、西藏寺庙和穆斯林清真寺被拆除；法轮功和其他 "邪教 "信徒继续遭受迫害，有的因酷刑死亡。
  <br/>
  <br/>
  三聚氰胺奶粉丑闻发生在2008年北京奥运会之前和期间，为了使奥运不受影响，中共刻意掩盖该事件，封锁媒体报道。于是，在家长不知情的情况下，数百万儿童继续食用毒奶粉。去年，中共隐瞒了新冠疫情的爆发：以处理疫情为名，政府对公民进行完全违法和超出必要限度的任意拘留、乃至公开羞辱。武汉市民陈和建在隔离期间逃出家门，被殴打致死。新冠病毒已经夺走了246万人的生命，现在，全世界正眼睁睁地看着奥运会再次成为中共侵犯人权的借口和契机。
  <br/>
  <br/>
  2008年，国际影星米娅-法萝称，北京奥运为 "种族灭绝奥运会"，因为中共政权是达尔富尔屠杀的帮凶。 “种族灭绝”一词，将完全适用于2022年的北京奥运。自2017年以来，已有100万至300万的维吾尔和哈萨克人被关进集中营，这是中共的邪恶计划。被关押者被强迫学汉语、唱红歌，放弃民族传统和宗教信仰。不断涌现的大量证据表明，集中营存在普遍的酷刑，有些人被折磨至死，大量的维吾尔人被强迫绝育，有的被强迫与汉人结婚，很多孩子被迫与父母分离，送入孤儿院。BBC最近的报道也完全证实了集中营里，针对维吾尔和哈萨克女性的系统性强奸的存在。我的维吾尔朋友们都告诉我，他们所认识的每一个维吾尔人中，都至少有一名家庭成员被关在集中营中或失踪。美国前国务卿蓬佩奥在离任前宣布，中国政府对维吾尔族犯下了种族灭绝和反人类罪。新任国务卿布林肯说："这也是我的判断。"
  <br/>
  <br/>
  北京已彻底摧毁了香港的“一国两制”。
  <br/>
  <br/>
  自08奥运结束以来，已有超过150名藏人自焚，要求获得基本的自由并允许达赖喇嘛回国。中共政权将50万藏人送入强迫劳动营。在内蒙古的学校，它正在剥夺蒙古人使用母语的权利。
  <br/>
  <br/>
  习近平大搞集权和个人崇拜，加快建立“高科技极权主义”。中共的黑手也伸向海外，跟踪、监视、骚扰、甚至袭击海外异议人士。中国的秘密警察甚至将海外华人或外国人从泰国、越南、缅甸等地绑架回国。想一想桂民海的案件：一个瑞典公民，因为他出版的书让中共高层领导人难堪，所以在泰国被秘密警察绑架到中国，被迫在电视上认罪，获刑十年，并在酷刑之下放弃瑞典国籍。
  <br/>
  <br/>
  去年10月，我作为160多个人权组织的五名代表之一，与国际奥委会官员会面。 我们得到的答复，与奥运官员早先为纳粹运动会辩护的答复相同："政治与体育应该分开"。
  <br/>
  <br/>
  ——"维护人类尊严 "这句话不是直接来自于《奥林匹克宪章》吗？ 奥运会应该与此无关吗？ 希特勒会把政治和体育分开吗？在大屠杀正在进行的时候，将体育与政治分开是选择完全地站在刽子手的一边。"不谈政治 "正是专制政权最想看到的政治态度。
  <br/>
  <br/>
  距北京冬奥会还有一年之际，我在《华盛顿邮报》撰文写到：把主办奥运会的荣誉，不是一次，而是两次，交给一个正在实施种族灭绝的独裁政权，这是奥林匹克运动的耻辱，并显示出对人权和尊严的全然蔑视。
 </p>
