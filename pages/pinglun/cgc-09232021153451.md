<div id="storytext">
 <p>
  最近，中共沦陷区内的共党喉舌们又在加大力度不断鼓噪所谓的“两个维护”。简单说来，就是“一要维护中央权威，二要维护习党魁的核心领导地位”。
  <br/>
  <br/>
  其实，诸如此类的说辞在共产党的章程里早就写明了：“下级服从上级，全党服从中央”。反观中共的历史不难发现，中共政权总是不断强调这一点，而每当中共中央强调这些的时候，往往就是又一次重大危机伴随的时候。根据以往的这一规律，也根据“中共的话要反着听”的我们这一生活经验，中共这次不断强调“两个维护”，想必是中共的中央权威和习党魁在党中央的核心领导地位遇到了挑战。
  <br/>
  <br/>
  仔细观察不难发现，中共喉舌们所说的“两个维护”，其实就是维护习党魁一个人的权威性，因为所谓的“中央权威”归根到底还是总书记的权威。不过，“新精神”所针对的对象……，我认为是有着明显不同的：“维护中央权威”主要针对的是从北京到地方，大大小小的中共党组织及其成员；而“维护习近平的核心领导地位”则主要是针对包括在中央担任委员或候补委员的各省地方大员在内的、习近平身边的中央官员们而言。中共认为，维护好了习近平在上述两部分人群中的“权威核心地位”，不仅习近平个人权力无忧，整个中共也就达到“团结一心”；企图以此实现共同防范人民造反、维护专制统治的目的。
  <br/>
  <br/>
  这里要特别提出的是，无论哪个“维护”都和我们老百姓没有任何关系。实现要人民拥护共产党的目的，那是归统战部管、需要和中宣部协同工作完成的目标。很多人可千万别不知道自己是谁，忙着紧跟，一厢情愿地自我感觉你似乎真的“姓赵”啦。
  <br/>
  <br/>
  社会若还没摆脱专制步入文明，那么统治者与被统治者的根本利益就仍然是控制与被控制、剥削与被剥削的关系，不可能利益一致。对此，人民必须有十分清醒的认知。但是，社会如果没有言论自由、新闻自由，人民没有任何发言权与说话的渠道或是平台；没有民主体制，人民手里没有选票来决定公仆——掌权者去留的政治生命，那人民的意愿就永远不会被重视，人民的声音就永远不会被政客们听到，这是已经被历史和现实反复证明的。因为忽视你，政客们没有顾虑，不必担心会为此付出代价。而只有掌权者是由人民授权，人民才能被尊重、被重视，才能够有尊严、切实拥有权利地活着；社会拥有公平公正，人民才是真正的“当家作主”。
 </p>
 <p>
  <br/>
  一百年来，中共一再把“让人民当家作主”挂在嘴上，其实人民要想真正当家作主，必须努力实现并巩固宪政民主制度。三权分立、司法独立的系统能保护人民的权利，规范和制约政客们的行为。上述系统虽然不是完美无缺，虽然需要不断完善和修复，但毕竟是迄今人类探索出的最不坏、能最大限度维护社会公正的社会制度。而中共在沦陷区推出的“两个维护”，维护的只是中共极权专制，也只会把中国和中国老百姓推入更加苦难的深渊。
 </p>
