 <p>
  自从美国总统拜登在匹兹堡演讲中展示了美国逾4万亿美元的宏大基建计划、誓言与中国竞争并重塑美国经济之后，两国媒体进入对基建计划的评议高峰。总的来说，两国舆论关切的重点完全不在一个频道上，美国关心税收、筹资渠道之类；中国方由于该计划的主诉是与中国竞争，除了关切美国的短板之外，刷屏式地嘲笑拜登“抄中国作业”。
  <br/>
  <br/>
  <strong>
   抄中国作业没错，作业本拿错了
  </strong>
  <br/>
  <br/>
  毫无疑问，美国这个基建计划受了中国2009年5万亿元人民币（一说4万亿）政府投资救市计划的影响。但中国那个2009年救市让中国深陷债务泥潭，负面影响甚大。这次“抄中国作业”的中文舆论场说的不是这个，而是2020年疫情期间中国启动的“新基建战略”。
  <br/>
  <br/>
  《瞭望》周刊确实于2020年4月30日发表过《新基建十大战略方向》，介绍过这个新基建战略，例如：智能化数字基础设施是主导方向、数字化科技创新基础设施是底层支撑、现代资源能源与交通物流基础设施是国民经济的大动脉、先进材料与智能绿色制造基础设施是制造强国和质量强国之基、现代农业和生物产业基础设施是生物经济之基，等等。但据此，说“拜登所推出的这个‘美国就业计划’其实就是抄了中国的‘新基建’计划”，则是拿错了自己另一本作业本（多停留在纸面）。
  <br/>
  <br/>
  “抄作业说”认为，拜登计划与中国新基建计划的内容大同小异，主要分为以下5大方向：一，要想富，先修路；二，制造业强国；三，投资3000亿美元升级电网、通信网络和饮用水体系；四，投资3000亿美元兴建楼房和升级学校，是美国版本的‘中国旧改’；5，投资4000亿美元改善老人和残障人士的护理，并称“美国没有国家医保体系，这是拜登对被半废的‘奥巴马医保方案’的一种回归”。
  <br/>
  <br/>
  作者高度称赞，“从计划本身来看，无疑，这是美国史上百年一遇的伟大计划。正如中国‘新基建’计划一样，都是立足长远，利国利民的超前规划”。然后再顺带称赞了中共一把：“拜登抄中国作业，也恰好印证了中国在战略部署上，已经比美国领先一步，赢在了起跑线上”。
  <br/>
  <br/>
  拜登的《美国就业计划》，虽然也谈了与中国新基建计划内容雷同的内容，但智能化、数字化本就是美国走在前面，绿色能源更是美国民主党心心念念要实行的战略。这些都不是中国原创，美国自然谈不上抄中国作业。真正“抄中国作业”的，其实就是政府大量印钞投资基础设施这点，因为美国从来就不主张政府投资。从西部开发时期开始，美国习惯走的筹资路线就是用政策撬动私人投资，整个铁路投资当中，私人资本占85%以上。而《美国就业计划》及其他的两个计划所需资金共逾6.2万亿美元，来源只有一个：联邦政府资金。这就意味着，只有印钞与税金两条路。
  <br/>
  <br/>
  这倒真是“抄中国作业”，但作业本是2009年中国5万亿救市计划。当时世界正值金融危机，中国政府投资5万亿加强基础设施建设，被西方不少媒体誉为拯救世界，国际地位大大上升。在匹兹堡演讲中，拜登阐明计划目的之时就有类似表述，比如要让“盟友看到美国的力量”，并将他雄心勃勃的国内议程与恢复美国海外影响力的计划联系到了一起。
  <br/>
  <br/>
  <strong>
   中国有意忘记的高铁另一半故事
  </strong>
 </p>
 <p>

  <br/>
  <br/>
  中国舆论肆意嘲笑美国铁路系统的陈旧老化与中国高铁的新与前卫，但却故意不提直到去年还在提的一个问题：高铁建设负债沉重，运输能力严重过剩。
  <br/>
  <br/>
  如果不看中国高铁的投资与产出（营运数据）之比，确实会赞叹中国高铁在短期内取得的辉煌成就。了解全部故事后，才知道中国为高铁修建付出了巨大的代价，其背负的债务成为中国金融风险的一头巨大“灰犀牛”。
  <br/>
  <br/>
  北京交通大学城镇化研究中心主任赵坚曾在2019年1月发表长文（财新网），列举了中国高铁项目的具体经营数据：截至2018年底，中国高速铁路营业里程已达2.9万公里。人们通常只看到中国高铁运营里程的世界第一和高铁的快捷，却对事件的另一面——高铁债务和运营亏损的世界第一，及中国交通运输结构的严重恶化视而不见。中国高铁主要靠债务融资，截至2018年9月，中铁总的负债已高达5.28万亿。但从中铁公布的负债和客运收入数据可得出如下判断：即使不考虑高铁的运营成本，高铁的全部运输收入尚不够支付建设高铁的贷款利息。
  <br/>
  <br/>
  赵坚还指出，除京沪、京广通道上的高铁运输能力得到较高利用外，其他高铁项目的运能大量闲置，存在严重亏损。最能反映铁路运输能力利用效率的指标是运输密度，即平均每公里铁路一年完成的运输周转量（每公里铁路创造年运输收入的能力）。2015年，中国高铁的平均运输密度在1700万人公里/公里左右，日本高铁的平均运输密度为3400万人公里/公里，是中国高铁平均运输密度的2倍。赵坚还指出，中国10多年间建设的高速铁路已是世界其他国家和地区半个世纪建设的高速铁路总和的2倍以上。世界各国的高铁几乎没有一条能够依靠客运收入支付建设和运营成本，大多处于亏损状态或靠政府补贴运营，中国的情况更严重。
  <br/>
  <br/>
  赵坚文章为中国高铁建设总结的几大问题，诸如巨额债务、低营运效率、对全中国交通运输系统的严重扭曲，以及造成的环境污染。而“抄中国作业”说的作者们基本不提，仿佛从不存在。
  <br/>
  <br/>
  <strong>
   美国应该如何看待中国的“抄作业说”？
  </strong>
  <br/>
  <br/>
  相比欧盟、日本等国对美国基建计划的沉默，中国舆论的活跃或过度活跃，我认为也有益。美国人也许不知道中国著名的唐太宗典故，这位明哲君主相信“兼听则明，偏信则暗”这个道理，在直言进谏的魏征去世之后，感叹说“夫以铜为镜，可以正衣冠；以史为镜，可以知兴替；以人为镜，可以明得失”，痛惜魏征去世，让他失去了一面镜子。但我相信，不少美国人会知道哈佛大学教授卡斯（Cass R. Sunstein）提出的一个著名概念“信息茧房”（Information Cocoons）：
  <br/>
  <br/>
  “如果你只关注自己选择的领域，如果你只关注某一种信息源，如果你只关注自己愉悦的东西，久而久之，便会像蚕一样，将自己桎梏于自我编织的茧房之中，从而丧失全面看待事物的能力。”
  <br/>
  <br/>
  中国2009年投入5万亿人民币，地方政府配套投入数十亿（无准确数字，官方数字有逾20万亿、逾30万亿两个版本），从而扭曲中国经济结构、深陷债务泥潭的经验在前，美国只要不困身于信息茧房，应该从中悟出怎么做才有利于美国的未来。习惯用极权专制、右与左来诠释美中争论的人士，这次请注意别陷在“信息茧房”里。无论美国还是中国，这次关注这个计划的人，其注意力多半非关政治体制，仅仅只是出于项目的可行性及其效果，尤其是对自身国家的影响。
