 <p>
  美国金融大鳄索罗斯最近在《华尔街日报》撰文，称习近平是全世界开放社会的最危险敌人，而全体中国人都是习近平的受害者。索罗斯说，习近平掌权后致力于消除邓小平的影响，威胁和实施摧毁大陆这只会下金蛋的鹅，要将中共返回列宁主义政党，习近平是面对不断变化的世界倒行逆施的独裁者。一生信奉并致力于开放社会的索罗斯，其实与社会主义颇多渊源并不全然是反感，如索罗斯对一手制造六四屠杀的邓小平就常有提及还颇多赞誉。索罗斯之所以在国际媒体上撰文直斥习近平，实在是因为习近平展现出来的与人类为敌的疯狂完全不可理喻并且构成了可见的危险，致使索罗斯对某种可接受专制的意识也难以认同。
  <br/>
  <br/>
  国际媒体和网络上有人说，习近平一天灭大陆一个行业。这话从深刻和警醒的角度来说，还真是恰如其分。原本要在港沪上市的蚂蚁金服集团万事俱备，但临门一脚被中共封杀，不但要创造世界上市圈钱最大奇迹泡汤，前母公司阿里巴巴被罚空前巨款，创办人马云也遭中共追打踪影不见。大陆另一网络巨兽腾讯公司和创办人马化腾，比之阿里巴巴和马云的下场也好得有限。腾讯已经落入中共反垄断机构打击对象的视野，马化腾则辞去腾讯征信等法人和执行董事高位，费尽心思力求避免成为马云第二的下场。其他动辄资产上万亿的巨兽公司或行业，如赴美成功上市圈钱的滴滴出行，仅数天后遭中共毁灭性打击风光变灾难；还有不久前风光无限，北大清华毕业生蜂拥入职的线上补教公司，在中共一纸公文痛击下九命丢了八命，眼看着数千万薪酬优渥的职工即将加入失业大军。总之，人们目接不暇和目瞪口呆地看到，大陆一个个巨无霸公司和蓬勃发展的行业在显然是习近平主导指挥下，惨遭灭顶之灾甚至灰飞烟灭。
  <br/>
  <br/>
  习近平对于大陆社会的一系列控制驯化，其程度比之对私企的粗暴打压毁坏用有过之而无不及的刻画也是贴切的。在习近平掌权之前大陆十分有限的一点空间，如红线明示的言论空间和生活空间，更从文革后挤撞出来的一点而回归毛时代的恐怖严密。中共鼓励的学校检举和线上检举，以及习近平思想成为小学生必修课，英语学习遭到打压歧视不列考试，则是习近平近十年掌权向毛回归的典型象征。习近平回归毛泽东恐怖统治的另一象征场面，就是索罗斯文章中提及的中共政治局常委们站在习身后，奴隶般重复对习的效忠誓言。这场面甚至超越了毛泽东时代的效忠表现，虽然可能是老迈毛泽东无法带领党的大佬们如此演出了，但也说明习近平追求仪式化的独裁，既有党徒更加表象的忠诚也有令习近平不安的危险阴影。
  <br/>
  <br/>
  而习近平在世界上表演的恶劣更是令人鄙弃不齿，反向回应了习近平要求讲好中国故事的指示。经济上，中共从来就不遵守承诺。习近平掌权更是变本加厉肆无忌惮，窃取专利智慧财产权是大陆弯道超车的成熟经验。在中共外宣影视剧中，不乏偷盗得手还酸讽老外的桥段。而中共将香港的中英条约承诺说是签署即是完成，更是令全世界见识了墨迹未干抹脸就不认账的中共嘴脸。中共对卫星拍照清晰可辨的新疆百万强制教育营，是这种死不认账的补充加强材料。而爆发于武汉的病毒甩锅恶行，是中共目前最受世界关注和愤慨的行径。习近平所谓为人类开辟最成功发展模式，并意图制定国际规则引领未来世界，则让国际对中共的狂妄野心逐渐认清，意识到世界有中共专制独裁祸害的风险。

  习近平这种作妖式的政治操作不仅令人瞠目结舌，诸如一天灭一个大陆行业、杀会下金蛋的鹅，是完全无视大陆经济已经出现的疲软和危险信号。习近平究竟打着怎样的算盘，难道真是白痴不成？或许这是习近平学识才能有限，而粗糙莽撞的表现。但不可否认的是，习近平早就下着一盘野心勃勃的所谓大棋，他要东升西降取代美国，改变世界格局治理全球。所有不合情理的、反常的作妖式行径，都是习近平要达成他野心的手段。所以看似习近平作妖式的国内外荒唐表演，但在习近平来说却是心思缜密的谋略。因为习近平也意识到，他将面对可能灭顶之灾的挑战。对他而言，有效的谋略就是快速加强独裁，消除任何不满的隐患和超出控制的政治经济力量。
  <br/>
  <br/>
  所以，不论是胁迫中共高官奴颜婢膝的宣誓效忠，还是搞运动式的社会告密、揭发、叛徒等恐怖氛围，打压外语学习，强制灌输习近平治国理念，真正用意都是将习近平置于全民跪拜的神龛，从而达到提前肃清可能对习近平独裁构成的威胁。至于看似荒唐的习近平每天灭一行业的经济自戕，不单是在实行习近平变私为公的思想，更是提前将具有巨大社会影响力的私人资本，尤其是网络科技这类难以预测的力量予以铲除根基。习近平绝不允许他应对国际国内危险的挑战时，大陆已经足以影响社会庞大群体的私人资本能够在他控制之外而形成不得不顾忌的威胁。习近平倒行逆施的目的可谓痴心妄想，但他宁可自戕也要拉上大陆波及世界，其中的危险性和破坏力切实不可小觑。
