 <p>
  在加拿大释放孟晚舟，德国和日本都进行领导人改选，台湾的国民党选出新的党主席等一系列焦点新闻之下，香港发生的一件大事多少被外界忽略了。那就是，香港最有影响力的民间社团之一香港支联会，在险恶的政治环境下，被迫宣布解散。作为长期得到香港支联会支持的中国民主运动中的一员，我认为我有义务呼吁外界，铭记三十多年来香港支联会为推动中国大陆和香港的民主做出的重大贡献。
  <br/>
  <br/>
  香港支联会成立三十多年来，做了很多的事情。但我认为，最有意义的是以下三个事情：
  <br/>
  <br/>
  首先，香港支联会从1989年起，坚持每年在香港维园举办纪念“六四”烛光晚会，三十年如一日，不论是人数减少还是天气恶劣，从来没有中断过一次。这件事使得八九民运在事实上并未被中共完全镇压，它以烛光晚会的形式继续在香港坚持了三十年，这是我们这些八九民运的参与者最应当感谢的，他们做了我们做不到的事情。现在的新世代香港年轻人对支联会有看法，有批评，这都正常，但我认为他们忽略了香港能够走到雨伞革命这一步的历史脉络。事实上，维园晚会不仅提升了香港民主运动在国际上的能见度，而且在很大程度上，这样的坚持是一种政治启蒙。香港的反抗精神，包括年轻世代的反抗行动，从发展线索上看，香港支联会每年举办的烛光晚会等，都是在这样的启蒙下成长起来的。维园晚会已经成为一种政治符号，写进了香港的历史中，这是港人不应当忘记的。
  <br/>
  <br/>
  其次，香港支联会创立初期提出的五大主张“释放民运人士，平反八九民运，追究屠城责任，结束一党专政，建设民主中国”，三十多年来始终没有放弃。这不仅是他们的口号，更是他们的具体行动。多年来，他们参与营救“六四”人士的“黄雀行动”，支持中国国内的“天安门母亲“运动，声援包括我在内的中国政治犯和维权律师群体，以及在海外的流亡人士的“我们要回家”运动。在中国政治长期处于黑暗期的时候，香港支联会长期不懈对于中国民主化的关注和支持，对于中国那些面对暴政仍然顽强抵抗的人来说，是巨大的精神鼓舞。从长期来看，对于中国的民主发展也起了重大的推进作用。这是中国所有追求民主的人士不应当忘记的。
  <br/>
  <br/>
  最后，支联会与其他香港的民主派团体一起，通过他们三十多年的努力，打造了“香港精神”。这种“香港精神”的内涵，包括长期的坚持，包括在薪火相传的精神下对于年轻一辈的影响，包括严谨的工作方式，包括对于人权自由理念的坚定信仰。所有这些，都使得香港在三十多年的岁月中，始终扮演了东亚民主的基地，对抗中共暴政的桥头堡的角色。支联会虽然被迫解散，停止活动，但他们参与建筑的这种“香港精神”不会消逝，提醒了全世界对于中共极权倒退的警觉，鼓舞了包括台湾在内的东亚地区的反抗运动，这是全世界爱好民主自由的人都不应当忘记的。
  <br/>
  <br/>
  香港支联会虽然在形式上解散了，但中共对他们的切齿痛恨，就是他们的荣誉勋章。他们三十多年的努力不仅已经创造了历史，也必将使得他们的工作被写进历史之中，历史是不会忘记他们的。
  <br/>
