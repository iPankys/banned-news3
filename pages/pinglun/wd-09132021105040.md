 <p>
  今年是震惊全世界的九一一恐怖袭击事件20周年，美国及其盟友都举办了庄重的纪念活动，并再次重申绝不屈服于恐怖主义的决心。唯有中国媒体的反应，不阴不阳，酸言酸语。举例而言，中国环球电视网发表署名文章，批评美国20年的反恐战争“浪费我们对三千美国死者的同情”； 《环球网》则发社论，斥责美国是“严重的美国中心主义在作祟”，说美国的精英们“直到现在也没有走出迷思”。 中国外交部发言人赵立坚则更是落井下石，指美国是阿富汗问题始作俑者，应深刻汲取教训。综合上述言论可以看出，中国对于美国深陷恐怖主义袭击的泥潭，有一种见猎心喜的暗爽的心态。
  <br/>
  <br/>
  中共乐见美国陷入麻烦之中，这原本并不是什么稀奇的事情。只是，回顾二十年前那场恐怖主义袭击，袭击之后美国和西方世界的反应，以及那次袭击给全世界的政治和军事发展带来的深远影响，中共其实没有什么见猎心喜的本钱，因为需要吸取教训的，绝不只是美国，也包括了中国。九一一事件的意义，中国也应当进行反思，吸取教训才是。为什么这么说呢？
  <br/>
  <br/>
  首先，20年前九一一事件发生之后，当时的美国总统布什曾经在国会发表重要演说，其中有一段的立场表述非常强硬且极为清晰。他说：“我们对那些对恐怖份子提供协助和庇护的国家绝不放过，现在是无论任何国家必须作出决定的时候：如果不是和我们一起，就是和恐怖份子一起。”这句话的意思非常明确，那就是任何协助过对美国发动恐怖袭击的组织或政权的国家，美国都将视为恐怖主义国家并予以沉重打击。了解美国历史的人都知道，这并不是布什总统的个人立场，而是美国传统上捍卫自己，打击敌人的一贯立场。
  <br/>
  <br/>
  而现在阿富汗局势发生重大变化，外界普遍预期恐怖主义团体将在阿富汗重振旗鼓，英国军情五处的官员已经发出了会出现新的恐怖袭击的警告。而在这样的历史时刻，中国却选择与塔利班政权紧密站在一起并给予大量的政治和经济协助。这对中国来说是一件玩火的行为。因为，除非中国能够永久控制住塔利班政权，确保其在阿富汗的统治不会助长恐怖主义，否则，一旦未来出现以阿富汗为基地的、针对美国的再一次恐怖袭击，美国的报复方针将一如既往地指向所有对阿富汗提供实质支持和协助的国家，中国必将面对与美国的正面军事冲突。大家都知道，这对中国来说，是不可承受之重。中国应当为避免这样的情况发生感到焦虑，而不是见猎心喜。
  <br/>
  <br/>
  其次，九一一事件留下的一个教训就是，现在的国际社会不再有强弱国之分，处于弱势的国家在局部战争中，可能会采取超限战的方式。因此，大国使用军事力量应当更谨慎。换句话说，大国军事力量再强大，也不能保证自己的安全。我想这一点，对于中国来说应当是一个是教训。尤其是那些积极主张用军事力量收复台湾的人，他们应当知道，一旦军事冲突爆发，台湾或许无力抵抗，但在被中共军队严重伤害之后，台湾一定会对中国进行类似超限战的报复，这样的报复防不胜防。如果说小小的基地组织都能迫使美国付出沉重代价的话，台湾的反击和报复能力就更会让中国也陷入类似美国的阿富汗困局中，本土安全也将面临重大威胁。九一一事件留给世界的一个重大反思就是，军事力量不能解决根本问题。同样的道理，也适用在台湾问题的处理上，这是中共必须严肃思考的事情。如果不去思考这些问题，以为九一一的意义只限于美国，那将是中共的严重误判。
