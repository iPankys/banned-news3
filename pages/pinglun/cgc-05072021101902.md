 <p>
  2021年4月28日，江苏昆山的王和英和张家港的陶红到河南信阳，去探望出狱两年多来却至今仍然被中共非法软禁在罗山县老家的江天勇律师。她们刚进江天勇家门，聊了没多久，中共公安就闻讯赶到门外，摇晃着大门要求客人出去，他们要“查验身份证”。一直以来，中共的爪牙以“查身份证”为名骚扰来访朋友的无耻做法从未停止过，这当然令江律师十份生气。唇枪舌剑之下，中共爪牙们胡搅蛮缠、知法犯法，面对人权律师有理、有据、有节的雄辩，自然是毫无招架之力。他们电话请示头头后，无奈地败走。如今的中共政权，除了无耻枉法耍流氓，还剩下什么呢？！
  <br/>
  <br/>
  公安离开后，江天勇律师和家人热情款待远来的客人自不必说。外面一直驻守在专门为了跟踪监视江律师，而建在江律师父母家附近平房中的八个地痞流氓，则警惕地通过早就在每个路口安装好的摄像头，严密监视来往的每一个行人。中共这样的非法行为，已经在江律师父母这里持续了两年多。无论是王鞘岭、李文足还是谢阳律师等等，所有到河南信阳探访过江律师的朋友都有过被公安以“查验身份证”为名骚扰的经历，有的还被强行带到了派出所进行盘问。这次王和英和陶红也不例外，当江律师送她们离开后，仍然有爪牙跟踪她们，并电话叫来了信阳的国保队长以及当地派出所公安，在宾馆附近抓住王和英和陶红要她们到派出所走一趟，之后再逼着她们出示身份证后才离去。
  <br/>
  <br/>
  中共这样的行为纯粹属于滥用职权，维护一党之私利。可是，在所有权力都被共产党垄断的共产专制之下，谈法律、法治就像在沙漠里要求人们必须遵守“红灯停，绿灯行，黄灯亮时等一等”的交通规则一样滑稽可笑，因为专政之下是没有法治的。

  尽管这一切对于我而言实在是太熟悉了，可是当我从江律师与中共爪牙讲理的视频了解到当时所发生的事情，当我读到王和英写的文章时，还是不由得胸中充满怒火。我经历了当年山东临沂东师古的壁垒森严，无数网友因为勇闯东师古而遭到中共雇佣的地痞流氓的抢劫，在公安遥控指挥下被殴打的一桩桩、一幕幕又都浮现在我的脑海中。网友们当年探访东师古的亲身体会，不仅让他们深度认识了专制政权的邪恶，也见证了中国共产党的无耻本性。
  <br/>
  <br/>
  记得我刚出狱不久，我请三哥帮我在临沂买了一个电话卡。我只用它打了几个电话给朋友后，中共就派公安驾车从沂南县到临沂市，把我三哥从打工的地方押回到东师古，逼着他把电话卡要回去拿走。许志永、李方平等，也因为要到东师古探访我而遭公安假扮的流氓一路追打。所谓的“公安”，这时的一系列作法都与土匪强盗毫无区别。换句话说，此时的“公安”已经不再是公安，而是中国共产党豢养的一群恶毒打手、党卫军而已。因此，他们所有的做法不仅是没有法律依据的犯罪行为，而且都和代表公权力没有任何关系，不再具有任何合法性和正当性。
  <br/>
  <br/>
  不幸的是，从那时到现在，又一个十多年过去了。从山东临沂东师古到河南信阳，中共不仅没有停止作恶，相反更加肆无忌惮、变本加厉地为了维护它的独裁政权，不断违宪、违法地迫害中国人民，就连遮羞布也不要了。

  如果哪位对我上述对中共的描述有不同意见，无需辩驳，请您去河南信阳探访江天勇律师。如果您去了，事实一定会大大提高您对中共的认识。您几十年来形成的认知，将彻底被颠覆。已经错过了去东师古的朋友们可千万别再错过了去河南信阳，您去了可能会后悔一天，但错过了认识中共本质的宝贵人生经历，则很可能会后悔一辈子。
