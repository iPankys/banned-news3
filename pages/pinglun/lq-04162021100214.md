 <p>
  持有澳大利亚国籍、但在中共中央电视台当主持人的成蕾女士，在中国于去年八月关押近半年后，终于被以“非法或意图提供国家机密给外国机构或个人”为由，宣布将成蕾逮捕也就是摆出要用法律惩治成蕾了。不过中共这公布有着让人难免不注意的笑点，即这样的罪名难道不应该拘禁前已经了然于胸，还需要费劲巴力地过上半年才憋出来吗？众所周知，孟晚舟几乎被捕的同时，犯有银行欺诈罪的罪名已经在全世界公开。中共这种相隔半年才憋出被捕外国人的罪名，不得不让人怀疑中共实施抓捕时究竟手里有点嘛？还是要通过逮捕后的操作，来知道点嘛？最近一名自称北国小甜瓜的作者，在网上发表名为央视主播实为他国间谍一文，详尽介绍了成蕾的过往和所谓间谍情节，尤其是满腔义愤地痛斥成蕾忘恩负义。
  <br/>
  <br/>
  小甜瓜在其洋洋洒洒的长文中，所谈及的间谍情节却让人一头雾水，根本不知道所言为何或何为间谍。小甜瓜指称成蕾的间谍故事情节，是二零一七年后成蕾在采访中“说漏了嘴”，成蕾公开称“美国、澳大利亚已经掌握了中国经济发展的动态” 。作者在此着重提醒读者注意“这是一个陈述句”，并立即诘问“你怎么知道他们掌握了？他们是怎么掌握的？”不过小甜瓜万分得意从成蕾说漏了嘴而抓到其间谍的实锤证据，却让人对其得意之笔的逻辑哭笑不得。
  <br/>
  <br/>
  如果掌握中国经济发展的动态算机密，那成蕾所说和泄漏的也是美国和澳大利亚的机密。由成蕾知道美澳掌握这机密，推导出是通过成蕾而掌握这机密的，这应是侦案才有的思维而不是断案逻辑。况且所谓中国经济发展的动态，不就是无视国际规则大肆坑蒙拐骗偷吗？这动态真算不上什么机密，包括许多大陆人的全世界关注经济的人，没有多少是不知道这机密的。当今世界正在形成的围堵制裁中国经济的形势，不就是因为无法再容忍中共以这经济发展动态坑害世界吗？
  <br/>
  <br/>
  小甜瓜对成蕾另一个重点攻击的是两面人，说成蕾面对大陆人是一副岁月静好的面孔，网上嗮的主要是工作和孩子；不过到了脸书账号里，成蕾却换成“灯塔斗士”的嘴脸。用小甜瓜的话说，“不断抨击生养她，给她饭碗和事业的中国”。但是小甜瓜所能列举的成蕾此类言行，也只是二零二零年一月至三月的时间段，内容全是对中共病毒期间表现的批评和感受，如“医生护士被告知不存在人和人之间的感染危险后，被生生感染死去”、“全中国的关键词只有一个感恩”、“官员没有羞愧地低下头，他们还要求当地人要感恩”；并说成蕾“一直恶意诱导新冠病毒污名化”，号召民众“要反抗！不要妥协要深究”。小甜瓜据此总结，成蕾是“吃中国饭，砸中国锅的香蕉人”，“吃相太难看了”。
  <br/>
  <br/>
  但是将小甜瓜通篇网文仔细读完，所得观感却与之结论南辕北辙，反而是这个小甜瓜才展现出了自家嘴脸，即想从中共残羹剩饭捞些甜头而不惜为恶之瓜。因为不论这个想骗些甜头之瓜如何删减编串，耀然于纸上的还是一个励志女士，一个敢于直言、为民为人类痛斥邪恶的勇士，看不到小甜瓜指斥的间谍的丝毫迹象。小甜瓜指责成蕾不安于平庸现状，想当记者、想当电视主持人，虽遭家庭反对和一时难以实现，但不放弃和不断自我完善，终于以小甜瓜网文中也得承认的水平，即流利的英文和财经专业的优势，在公开招聘中通过层层筛选，而成为《财经回顾》、《中国财经》等央视节目主持人。
  <br/>
  <br/>
  仅从小甜瓜所写文章和所谓的间谍情节看，中共逮捕成蕾的真正原因可能更趋重于两点：第一是中共病毒在武汉爆发后，澳大利亚是世界上最早和最强烈要求向中共追究病毒溯源的国家，而中共意图通过各种施压，逼迫澳大利亚妥协退让，对澳经济的种种加税、取消采购或消减进口等最极明显。而通过扣押、驱赶、刁难、骚扰澳籍公民也是手段之一，成蕾大概率就是中共这手段的牺牲品。第二，中共之所以选中成蕾当牺牲品，则与成蕾的言论戳到中共痛处相关。正如小甜瓜痛骂成蕾忘恩负义所言，表达了中共将央视招聘成蕾视为对她天大恩惠，而成蕾并不认可这恩惠并率性批评，令中共不痛击成蕾就难泄心头的无名火，也唯恐群起效仿而形成没有顾虑的局面。
  <br/>
  <br/>
  成蕾的外甥女路易莎在接受媒体采访时说，她不认为成蕾会做意图伤害国家安全的事。这既是对成蕾人品和思想意识的了解，也是如她所说，对情况毫无了解下仅能做的。其实成蕾案件一如许多被中共拘押的案件，仅仅是中共在国际操作中要抓些谈判筹码。这从中共拘捕人后长期不能公开罪证，只有笼统随便安置的如间谍这样的罪名便可以基本判断出来了。例如，澳大利亚杨恒均，还有孟晚舟被捕后中共火速抓捕的两名加拿大公民，以及小甜瓜所称成蕾最铁的闺蜜范若伊，也就是与成蕾前后被中共拘捕的彭博社派驻北京的记者，全是中共以间谍罪名关押起来的。但是两名加拿大人已经关押超过二年，而且闭门审判也已经结束，中共仍然是没有任何具体罪行公布。
  <br/>
  <br/>
  其实，朝鲜曾经抓捕审判的美国人就是这些共党国家所谓间谍的最好写照。奥托·瓦姆比尔在朝鲜旅行时试图带回一张宣传海报，被朝鲜以间谍罪名判刑十五年，在关押一年半后成为植物人释放回美。旅行时年仅二十岁的这名大学生，几天后死于美国医院。在共党国家，间谍罪名可以扣在每一名外国人头上，并可以残酷到无可言喻地让其失去生命。
