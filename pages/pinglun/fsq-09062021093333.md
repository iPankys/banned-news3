  <br/>
  <br/>
  白皮书称：“共产党领导、多党派合作，共产党执政、多党派参政”，是“为人类政治文明发展作出了重大贡献” 的新型政党制度。中共就敢厚颜无耻，睁着眼睛向世界描述这样一个动人的童话故事。
  <br/>
  <br/>
  白皮书称，中共始终“把为中国人民谋幸福、为中华民族谋复兴作为初心和使命”。说“为中国人民谋幸福、为中华民族谋复兴”这没什么特别，每个国家的政党或政治团体都这么说，没有说“为本国人民的痛苦，为本民族的衰败”而成立的，即便是希特勒的社会主义工人党也是宣称为德国人民的幸福、为德意志民族的复兴而奋斗的。但说这是中共的“初心和使命”，就是弥天大谎了。
  <br/>
  <br/>
  中共的初心和使命在一大纲领里，作了明确表达：中共领导的“革命军队必须与无产阶级一起推翻资本家阶级的政权；承认无产阶级专政，直到阶级斗争结束，即直到消灭社会的阶级区分；消灭资本家私有制，没收机器、土地、厂房和半成品等生产资料，归社会公有。”这和马克思、列宁一再表达的共产党人的使命，就是消灭私有制、消灭阶级和国家是完全一致的。
  <br/>
  <br/>
  如今变异的幽灵在改革开放的名义下，造就了特殊的权贵私有制。声称要消灭资本家私有制的红色家族，自己成了资本家或资本家的看家狗！他们再也不敢举起“消灭私有制”，这个中共的初心和使命了！
  <br/>
  <br/>
  如今的中共知道，前辈的初心和使命是不可能实现的。断然加以抛弃，这本来是件好事，是中国回归市场经济，并再造与之相应的多党民主制度的契机。但尝到特权甜头的中共党人紧紧抓住用暴力和谎言获得的既得利益，巧言掩饰抛弃初心和使命的事实，维护自身的特殊利益。更可恶的是，为了维持统治利益能长久不衰，以无产阶级专政的名义，创造了名为多党合作的一党专政制度。这个政党制度及其统治的社会，就是一台毫无人性、囊括了所有人的“内卷”绞肉机，即便是最高层人员也会在瞬间成为牺牲品。因此，这个政党制度是新型的，但却是邪恶的，不仅不能为政治文明作贡献，相反是危害人类文明的毒瘤。
