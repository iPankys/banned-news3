 <p>
  各位听众你们好：
  <br/>
  <br/>
  首先在农历新年之际，我要借自由亚洲电台“王丹热邮”栏目，向各位自由亚洲电台的听众朋友说一声新春快乐，也谢谢你们长期以来对我的节目的支持。
  <br/>
  <br/>
  新的一年开始的时候，往往也是我们可以对自己重新有所要求，有所期待的时候。在这样的时候，我说说我对各位听众，尤其是在中国大陆生活的听众的一点小小的寄语，那就是：新的一年，我们要做一个什么样的人？
  <br/>
  <br/>
  我祝愿大家都要做一个健康的人。新冠病毒肆虐全球已经一年多了，对全世界，对整个人类都造成了重大的破坏。新的一年，祝愿大家能够早日走出疫情的阴影，保护好自己的健康。有一个健康的身体，不仅是对自己负责，也是对自己的家人负责。
  <br/>
  <br/>
  我祝愿大家做一个有记忆的人。想想中国政府对于所谓“抗疫成绩”大张旗鼓的表扬与自我表扬，每个中国人都应当问问自己：是否还记得一年前武汉封城的时候，传出的各种令人揪心和痛心的事？是否还记得，有一个叫李文亮的医生为什么会被警察警告？是否还记得，有一个叫方方的作家每天的日记？是否还记得陈秋实、方斌、李泽华等现场记录的公民记者？一个有记忆的民族才有未来，同样，做一个有记忆的人，我们才对得起这个民族。
  <br/>
  <br/>
  我祝愿大家做一个不说假话的人。在今天的中国，说真话越来越危险。规避危险，保全自己，是可以理解的。因此，面对专制的政权，面对不公义的事情，我不会说每一个人必须要勇敢站出来表达抗议。但是，不能说真话，我们至少可以不说假话。当我们看到中国仍然还有很多低收入的人口过着贫穷生活的时候，你可以不说话，但你不应当跟着政府说“中国已经完成了脱贫的任务”；当clubhouse秒速在中国被禁的时候，你可以不抗议，但不要自欺欺人，认为中国还有一些自由言论的空间。如果新的一年，中国人都开始不说假话，这个国家的变化就会发生。
  <br/>
  <br/>
  最后，也祝愿大家做一个有公共意识的人。我们应当知道，在中国这种党统治国家的体制下，在国家可以任意剥夺个人，甚至是富豪的财产的制度下，没有什么完全的个人生活，你的生活是与大环境联系在一起的。正因为如此，即使是为了个人的利益，我们也应当做一个公共人：关心公共事务，参与公共讨论。这个国家的改变，不可能靠中南海里面的那些人，希望在每一个中国人身上。改变自己，就是保护自己利益的开始。因此，新的一年，让我们做一个公民，而不仅仅是居民和国民。
  <br/>
  <br/>
  这是我个人的一点小小期待，仅供参考。谢谢。
  <br/>
  <br/>
  王丹
