 <p>
  前不久，智库“对话中国”出版了海外著名的反对运动人士王军涛的新书《行动创造转型：中国民主化的思考笔记》。曾经长期参与中国反对运动，王军涛在书中对一些有关中国民主转型的问题提出了一系列发人深省的观点，在这里我向外界简要介绍其中一部分，希望给关心中国民主化问题的听众朋友提供一些参考。
  <br/>
  <br/>
  王军涛在他的新书中开宗明义地指出：当中国成为世界上第二位经济强国的时候，中国的民主化对世界的和平与安全就有了决定性的意义，而改造中国成为二十一世纪的世界和平和自由民主国家安全的至关重要的任务。我认为，指出这一点十分重要。因为今天不论是在台湾、香港，还是在美国和西方国家都有一种错误的观点，似乎认为中国民主化是中国的事情，其他国家和地区没有义务帮助中国的民主化。这样的观点忽视了一个重要的现实，那就是正如王军涛在书中指出的：中国问题已经不仅仅是中国问题，中国问题就是世界问题，台湾问题，香港问题，也是美国问题。
  <br/>
  <br/>
  在分析当今中国的政治形势以及未来转型的可能路径的时候，与外界普遍的期待相反，王军涛认为中国的未来不可能出现和平转型的前景，而出现社会革命的可能性更大一些。做出这样的判断，基于王军涛提出的一个概念 -- “暴力依赖“。他指出：中共已经习惯了用暴力统治，并且已经形成了对暴力的依赖。一方面，政府尝到了暴力的立竿见影的效果，越来越没有耐心用讲道理的方式治理国家；另一方面，暴力造成的后果会激发起国家与社会之间对立，久而久之，民间社会也会且只能倾向于暴力反抗。他认为，在朝野都相信暴力的情况下，和平转型的希望已经越来越渺茫。尽管这不是我个人希望看到的状况，但我也必须承认，在今天的中国，从“和平、理性、非暴力”的口号受到越来越多的质疑，到民间对杨佳这样的暴力反抗案例的推崇，都已经在验证王军涛的判断。
 </p>

  <br/>
  <br/>
  在民主转型出现的前提条件的问题上，王军涛在新书中特别提醒我们，从历史的经验角度看，大多数社会的民主转型都不是在民主理念深入人心的情况下发生的，这些转型大都是利益驱动导致的。基于这样的判断，王军涛对于中国的海外民运或者反对运动给出了一个具有指导性的建议，那就是：民主力量要将民主解释为符合这些力量的利益的最佳解决方案。对此，我的理解是：作为建设性的反对派，民主力量必须提出符合人民的利益期待的主张。例如，如何更好地分配社会资源等等，而不能仅仅停留在理念上的民主宣传和口号式的民众动员上。
  <br/>
  <br/>
  最后，针对外界对于海外民运的种种批评，王军涛也提出了一个比较新颖的观点。他认为，民运是建立全民参与的政治制度，一个国家的民主运动是全体公民的事情。民运在转型前的政治博弈中，只传播民主理念和推动转型发生，他们不能对这些民族或者国家整体不接受民主理念承担责任。那些没有主动参与的人，不仅没有道义资格指责民运做得不好，而且应该承担国家未能进入民主转型的责任。民运往往以外界的指责来反思、自责，这是定位不清，好像国家民主制度是他们自己就可以办好的事业，夸大了自己的责任。这个观点也许见仁见智，但值得外界参考。
  <br/>
  <br/>
  在中国政治局势充满不确定性的时候，我们需要有更多对于未来转型的思考。我向大家推荐王军涛的这本新书《行动创造转型：中国民主化的思考笔记》，就是希望激发起更多的相关讨论。
