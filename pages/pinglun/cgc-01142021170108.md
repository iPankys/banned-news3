  在病毒来源仍然需要调查尚且没有结论的情况下，在中国沦陷区究竟有多少人因为这场瘟疫而死亡都还没弄清楚的情况下，最近在中国河北、北京、辽宁、山东、河南、上海等多地，武汉病毒又杀回来了。事实上，也许从去年到现在病毒就没有从根本上被消除，只是主动或者被动地隐藏起来了而已。
  <br/>
  <br/>
  回想起去年的此时此刻，中共正在通过它的喉舌向全世界宣传“病毒是可防可控的”，但是转眼，中共就开始了所谓的“有效抗疫方式”——封城锁国，采取了不顾人民死活地把住户的家门、单元门用木板钉死，甚至用钢筋、铁链焊死的方式，让病毒和居民在屋子里一起自生自灭，并且对所有在网上提及疫情真相的言论进行审查过滤，结果导致所谓“可防可控”的武汉病毒迅速传遍了全世界。这样的“可防可控”实在可笑，却令人笑不出来，只想哭。

  不幸的是，即使到现在中共依然没有停止对所有敢于揭露疫情真相、调查因封城锁国导致灾难性后果的人进行打压迫害。去年调查揭露出武汉封城后，疫情导致大量市民死亡的公民记者方斌至今下落不明；陈秋实和李泽华也杳无音信；张展女士因为亲自赴武汉调查疫情真相而成了党国的敌人，被中共以“寻衅滋事罪”跨区抓捕，带回上海构陷关押判刑四年……。
  <br/>
  <br/>
  而更加不幸的是，就在世界各国几乎都错误地认为“中共的抗疫方式十分有效，应该效仿”，并藉此批评他们本国政府抗疫不利的时候，武汉病毒却鬼使神差般地在好多地方又回来了，甚至发生了变异，毒性比以前大幅度提高。
  <br/>
  <br/>
  面对病毒的捲土重来，中共的抗疫方式毫无新意，依然采取封城加封口进行应对：一方面通过宣传机器告诉疫区以外的人民，“全国只有几十例感染者，况且全都是外国输入”；另一方面则故伎重演，加紧封城、封村、封小区、封单元，对人民进行暴力管控。有网友放出一段某地村党委书记在村里高音喇叭上的讲话说：“多准备洋镐把，谁不听，出来就打死他……。”
