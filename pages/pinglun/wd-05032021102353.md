 <p>
  最近几年，随着中国在南海、东海和台海不断制造紧张局势，国际政治上传统的“亚太地区”的概念越来越被“印太地区”替代。而印太地区作为最有可能发生军事冲突的战略区域，受到的外界关注也越来越大。这个地区的重要性在哪里？哪些国家左右这个地区的局势？要如何确保印太地区的和平、繁荣和稳定？如果想对这些问题有初步的了解，不妨可以看看去年9月在台湾出版的一本新书《印太竞逐》。
  <br/>
  <br/>
  《印太竞逐》一书的作者罗里·梅卡尔夫（Rory Medcalf）是澳洲国立大学国家安全学院院长，曾经担任外交官，派驻新德里和东京，也曾经担任过澳洲顶尖情报机构“国家评估办公室”的资深战略分析师，他对于印太地区的局势的分析有相当的权威性。在他看来，印太（Indo-Pacific）局势非常重要，是中美冲突的前线，也是全球战略竞争的新热点。目前在印太地区已经众兵云集，局势进入高度不稳定状态，很可能是第三次世界大战的引爆点。因此，作者认为看懂印太地区的局势很重要。他在书中主要介绍了中国在东海、南海、台海、中印边境的挑衅，美国在印太地区的布局，以及日本、印度、澳洲、印尼等国的动作。
  <br/>
  <br/>
  作者指出：中国的外交企图心主要表现在三方面：战狼式外交、一带一路、争取国际组织的领导权。我认为这个总结虽然并无新意，但简单中透出精准，同时也对于我们评估中共的扩张动态给出了重要的参考指标。换句话说，我们仔细分析这三方面的发展状况，就可以大致判断中国对外扩张的成败。另外，与外界主流意见相反，对于中国是否能与美国正面抗衡并试图成为全球第一强权这件事，作者持相当保留的态度。对于中国的军事实力，以及中国如果进攻台湾是否存在绝对的军事优势，作者也存有疑问。他认为，军队必须花费一半的经费用来维持军备，以中国已经开始走下坡路的经济发展，是否能一直维持庞大的军事优势令人质疑。更何况，这是一只没有战斗经验的军队。相比之下，美军作战经验十分丰富。
  <br/>
  <br/>
  作者在书中花主要篇幅介绍了“印太战略”，从概念的提出，到具体政策的落实过程指出：不仅是美国，其他国家对于印太战略的设计和推动作用也很大，尤其是日本的安倍晋三。印太战略的核心机制之一，就是美、日、澳、印四国的“四方安全对话”，这个机制的最早的倡议者就是安倍晋三。另一个国家就是澳洲，安倍虽然最早提出四国联盟，但有意识地提出“印太战略”一词的是澳洲，早在2013年就提出来了。在澳洲争论激烈，作者代表鹰派立场，也就是现在的澳洲政府的立场。他认为，中国本身与澳洲的价值存在差异，而且中国并未崛起到那么强大，甚至在某些领域已经出现衰退的征兆。澳洲要先确立海洋的自由连通，避免落入大陆强权的控制。支持他的，是曾经担任澳洲国防部副部长的狄布。他认为，澳洲必须与美国积极维持有利于民主与自由开放的印太战略平衡，对中国破坏现状的作为必须强力回应。他还公开主张台海有事而美国卷入的时候，澳洲必须也加入，不能置身事外。
  <br/>
  <br/>
  总之，这本书给我们详细介绍了印太战略的内容，以及印太战略同盟的几个主要国家的战略意图，极具参考价值。对这个议题有兴趣的朋友，不妨找来一读。
