 <p>
  实际上，日本的地缘条件和战略现状注定无法在东亚大冲突中独善其身。这一点，与韩国还不同。台海冲突，西方不宜卷入驻韩美军，否则朝鲜很可能趁机生事，而且韩国恐怕更不愿为台湾而赌上国运，最多敲敲边鼓。
  <br/>
  <br/>
  那么，日本有与中国对抗的战略吗？完全没有，一切都先看美国怎么做。可是美国如果无法确保，哪怕在军事冲突条件下日本也能全力配合，美国在东亚的军事干预能力也难为无米之炊。
  <br/>
  <br/>
  之所以有这个局面，是因为在从中美建交到习上台的时期，中国会和平解决台海问题的可行性，一直是美日普遍预期，并基于此构建了紧密的对华经贸利益。
  <br/>
  <br/>
  可惜，这种小确幸一戳即破。日本只是对台海问题稍有态度，中国民间就爆发出核平日本的叫嚣。这当然不是官方政策，也极其幼稚可笑，然而中国官方对这种声浪不说喜闻乐见，至少是颇为享受的。
  <br/>
  <br/>
  正是中国从上到下的一种疯狂，构成了巨大的风险。如同朝鲜一样，核武不能灭美，常规武力远不足胜韩，但在朝鲜面前，美国即使不焦头烂额也要算疲于应付了，而中国是一个在能力上放大百倍的朝鲜。
  <br/>
  <br/>
  这又从另一个角度对日本构成压力。如果美国在东亚认怂，或者因为不愿与中国决战而放弃保护台湾，美日同盟自然也形同虚设。这种可能性不管大小，仅仅因其后果的灾难性就足以令日本不安。
  <br/>
  <br/>
  更大的压力在于日本一现状。与中国长期渲染历史问题，指责日本有军国主义复活趋势和战略野心完全不同，日本应对地区冲突的准备是非常令人担忧的。
  <br/>
  <br/>
  <strong>
   日本人早已失去控制东亚的野心
  </strong>

  日本有一支装备精良、训练专业的事实上的军队，工业科技实力支撑下的军事潜力相当可观。可是这支力量从人力、士气和地位方面都捉襟见肘，日本政治体制和民间舆论也完全无意使之用于守卫本土以外的任务。修宪、入常、海外派兵等动作，代表着一种适度进取的政治企图，但实施起来阻力和折衷之大，足以显示日本社会的主流心态仍是“岁月静好，巧妙生存，少管闲事”。即使在一度如日中天的经济领域，日本也早就没有了执世界之牛耳的宏图大略。
  <br/>
  <br/>
  在这种国情下，日本自卫队一方面在战略上居于对美彻底依赖和被动跟进姿态，一方面最远的实际军事准备止于离岛被占后予以夺回，或者跟随美国舰队提供后勤，最多执行护航服务，更多的战争假设都还只是动漫题材。事实上，不管是台海冲突还是钓岛有事，日本这样的对策不过是听天由命。
  <br/>
  <br/>
  反观中国，虽然实力上一直不无问题，但在战略想定和准备上的确魄力惊人。台海危机以来，中国部署在东南沿海的庞大常规弹道导弹力量，除了战时瘫痪台军要害，将驻日本冲绳的美军基地化为火海，也是题中应有之义。你赌中国不敢吗？那美军为何将驻冲绳的海军陆战队多数，调整到更远的二线基地了？别忘了，冷战高潮也是苏联以质量上远不如美国的军力令美国疲于奔命。
  <br/>
  <br/>
  这也是一个可笑现象的结果：美国对中国总是料敌从宽，而中国对美国却长期料敌从严。这从战争准备上当然是美国严谨而中国无知，但从政治和心理上却有与朝鲜一样的效果。众所周知更深层的原因是，美国的社会制度决定了死不起人，而中朝炮灰无穷，一切都可以牺牲。“狭路相逢勇者胜”不过是文学语言，真正的江湖上是“狭路相逢蛮/贱者胜”。
  <br/>
  <br/>
  日本与中国比起来，最要命的也是这一点。和平主义深入骨髓，凡事讲规则、讲道理、讲人性，还总想和气生财，这与动不动“西安以东都可以不要”能对抗吗？你可以说中国不过是唬人，但可以视人命如草芥，不能不说是一种战略优势。日本若出战，只要稍微死几个人，国内必然沸反盈天。
  <br/>
  <br/>
  不管中国真能做到多少，也或许日本没那么怂，但现在是中国把危局和难题制造出来了，而哪怕是拜登也才刚刚开始严肃对待，能否力挽狂澜尚属未定，以日本在台海以及整个东亚大局中的关键重要性，很多问题还想都没想过，这才是一个问题。或许正是因为如此，中国对日本还暂时没有发难，但这绝不是日本的机会，只不过是看扁了日本而已。
