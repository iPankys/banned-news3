 <p>
  各位读者朋友：
  <br/>
  <br/>
  五月份，“对话中国”智库推出了一个“自治领”项目，试图先从虚拟空间开始，尝试在海外建立宪政试验区。有一些网友不是很了解我们这么做的动机是什么？为什么我们要推动这样的一个计划？我现在简单介绍一下我们的想法。
  <br/>
  <br/>
  从历史上看，公元前221年，秦王攻灭六国并以帝制取代三代以来的分封制，从此，中国这片土地上便进入了长达两千多年的中央集权君主专制时期，直到今天仍未彻底结束。漫长的集权专制历史，使得中国社会长期缺乏现代文明国家应有的生气和活力，以至戾气横生，物慾横流。虽然近代中国曾有过宪政民主的昙花一现，但无奈专制思想的根深蒂固，外加时运不济，导致一切民主革命的努力均以失败而告终：文化大革命、六四屠杀、“一国两制”谎言、少数民族暴力同化等等，集权专制的阴霾历历在目。
  <br/>
  <br/>
  在21世纪都已过去20多年的今天，人类政治文明早已进入了以宪政民主、法治为绝对主流价值观的时代。自二战结束之后，世界上绝大多数国家和地区已承平近80年，其政治、科技、经济、文化等各项文明日新月异。而中国仅在政治文明这一项指标上的远远落后，就导致了绝大多数华人一直未能跻身主流文明国家的国民之列。所以，华人是要继续苟且抱残守缺，还是要另谋出路自图未来？答案如果是后者，那又应该如何去探索和实践？这，或许就是本自治领项目诞生的原因和意义所在。
  <br/>
  <br/>
  我们号召创建这样一个自治领，意在发出这样一种声音：借助自由民主世界的广阔天地，依靠有志华人的戮力同心，一起证明华人也可以自主实现并享有自由民主社会，而不只是一厢情愿地期待中国爆发所谓的鼎革变局。所以，创建本自治领的目标有二：一是对华人自主开拓自由民主之路的再探索和再试验；二是对从专制垮台到民主建立的中国未来新秩序的立宪预备和民主演练。
  <br/>
  <br/>
  总之，我们这个自治领项目的愿景就是：接续二十世纪前辈们的未竟事业，号召二十一世纪的华人一起努力，改变中国专制轮迴的路径依赖。在造福我们子孙后代的同时，也为人类文明的进步尽一份绵薄之力。
  <br/>
  <br/>
  最后也要介绍一下，我们这个自治领项目准备分为三个阶段来完成：
  <br/>
  <br/>
  第一个阶段为“创建者招募阶段”，由自治领规划小组负责起草项目介绍、招募要求、议事规则，以招募合格的创建者来进行本项目的创建。本阶段在线上进行。
  <br/>
  <br/>
  第二个阶段为“创建阶段”，由创建者在自治领监督委员会的监督之下，按照议事规则，召开包括制宪会议在内的各项议题会议。监督委员会最初由自治领规划小组担任，之后从创建者中定期选举出来轮流担任。本阶段在线上进行。
  <br/>
  <br/>
  第三个阶段为“运行阶段”，当各项议题的会议结束并制定出包括自治领宪章在内的各项法规等规章制度之时，海外华人自治领便正式成立。前两个阶段中的所有项目规则和成员身份将自动失效，除非新的自治领功能部门另有安排。所有创建者将以自治领公民的身份，与随后新加入的公民一起，正式参与自治领的运行。本阶段又分为两个时期，前期为线上运行，后期为线上、线下联合运行，即本章说明中所说的两种“载体”。
  <br/>
  <br/>
  我们欢迎各界朋友参加我们的尝试，共同为推动中国的宪政民主努力。
