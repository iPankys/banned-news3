 <p>
  2019年末新冠疫情在武汉爆发后，因中国官方禁言，延误了疫情的控制。武汉采取极端措施封城之后，国内各大城市的交通都进行了严格控制，但中共当局却放任国际航班满载运行，致使病毒迅速在全球蔓延。世卫组织却和中共唱一个调子，为中共的言行背书点赞。许多国家尤其是美国，对中共的隐瞒和拖延以及世卫组织的袒护进行强烈谴责，甚至切断给世卫组织的经济援助并退出世卫组织；全世界超过100个国家要求世卫组织对疫情源头进行独立调查。但中共不断用各种借口拒绝。直到疫情爆发一年后，中共才勉强同意世卫组织专家组进入武汉。
  <br/>
  <br/>
  世卫专家组预期在武汉停留两周，准备访问实验室、野生动物市场和有关医院。世卫专家组成员彼得·达扎克在推特上发文说：第一次进行现场访问极为重要。但如今的武汉，华南海鲜市场早已整顿得面目全非；P4实验室的设备、材料、数据该销毁的已销毁，该隐藏的已隐藏；到访医院都做好了万全准备。专家们能调查到什么呢？专家们对此也心知肚明，所以世卫组织紧急事务负责人迈克·瑞安表示：并不能保证调查能够得到答案。世卫组织不想让外界对这次调查有过高的期望。中共官方的说辞更是不加掩饰，明确说：世卫组织的这次行动不是调查，而是和中国的专家一起就新冠病毒溯源问题进行的一次交流合作，是全球研究的一部分。
  <br/>
  <br/>
  鉴于在过去一年多时间里，中共不断编撰奇葩的故事，向许多国家甩锅，声称中国的病毒来自美国、意大利、西班牙等国家，也可能来自中国进口的冷冻食品，因此中国也是新冠病毒的受害者。可以预期，世卫组织只是把这次调查当作一次走马观花的例行公事，向世界各国交差。从而可以继续向全世界化缘，继续维持其独尊的话语权。而中共则很希望把这次调查变成交流，并引导出武汉不是病毒源头的论断。很可能，专家组的报告要么是天马行空泛泛而谈，要么是隐隐约约为中共甩锅背书。是否如此，两周后就见分晓。
  <br/>
  <br/>
  新冠病毒究竟是自然生成还是人工合成的？科学迟早会得出结论。但有一点可以肯定：公开的信息显示，P4实验室确实在研究冠状病毒。因此病毒无论是自然生成的还是人工合成的，依据常识就可以判断：病毒突然现身武汉决不是偶然的，P4实验室泄漏的可能性是最大的。
  <br/>
  <br/>
  这是许多人的直觉和常识判断，没有证据！但证据迟早会被披露，事实终究会大白天下！
  <br/>
  <br/>
  但确实不能对这次被转化为交流的调查，抱有不切实际的期望！
