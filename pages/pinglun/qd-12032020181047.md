 <p>
  各位听众你们好，我是王丹。趁着美国权力交接，无暇顾及太多国际事务的机会，中共在前一段时间外交受挫暂时低调之后，展开了新的一轮外交攻势，试图恢复在国际社会的影响力。这样的攻势表现在几个方面：一是推动签署成立了区域性自由贸易区RCEP，并表态考虑加入另一个国际贸易协定CPTTP，积极介入国际组织；二是派出老牌“日本通”，外交部长王毅访问日本，试图说服新任首相菅义伟拉近与中国的关系；三是通过调高关税等经贸措施和挑衅性质的外交语言，下大力气打击澳大利亚，试图杀鸡儆猴，威慑其它美国的盟友；四是在香港问题上动作频繁，重判黃之锋等三名学生领袖，收押媒体老板黎智英，反击西方国家对中国制定《国安法》进行的制裁。这一系列外交攻势，有组织有步骤，显然是策划好的一盘大棋，其目的，很可能是在拜登新政府上台之前，做成某些既成事实，看2月份以后的美国如何反应，以此测试下一届美国政府的对华政策的底线在哪里。
  <br/>
  <br/>
  全球疫情爆发之后，中国在外交上处于被动；现在疫情的结束，随着疫苗的开发已经露出曙光，美国政治发生重大变化，中共见猎心喜，蠢蠢欲动，其实并不意外。但是，这样的新一轮外交攻势是否能够收到成效？目前来看，答案显然是否定的。而且，很有可能不仅没有成效，还会适得其反，自己给自己挖坑找麻烦。这从国际社会的反应就可以看得出来。
  <br/>
  <br/>
  举例来说，王毅曾经担任驻日本大使，据说在日本政界曾经颇有人缘。这次访问日本，原本打算利用他的人脉获得外交成果。但日本政府的态度不能说不礼貌，但也并没有送什么外交大礼给王毅。王毅访日的主要议题之一，应当是一再耽搁的习近平对日本的访问，但这一次日本对这个话题避而不谈，让王毅碰了个软钉子。更有甚者，针对王毅提出的处理钓鱼岛争议的新建议，菅义伟断然拒绝，还反过来要求中国方面对中国海警船在钓鱼岛附近海域的活动加以约束，在这个议题上双方可以说是不欢而散。王毅拉拢日本的尝试，可以说无功而返。
  <br/>
  <br/>
  另一方面，中共对澳大利亚进行外交打击，引发了国际社会的反弹。英国国际事务评论人士、泰晤士报专栏作家爱德华·卢卡斯(Edward Lucas)本月早些时候在推特上发起了“买澳大利亚葡萄酒，抵抗专制”的“公民运动”，得到来自世界各地的推友的热烈响应，并获得跨国议员联盟的支持。澳大利亚也并未因此屈服，澳大利亚参议院外交、国防、贸易议事委员会主席、来自工党的金伯丽·克琪英对媒体表示，澳大利亚作为一个主权独立国家，绝不能允许外国政府以为他们可以任意改变“我们所秉持的价值和理念”，并指出，这一点上，工党和目前执政的保守派自由党是完全一致的，议会当中其他一些小党和独立党派也无一例外。
  <br/>
  <br/>
  《金融时报》发表的一篇社论说，中方在处理澳大利亚问题上，开了一个很让人不安的先例。全球民主国家都应当密切关注这一局势的动态，并要做好准备，共同反制来自中方的压力。这篇社论还呼吁，在中国政府如此咄咄逼人的态势下，各个民主国家要么团结起来，共同行动捍卫彼此，要么将会被各个击破。这样的声音相当有代表性，也绝不仅仅针对中国打击澳大利亚这一个单一事件而发，它说明，中共要想重新在外交上恢复强势地位，恐怕只能引起西方国家的反感和警惕，到头来变成作茧自缚。
