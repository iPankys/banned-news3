 <p>
  【简述相关介绍：2020年四月间，国际文学杂志《渐近线》（ASYMPTOTE）网站发表了捷克学者、翻译家Kamila Hladíková对我的访谈。她也是我的散文集《西藏笔记》译者，于2015年在捷克出版。而这个实际上我用了一个多月完成的访谈，原本关涉的话题更多也更深入，译成英文发表时因限于篇幅做了较多删减，为此我将中文原文在此连载。】
  <br/>
  <br/>
  卡米拉：你曾经在与美国诗人/艺术家Ian Boyden的对话里，你们谈到了“空”的象征性。据我的理解，“空”也可以代表现实和历史被审查的部分。我在拉萨的时候，我感受到“被空的”那些人物和事件好像一直在沉默地喊叫，生活在那里的人们能听见吗？
  <br/>
  <br/>
  唯色：“空”所象征的事物很多，大到一段又一段历史，小到佛殿里的壁画上一个人物画像，比如尊者达赖喇嘛的画像。“空”又像一个个空白，确如你说，代表的是现实和历史被审查的部分。不只是被审查，很多都被消失，被抹除，被缺席，变成了一种不在场。而“不空”则是用事实填补空白之后的真相，那种努力的动作是一种拒绝、一种抵抗和一种不合作，反而变成了在场，永远在场。
  <br/>
  <br/>
  我作为一个写作“看不见”的人，也属“被空的”人之一。我与许多“被空的”族人，一直在以自己的方式抗拒各种人为的，或者说来自他者的“空”。是的，我们各有各的方式。比如我2018年在拉萨期间写的一首诗：
  <br/>
  <br/>
  反抗或抵抗有多种，
  <br/>
  不需要大声地宣布，
  <br/>
  也不必慨然殉身，
  <br/>
  我不是斗士，而是诗人，
  <br/>
  自有属本土的个人记号。
  <br/>
  <br/>
  比如在至尊依怙主的寿诞日，
  <br/>
  去祖拉康面向觉沃佛诵念长寿祈请文，
  <br/>
  去罗布林卡面向独一无二的金法座磕长头，
  <br/>
  眼前幻现沦丧之前的民意多么决绝而忠诚，
  <br/>
  恰如相遇怀抱鲜花、手捧哈达的族人，
  <br/>
  无需表白，相互微微颔首足以会意，
  <br/>
  而这一切全被走卒看在眼里。
  <br/>
  <br/>
  任由盯梢多久就盯梢多久吧，
  <br/>
  包括昧了良心的各种技术，
  <br/>
  才能在至暗时刻，细细地领略到
  <br/>
  那静静下雨的，仿佛往昔的，
  <br/>
  拉萨好时光。
  <br/>
  <br/>
  我还有一首诗是在尊者达赖喇嘛83寿诞日写的，涉及到了你想知道的是否“还能听见”这个问题：
  <br/>
  <br/>
  等待的方式有很多，
  <br/>
  一种是把您画在佛殿外的墙上，
  <br/>
  哪怕被干部认出、报告，
  <br/>
  画上胡子，把您变成十三世的样子，
  <br/>
  但十三世也是您，
  <br/>
  您是一世至十四世，
  <br/>
  您是之前之后的每一世。
  <br/>
  <br/>
  等待的方式有很多，
  <br/>
  一种是守住幸存的每一座佛殿，
  <br/>
  在空空荡荡的遗迹上，
  <br/>
  堆积从山脚下背回的泥土和石块，
  <br/>
  重又盖起跟往日一样的僧舍、厨房，
  <br/>
  坚信有一天您会重返故土，
  <br/>
  随您回来的众僧将住满往昔的康参。
  <br/>
  <br/>
  “我们一直在等、等、等，
  <br/>
  很多人在等待中去往了轮回的长路……
  <br/>
  我们的依怙主，原本有自己的宫殿、寺院，
  <br/>
  有自己的人民、土地，这里的一切属于他，
  <br/>
  每个人的今生和来世都属于他。”
  <br/>
  甜茶馆里，与您同龄的老人握着我的手，
  <br/>
  用敬语低声说着，眼里全是泪。
  <br/>
  <br/>
  “衮顿，拉萨见啊！”
  <br/>
  是那个冬天，那个独自去往菩提伽耶
  <br/>
  领受时轮金刚灌顶的拉萨青年，
  <br/>
  朝着缓步走来的绛红色老人
  <br/>
  合十高喊，热泪奔涌。
  <br/>
  <br/>
  还有一位安多青年，
  <br/>
  将去西方的学府读博士，
  <br/>
  在手臂上刺了几个藏文的数字，
  <br/>
  那是尊者您提及的这一世的寿命之数。
  <br/>
  <br/>
  （补充诗里的两个藏语词汇的解释：“康参”是藏传佛教寺院中依僧徒来源地区划分而成的僧团单位。“衮顿”是敬语，意为虔心呼喊即出现眼前，简译尊前或殿下，这里是对达赖喇嘛的尊称。）
  <br/>
  <br/>
  其实是能够“听见”的。生活在空缺之地或无主之地的人们，只要出于“信”，即能发出无声之声，也能听见“被空的”人物和事件的故事。只要虔信，只要信仰，只要信念不变。
 </p>
