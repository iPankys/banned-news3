 <p>
  今年九月份以来，中国再次发生大面积而持续的限电、断电、停电，这是近几年来最严重的一波中国电荒，出现在倒行逆施、轮番扫荡各个行业的习近平时代，毫不奇怪。令人疑惑的却是，大瘟疫还未完全过去，中国经济并不兴旺，各行各业均不景气、开工不足，按理说，同前些年相比，当下中国的用电量不至于如此急需和短缺。大停电现象太诡异！
  <br/>
  <br/>
  那么，究竟是什么原因导致这波严重电荒？国内外说法不一，版本众多。
  <br/>
  <br/>
  说法之一，也是最通常的说法，澳煤说。因澳大利亚去年带头提出，要求国际社会调查大瘟疫来源，中共恼羞成怒，对澳大利亚搞经济制裁，以为以大欺小、恃强凌弱，经济战必胜。却不料，中国反受其害。其中就涉及澳大利亚煤炭进口受限和铁矿石涨价。澳大利亚煤炭占中国进口市场的28%，仅次于印尼，居第二位，而澳大利亚煤炭优质、发电量高，更是不可取代的优势。但习近平充硬气，阻止或减少进口澳大利亚煤炭，直接导致煤炭荒，进而导致电荒。因中国70%的电力仍然依靠火力发电，即烧煤发电。中澳经济战，习政权意气用事，杀敌三千、自损一万。
  <br/>
  <br/>
  其他说法：
  <br/>
  <br/>
  环保节能说。中国废气排放占全球28%，是美国的两倍，是西方工业国的总和。为了履行巴黎气候协定的义务，必须节能减排。但中共从来就是厚黑集团，为了统治利益，从来无视国际规则，怎可能忽然认真履行所谓国际义务？除非太阳从西边升起。
  <br/>
  <br/>
  涨价铺垫说。中国拉闸限电，是为供电涨价造势、做铺垫，先让企业和民间感受到缺电之苦之后，在有电和无电的选项之间，不得不接受供电高价。
  <br/>
  <br/>
  电力政变说。类似于2015年的金融政变，如果说当年党内政敌用策动上海股市暴跌打击习政权，今年的电力政变，则是党内政敌策动全国性电荒打击习政权。
  <br/>
  <br/>
  中美竞争说。习近平下令限电，是为了损害美国，因为减少中国商品的生产和出口，可以导致美国物价上涨、通货膨胀、进而酿成美国经济危机。鉴于又是一个杀敌三千、自损一万的思路，网民总结：习近平经济学，就是自杀式经济学。
  <br/>
  <br/>
  备战演习说。习当局大搞限电、断电、停电，尤其在东部各省市，是一种备战演习。一旦台海开战、南海开战或中美开战，可能出现断电和停电。为此，须让中国民众提前习惯，以经受未来战争的残酷洗礼。
  <br/>
  <br/>
  新式武器测试说。多国传言：习近平当局正在测试一种陆基激光反卫星武器系统，据称这个系统是中共唯一，可以在六小时内，摧毁中国周遭的所有卫星。一旦开启战争，中共将在这一领域占据优势，即，让其他国家卫星侦查失灵。据称，测试和使用这一庞大系统，会大量耗电，故而牺牲工商业和民用电，以优先保障共军测试和运作这类新式武器系统。
  <br/>
  <br/>
  N种说法，众说纷纭。不过，这一回，东北民众有一个质疑，颇能引起注意。他们抱怨：位居东南沿海的福建省，是耗电耗能大省，为什么没有限电？反而是东北三省限电？成为大停电的最大牺牲品。而且，东北地区遭无预警、无通知限电、断电
  <br/>
  、停电，当局手段极其粗暴，被东北民众怒斥为暴力断电。
  <br/>
  <br/>
  这表明，习近平当局力保福建用电，因为那是对台开战的前线。故而，备战说或新式武器测试说，似乎更能解释中国持续而大面积限电、断电、停电的诡异现象。
