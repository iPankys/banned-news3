 <p>
  与习近平自小玩闹在一起的王岐山，近来一系列对习近平拍马溜须的举措因其反常，而引起媒体和网络上的热议。王岐山近来在中共的重要会议上几乎是言必提习近平表忠心，更为离谱的是，在讨论国际经济的博鳌论坛上，否认自己是会议介绍的主持人而称自己是习近平的报幕员，习近平才是博鳌论坛的主持人，只有如此才能凸显对习近平的高度尊重。
  <br/>
  <br/>
  王岐山与习近平的关系非同一般，他不仅可谓习近平的发小，实际上还曾是滚一个被窝的亦师亦友。而且王岐山在习近平掌权中，如果独裁可以论功行赏，他无疑是助习近平抢权的第一功臣。正是靠了王岐山所谓的大肆反腐，为习近平清除了派系林立的非习派大量官员，为习近平的习家军腾出了大量关键职位，并且震慑住无官不贪腐淫乱的共党官场。
  <br/>
  <br/>
  王岐山之所以对习近平如此奴颜卑膝，与他一些最亲近的人遭受清洗打压密不可分。众所周知，任志强与王岐山关系密切，但是因为公开批评习近平专制横蛮，先遭审查处分后被逮捕重判。而王岐山执掌中共纪委大权时的大秘、被称为王岐山大管家的董宏，也被以“涉嫌严重违纪违法”陷入险境。王岐山曾经重用的旧部蒋良超，此前也遭没有讲述原因的免职。
  <br/>
  <br/>
  根据中共官场的潜规则，这种状况无疑表示王岐山处境不妙。任志强最初批判嘲讽习近平的党媒姓党时，仅受到没有伤及根本的党纪处理。社会有议论说，是因王岐山的关系使任志强这位学弟兼友人得以全身而退。而任志强大批习近平是光屁股争皇位后，习近平直言不许任何人为任志强求情，似乎也是表达了对之前处理的不满和警告。这基本可以证实，王岐山保任志强并非空穴来风。
  <br/>
  <br/>
  而王岐山前后任大秘，在短时间内或锒铛入狱或身处险境，更是中共官场规则中不可小觑之事。中共高官只要跟对主子，不论如何作奸犯科皆稳如泰山。张春桥、陈永贵有实据是叛徒和汉奸，但在毛泽东那里皆不值一提。对于远超毛泽东拉帮结派的习近平而言，追究官员腐败淫乱只是清洗他派，用来培植亲信党徒的手段。以王岐山前后任大秘腐败为名惩治，在无官不贪腐淫乱的中共官场，这种指向的真实用意是没有谁感受不到的。
  <br/>
  <br/>
  王岐山面对如此凶险的官场局面，采取自我贬损、无耻吹捧独裁暴虐者以求自保，也是中共官场生存的惯常和有效手段。只要看看邓小平如何吹捧毛泽东和贬损自己，逃过重重灭顶之灾而熬到毛泽东死翘翘，终于熬为大权在握的太上皇，便能体味个中三味。而刘少奇虽然也被迫向毛违心认错，但是尚未贬损到令毛满意，终至死无葬身之地。浸淫中共官场权斗、练成金钟罩的王岐山，当下这种自我贬损和吹捧一尊也不过是中共历史中权臣的重演。
  <br/>
  <br/>
  王岐山对习近平对吹捧是否发自肺腑，是不能完全排除其真实程度的，毕竟斯德哥尔摩综合症确是一种社会现象。为了维持中共专制独裁的延续，不能排除真有信徒认为，维护定于一尊的独裁是专制政体的必要保障。只是让人生疑的是，以王岐山的经验和见识捧一个只识宽衣的血腥暴虐者，有多少是为了中共暴力政权得以维系？有多少是贪恋权势恐惧自保？怕是王岐山内心也分辨不清了。
  <br/>
  <br/>
  不过，鉴于王岐山对习近平的极端吹捧，其中的捧杀成份也是若隐若现，所以也难以排除这样的吹捧含有挖坑及效果适得其反的含意。当年赵紫阳告诉戈尔巴乔夫，中共真正的掌舵人其实是邓小平，就被邓认为是暗藏玄机的明捧，实则暴露中共秘密内情、甩锅邓小平。这次王岐山自贬只是习近平的报幕人，就有双刃剑里外皆伤的社会效应。固然，如此吹捧习近平有加固独裁的恐惧效应，但也有习近平疯狂到令友人和助他独裁的最大功臣也只有仰其鼻息、战战兢兢的活着之想。这无异会让不少人对习近平心生厌恶，而且也暴露了中共决策层的内斗躁动之危境。
