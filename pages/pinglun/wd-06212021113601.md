 <p>
  七月一日是中国共产党建党一百周年，中共方面很早就开始借机进行自我歌颂，不断强调中共的“伟大光荣正确”。这样的自我吹嘘当然已经没有市场，恐怕中宣部的写作班子的成员，自己都不相信自己写下的每一个字。但是另一方面，中共成立百年，对中国社会这一百年来的发展产生过而且依然正在产生巨大影响。我们当然不同意“没有共产党就没有新中国”这样的说法，但是，没有共产党，今天的中国肯定是另一番景象，这倒是可以肯定的。对于这样一个左右着中国发展命运的百年老党，在它成立百年之际，对它做一些分析和认识也是必要的。我有一些简单的想法，愿意分几次与朋友们分享。
  <br/>
  <br/>
  其一，关于暴力性。我们必须认识到，中国共产党从其建立的第一天起一直到现在，都具备一个鲜明的特质，那就是“暴力性”。这是一个崇拜暴力、迷信暴力，以暴力为号召而成立，靠暴力扩展政治力量，用暴力夺取政权，凭借暴力维持和巩固统治的政党。这个党，因为一切依靠暴力，所以天生就不具备自然改良为现代性政党的条件。我们也不能幻想它会愿意用现代文明的方式，解决社会矛盾。即使在它已经执政四十年之后的一九八九年，当人民对它提出改革要求的时候，它还是采取了暴力手段。这就证明，暴力已经成了这个党解决问题的习惯性做法。
  <br/>
  <br/>
  更重要的是，暴力已经不仅仅是行为，暴力行为深刻地影响了中共的思维模式，使得中共的政治路线也具备暴力特点。这样的路线已经渗透进这个党的灵魂中，成为中共各种思想和政策的最深层次的基础。暴力对中共的影响从其婴儿时期就已经开始，一百年来不断强化和加深，可以说，中共已经对暴力产生了路径依赖，不可能摆脱其影响了。认识到这一点非常重要，因为只有认清中共的暴力性，我们才能丢弃那些错误的幻想，正视结束中共一党专政统治的艰巨性和必将付出的代价。
  <br/>
  <br/>
  其二，关于合法性。中共成立到现在已经一百年，在中国执政也已经七十二年了，我们似乎已经习惯了中共作为执政党的合法性。但是其实我们应当认识到，中共在中国行使统治权力是没有任何程序上的合法性的。中共建政使用的是战争的手段，打败国民党之后，中共自行宣布建立了中华人民共和国。这个过程，从来没有经过选举这个建立合法性的最低的标准。
  <br/>
  <br/>
  尽管世界各国承认了中华人民共和国，但一个政权的合法性从来不仅仅是建立在其他国家的承认的基础上的，更需要的是本国人民的承认。在这方面，中共宣称自己得到了全国人民的拥护，但拿不出任何具体的证据。七十二年以来，中共从来没有通过一个合法的方式证明自己的统治的合法性。中共号称“为人民服务”，但中国人民从来没有过一次机会用投票的方式，自由地表达自己是否愿意接受这样的“服务”。更何况，作为一个具备各级组织机构的政党，中共甚至都没有到民政部进行政党登记。因此，严格地说，中共本身就是一个非法组织。一个非法组织的统治，当然没有合法性而言。
  <br/>
  <br/>
  习近平上台以后的中共不断地强调各种各样的自信，如果他们真的有自信，就应当开放选举，让选举结果建立他们统治的合法性，证明他们确实得到人民的拥护。不管他们嘴上多么自信，对于自己的合法性却始终不敢通过正规的方式确认，这说明，他们自己对自己的合法性都没有信心，那么我们凭什么承认它的统治的合法性呢？这是我要讲的第二个问题。
  <br/>
  <br/>
  其他问题，我下周继续跟大家分享。
