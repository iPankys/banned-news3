<div id="storytext">
 <p>
  第245个美国独立日刚刚过去，可是由美国独立日引起的思考和给我们的启发，却永远不应该也不会过去。美国的立国之本——普世价值、民主自由、尊重人权、宪政法治、权力制衡等等基本价值，将永远激励、规范著人类奋勇向前的发展方向。
  <br/>
  <br/>
  仅仅建立240多年的美国，不仅创造出前所未有、领先世界的高科技，比这更为重要的是还创造出“把权力关进笼子”的“三权分立”权力制衡的社会制度，由此人类社会走出了“家天下”王朝更替、人民遭殃的独裁专制的恶性循环，开历史先河地创造出了公权力只能经由人民投票授权，才具有合法性和正当性的服务性政治机制。
  <br/>
  <br/>
  美国国父先贤们根据人性的弱点，设计出两条社会规则——
  <br/>
  <br/>
  一条是在世俗间，通过宪政法治把多种不同的社会观点逐渐形成以致划分出两大类，再通过持有水火不容观点的两党（或两党以上多党）人群把监督制衡发挥到极致，从而达到对人性中自私贪婪、权力滥用等等的弱点进行最有效的监督制衡，最大限度地避免了权力的私有。
  <br/>
  <br/>
  另一条是在形而上，也就是人的精神层面，通过捍卫信仰自由使人们保持对浩天上帝的敬畏之心，使人认识到：你、我相对于浩瀚的宇宙都不过是沧海一粟，认识有限，更不要奢想谁是完人，带有原罪的人没有什么值得骄傲的。宗教信仰能使世人知道，做任何事情可以瞒得过人，但是无论如何也瞒不过神，从而最有效地维护了社会道德，持续提醒人们做事要有原则底线，很大程度上避免或者减缓了人类走向堕落。因为这样厚实的立国根基，使得美国伟大如此。
  <br/>
  <br/>
  相形之下，再看看在中共绑架挟持之下的中国沦陷区，无论是党庆还是所谓的国庆，都是由共产党组织……，甚至是命令基层组织进行庆祝。尤其是这次的中共百年庆生，简直是全党作秀、劳民伤财。有人说，和慈禧太后过生日有得一比。
 </p>
 <p>
  尽管为了引导舆论，中共喉舌开足马力进行宣传，也布置了大量的网军帮兵助阵，可是纵观全国，发自内心相信党的宣传而“爱党与党国”者又有几人？这一点，从中国沦陷区没见一家人在自己家里悬挂国旗就足以证明。而在美国，每年的独立日庆祝活动绝大多数普通百姓都会自发进行庆祝：邀请朋友一起聚餐；住家房子的位置近水楼台便于观赏烟花者，会邀请朋友前来一起看烟火。政府会顺应民意，在广场或者河面的船上组织大型燃放烟花的活动，各种民间组织也会扮演重要角色，参与筹备和现场服务……；观看烟花的民众都是怀着感激和欢乐的心情自发前往参与，热闹非凡，真正作到了黎民百姓举国同庆。大概这是美国每年唯一一次，政府会有限度地花纳税人的一些钱与民同庆的公开活动。因为这是独立日，是人民获得自由的开始，是真正的国庆节。
  <br/>
  <br/>
  反观中共国，中共为了庆祝它成功附体中国百年，庆祝它颠覆中华民国政府成功抢夺到政权72年，敞开国库大肆挥霍民脂民膏，把国库当作党库肆意挥霍。甚至直接拿国库的钱给中共老党员发放每人五千元的补贴，这是明目张胆地利用执政之便公款私用，无异于监守自盗。要补贴你的老党员只能用你们的党费才对，凭什么要人民买单？！别忘了，中国沦陷区月收入不足一千人民币的超过六亿人，不足两千人民的超过九亿人，你中共有想过给他们发钱补贴家用，改善他们的生活条件吗？！
  <br/>
  <br/>
  其实，中国的社会贫富差距越来越大都是中共的“二元社会制度”分配不公所致。社会配置的错位，导致了社会的机会不均等和资源的浪费。
  <br/>
  <br/>
  随着网络的发展、人民视野的开阔，对比之下，不同的制度对社会发展的作用一目了然。面对人民的质问，中共只好采用“宣传时间差”手法：每当谈及“中国发展不如美国”时，中共就会说 “美国已经成立240多年了，中国才70年”——难道七十年以前，中国不存在吗？而每当需要说“中国优于美国”时，中共又会说，“中国历史上下五千年，美国才二百多年而已”——这不只是双重标准，显而易见就是胡搅蛮缠。
  <br/>
  <br/>
  可是为什么，最近人类二百多年却创造出了超越人类几千年的成果？一说到这背后的制度性原因，中共就会通过审查不让人民继续探讨下去了。因为进一步做探讨，人们当然就会提出“从1940年代到1970年代，全世界都在发展科技，你中共为何在搞阶级斗争”的问题来。这样，中共政权阻碍社会生产力发展的历史铁证就摆在人民面前了。无论是从经济上还是政治上，中共都是阻碍中国发展进步、给中国人民带来无穷灾难的罪魁祸首。这是中共最不想让人民知道的。
  <br/>
  <br/>
  古人云，“此心安处是吾乡”；现代人说，“哪里有自由哪里就是祖国”。通过对美国国庆和中共党庆的对比，比出的是人民的生活质量。中共高压下的“爱党爱国”是虚假可悲的；生活在自由国度美国的人民很幸运，人民真心支持的美国是幸运的美国。
 </p>
