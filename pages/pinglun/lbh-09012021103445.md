<div id="storytext">
 <p>
  8月29日，中共中央与地方党的媒体都转载了李光满的一篇文章《每个人都能感受到，一场深刻的变革正在进行！》的确，我感受到目前的情况就如1965年11月，中国上下媒体转载姚文元“评新编历史剧《海瑞罢官》”的氛围。那时吹响文革的号角，如今的中国正在发生文革2.0。
  <br/>
  <br/>
  当时，中共吹捧文革是触及每个人灵魂的大革命；现在，“深刻的变革”也是如此。灵魂就是精神，就是思想意识，这是每个正常人所具备的。即使大多数人对政治缺乏兴趣，但是共产党的政治还是要触及到每个人，透过文化娱乐、透过学校与社会的教育。
  <br/>
  <br/>
  1963年12月12日，毛泽东对文艺工作批示说：“许多共产党人热心提倡封建主义和资本主义的艺术，却不热心社会主义的艺术，岂非咄咄怪事。”以此，揪出“文艺黑线”。文革爆发前夕的1966年5月7日，毛泽东又发表对教育工作的“五七指示”说：“学制要缩短，教育要革命，资产阶级知识分子统治我们学校的现象再也不能继续下去了。”由此，又去揪了“教育黑线”。
  <br/>
  <br/>
  文革触及每个人灵魂的结果，是大家看透毛泽东的本质。毛泽东最后众叛亲离，在孤独中翘辫子。习近平又会如何？
  <br/>
  <br/>
  目前习近平在大整艺人，点名批判；在“改革”教育，打击补教，修改课本，压缩英文。这也是在文化艺术与教育领域发动“革命”，与毛泽东如出一辙。这是精神领域里的问题，但是这次习近平还加上抢钱，涉及到物质领域。整肃文化艺术与教育领域已经无人可以回避，然而再涉及经济领域，其影响就更加广泛与深刻，被触及的灵魂大大超过当年文化大革命。
  <br/>
  <br/>
  “改革开放”已经40多年了，私有财产合法存在并且受到宪法保护，全党全民已经向钱看，习近平什么能耐能够推翻这一切？所谓“共同富裕”的三次分配，不但强迫捐献，还试点要严查十万元以上存款的存户，习近平要得罪的是几乎全部小康之家；要进行四次土改，革除四十年前的承包制，又会触犯多少农民？教育双减，引发多少望子成龙学生家长的不满？游戏机限时上网，又触怒多少青少年？毛泽东的“一小撮”加起来是一大撮，习近平的“一大撮”加起来难道不会翻天覆地？
  <br/>
  <br/>
  钱财已经是中国每个人的命根子，被剥夺还不会拼命？教育双减却增加学“习”，就是学习近平思想，哪一个对学生更有用？习近平以为民众同他一样愚蠢吗？
  <br/>
  <br/>
  现在的中国已经不是当年毛泽东的闭关锁国，可以推行自给自足的计划经济；习近平不管要单循环还是双循环，已经不是由他个人可以决定，也不是中国单独可以决定，尤其涉及习近平一窍不通的金融。中国的经济成长正在萎缩，股市暴跌，资金外逃，香港的金融市场正在被掏空。习近平刮起新的共产风，不是中国再来个大饥荒，就是他被推翻。
  <br/>
  <br/>
  以上是自由亚洲电台特约评论员林保华所做的评论
