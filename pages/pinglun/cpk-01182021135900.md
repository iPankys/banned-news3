 <p>
  美国政府换届前夕，中共总书记习近平写信给美国星巴克前董事长舒尔茨（Howard Schultz），一则诱惑道：中方“将为包括星巴克等美国企业在内的，世界各国企业在华发展提供更加广阔的空间”；二则请求道：“希望星巴克公司为推动中美经贸合作和两国关系发展继续发挥积极作用。”
  <br/>
  <br/>
  此前，舒尔茨曾给习近平寄去他在中国出版的新书《再出发：重新想象美国未来》。收到习近平的信件后，舒尔茨声称：“我坚信，星巴克在中国最好的日子就要到来了！”
  <br/>
  <br/>
  舒尔茨曾任星巴克首席执行官和董事长，现已退休，任荣誉董事长。星巴克于1999年开始在中国落脚，现已在中国175个城市开设分店近5000家。星巴克在中国的成功，就如它在其他国家的成功一样，不足为奇。但星巴克在中国的成功故事，只是美国企业在中国的故事之一，更多美国企业经受的却是：市场准入限制，强制转移知识产权，甚至遭到大规模的抄袭、剽窃、盗版，产权损失巨大惊人。
  <br/>
  <br/>
  即便星巴克公司本身，也曾在中国有不愉快的经历。2007年，前中共央视主持人芮成钢以星巴克“侵蚀中国文化”为由，发表讨伐檄文：“请星巴克从故宫里出去”。随后，星巴克的故宫分店果然被中共官方驱离。那个私下崇洋媚外却假装公开捍卫中国传统文化的芮成钢，为此大大地出了一把风头。多年后，芮成钢却因卷入中共高层权力斗争而锒铛入狱，遭判刑六年。
  <br/>
  <br/>
  习近平赶在美国政府换届之际，发挥中式关系学，写信给舒尔茨自有其用意：利用中共与美国商界的关系，图谋重启中美关系，最好能让中美关系回到特朗普当政之前的旧格局，即期待美国民主党政府奉行绥靖主义，继续让中共占美国的便宜。
  <br/>
  <br/>
  去年，在中共智库举办的论坛上，舒尔茨曾表态： “俄罗斯是美国的敌人，但我从未将中国视为美国敌人，中国只是一个强烈的竞争对手。”这种认知与说法，都是上世纪冷战时代的典型美国人思维，习惯把苏联或俄罗斯当成最大敌人。在共产中国崛起而俄罗斯衰落的今天，这种认知就难免显得落后、过时，而与时代步伐和世界现实对不上号。
  <br/>
  <br/>
  习近平写信给美国富商，希望他们为中美关系改善而发挥影响力。但习近平有没有想过：如果美国总统写信给中国富商，比如马云（阿里巴巴创始人），希望他发挥影响力调整美中关系，习近平将作何感想？
  <br/>
  <br/>
  这恰恰反映了美中两国的本质区别：美国是多元化的民主大国，用多种声音和多种途径保持与世界的联系，灵活自如。而中国是一元化的专制大国，只有一种声音、一种途径，即官方的声音、官方的途径，与世界打交道生硬之至。
  <br/>
  <br/>
  舒尔茨声称，收到习近平的来信“倍感荣幸”，但舒尔茨有没有想过：幸好他是外国人，是美国企业家，如果他是中国人、中国企业家，习近平会给他写信并请托吗？中国商人的处境，比如马云当下“失联”的处境，不知舒尔茨作何感想？
  <br/>
  <br/>
  说起来，习近平对中外企业家的不同态度也算得上是崇洋媚外、挟洋自重。其实，在专制者习近平的潜意识里，中国或中国人，不过就是他的私产，想怎样处置就怎样处置。只是对他鞭长莫及的外国人，才不得已强作笑颜，图谋利用而已。习近平给舒尔茨写信落款1月6日，正值美国发生国会山事件。习近平政权的曲折用心，由此可见一斑。
