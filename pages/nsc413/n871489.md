 【大纪元3月30日讯】3月28日下午，西安火车站派出所查获了男子随身携带的两个尼龙编织袋中发现6具完整的女性尸骸。
 <br/>
 *身背6具尸骸乘车
 <br/>
 据西安晚报29日消息，3月28日下午3时许，西安火车站派出所执勤民警将一名迟迟不愿出站，欲躲避治安检查的中年男子带回派出所查问时，竟在他随身携带的两个尼龙编织袋中发现6具完整的女性尸骸。经查问此人是为了将尸骸变卖换钱，以成全配“阴婚”的陋习。
 <br/>
 29日上午10时许，西安铁路公安处车站派出所民警在站台上执勤时，发现由成都到达西安的2120次列车下来一名男子，身背两个鼓鼓囊囊的尼龙编织袋。令人奇怪的是，此人并没有走出站台，而是沿着铁轨往东走，民警以为男子找不到出站口，急忙叫住了他，询问他要去哪？并告诉他出站口的方向。可是这名中年男子仍不肯出站，并吱吱唔唔说不出原因。民警观察到，他的手一刻也没有离开编织袋的袋口。民警发现此人形迹可疑，就将他带回了车站派出所检查。
 <br/>
 在派出所里，此人自称叫杨锦玉，民警打开编织袋进行检查时看到6具完整的尸骸！
 <br/>
 *疯狂盗掘6具尸骸
 <br/>
 经审查此人名叫杨锦玉，今年43岁，是陕西省西乡县人。曾因拐卖人口被山西省汾阳中级人民法院判处17年有期徒刑，减刑6年后，于去年被释放。其后在山西省汾阳一家煤矿打工。
 <br/>
 2004年8月，杨锦玉回到家乡，在家中的日子，他既不下地务农，也没有找工作，而是整天往山坡的坟地上跑。据杨锦玉交代，他在坟地里是为了寻找女性坟墓，继而盗掘女性尸骸。他跑遍了本村及邻村的坟地，找到了6处适合下手的“目标”。3 月19日傍晚开始不到一周的时间里，杨锦玉先后盗掘了6具女性尸骸。
 <br/>
 *盗掘女性尸骸为卖钱
 <br/>
 在对杨锦玉的审查中，民警了解到，这些尸骸是要卖到山西省汾阳市一个叫阳泉的村子。杨锦玉说,去年他在汾阳一煤矿打工时，认识了一个烧锅炉的老汉，老汉告诉他，当地有一种配“阴婚”的陋习。就是为那些未婚而死去的男性找到一具女性尸骸，在清明节时将女性尸骸与其合葬。当地人相信，“阴婚”可以使死去的人在阴间不寂寞，所以愿意高价购买女性尸骸。而这个老汉愿意充当“中间人”,由杨锦玉回老家盗掘尸骸，老汉则为其寻找买家，一般年轻的女性尸骸可以卖到500元，年长的尸骸也卖300元。老汉还提出卖出一具尸骸，自己从中抽取50到100元不等的“中介费”。
 <br/>
 目前犯罪嫌疑人杨锦玉因涉嫌《刑法》第302条：盗窃、侮辱尸体罪，被西安铁路警方予以刑事拘留。
