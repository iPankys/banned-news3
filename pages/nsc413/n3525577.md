<div><p>
 【大纪元2012年02月29日讯】中国新疆喀什地区叶城县28日18时许，发生群体冲突事件导致12人死亡。
 <br/>
 <br/>
 据官喉新华社报导，事件发生在叶城县幸福路市场。有目击者称，冲突造成12人死，多人受伤。警方当场至少打死2人。
 <br/>
 <br/>
 新华社的报导没有得到独立方面的证实。对于类似事件，中共当局一直采取资讯封锁措施，外界很难判断真相。
</p>
<p>
 英国广播 公司的报导说，该消息发布之后不久，“喀什”即被微博屏蔽，相关消息也被过滤。
</p>
<p>
 新疆维吾尔自治区最近几年暴力不断， 2009年7月日发生乌鲁木齐七五事件，该事件导致至少156人死亡。去年7月18日，新疆和田地区曾经发生针对派出所的暴力事件，中共官方媒体报导称是“恐怖袭击”，20人死亡。而海外维族组织则认为实际死亡人数更多，并指是愤怒的普通维族民众的抗争活动。此后，距离新疆和田派出所遇袭事件发生不到两周，南疆喀什又接连两天发生连环暴力事件，造成9人死亡，至少28人受伤，警察当场打死5人。
 <br/>
 <br/>
 对于新疆不断出现的导致多人死亡的暴力冲突事件，中共官方对有关事件的报导一直是模模糊糊。此前美国之音分析文章表示，官方的报导给人的印象是新疆时常有恐怖份子自天而降，兴风作浪，乃至行凶杀人。
 <br/>
 <br/>
 路透社记者克里斯‧巴克利发表一篇分析性报导，指出新疆近来频繁出现的暴力冲突根源在于中国内部。报导说：“北京掌控西部边陲地区（新疆）所面临的最大威胁来自中国内部的愤怒爆发为暴力，而不是来自以巴基斯坦为活动基地的恐怖主义分子。中国有关官员早先把（新疆）最新的流血冲突归咎于这样的恐怖份子。”
 <br/>
 <br/>
 对于北京将新疆暴力事件归于“东突厥斯坦伊斯兰运动”策动。巴克利表示，“很多观察家说，东突运动处于困境，阵脚早已大乱，甚至完全不起作用了。这些观察家表示，（新疆的）民族冲突的原因以及冲突的解决之道主要在中国国内。”
 <br/>
 <br/>
 总部设在德国的维吾尔族人权益组织“世界维吾尔代表大会”的发言人迪里夏提此前也表示，中国的问题就在于这种所谓的“严惩不贷”式的镇压。
 <br/>
 <br/>
 迪里夏提说，北京当局以及当局控制的媒体一直在竭力回避近来新疆暴力冲突的基本原因，这就是自2009年乌鲁木齐7.5事件以来，北京当局对新疆维族人的镇压变本加厉；从和田到喀什，都显示了北京当局堵塞了维族人维护自己权益和权利的一切通道，其残酷镇压达到了维族人所能承受的极限，导致越来越多的维族人绝望，采取了令人担忧的绝望抗争手段。
 <br/>
</p>
