### 一男子被撞再遭碾死 家属称司机是人大代表

<div><p>
 【大纪元3月30日讯】3月28日，抚顺站附近的百货大楼门前，一辆白色越野车将一名男子撞倒后，掉头又从男子的头部碾过，将其碾死。死者的妻子目睹了惨剧全过程，据死者家属说：“肇事司机是抚顺一个知名企业家，还是一名人大代表。”
</p>
<p>
 惨剧： 越野车从他头上碾去
</p>
<p>
 据华商晨报报道，28日晨3时，家住抚顺东洲区东洲街56岁的裘吉和妻子带着旧电视机，到抚顺站前的百货大楼要以旧换新，准备抢个靠前的位置。当时还有两对老夫妻也来到了现场。“我先到附近的火车站候车室暖和暖和。”裘吉对妻子说。
 <br/>
 <br/>
 “10多分钟后，裘吉跑着回来了，身后还跟一辆白色的越野车，没等裘吉到我身边，这辆越野车就将裘吉撞倒，我还没反应过来，这辆越野车在商场门前的步道上划了个圈，从倒在地上的裘吉头上轧了过去……”裘吉的妻子蔡宝芹泣不成声。
 <br/>
 <br/>
 抓捕： 越野车司机被警方控制
 <br/>
 <br/>
 “为什么撞我老头？抓住他！”蔡宝芹见到自己丈夫被轧后，本能地去追赶越野车。结果越野车快速逃掉，借着路灯，蔡宝芹清晰地记下越野车的牌照号：辽D19995。随后，另一对老夫妻借给蔡宝芹电话报警。
 <br/>
 <br/>
 抚顺市公安局新抚交警大队赶到现场后，迅速在全市各路口布控堵截逃跑车辆，并向上级汇报情况。随后新抚刑警大队来到现场，按照刑事案件处理此事。1个多小时后，警方发现一辆同牌照越野车在望花区丹东路又发生肇事，后来车内司机被警方控制并进行调查。
 <br/>
 <br/>
 蹊跷： 撞人惨剧缘何发生
</p>
<p>
 “我现在腿都发软，看到当时的惨景，我实在感到太可怕了。”和蔡宝芹一起排号换电视的邵女士说。是什么原因造成越野车司机竟要驾车撞死裘吉呢？一位知情的出租车司机告诉记者，事发前看见白色的越野车司机和被轧男子在抚顺大酒店门前争吵过，越野车里副驾驶位置上还有个黄头发的女子。
 <br/>
 <br/>
 疑问 司机是人大代表？
 <br/>
 <br/>
 “经公安部门调查，得知驾驶越野车的司机是抚顺一个知名企业家，还是一名人大代表。”蔡宝芹的哥哥蔡先生告诉记者。随后，记者来到抚顺市公安局新抚分局，局长邵希默说：“我们正积极调查此案件。”
 <br/>
 <br/>
 记者向其求证越野车的司机到底是不是人大代表，是哪一级的人大代表时，邵局长表示：“暂时不能披露，到时就可以公开，此案已经引起抚顺市副市长、公安局长关飞的重视。”
 <br/>
 <br/>
 记者在新抚区刑警大队看到，涉嫌撞死裘吉的白色越野车已被拖进了车库，民警称“等待技术鉴定”。记者发现，越野车只是两个前轮的轮胎橡胶已经破损，车的四周没有破损的痕迹。
