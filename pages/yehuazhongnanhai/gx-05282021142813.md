 <p>
  自美国《华尔街日报》率先炒出中美全面经济对话中方牵头人“阵前换将”的消息，并立刻被中方商务部发言人高峰在记者会上用 “不属实”三个字给予否认之后，我们《夜话中南海》节目已经刊登和播发了四篇系列文章《习近平重用的刘鹤到底是不是“右派 ”？》，《习近平的两个核心智囊王沪宁和刘鹤谁左谁右？》，《刘鹤与习近平莫须有的“同窗之谊”》和《刘鹤： “大器晚成”的中共学术官僚》 。
  <br/>
  <br/>
  最新动向是，北京时间的昨天（5月27日）上午，中共新华社和商务部同步报道了“中美全面经济对话中方牵头人刘鹤与美贸易代表戴琪通话”。对此，外部媒体关注的焦点仍放在中方“牵头人”并未由刘鹤换成胡春华，而中国大陆的官民媒体则都不约而同地把关注焦点放在了“双方认为，发展双边贸易非常重要”等表述上，似乎是在暗自庆幸“中美不会脱勾了”。
  <br/>
  <br/>
  新华社转发一的篇评论文章《中美最新通话，传递什么信息？》中，列举了2021年一季度中美贸易高速增长，其中中国对美出口同比增长62.7%，自美进口增长57.9%，能源、农产品、汽车及零件等进口都快速增长。3月份当月，中国对美出口同比增长41.6%，自美进口增长61.8%，进口进一步加快等商务部发布的一手数据，以此证明，“中美关系加速恶化裂变，跌至历史低点的时候，经贸合作依然是中美之间相对稳定的领域，对中美两国关系正常化、保证中美关系不出轨、不越红线十分重要”。
  <br/>
  <br/>
  总部在北京的多维新闻网发表的评论《刘鹤与美国贸易代表通话，胡春华为何未接替刘鹤》认为：“如没有意外，在中共二十大前，刘鹤将继续担任中美全面经济对话中方牵头人”。理由是现在无特殊情况，确实没有理由，也没有必要让胡春华接替刘鹤担任中美全面经济对话中方牵头人。两人的分管领域不同，刘鹤分管商务、金融和科技；胡春华分管三农和扶贫。从两人分工看，刘鹤担任中美全面经济对话中方牵头人更合适，并且2018年以来，刘鹤一直担任中美全面经济对话中方牵头人，对中美经济对话十分了解，中方不需要在此时换人。
  <br/>
  <br/>
  多维的这篇评论文章还分析说：再说胡春华分管三农和扶贫工作，任务十分吃重，他更需要全心全意做好这方面的工作。2020年，中国刚刚宣布现行标准下农村贫困人口全部脱贫，要巩固脱贫成果，真正让中国原来的贫困人口实现全面小康，还有很多艰巨的任务。乡村振兴是中国实现社会主义现代化的最后一里路，也是最艰难的一里路，胡春华作为主管三农和扶贫的副总理，更应该在这块“最难啃的骨头”上尽心尽力。
  <br/>
  <br/>
  多维的分析文章总结说：当然，中美全面经济对话也很重要，它事关中国最重要的双边关系中美关系的健康发展。不过中国复兴的核心还是在中国自身，只要把自己的问题解决了，中国可以持续稳定复兴，那么发展中美关系就会有坚实的基础和后盾。因此，从战略层面考量，胡春华做好乡村振兴工作更符合中国的长远利益。
  <br/>
  <br/>
  从现有中国高层人事布局看，胡春华未来有希望成为中国高层领导人的接班人，大概率在明年的中共二十大上，升任中共政治局常委。身为中国领导人的接班梯队成员之一，胡春华没必要走到中美谈判对话的最前沿。是否担任中美全面经济对话中方牵头人，对胡春华的仕途也没有影响。因此从各方面说，胡春华暂时没必要接替刘鹤担任中美全面经济对话中方牵头人。
  <br/>
  <br/>
  笔者也认为，如上文章中“现在无特殊情况，确实没有理由，也没有必要让胡春华接替刘鹤担任中美全面经济对话中方牵头人”的分析，理由并非刘鹤和胡春华在国务院里“分管领域不同，刘鹤分管商务、金融和科技；胡春华分管三农和扶贫”。
  <br/>
  <br/>
  关于中共本届国务院副总理的责任分工，一直都是被媒体总结为：韩正分管发展改革、财政；孙春兰分管文教卫体；胡春华分管三农和扶贫；刘鹤分管商务、金融和科技。但事实上，至少是对外经贸这一摊，一直都是胡春华与刘鹤共同分担。下面以最近一年里，新华社对外报道的胡春华在商务领域的活动轨迹和他具体扮演的角色为证：
  <br/>
  <br/>
  本月27日，也就是刘鹤以“中方牵头人”身份与美方贸易代表最新一次通话的当天，胡春华以“中法高级别经济财金对话中方牵头人“身份”，应约与法国经济和财政部部长、对话法方牵头人勒梅尔举行视频通话”。

  仅从如上最近一年来的公开信息看，胡春华在中共中央政治局和国务院内部的责任分工，或者说“分管的领域”，绝不止外界知道的“三农”和“扶贫”。而刘鹤的所谓“分管商务”，更不是全部，或者解释成刘鹤和胡春华共同分管商务，但各有侧重更为准确。
  <br/>
  <br/>
  在对外经贸方面，刘鹤以所谓“牵头人”身份主要负责与美国打交道是毫无疑问的。但除了前面归纳的胡春华近一年时间来，在对外经贸领域的活动内容，如果再往前追朔的话就会发现，在中共国务院的本届内阁，也就是胡春华和刘鹤一同上任国务院副总理的2018年3月之后，胡春华在在对欧、对俄，甚至对非洲和对美洲的经贸领域都扮演着“中方牵头人”的角色。
