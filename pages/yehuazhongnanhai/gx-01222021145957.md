 <p>
  我们在本专栏的上周五，刊发和播讲了《胡怀邦和赖小民居然是出自同一个“伯乐”》一文之后，关于赖小民的最新消息是，昨天，也就是2021年1月21日，由天津市高级人民法院发布的二审判决书公布：裁定驳回赖小民上诉，维持死刑原判 。
  <br/>
  <br/>
  新华社奉命播发的相关报道中说：赖小民受贿、贪污、重婚一案，天津市第二中级人民法院于2021年1月5日一审宣判，认定赖小民受贿财物折合人民币共计17.88亿余元，贪污人民币2513万余元……。决定（对他）执行死刑，剥夺政治权利终身，并处没收个人全部财产。宣判后，赖小民提出上诉。天津市高级人民法院依法立案受理……。赖小民充分发表了自己的辩解意见，辩护人充分发表了辩护意见，赖小民还进行了最后陈述。天津市高级人民法院经合议庭评议、审判委员会讨论，依法二审做出裁定：虽然赖小民具有重大立功表现，但综合其所犯受贿罪的事实、性质、情节和对于社会的危害程度，不足以对其从宽处罚。……天津市高级人民法院遂作出驳回上诉、维持原判的裁定，并依法报请最高人民法院核准死刑。
  <br/>
  <br/>
  报道中还称，“人大代表、政协委员、新闻记者及各界群众旁听了庭审”。
  <br/>
  <br/>
  不知道这些旁听者在现场听到的关于赖小民所谓“重大立功表现”的具体内容，除了一审判决的相关报道中仅仅提到的那一句“赖小民提供下属人员重大犯罪线索并经查证属实”，是否还有其他内容？比如，他和他当年在华融的下属们是如何把国有资产华融旗下子公司廉价“出让”给曾庆红的家族公司，由他的侄女曾宝宝出面经营的“花样年”的。
 </p>
 <p>

  <br/>
  正如我们本专栏的上篇文章中引述过，中国内地财经网站上的《赖小民幕后小叙》一文中所说：关于赖小民“赖六吨”的靠山……，他这辈子，最爱和两种人打交道，老乡和二代。老乡围成的圈子让他感到安全，“到处都是我们的人”这种感觉很舒适。二代则是他获取安全感的另一种路径，他总幻想着摆平了牛逼老子的家属，就等于摆平了牛逼的老子，就相当于给自己买了最靠谱的保险。从同学、同乡、战友、同事中去寻找出几个官最大的，就是坊间一直传言的庆王爷和吴家。
  <br/>
  <br/>
  该文里的“庆王爷”暗指何人，在中国大陆尽人皆知，全都因为是中共政权自己的中纪委和监察部的官网上，六年多前就曾张贴出一篇标题为《大清"裸官"庆亲王的作风问题》的借古讽今文章。被读者们坚信是在以史为鉴，暗示习近平政权继周永康等之后，下一个被摘掉“铁帽子”的可能会是当今中共政权的“庆亲王”曾庆红。
  <br/>
  <br/>
  当时的中纪委网站上“突兀”地帽出这样一篇文章的一个重要背景是，习近平刚刚在王歧山主持中纪委全会上的讲话中，八哥学舌了王歧山的一句“腐败没有‘铁帽子王’”，意在用已经抓了周永康、徐才厚、苏荣和令计划的“反腐成果”，来说明即使是官至“党和国家领导人”，只要腐败了，就要被摘除“铁帽子”。
  <br/>
  <br/>
  话说在中国大清史上，一八二零年永璘晋封庆亲王，其子依例降为郡王。后来其孙奕劻在光绪二十年，即一八九四年被慈禧太后懿旨封庆亲王，是清朝第十二位铁帽子王。
  <br/>
  <br/>
  这十二位“铁帽子王”说起来，都算不上清官。那么六年前中纪委网站署名“习骅”的文章《大清"裸官"庆亲王的作风问题》，为什么单单挑出“庆亲王”说事？所以当时有中国内地网友感慨，“庆亲王所指太具体了，下一只老虎的尾巴露出一点‘红’来了”。
  <br/>
  <br/>
  这篇《大清"裸官"庆亲王的作风问题》的作者“习骅”也曾经著有《中国历史的教训》，该书曾经被中纪委网站列为二零一五年推荐书目，对他的介绍是：在“反腐第一线”工作多年，现在中央纪委派驻机构工作，法学博士。曾先后师从南京大学林仁栋教授、北京大学黄宗良教授，研究廉政立法和苏共问题。
  <br/>
  <figure class="image-richtext image-inline captioned" style="width:622px;">

  <br/>
  习骅在这篇文章中说：庆亲王奕劻工作能力很差，名声也不太好，官运却好得出奇……。但是庆亲王业余时间更忙，忙啥呢？吃饭、打牌、投资。请客和吃请是他每天必修课，主题多是庆贺生日。当时成功人士都是老婆一群、孩子一堆，所以庆亲王不是在主办宴会，就是在出席宴会的路上；然后就是打麻将。为了讨老佛爷喜欢，他竟然把麻将牌引入宫中，手把手地将搓麻技艺推广到人，从宫女到太监，无不乐此不疲。当时大清正是内外交困，慈禧需要舒解心情，所以对麻将引进工作十分满意。
  <br/>
  <br/>
  当然，这只是第一步。庆亲王不时让他的一个小老婆进宫，不是陪老佛爷、就是陪老佛爷身边工作人员来两圈。这位女士每次进宫陪打，随身带去的银票总是不够输。功夫不负有心人，庆亲王因此不断得到提拔重用。有人实在看不下去，郑重提出批评意见。慈禧耐心地做人家思想工作：“他啥水平我能不知道吗？但是看来看去，还是他贴心。”
  <br/>
  <br/>
  有内地网友点评上文中的这一句说：“曾经的庆亲王红得很，都是让慈禧宠惯的，结果彻底毁了大清江山。”把曾庆红喻作如今的“庆亲王”，那宠坏曾庆红的“慈禧”，当然就是江泽民了。
  <br/>
  <br/>
  笔者在这里需要特别提示一下，曾庆红对麻将的酷爱，中南里尽人皆知。其“麻友”虽多，但哪个也比不上当时的政治局委员回良玉更令他贴心。回良玉当年能够巴结上江泽民，并不是因为外界所传闻的把自己的女儿嫁给了江泽民的儿子，而是始于陪同曾庆红“搓麻”。
  <br/>
  <br/>
  习骅的文章中还写道：庆亲王理财很有天赋，也有超前意识。虽然外国银行当时已经进入中国，但是八国联军侵华硝烟未散，人们的爱国热情空前高涨，大家耻于与外国人打交道，更愿意到中国人的银行或钱庄存钱。庆亲王却格外青睐外资银行，特别是英资汇丰银行，民族金融机构里没有他一厘钱。要是迟生一百年，庆亲王肯定是个家小在外、见首不见尾的“裸官”！
  <br/>
  <br/>
  众所周知，数年前曾庆红的儿子，澳大利亚籍华人曾伟在澳洲花3240万澳币购买超级豪宅，并提出再花500万澳币翻新豪宅被当地政府拒绝事件，就已经轰动整个澳洲。当地媒体对此事的详情披露有图有真相，容不得当事人抵赖。
  <br/>
  <br/>
  有内地网友对习骅的文章拍案称奇，评论说，当今共产政权里的“铁帽子王”也就是退休后仍然还活在世上的“前政治局常委”，只有两个带“庆”字的：一是贾庆林，二是曾庆红。但贾庆林显然还没有坏到历史上“庆亲王”的腐坏程度。有外界媒体引述“中国搜索评论员”陈方的评论说：“庆亲王所指是谁？谁满足（习骅）文中所指四条：一、长期身处中枢，得一把手欢心，却毫无治国才能；二、海外资产不菲，连外媒都看不下去，予以公开报道；三、颇善于摆平事，再大的丑闻也能稳坐钓鱼台；四、热衷搞圈子，一个人带出一批贪腐巨蠹。
 </p>
 <p>
  用独立观察人士郑经纬的话说：如果用慈禧来比喻恋权乱政的江泽民，那么曾庆红就是最契合的〝对应〞。二者无论勋贵家庭出身、上位经历、裸官身份以及敛财途径，都极为相似。
  <br/>
  <br/>
  正如一篇中共官媒上的评论文章所说：“中纪委网站代表官方，不同于商业网站，于羊年春节后刊登这样一篇文章应该是意有所指。2015年的中纪委将拿下让世界瞠目，让国人欢呼的超级巨虎。”所以当时曾有不少人因为中纪委网站上的这篇文章，大胆预测曾庆会是继周永康之后，第二个被摘掉“铁帽子”的正国级。但此预测的话音未落，中纪委和监察部的官网上即把习骅的这篇文章悄悄删除了。
 </p>

  <br/>
  日后发生的故事之一是，正好是去年的今天，中国内地的多家财经网站都转发了《花样年创始人曾宝宝任地产集团CEO 回归一线》的报道文章，说是曾庆红的胞弟曾庆淮的女儿曾宝宝是花样年集团创始人、花样年控股股东之一并为公司最大股东，也就是公司真正的老板、实际控制人。多年来，她推动（台面上的董事长）潘军成为花样年操盘手，全权负责公司的业务，自己则隐在幕后居多。在做了多年的“潇洒老板”后，曾宝宝决意重回一线，担任花样年地产集团CEO。此外，曾宝宝仍担任花样年中国集团战略规划委员会主席……。
  <br/>
  <br/>
  报道中还介绍了迅速扩张的花样年的五大“封疆大吏”，都已经被曾宝宝逐一任命。这“五大”分别指北京、上海、深圳、武汉和成都这“五大区域”。
  <br/>
  <br/>
  此后，曾宝宝仍然十分乐见被中国境内媒体追踪报道。最典型的一则报道莫过于她去年大年初三，坐在北京飞住罗马的飞机头等舱里品着红酒把玩“抗疫”的细节描述。说的是“就任花样年地产集团CEO之后，手撕绿城不在话下的曾宝宝，遇到的更为棘手的事便是来势汹汹、席卷全国的肺炎疫情。一向棱角分明，爱谁谁的曾宝宝也难置身事外”。报道中贴出了曾宝宝当天写的微博，说是“全面抗战，需要军统、中统，需要中央军正规军、游击队，也需要黄金荣、杜月笙，这不矛盾不冲突啊。……我们集团自行采购的物资囤积在武汉，发不到湖北周边……。麻痹！”
  <br/>
  <br/>
  赖小民落网之后曾宝宝对外更加高调，甚至干脆从“幕后”走到前台，以及由她出面经营的曾氏家族的企业规模已经遍及整个中国“五大区域”的公开报道，与被外界曝光其“离岸公司”之巨额财富之后，火速隐身的习近平亲属形成显明对比。如此现象，只能说明当代“庆亲王”曾庆红的“铁帽子”仍然是牢不可破。
  <br/>
  <br/>
  至于习近平的“反腐打虎”为什么不会殃曾庆红及其家族，后续的文章会有详细的分析和介绍 。
