<div id="storytext">
 <p>
  <span style="-webkit-text-size-adjust: 100%;">
   我们在本专栏上期节目播出的《从金人庆之死看中共退休高官的“合法腐败”》一文中，只着重介绍了中共正部长级干部待遇的较详细内容。
  </span>
 </p>
 <p>
  <br/>
  当然，正部长级和副部长级退休干部之间的待遇区别，首先是体现在一个钱字上。除了退休工资的数额不同，每月得到各类补贴的项目和数额也不同。比如保姆费一项，网上疯传过的一份中共前外经贸部长郑拓彬的离休工资单，其中的所谓“自雇费”就是自己请保姆的月补贴额：一个月是3500元。而副部长级的退休待遇中，“自雇费”这一项只有1500元。
  <br/>
  <br/>
  这也是中国国内网友纷纷质疑金人庆死因的一个重要问号：不但国家配给的贴身秘书、专职司机都不在身边，一个七十七岁、动过心脏大手术、行动已经必须靠轮椅代步的正部长级退休“老同志”，怎么会身边连个保姆都没有？很容易理解的是，作为离退休的老干部，其所有生活和经济待遇中，除了每月直接到手的人民币数额，最重要的应该就是医疗待遇了。
  <br/>
  <br/>
  正省部级与副省部级干部无论退休前还是退休后，正的住院可以住套间，副的只能单间，司局级干部住双人间，级别更低的干部只能住普通病房了。不过，到了基层以后，副省级以至副省级以下的在位和退休官员的医疗待遇，特别是住院待遇，几乎全都比在京官员要优渥许多。《南方周末》曾有报道说，该报记者接触到的市、县地区医院也有“干部病房”，如果市委书记、县委书记退休后在本地医院看病，虽然他们级别不高，但仍旧可以住到单间。
  <br/>
  <br/>
  当然，正省部级与副省部级的在位和退休官员的医疗费用报销额度也不一样。
  <br/>
  <br/>
  首先说所谓“离休”干部。“离休”的概念是邓小平时代才出现的，凡是1949年10月1日之前“参加革命”的，退位之后称之为“离休”。该时日之后“参加工作”的，退位之后才称之为退休。大概从二十年前的江泽民时代开始，中共政权的离休干部无论是什么级别，其医疗待遇的特别就是所有医疗开销，包括所谓“保健品”的开支，基本能实现实报实销。
  <br/>
  <br/>
  至于所谓退休干部，以北京的部级干部为例，退休前和退休后的医疗待遇是一样的。正部级医疗证封皮是绿色的，副部级的封皮是蓝色的。正部级所有医疗费用都可以报销，而且不是在本单位或者退休前最后任职的“原单位”报销，而是由国家卫生部门直接与医院结算。副部级干部的医疗费用在本单位或者退休前的“原单位”报销。与正部级的区别之一是，副部级的个别进口药的报销原来是有点限制的，不过在2012年，中央保健委员会办公室下发了《关于适当提高副部级人员医疗保障水平的通知》后，副部级医疗待遇人员的药品费用基本能全额报销了。
  <br/>
  <br/>
  到目前为止，除了“离休”前即已经是副省部级以上者，其他级别的所有“离休”干部的生活和医疗待遇都会比他们在位时的待遇从优。从很多中共中央和地方媒体报道出来的一些“离休老同志”去世之后的生平介绍中，常常可以看到一句“享受正部长级医疗待遇”或者“享受副部长级医疗待遇”之类。最常见的主要是“享受副部长级医疗待遇”，意思是此人离休前的最高官职只是正局级甚至更低，只是因为“参加革命”的时间早，所以离休后的医疗待遇从优。
  <br/>
  <br/>
  2009年中组部规定，1937年7月6日前参加革命工作、尚未享受副省（部）长级医疗待遇的离休干部，全提高至享受副省（部）长级医疗待遇。1937年7月7日至1945年9月2日参加革命工作、离休前为正司局级的离休干部，也可以提高至享受副省（部）长级医疗待遇。不过，对于这批人，其离休后所享受的“政治待遇”还是维持在离休前原级别的基础上。
  <br/>
  <br/>
  当年邓小平提出废除领导干部职务终身制，商量对老干部的安排时，陈云强调要使人心安定，要准备花点钱，但还要有思想工作，“有许多是可以不花钱的，看戏坐头排，开会上主席台，并不要花钱。”
  <br/>
  <br/>
  “坐头排”、“上主席台”，指的是离退休干部的政治待遇。1982年2月20日，中共中央颁布《中共中央关于建立老干部退休制度的决定》里明确规定：对一切离休退休老干部，他们的政治待遇，包括阅读文件、听重要报告、参加某些重要会议和重要政治活动等等，应当一律不变。作为保障措施，各单位都要安排专人负责老干部阅读文件。一般来说，同级别在职干部可以阅读的，离退休干部也可以阅读。
  <br/>
  <br/>
  一位国务院某部门老干局的负责人曾对《南方周末》记者透露，除了阅读文件，部里重要人事变动也会及时通知老干部。对于文件，身体好的到部里阅读，年老体弱的，则由部里派专人送到家里阅读。一位由部委改为央企的老干局工作人员曾对《南方周末》记者介绍说：单位里为每个离退休（副）部级干部准备了一个阅文室，实际上为每人保留了一间办公室。副部级以下待遇的老干部也享受阅读相应级别文件待遇，但没有独立阅文室，大家合用一个。各级老干局一般至少有两个处负责老领导服务，一是生活待遇处，二是政治待遇处，但这两个处面向所有离退休干部。
  <br/>
  <br/>
  在中央和国家机关里，有的部委为体现对老领导的关心，还会设立专门的老领导处来负责落实他们的退休待遇。这里的所谓“老领导”，是专指在该部委的正副职领导岗位上退下去的“老同志”。
  <br/>
  <br/>
  至于国家级领导人，也就是所谓副国级和正国级高官的离退休待遇，那就更令人瞠目咋舌舌了。网上有一篇标题为《国家领导人长寿秘诀和退休后待遇》的综述文章，内容非常详实。文章开篇即以中央军委原副主席张震去世时享年101岁为例，认为张震之所以活到100多岁，决不是因为他有什么养生秘诀，养生秘诀之于常人是不管用的。张震真正的“养生”法宝是他的高官身份，党和人民“不惜任何代价”把他养着，他不长寿还不行。早就有人说，中国领导人“以高寿着称”于世。毛泽东时代，领袖们已经开始长寿，毛本人活了83岁，周恩来78岁，朱德90岁。当时中国人均寿命65岁，他们算是鹤立鸡群。但现在的中国领导人使他们相形见绌，几乎人人都是或者至少接近百岁老人。比如，邓小平93岁，万里99岁，其他如原中共中央副主席汪东兴100岁，中央书记处原书记邓力群100岁，原国务委员张劲夫101岁，全国政协原副主席吕正操、雷洁琼分别活了105岁和106岁。
  <br/>
  <br/>
  该文中还特别介绍了前中常委，1917年4月生人的宋平。该文发表的那年宋平98岁，精神矍铄地出现在在天安门城楼上陪同习近平抗战阅兵。文章作者幽默道：“（这宋平的精神状态）一下子把江泽民都比下去。谁道宋今年已经高寿98岁。这一亮相，让全国人民惊叹不已，甚至目瞪口呆。”有道是，如今的宋平已经是104岁高龄了，仍然还健康地活着。
  <br/>
  <br/>
  如上文章的作者说：现在中国媒体上充斥着离退休领导人的养生秘诀，人们似乎忘了一个基本事实，即中国领导人长寿本质上既是特权的表现，也是特权的结果。一遇到地震、矿难或其他安全生产事故，领导人经常指示说，要不惜一切代价抢救被困者的生命安全。其实国家真正不惜一切代价的，唯有保障领导人离退休后的健康、安全和生活。
  <br/>
  <br/>
  副部级以上高干，可以成年累月住在专门医院或高干病房里，享受无微不至的医护，用世界上最先进、最贵的药，每天花个几千上万元不在话下。2013年，官媒曾报道一位退休省级干部住一次院花费高达300万元。党和国家领导人级别花费如何，我们可想而知。
  <br/>
  <br/>
  这位作者还介绍了他知道的中国内地某座城市，有一个广州军区驻当地首长服务处，大院有一两平方公里，住着兵团以上级别的离休将领二、三十人。这些人每人一栋别墅，配有秘书、警卫、司机、保姆、厨师等五到七人。每年暑期上河南鸡公山避暑一个月，每位首长及其老伴住山上别墅一栋，都是蒋介石等国内外达官贵人的遗产。一家专门为首长服务的军队疗养院抽调一整套班子，跟着首长上山，每天给他们量血压，每周血检尿检一次。每年冬天，首长们到海南等地疗养一个月。首长服务处是正师级单位。一个正师级军事单位，就为这么些“老革命”们的健康、安全与吃喝拉撒存在着。党和国家领导人退下来以后享受什么样待遇，我们不得而知，但可以猜测得之：他们有自己的宅子，但不称为家，而称为“驻地”；有一大套班子服务于他们的健康、安全和日常生活。中国领导人比外国领导人还长寿，道理很简单，秘密就在这里。
  <br/>
  <br/>
  两年前的9月15日，中国微信上热传一条来自北京301医院的微信广告，其中重点宣传中共领导人的“健康工程”，透露中共领导人延寿目标为150岁。因过于敏感，广告次日即遭屏蔽。众所周知，解放军北京301医院是中共中央的重要保健基地，多年来一直负责中共高层的医疗和保健。
  <br/>
  <br/>
  据当时我们自由亚洲电台取得的影片片段，301医院的该广告词介绍，自从中共建政后，中共的首长医疗保健在60多年的“探索和实践”中取得“长足进步”，形成了“集预防、医疗、保健、康复、疗养”为一体，以“防癌、防心脑血管疾病，抗衰老，慢病管理、器官功能再生和健康生活方式”为重点，“中国养生精华同西方先进医疗技术深度结合”的独特体系。
  <br/>
  <br/>
  广告还说，这个医疗体系“全球第一”，中共领导人的平均寿命普遍高于西方发达国家领导人，2008年中共领导人平均寿命为88岁。广告结尾透露，2005年已启动“981首长健康工程”，延寿目标为150岁。视频内容显示，981首长健康工程设有三大工程，包括“150岁长寿工程”。
  <br/>
  <br/>
  百度资料也显示，981首长保健工程中心“拥有一支多年服务于中央首长的医疗保健专家及优秀的服务人员团队，拥有国际上最前沿的高精尖预防医疗保健设备和分子诊断实验室”。资料还显示，981模式共有三大工程：“健康促进工程（不老）；青春再现工程（不老）；150岁长寿工程（长寿）”。其中的150岁长寿工程，与301医院微信广告被删除的影片内容完全一致。
  <br/>
  <br/>
  该健康工程数据透露，2000年时，中共领导人的平均寿命同美国相比只多出2～3年，但在之后中共领导人的平均寿命呈现出非直线的飙升。到2010年，中共领导人的平均寿命已经高出美国领导人平均寿命逾10年。
