 <p>
  最近几天，澳门一家媒体借今年清明之际所发表的中共前总理温家宝悼念自己母亲的回忆文章，遭内地媒体“禁止分享”一事引发外界强烈关注。
  <br/>
  <br/>
  温家宝的这篇原标题为《我的母亲》的文章，本是去年年底他母亲杨秀云女士去世后的有感而发，日后如何被《澳门导报》拿到手并于今年清明前后发表的过程，外界不得而知。《澳门导报》同时发布了数张温家宝与母亲的照片。但在凤凰网、网易等新闻门户网络，都抢先转载了温家宝的这篇文章。北京时间4月17日，微信公众号“学人scholar”也转载了温家宝的这篇文章后，中共监管部门立刻采取了行动。截止本文发表的两天之前，通讯社交网络微信上虽然可以通过搜索关键词“温家宝”以及“我的母亲”，找到包括“知政前沿”和“东川发布”在内少数公众号转载的这篇文章，但相关文章的分享、转发、收藏和留言评论功能均已经被禁止，理由是“违反微信公众平台运营规范”；凤凰网等媒体的转载内容，也全数被删除。
  <br/>
  <br/>
  中国社交网络为何给前总理的文章设置障碍？法广的报道中引述了前中央党校教授、目前在美国的学者蔡霞，分析微信群里不许传播温家宝文章的原因：细读温写的全文，其实文章已经做了“自我审查”，全文没有“民主”“法治”四个字。即便这样，文章依旧被禁止转发分享，可见大陆极权统治当局对民主、法治有多恐惧，他们恐惧人民的权利。
  <br/>
  <br/>
  其实，虽然温家宝没有在自己主题是悼念母亲的文章里论及“民主”和“法制”，但仅仅其中对自己全家在“文革”中的苦难遭遇的“不堪回首”，就已经犯了习近平 的大忌。更何况，他温家宝“擅自”在境外媒体发表个人文章这一举动的本身，就已经严重地违反了习近平口中的所谓“政治纪律和政治规矩”。
  <br/>
  <br/>
  中共当局给“老同志”们的退休生活本来就是有很多清规戒律的，包括未经组织允许不得擅自接受境外媒体采访，不得在境外发表文章和出版回忆录；未经组织同意不得私自出境旅游和探亲访友，等等。
  <br/>
  <br/>
  到了习近平时代，对“老同志”们的“政治纪律和政治规矩”变得更为具体，更为严格。比如相关内部文件中，就已经把“未经组织允许，不得擅自接受境外媒体采访”这一句中的“未经组织允许”几个字去掉了。意思无非是把接受境外媒体采访，规定为没有任何变通可能的政治禁区。
  <br/>
  <br/>
  众所周知，为了确保所谓“两个维护”即“维护习近平党中央的核心、全党的核心地位，维护党中央权威和集中统一领导”，习近平这些年来对其内部强调最多的，就是所谓“严守党的政治纪律和政治规矩”，不止一次恶狠狠地警告“无论地位多高权力多大”，“谁都不能拿政治纪律和政治规矩当儿戏”。所以，如今温家宝此举彻底惹毛了习近平是毫无疑问的。
  <br/>
  <br/>
  已经有外界媒体发表评论说：在提到自己父亲的时候，说自己父亲在十年文革中吃了很多苦，经常受到野蛮审讯和打骂， 温家宝这篇文章目前被大陆的微信平台封杀，其封杀原因很可能在回忆自己父亲的文革苦难的部分，因为这刚好与习近平当局目前的官方口径相抵触。习当局现在提出要树立正确的所谓“党史观”，意思就是少提类似文革这种中共历史上的错，而是要大力宣传中共的所谓伟光正形象，目前已经大幅度修改党史。如果说，温家宝说的别的话中共还能容忍的话，那关于文革的回忆就刚好撞了枪口。
 </p>

  <br/>
  笔者注意到，温家宝在悼念母亲的文章中不但是突出强调了“史无前例”的“文化大革命”给他们全家带来的灾难，而且特别把“文革”对他父亲的迫害，和当年他父亲险些死于日寇的刺刀之下对照来写。温家宝的文章中有这样一段：“父亲能忍，但有个性。我刚满周岁时，正赶上日寇清乡、扫荡，行人过道卡，均要严格盘查，不准带一粒大米。有一次，妈妈因奶水不够，让父亲买点米带回家。天黑了，爸爸怀揣一包大米过路卡，日寇持枪盘查，非要搜身不可，父亲宁死不依，差点被刺刀挑死。”
  <br/>
  <br/>
  紧接下来是这一段：“ ‘文革’期间，父亲被关在学校，经常遭受野蛮的‘审讯’和打骂。一天，造反派一拳将父亲的脸打肿，眼睛被‘封’得看不见东西。父亲忍无可忍，指着心口说:‘小子，朝这儿打!’”
  <br/>
  <br/>
  两年多前，笔者曾在本专栏接连发表《政治改革，温家宝退位前的“谢场绝唱”》和《比比习近平 ，念句温家宝的好！》两篇文章。文中介绍 了赵紫阳1987年11月在十三大政治报告中说过，“应当通过改革，使我国社会主义民主政治一步一步走向制度化、法律化。这是防止‘文化大革命’重演，实现国家长治久安的根本保证。”从那以后，这段内容只被中共高层领导人公开强调过一次，那就是温家宝在20012年3月人大会结束时的中外记者执招待上所说：粉碎“四人帮”以后，我们党虽然作出了若干历史问题的决议，实行了改革开放，但是“文革”的错误和封建的影响并没有完全清除。不仅要进行经济体制改革，而且要进行政治体制改革，特别是党和国家领导制度的改革。没有政治体制改革的成功，经济体制改革不可能进行到底，已经取得的成果还有可能得而复失，社会上新产生的问题也不能从根本上得到解决，文化大革命这样的历史悲剧还有可能重新发生。
 </p>
 <p>
  日后因为有了习近平复辟文革的鲜明对比，笔者更认为，温家宝当年在中外记者面前对“文革”悲剧重演的忧心和警告难能可贵。习近平上台之后在政治路线上的倒行逆施，证明当时的温家宝利用他最后一次中外记者会机会，所发出的警世箴言似乎并非仅仅针对薄煕来一人。
  <br/>
  <br/>
  习近平上台之初，即有网友在笔者的相关文章之后发表评论说：温家宝反薄完全是讨厌薄搞“文革”那一套，薄上台和习也不会有太大差别，因为他们受“文革”影响很深。温家宝非常讨厌和反对“文革”。
  <br/>
  <br/>
  其实，笔者在过去七、八年发表的相关文章里一直都坚持认为，假如不是夫人薄谷开来亲手毒杀国际友人的罪行败露，导致薄熙来走出一步一巴掌把个王立军打进了美国领馆的臭棋，2012年产生的十八届中共中央总书记习近平之下的“副总书记”的角色，本应该是由薄熙来扮演……。不过话又说回来，薄熙来虽然时运不济，但他所开创的“回归毛泽东”的伟大事业，已经在习近平手上发扬光大。习近平已经成功地在薄熙来重庆整座“山城红”的基础上，实现了“全国山河一片红”。
  <br/>
  <br/>
  温家宝还回忆了自己在中南海的日子：“对我这样出身的人来说，‘做官’本是偶然之事。我奉命唯谨，如履薄冰、如临深渊，受事之始，即常作归计。”
  <br/>
  <br/>
  请读者听众们注意，温家宝在这里所强调的“我这样的出身”，就是要区别另外一种“出身”者“做官”的必然。 他在文末写道：“我同情穷人、同情弱者，反对欺侮和压迫。我心目中的中国应该是一个充满公平正义的国家，那里永远有对人心、人道和人的本质的尊重，永远有青春、自由、奋斗的气质。我为此呐喊过、奋斗过。这是生活让我懂得的真理，也是妈妈给予的。”
  <br/>
  <br/>
  英国BBC中文网今天刚刚发表的评论文章《中国前总理温家宝撰文忆母谈公平正义 遭禁止转发》，引述中国政治学者凯博（化名）的评论说，中国前总理温家宝选择在胡耀邦去世纪念日附近发表文章，从自己苦难的家史谈到希望中国有公平正义，暗含强烈的政治信号。
  <br/>
  <br/>
  这位凯博具体分析说：“（温家宝的）文章结构基本上是4个部分，第一部分是抗日战争时期；第二部分是谈他父亲在五六十年代作为小知识分子经受的苦难；第三部分讲他在英国演讲被扔鞋，母亲很担心，但尽管如此他坚持中英友好，与现在外交战狼化形成鲜明对比。结尾最关键，谈公平正义，对人的尊重和自由。”凯博说，“这某种意义上是中国主流社会的共同心声，对现在社会政治状况不满，要求恢复最低限度的道德或最低限度的政治要求，要有公平正义。这是他想谈的价值观，跟中国过去八、九年的社会情况形成鲜明对比。”
 </p>
 <p>

  凯博认为，目前正值中共党建百年纪念强化权威之时，为未来二十大做准备，温家宝选择这个时间发表这篇文章是公开表达对当前执政者的不满。
  <br/>
  <br/>
  凯博的这番评论，令笔者想起当年曾经与温家宝一起参与了中共十三大政治报告起草的吴国光先生2011年10月的答记者问内容。当时的香港《动向》杂志发表《为温家宝鼓个掌――吴国光就温家宝"九一四"政改讲话答《动向》记者问》一文。《动向》记者的问题是：（2011年）九月十四日，中国总理温家宝借达沃斯夏季经济论坛在大连举办之际，再次高调谈论了政治改革。这已经是过去一年多来，温家宝第十几次谈论政治改革这个敏感话题了。一方面，这个话题依然敏感，中国官方舆论对温这次讲话还是采取了基本不予报道的冷处理态度……。至于民间和国际舆论，则不外乎嘲笑温的作秀和无力，甚至说中国政治改革已经成了一幕滑稽戏和讽刺剧，都不相信温有诚意、有能力真正推动中国的政治改革。
  <br/>
  <br/>
  吴国光先生回答说：去年年底，我对温家宝关于政治改革的讲话也有比较尖锐的批评。不过，这一次，我的看法比较积极。我认为，与他此前的十多次同类讲话相比，温家宝这一次讲话有实质进步，显示了他确实有推动政治改革的诚意。我看，可以说温家宝的大连“九一四”讲话是中共领导人自一九八九年“六四”以來，关于政治改革的最大胆、最深入、最有內容，也最有政治背景和意涵的一次讲话。
 </p>
 <p>
  当年和温家宝一起参加过十三大政治报告起草的吴国光先生具体分析说：总的来说，温家宝这次讲话有关政治改革的论述还远远没有超出二十多年前，中共十三大关于政治改革的部署。不过，他这次是沿着当年那个方向展开论述的，他确实提出了一些具有实质涵义的东西：第一是在强调依法治国的大前提下，重提“党政分开”；第二是突出地论述了选举的重要性；第三，也是与选举有关的，他讲到了在乡镇乃至县级实行有真正意义的选举；第四，则是实际上提出了“司法独立”的问题；第五，他强调了以政务公开来反对腐败，并直接论述了官员财产公开等实际措施；第六，温也讲了党内民主——尽管这一条我不大以为然。所有这些，都不再仅仅是泛泛地、口号式地谈论政治改革呀、民主呀，只给一些好听的话。这次他提出了一些实际的改革举措，当然并不系统，但是大都具有重要的、重点的意义，有目标，有措施。在这个意义上，我说这是给出了中国政治改革的“温氏路线图”。
  <br/>
  <br/>
  吴国光先生对温家宝的评论内容，下篇文章里还会继续介绍。这里先用吴国光先生如下 一段话做本文的结尾：“……赵紫阳成为改革与反改革兵戎相见的牺牲品。现在的既得利益更要大的太多太多了。我看温家宝也是明知做不到，但不甘心在名声上成为这个制度的殉葬品，至少要表示一下‘我不是个糊涂人’吧。我总说，中国这种制度是侮辱正常人的智商的，看来温家宝并不甘心受这种侮辱就是了。应该给他鼓鼓掌，因为他在进一步向民众靠拢，向真正的政治制度转型那个方向靠拢，这是应当受到鼓励的。”
 </p>
