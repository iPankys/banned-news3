<div id="storytext">
 <p>
  近日，一篇标题为《按惯例中共政治局将新晋一名女性成员》的外界报道文章分析说：从历届中共的最高权力机关——中央委员会，以及中共领导机构政治局的人事布局来看，女性政治人物所占的比例一直偏低。鉴于孙春兰在中共二十大时将年满72岁，料将卸任退休，政治局中必然要补充新鲜血液。谁将成为她的继任人选，可供的选择其实不多。2022年的中共二十大的人事布局中，如果没有意外，也将沿袭惯例，至少有一名女性高官进入中共政治局，并出任中共国务院副总理。
  <br/>
  <br/>
  该分析文章中回顾历史总结说，从中共十六大吴仪进入中共政治局，并出任国务院副总理之后，似乎政治局之中出现女性官员并担任国务院副总理已经成为惯例。中共十七大，刘延东成为政治局中的新晋女性成员，担任国务委员。刘延东在中共十八大连任政治局委员，并成为国务院副总理。孙春兰也在中共十八大进入政治局，政治局出现两位女性成员也是“文革”结束以来的首次。中共十九大后，孙春兰延续了吴仪和刘延东的角色，在政治局委员的第二任期出任国务院副总理。
  <br/>
  <br/>
  正如多维新闻网的一篇分析文章中所说：在过往，中共政治局女性成员并非标配，但过去近20多年未曾间断。从吴仪、刘延东到目前的孙春兰，他们都在中共高层政治中扮演了重要的角色，因此中共二十大大概率不会出现女性成员缺席的局面，特别是在女性地位高企的今天。
  <br/>
  <br/>
  毫无疑问，下届，也就是明年十月召开的中共二十大上将产生的新任政治局委员中，绝大多数都会是从适龄的现任省级党委书记中产生。有港媒分析，两年后的中共中央女性政治局委员，应在现时正部级或以上的女性官员中产生。
  <br/>
  <br/>
  其实这句分析稍欠严谨。严格说，应该是会从现有的省级党委书记或已经是非政治局委员的副国级女性官员中产生。因为很难想象，一个也是正部级的女性省长会在此基础上直升中央政治局委员。
  <br/>
  <br/>
  多维的分析文章中还说：作为中共政治高地贵州的现任省委书记，不少声音认为，1959年出生的白族女干部谌贻琴，极有可能成为现任政治局委员、国务院副总理孙春兰的继任人选。她是中共政坛目前唯一的女性少数民族省委书记，长期在贵州官场履职，先后与五任贵州省委书记共事，其中有三位成为中共的“党和国家领导人”。包括现任政治局常委、全国人大常委会委员长栗战书，政治局委员、重庆市委书记陈敏尔，以及国务委员、公安部部长赵克志。但是也有人认为，她从未离开过贵州，视野有限。但目前来看，她是最接近中共二十大政治局女性高官的候选人。
 </p>

  有内地的中共官媒刊登《中国现任唯一女省委书记，谌贻琴创造多项纪录》一文，对此女进行了详细的追踪报道 ，说是2017年9月，谌贻琴履新贵州省代省长（凭此职位顺利进入十九届中央委员序列），2018年1月去代转正。在2020年11月晋升贵州省委书记前，谌贻琴担任贵州省省长已经2年多。
  <br/>
  <br/>
  令人关注的是，目前谌贻琴是中国31个省份中，唯一一位女性省级党委书记。同时，谌贻琴也是中共建政以来，贵州第一位女省长，第一位白族出身的省长，也是1993年以来贵州首位本省出身的省长。此外，谌贻琴还是中共建政以来第四位女性省级党委书记。此前三位，分别是河北前省委书记吕玉兰、江西前省委书记万绍芬，以及分别于2009年担任福建省委书记和2012年担任天津市委书记的孙春兰。孙春兰出任天津市委书记是与晋升政治局委员同时。
  <br/>
  <br/>
  但是，虽然现已浮在台面上的唯一的女性省委书记谌贻琴已经具备了晋升政治局委员的必要条件，另外一名已经享受副国级待遇的中共女性领导人沈跃跃，在二十大上成为政治局中的女性代表也是有可能的事情。
  <br/>
  <br/>
  早在上个世纪八十年代初，即已经在贵州当地受到广为关注的“白族女大学生”出身的谌贻琴从2007年开始就晋升为贵州省委常委，开始享受副省部级待遇，至今已经是十七届、十八届中央候补委员和十九届中央委员，在同龄干部中也称得上“资格甚老”。但是，这也是相对而言，至少不能与沈跃跃相提并论。
  <br/>
  <br/>
  只比谌贻琴年长两岁，出生于1957年的沈跃跃从上个世纪的1997年，历任中共第十五届、十六届中央候补委员，十七届、十八届、十九届中央委员。也就是说，她已经在中央委员会的层面经历了江泽民、胡锦涛和习近平三任总书记，而且已经连任了第十二届和十三届两届全国人大常委会的副委员长。
 </p>

  <br/>
  在过去的节目里，我们已经多次分析过所谓“七上八下”的年龄潜规则是怎么回事。明年二十大召开之前的习近平当然有可能不再从年龄角度搞“一刀切”，决定让沈跃跃在任满第二届全国人大副委员长之后即直接退休。而另一种可能则是，1957年出生的沈跃跃因为已经是在位副国级领导人，那么在明年十月召开的中共二十大上理应因届时才年满65岁，而继续连任中央委员，那么接下来的安排可能会是晋升政治局委员，继而充当一届国务院的女性副总理。当然，在继任第二十届中央委员的前提下，也有一种可能是仍然不进政治局；但在次年，也就是2023年三月召开的两会上，在卸任全国人大副委员长的同时改任全国政协副主席。
  <br/>
  <br/>
  现在的沈跃跃是以中华全国妇女联合会主席身份，出任全国人大副委员长的。众所周知，中共政权治下的中华全国妇女联合会和中华全国总工会的主席都是副国级配置。历任主席中，只有康克清一任同时是全国政协副主席，其他都是全国人大的副委员长。
  <br/>
  <br/>
  直到前不久，现任蒙古自治区政府主席布小林身体出了问题的消息传出之前，中共政权内盛传这位布小林将会在明年的二十大上继任中央委员，然后便安排她陆续接替沈跃跃的全国妇联一把手和全国人大副委员长两项职务。
  <br/>
  <br/>
  与此同时，布小林的内蒙古自治区政府主席的位置将会交给现任湖南省委副书记、蒙古族的女性干部乌兰。
 </p>
 <p>

  <br/>
  去年九月，我们在本专栏连续发表了《内蒙强推汉语教学是习皇诏令》和《习近平彻底背叛了他父亲的宽容民族政策》两篇文章 ，其中引述了维基百科对布小林的介绍：“祖父乌兰夫为开国上将、原国家副主席、国务院副总理、全国人大常委会副委员长、全国政协副主席、内蒙古自治区首任主席、书记。父亲布赫为原全国人大常委会副委员长、内蒙古自治区主席。其家族为中华人民共和国第一个祖孙三代执政同一省级行政区的家族……。”
  <br/>
  <br/>
  这段话中的“执政”二字就看怎么理解。虽然从乌兰夫到布赫再到布小林是被称之为祖孙三代“蒙古王”，但事实上，当年的乌兰夫是蒙古自治区的党政一把手“双肩挑”，而日后的布赫也好，布小林也好，都只能是担任自己民族地区的行政一把手，党的二把手。区别甚大。
  <br/>
  <br/>
  五年多前，时年58岁的布小林在自治区党委常委中原本排名第8，既不是中央委员，也不是中央候补委员，突然被任命为自治区代主席，一下从内蒙古常委的排名中跃升第2。外界认为，布小林可谓超常规晋级，被总部在北京的多维新闻网相关分析文章称之为“弯道超车”。由此也就炒出了习近平及其一家，与“蒙古王“乌兰夫家族的特殊关系。
  <br/>
  <br/>
  曾经长年给习仲勋当秘书、先后陪伴习仲勋及其一家长达二十年之久，日后被习近平尊称为“张老”的张志功曾出版一本《难忘的二十年——在习仲勋身边工作的日子里》，其中有专章揭秘“内蒙古自治区创立者乌兰夫与习仲勋‘感情至深’”。1988年12月，乌兰夫病逝后，习仲勋亲笔撰文《乌兰夫永远活在各族人民心中》。文中称，乌兰夫是自己“最亲密的老战友” ，“深为敬佩的良师益友”。
 </p>
 <p>
  文章介绍说：20 世纪50年代至60年代初，习书记住在北京后圆恩寺胡同甲6号，与之一墙之隔的6号则住着时任国务院副总理乌兰夫。我的印象中，他们的关系很密切。习书记热情谦和，乌兰夫和蔼慈祥，家处近邻，说话之声隔墙可闻，两家人经常互访走动。习书记以他那惯有的幽默，戏称乌兰夫为“王爷”，经常让我给“王爷”送上一些陕西土特产。乌兰夫也送一些内蒙古的土特产，如黄油之类给习书记。
 </p>
 <p>
  正如明镜网的一篇文章中所形容的那样：“乌兰夫跟当今的习近平的父亲习仲勋的关系，也可以说是铁杆兄弟”。
  <br/>
  <br/>
  “文化大革命”中，习近平到延安进入 “梁家河大学”的那段时间里，他的姐姐习桥桥则是被发配到地处内蒙古巴彦淖尔盟的“生产建设兵团”某部。因为生活和劳动条件恶劣无比，年轻的齐桥桥患上了很严重的关节炎，同时肺结核复发，还得了风湿热，侵犯心脏。习桥桥日后回忆说：“……有时候只能捧起渠里的污水喝。那个时候，痢疾就像伤风感冒一样，是我们的常见病。”
  <br/>
  <br/>
  一九七五年，虽然当时的乌兰夫还被毛泽东和江青下令继续关押在秦城监狱，但他的大女儿云曙碧还有长子布赫，已经因为被周恩来批示为“可以教育好的子女”，被先后从内蒙当地的监狱里释放并“恢复工作”。
 </p>
 <p>
  习仲勋秘书的回忆文章中透露：“由于患病得不到及时治疗，桥桥身体状况很差。1975年，刚刚恢复工作的云曙碧便把桥桥的关系从巴彦淖尔盟，转到她所工作的哲里木盟的科左后旗，并把她接到自己家里休息养病，后来又设法送桥桥回北京治病。”
 </p>
 <p>
  有回忆录说，乌兰夫的长子、布小林的父亲布赫早在1966年就被关押起来。布小林与双胞胎姐姐布大林、哥哥青戈三人，随祖母云亭与大姑云曙碧一起生活。习桥桥被接过去后，小林和大林都和习桥桥成了“无话不谈的好友”，继续了乌兰夫家族与习仲勋家族第二代之间的友好情谊。
  <br/>
  <br/>
  仅从这段两个红色家庭的私人情谊来看，如今的习近平不对布小林另眼相待，自己的亲姐姐习桥桥都不会允许。当然，大前提必须是布小林无条件服从习近平，打着“民族融合”旗号，事实上推行的“全民汉化”政策。
  <br/>
  <br/>
  现如今，布小林到底是本人身体有病，还是对习近平下令在内蒙古强推汉语教学的“民族 融合”政策内心有所保留，她本人在明年中共二十大之后的政治前途已经变得不再明朗。但无论布小林的未来出路如何，因为宪法的规定，沈跃跃在不可能连任三届全国人大委员长的前提下，是否能够成为下届政治局委员中的女性代表，将是明年二十大高层人事安排 的一大看点 。
  <br/>
  <br/>
  当然，已经在高层人事安排方面一再不按牌理出牌的习近平，在明年召开的二十大上干脆不再给新一届政治局里安排女性代表，或者不再给下届国务院内阁里安排女性副总理的可能性，也不是没有。而即使在政治局里继续保留一名女性委员，那么这个女性委员也不见得就一定出任国务院副总理。比如，前面介绍过的刘延东担任了两届政治局委员，第一届是兼任国务院的国务委员，第二届则是兼任副总理。
  <br/>
  <br/>
  更多的这方面的分析内容，留待本专栏的下篇文章继续向读者和听众们介绍。
