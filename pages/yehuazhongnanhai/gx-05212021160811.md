<div id="storytext">
 <p>
  我们在本专栏的上篇文章《习近平的两个核心智囊王沪宁和刘鹤谁左谁右？》里，已经向听众和读者们介绍了中共官媒刊登报道刘鹤的文章《揭秘习近平智囊刘鹤: 为三任总书记起草经济讲稿》。很自然会令人联想到，如今已经身居中共中央政治局常委高位被称之为“三朝国（帝）师”的王沪宁。
  <br/>
  <br/>
  被习近平在公开场合照稿念出的主要高频词中，“民族复兴”是王沪宁先教给胡锦涛，而后又教给习近平的；“中国梦”三个字则是习近平登基之初，王沪宁专为习近平 “发明”的……；至于“一带一路”，其实不是王沪宁的首创，专利权当属陈云的长子陈元。而自习近平上台之后的“经济论述”中，常常出现甚至早已经为全中国人耳熟能详的诸如“顶层设计”、“底线思维”、“新常态”…….等等，原创都是刘鹤。所以中共内部甚至有人讽刺说：习近平就是为给王沪宁和刘鹤念稿子的。
  <br/>
  <br/>
  查维基百科，王沪宁词条中称他为“习近平及前任中共中央总书记胡锦涛和江泽民的主要撰稿人”。这话当然符合事实，但更准确的说法似乎应该是“江泽民三个代表，胡锦涛科学发展观和习近平‘新思想’的真正创立者”。
  <br/>
  <br/>
  而维基百科刘鹤词条中，对其生平介绍的第一句就大错特错，即：“刘鹤亦是中共中央总书记习近平的中学同学和重要财经智囊，被视为习近平派系的重要成员”。该词条还详细介绍说：“刘鹤中学毕业于有不少红二代就读的北京市海淀区一零一中学，与习近平为同一中学毕业生”。
  <br/>
  <br/>
  事实是，习近平从来没有就读过一零一中学。他当年和确实是一零一中学初中毕业的刘鹤既不相识，更没有过半天的“同窗之谊”。
  <br/>
  <br/>
  如果按照维基百科此词条的索引查寻，可见找到最早把刘鹤说成习近平中学同窗的是《端传媒》记者吴婧 2015年9月25日，发自香港的《解密刘鹤 ─ 中国新经济计划的顶层设计师》一文。文章开篇先介绍刘鹤成名之作《两次全球大危机的比较研究》，说是“在亚马逊中国的网站上，刘鹤的书《两次全球大危机的比较研究》只获得15条评论，但10条给了满分五颗星。其中一条评论说：‘懂得人就懂得此书的价值’。”
  <br/>
  <br/>
  文中还介绍，今年63岁的刘鹤与习近平的交往可以上溯至上世纪60年代，俩人同在红二代云集的北京一零一中学读书。建于1946年，一零一中学是中国共产党在革命老区创建并迁入北京的唯一一所中学，1951年经周恩来特批从河北张家口迁至圆明园南侧，生源多为高级干部子弟和烈士孤儿。前国家副主席曾庆红、前人大副委员长李铁映、刘少奇之子刘允若等人，都曾在这所学校就读……。
  <br/>
  <br/>
  按照维基百科的词条索引，能够查找到的第二篇把刘鹤说成习近平中学同窗的报道文章，是《香港经济日报》2017年10月16日刊登的《习中学同窗 “财金国师”刘鹤或走到台前》。文章首段即是“在阅读刘鹤的仕途轨迹时，不能不先了解他与习近平的关系究竟有多深厚 － 原来刘鹤是习近平少时在北京一零一中学的同期同学，两人拥有一段同窗情谊。”
  <br/>
  <br/>
  待刘鹤在2017年10月的中共十九大进入中央政治局，继而又在次年三月，被正式宣布“当选”为国务院副总理之后，中国境外大部分介绍刘鹤的文章都把着重点放在他与习近平的“发小”关系上。比如，当时德国之声的报道文章《少年知交到经济智囊：刘鹤升迁轨迹》中的第一句话，就是“习近平的‘发小’刘鹤本次人大会议上，顺利当选副总理。观察家们普遍认为，这是中国为整顿市场秩序而发出的信号。”文章中说，“刘鹤中学毕业于红二代云集的北京一零一中学，是现任中国最高领导人习近平的同班同学…..。”
  <br/>
  <br/>
  以讹传讹，“ 同期同学”又演变成了“同班同学”。再传下去的话，就可能直接描述成“ 同居长干里，两小无嫌猜”了。
 </p>

  <br/>
  <br/>
  事实上，如上两家中文媒体并不是把刘鹤硬说成是习近平的中学或者“发小”的始作俑者。这两家中文媒体的相关文章作者，应该依据的都是美国《华尔街日报》2013年10月07日的文章《刘鹤的艰巨任务：让中国经济再次腾飞》。文章作者称，“本文基于对国内外诸多政党和政府官员，以及参与改革进程的学者的采访，并且参考了刘鹤所写的文章。刘鹤的办公室在回复《华尔街日报》的电子邮件中说，刘鹤认为，外界对他在中国经济政策制定方面扮演的角色有许多误解。实际上，中国的经济政策是通过一个集体决策体系制定的，任何个人发挥的作用都是相当有限的。”听上去整个为文过程是多么的严谨，但却又毫无根据地说，“但刘鹤更大的权力可能在于他与习近平长期以来的关系。他自上世纪60年代，两人在北京一零一中学上学时就与习近平相识了”。
  <br/>
  <br/>
  我们只要根据中共官方公开的官员简历就可以考证出，目前仍在位、在习近平所有的省部级以上政治亲信，以及高级智囊或核心幕僚中，只有一个人曾经是他的同班同学：就是与刘鹤一同在十九大上，被习近平拉入中央政治局的陈希。但是他们两人不过是工农兵学员时期的大学（普通班）同学，而不是中学同学。
  <br/>
  <br/>
  笔者过去曾有一篇介绍习近平与陈希关系的文章《陈希与习近平既是同窗更是同党》，被网友质疑。说是陈希的官方简历中，清楚介绍他是“清华大学化学与化学工程系催化动力学专业毕业”；而习近平的官方简历中，清楚写明他是“1975-1979年清华大学化工系基本有机合成专业学习”。根本不是一回事。其实“清华大学化学与化学工程系催化动力学专业”，是陈希与习近平一起于“清华大学化工系基本有机合成专业”完成“大学普通班”学业后，返回该校就读硕士研究生时所选的专业。
  <br/>
  <br/>
  回过头来，再说刘鹤当年就读的北京一零一中。该校的“红色基因”始于1946年在张家口建校，校名先后易名为张家口市立中学、晋察冀边区联合中学、华北育才中学、北师大附中二部等。1955年，学校定名为北京一零一中。
  <br/>
  <br/>
  该校当年“高干子女云集”毫无疑问，但并不是唯一一所“高干子女云集”的学校，更不是“中国共产党在革命老区创建并迁入北京的唯一一所中学”。习远平与这所一零一中学更是没有任何关系。习近平当年就读的北京八一学校，即是和北京育英学校、北京育才学校等一样，另外一所在“革命老区创建并迁入北京的”红色学校，前身是1947年3月1日在河北省阜平县向阳庄，由聂荣臻元帅亲手创办的中国人民解放军晋察冀军区荣臻学校。
  <br/>
  <br/>
  1958年8月25日，这所学校由原名八一小学改名八一学校。而就在1965年积极筹建高中的时候，“文革”开始了，八一学校遭到了毁灭性的破坏。直到1979年，《人民日报》刊登了记者纪希晨的署名文章《一场捍卫党的原则的伟大斗争》，“为具有光荣传统的八一学校恢复了名誉”，八一学校由此重获新生。
  <br/>
  <br/>
  2016年9月9日，习近平于教师节之际到他的母校北京八一学校“光宗耀祖”，被中共官媒连篇累牍地高调宣传了好一阵子。
 </p>

  一篇《夜读：北京八一中学是怎样一所学校》的官媒文章介绍说：新中国成立前后，各路解放大军，南下的、西征的、开赴祖国边疆的，以及调入中央和中央军委领导机关的大批干部年幼的孩子，均选择在“八一学校”读书。当时党和国家领导人和军队高级干部，包括毛泽东、周恩来、朱德、任弼时、陈云、邓小平、彭真、董必武、李富春、习仲勋、张闻天、薄一波、刘澜涛、滕代远、吴玉章、徐特立、谢觉哉、彭德怀、贺龙、徐向前、叶剑英、粟裕、黄克诚、罗瑞卿、谭政、肖劲光、王树声、许光达等的子女亲属，也都在这所学校入学……。八一学校管理严格，师资优秀，教学质量高，条件待遇优越。共和国的精英们，纷纷将自己的后代放心地托付给这所学校。
  <br/>
  <br/>
  和大部分当年八一学校的红二代一样，习近平“文革”前也是住校生。该官方介绍强调说：在实行供给制的年代，据说唯有八一学校的学生是享受排级供给标准。这里所说的“排级供给标准”，指的是当时的军队干部中，各级军官的粮食和服装供给标准的初始级。按照当时的中共军队官衔制度，班长是所谓的“兵头将尾”，编制上与士兵无异，排长才算“干部”。也就是说，当年包括习近平在内的所有八一学校的“子弟”们都是吃喝穿、拉撒睡等生活和学习的全部开销，都是和军队干部一样 “国家包干” -- 现在的新名词叫“国库供养”。
  <br/>
  <br/>
  1953年6月出生的习近平，按照他当年老师的回忆，1965年12岁进入初中。说明他习近平是6岁入读小学。他日后的中学同班同学聂卫平等人，都是年长他一岁。
  <br/>
  <br/>
  说到聂卫平，就得从习近平曾经进入过的第二所中学 -- 北京市二十五中说起。按照聂卫平的回忆：那时中学的红卫兵已经分化成好多派，各派之间经常发生一些“派仗”。这时，我们班分来了两个学生，一个叫习近平，一个叫刘卫平。他们原是八一学校的，后八一学校因所谓“高干子弟学校”被解散，他们才被分到我们二十五中。习近平是习仲勋的儿子，他爸爸当时可是有名的大“黑帮”……。当时班上的人都看不起我们，也不敢沾我们。我们也看不起他们，但是和校外的老红卫兵联系很多，这主要是习近平和刘卫平的关系……。
  <br/>
  <br/>
  包括这份聂卫平自己撰写的回忆录一手材料在内，习近平先从景山学校的小学到本校的初中一年级，初中一年级未读完即开始“停课闹革命”；“停课闹革命”及“革命大串联”再加“打派仗”的过程中，学籍又被转到北京二十五中；继而就是“上山下乡闹革命”，那几年经历的所有内容都有公开资料可考，甚至还有习近平回到自己母校八一学校的官方吹捧文章中，更是介绍和描述得不厌其详。而刘鹤早在习近平登基之前的个人文章及著作被介绍时，其简历中也把他中学就读于一零一中的经历标记得清清楚楚。所以笔者倾向于相信，外部世界最早炒作刘鹤与习近平“发小之谊”的记者，有“臆想”之嫌。
  <br/>
  <br/>
  以讹传讹之后 ，几乎所有媒体都是将错就错。还是那份可能是硬把刘鹤与习近平编排成六十年代“同窗发小”的始作俑者《华尔街日报》，日前在它的《中国考虑是否更换中美全面经济对话中方牵头人刘鹤》一文中，再次强调了“刘鹤是中国国家主席习近平少年时代的朋友”，所以“在之前的中美贸易战期间，习近平挑选他牵头与特朗普政府谈判”。说是“与习近平之间的密切私人关系让刘鹤获得授权与美国磋商‘第一阶段’贸易协议；第一阶段协议于去年达成，起到了缓解双边紧张关系的作用。但这项协议也在中国国内招致了一些批评，因其做出了贸易让步而未能让美国大幅降低关税”。
  <br/>
  <br/>
  那么既然压根没有所谓“同窗之谊”和“两小无猜”这回事，习近平与刘鹤之间的情感关系，或者说习近平对刘鹤的非常信任，到底是建立在什么基础之上？这将是我们本专栏下篇文章的内容。
 </p>
