 <p>
  我们《夜话中南海》上期节目刊登和播出的《被全人类声讨的治疆之恶是陈全国晋升政治局常委的最大政治本钱》一文中，向读者和听众们介绍到现任中央政治局委员和政治局常委中，除了不受年龄限制的总书记习近平，共有十一个符合“七上八下”年龄限制的“七上”标准，他们是：王沪宁、汪洋、赵乐际、陈敏尔、胡春华、丁薛祥、李强、李希、李鸿忠、黄坤明、陈全国。
  <br/>
  <br/>
  先要在本文里更正的是，如上名单中漏掉了1955年7月出生的李克强和1955年12月出生，现以十九届中央政治局委员身份兼任北京市委书记职务的蔡奇。
  <br/>
  <br/>
  那么，如果按照中共十六大和十七大时的政治局委员“一刀切”的换届年龄规则，即在党代会召开那年的实足年龄是六十七岁和六十七岁以下的全留 ，年满六十八岁和超过六十八者全下，那么如今的十九届中央政治局不包括军方代表在内的二十三人里，将会在明年二十大上可以留任的总共有十四人，即如上十三人加上习近平。
  <br/>
  <br/>
  我们再假设，二十大上产生的政治局仍然是25人的编制，不包括军方代表是23人。那么可以有九个位置留给政治局里“新来的年轻人”。
  <br/>
  <br/>
  当然，如上只是假设。参考习近平2017年决策十九大高层人事换届的详细内容，有好几个是当年刚满六十七岁，甚至不足六十七岁者或被一退到底，或者被“退居二线”。
  <br/>
  <br/>
  不过摆在习近平面前的最现实问题是，相比于十八大至十九大之间的五年时间里，习近平最看不惯的李源潮和当时因为“柔性治疆”而被习近平赶在十九大召开前一年就提前免职的张春贤，如今的这十九届中央政治局委员里，全部都已经是对他习近平忠心耿耿。正像我们本专栏上篇文章中已经分析过的，真的是手心手背都是肉，留用谁，不留用谁；提拔谁，不提拔谁，实难抉择！
  <br/>
  <br/>
  另外，中共当局每届政治局委员里的新鲜人，大都出自省委一把手 -- 当然军方代表除外。那么我们看看眼下在位的全国三十一个省市自治区的党委一把手，包括本来就是以政治局委员身份兼任的那几个，剩下的还是以五十年代中后期出生者居多。六十后包括陈敏尔在内，只有八个。
  <br/>
  <br/>
  虽然像1961年11月出生的现任黑龙江省委书记张庆伟，1964年8月 出生的现任辽宁省委书记张国清，1962年9月出生的现任浙江省委书记袁家军，以及现任上海市长、1960年3月出生的龚正，现任北京市长、1964年2月出生的陈吉宁等，几乎都是板上钉钉的下届中央政治局委员，但也不太好想象，明年二十大上除了将陈敏尔晋升政治局常委的同时，还会把现在七个六十后的省委一把手全部晋升为政治局委员。所以有足够的理由相信，明年的二十大上产生的所有政治局委员里，只会以五十后为主。
  <br/>
  <br/>
  在2017年10月召开的中共十九届一中全会上产生的新一届中央政治局里，从十八届政治局委员晋升十九届政治局常委的最高年龄者是栗战书，出生于1950年，时年六十七岁。连任政治局委员中的年龄最长者，即女性点缀孙春兰，以及军方代表许其亮，也都是出生于1950年。而新任政治局委员里的年龄最长者也都是出生于1950年，即军方代表张又侠，以及王晨和杨洁篪。

  以此类推，明年二十大上对新任和连任政治局委员，以及连任和升任政治局常委的年龄坎，最年轻也会被设置在1955年。只会有可能以“工作需要”为名向前推一两岁，比如对个别人选放宽至1954甚至1953，但没有可能会把年龄坎降到六十七岁以下。
  <br/>
  <br/>
  也就是说，即使不把年龄坎往上升，过往一直执行的“七上”的基础上，不但现任十九届中央政治局委员中的一九五五年及一九五五年以后出生者，大都会在明年二十大上继续留在一线 -- 或在政治局委员位置上连任，或者由十九届政治局委员晋升二十届政治局常委；将会在明年二十届一中全会上新晋升的政治局委员里，应该也还会有数名五十后，比如现任四川省委书记、1957年4月出生的彭清华，从上海市长调任湖北省委书记、1957年11月出生的应勇等。
  <br/>
  <br/>
  在仍然是六十七岁“封顶，但”对个别因“工作需要”而放宽年龄限制的可能性会实现的假设前提下，最可能的受益者会是哪一个或哪几个，是我们日后文章所要分析的内容。这里先要说的是，在六十八岁封顶的前提下，现任十九届中央政治局常委和委员里的所有一九五五年出生者，全都留任的可能性不是很大。
  <br/>
  <br/>
  我们在过去已经播讲过的文章中介绍过了，中共新华社事后对外披露的十九大上的人事安排，是“参照往届做法，根据党和国家事业发展需要和中央领导机构建设的实际，对推荐人选的范围、年龄和结构提出明确要求……。（但是）党和国家领导职务也不是‘铁椅子’和‘铁帽子’，符合年龄的也不一定当然继续提名，主要根据人选政治表现、廉洁情况和事业需要，能留能转、能上能下。”
  <br/>
  <br/>
  所以，虽然现任十九届政治局委员中的所有“符合年龄者”，对习近平来说“手心手背都是肉”，但仅仅从“事业的需要”角度，总还是要有取有舍的。
  <br/>
  <br/>
  2017年10月下旬，中共央视在十七届一中全会闭幕当天发布一则报道文章 ，标题是《当选党代表、当选中央委员、当选总书记，习近平都是全票》。报道文章中披露，2017年10月召开的中共十九大“新一届中央领导机构人选问题”，是从当年1月，也就是提前10个月左右开始讨论的。在当时的那次常委会专门会议上，“大家一致赞成”习近平提出的，采取谈话调研的方式，就新一届中央政治局、常委会、书记处组成人选，中央军委组成人选以及需要统筹考虑的国务院领导成员人选和全国人大、全国政协党内新提拔人选等，在一定范围内面对面听取推荐意见和建议。而且，谈话调研重在集思广益、统一认识，不限定推荐人数，人选推荐票数作为参考，不以票取人。

  之所以强调了“不以票取人”，是因为“党的十七大、十八大探索采取了会议推荐的方式，但由于过度强调票的分量，带来了一些弊端：有的同志在会议推荐过程中简单‘划票打勾’，导致投票随意、民意失真，甚至投关系票、人情票。中央已经查处的周永康、孙政才、令计划等就曾利用会议推荐搞拉票贿选等非组织活动。”
  <br/>
  <br/>
  如上央视报道中还透露说：从2017年年初开始，习近平先就如何酝酿产生新一届中央领导机构人选问题，听取中央政治局常委的意见。然后从2017年4月下旬至6月，习近平专门安排时间，分别与当时在位的党和国家领导人、中央军委委员，以及“党内老同志“谈话，充分听取意见。当时先后被习近平谈话的总人数是57个。与此同时，根据中央政治局常委会的安排，中共中央相关领导人分别听取了正省部级、军队正战区职党员主要负责人和其他十八届中央委员共258人的意见。中央军委负责人分别听取了时任正战区职领导人和军委机关战区级部门主要负责同志共32人的意见。
  <br/>
  <br/>
  央视和当时的新华社在相关报道中，都透露了这个“征求意见”过程的细节内容，说人在外地有幸被“征求意见”者，都是应召进中南海，被按照谈话调研工作程序，安排充分时间阅读材料，具体是《谈话调研有关安排》、《现任党和国家领导人党员同志名册》和《正省部级党员领导干部名册》。 被“征求意见”者阅读完全如上三份材料，“独立认真思考准备”之后，“中央领导同志以面对面谈话的方式，听取了这位干部关于新一届中央领导机构人选的推荐意见”。
  <br/>
  <br/>
  试想，如果从明年一月开始就二十大新一届中央领导机构人选问题的“听取意见”过程中，每一个被征求意见者面对《现任党和国家领导人党员同志名册》，对其中“符合年龄者”谁去谁留发表自己意见时，具体到比较两个出生于1955年，也就是明年二十大召开时已经年满六十七岁的蔡奇和陈全国谁去谁留、谁上谁下时，认可陈全国的可能性肯定要比认可蔡奇的可能性大得多得多。原因就是我们本专栏的上篇文章的标题所说，“治疆之恶是陈全国晋升政治局常委的最大政治本钱”。
  <br/>
  <br/>
  日前美国之音发表一篇报道文章《自由西藏称西方对中国新疆官员的制裁避开了罪魁祸首陈全国》，说是美国、欧盟、英国和加拿大已宣布就中国新疆人权问题对几名中国官员实施制裁，但总部设在英国的“自由西藏”（Free Tibet）组织表示，他们避开了新疆维吾尔自治区党委书记陈全国。该组织在一份声明中说：“被制裁的官员包括陈全国的直接副手朱海伦和其他几名级别较低的官员，但明显缺少了陈全国本人。”
  <br/>
  <br/>
  声明还说，“自由西藏”组织自去年11月以来，便一直在争取对陈全国实施制裁，因为他是“针对维吾尔族人民暴行的设计者”。

  <br/>
  <br/>
  其实，虽然有外界的专家和外交人士认为，一些西方国家近日宣布的制裁名单上没有陈全国是为了避免发生更大的外交争端。但实际上，虽然陈全国这一次没有被列入欧盟、英国和加拿大宣布的制裁名单，但他早在去年7月就便受到了美国的制裁。时任国务卿蓬佩奥也在同日发表声明，对侵犯人权的陈全国等中共高官施加制裁和签证限制，这些官员及其直系亲属被禁止入境美国。
  <br/>
  <br/>
  为此，当时的新华社奉命播发了对陈全国及其手下几个一起被制裁者的“接受记者采访”，个个都强烈表示“以此为荣”。
  <br/>

  众所周知，当前“新疆棉”背后的人权问题在全世界广受关注的对立面，是中共政权借机掀起的又一轮“爱国热潮”。而前一段时间已经鲜少对外发言的陈全国则再次被中共当局安排对外发言，特别为他提供了上海合作组织秘书长和有关国家使节访问新疆的机会，以让他借机对外公开发言，抨击国际上“一些势力”抹黑新疆形象是“企图破坏”他本人创造出的“新疆安全稳定”的大好形势。
  <br/>
  <br/>
  回想当年，在经济大权已经由朱镕基独揽的前提下，李鹏为什么还会连任一届国务院总理？继而又在任满两个整届总理职务的前提下，还能继续留任政治局常委并改兼全国人大委员长？最重要因素，就是他李鹏是“六四”镇压的标志人物。
  <br/>
  <br/>
  同样道理，无论陈全国的水平高低、能力大小，未来二十大上留用他甚至提拔他绝对是“事业需要” -- 习近平对少数民族施行铁血政策的“事业需要”！
