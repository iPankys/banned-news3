<div id="storytext">
 <p>
  我们《夜话中南海》专栏上周五播讲的文章《李克强修订简历暗含对习近平学历的不屑？》中，已经向读者和听众介绍了习近平2012年10月在中共十七届一中全会上进入中央政治局常委会，又在次年三月的全国人代会上接替了曾庆红的国家副主席职务之后 ，中国境内的官方媒体曾掀起过一场对中共官场上，以习近平最为典型的“论文博士”的舆论围剿。
  <br/>
  <br/>
  除了我们本专栏上篇文章中已经引述过的，至少还有一篇当时《中国青年报》的文章吐槽更狠，说是现在的官员们一窝蜂地“攻读博士”，过去的以权谋私只在金钱、职位和美色上，现在又扩展为以权谋学位，不仅要权财色应有尽有，更要用高学历的知识标志来包装，这样才算真正的功德圆满。至于由此造成的学术腐败则不在他们的关心之列，反正腐败已经深入和普及到所有领域，高校和学术界有什么理由保持清高！
 </p>
 <p>
  该文揭露说：官员们读学位、拿博士文凭决不会“十年寒窗苦”，而是轻松加愉快，只要报上名，被学校列入博士生名单就算大功告成。他们既不用参加统一的正式考试，也不用按时上课和埋头于学术，更不用为毕业论文发愁。党和国家的大事都忙不过来，哪有时间准备考试、上课、读书、写论文？毕业时，想找个操刀手简直太容易了，有太多的饱学之士愿意效劳，做出一篇优秀的博士论文。至于学术尊严和学者良知，怎么能比得上权力好用！
  <br/>
  <br/>
  当时，人民网也加以转载的一篇标题为《图说中共中央政治局常委和他们的老师》的吹捧文章，有关习近平的描述内容是：时任福建省委副书记、外出均有秘书陪同的习近平，每次到北京跟博士论文指导老师刘美珣讨论论文，都会请秘书离开。
  <br/>
  <br/>
  该文还详细介绍说：“习近平就读的这种在职博士与普通博士略有不同，一是入学不需要参加全国考试……，二是不需要在课堂上听课，学校会对每门课程进行教学录音，然后寄送给学生。”
  <br/>
  <br/>
  讽刺的是，正是在如上吹捧文章发表之前 ，新华社刚刚发表过我们上篇文章已经介绍过的标题为《“论文博士”是为何人量身定做》的评论员文章，揭露所谓“论文博士”的特殊“培养”制度，根本就是为官员量身定做……。
  <br/>
  <br/>
  笔者在本专栏上篇文章中的一句，“在如上当时中国境内公开发表的文章内容，让人读起来的感觉，就是句句都是在讽刺“法学博士习近平”。被转载笔者文章的某网站读者反问了一句，“何以见得不是在讽刺李克强？”同时也有另一位读者认为，“一样都是‘论文博士’，拿李克强比习近平是五十步笑百步……”。这位读者的评论令笔者想起了几年前曾读到的一则网评，大意是说，习近平的博士论文至少篇幅足够，不像李克强，就凭一篇十七页长的文章就能拿到经济学博士学位。
 </p>
 <p>
  <figure class="image-richtext image-inline captioned" style="width:620px;">

  <br/>
  <br/>
  其实，网上传播的那份李克强公开发表过的论文《论我国经济的三元结构》，确实只有17页的篇幅，但只是他发表在《中国社会科学》1991年03期上的一篇文章。该文章虽然日后获得了1996年度(第七届)孙冶方经济科学奖的论文奖，但在发表的当时，连李克强的硕士论文也不是。李克强的经济学硕士论文的题目是《农村工业化：结构转换中的选择》，论文指导教授是现已经去世的北京大学经济学院的另外一位教授萧灼基先生。而李克强1996年完成的博士论文主标题是《论工业化进程中的结构转换》，副标题是&lt;国际比较与中国的经验&gt;。
  <br/>
  <br/>
  根据中国国家图书馆学位论文收藏中心提供的资料显示，李克强的学科专业为政治经济学，研究领域为社会主义经济理论，学位授予单位为北京大学。他的学位授予日期1994，论文页码总数183页，馆藏号为bslw/1998/f40/6.。
  <br/>
  <br/>
  “根据国务院学位委员会的规定，博士学位论文通过后，必须提交一份给中国国家图书馆，由中国国家图书馆学位论文收藏中心收藏整理”。一位详细介绍如上内容的中国境内作者奇怪，“在国家图书馆学位论文收藏中心，查不到习近平的博士论文，不符惯例。所以使人怀疑，他并没有通过博士论文，或者根本没有提交博士论文。”
  <br/>
  <br/>
  当然，习近平的“博士论文”确实还是有的，虽然不知什么原因未在国家图书馆学位论文收藏中心被收藏，但在多年前还是可以在清华大学的图书馆里被找到的。
  <br/>
  <br/>
  大概是2014年年中的事情 ，署名是“新京报首席记者关庆丰”的文章介绍说：在清华大学图书馆的学位论文阅览室，可以查询到一篇十三年前的博士论文—《中国农村市场化研究》，全文共169页，淡黄色封皮上的内容是：申请清华大学法学博士论文/培养单位：人文社会科学学院/专业：马克思主义理论与思想政治教育/研究生：习近平/2001年2月。指导教师为刘美珣。
  <br/>
  <br/>
  不过，也就是因为习近平进入政治局常委之后，中国境内的官方媒上都曾围剿过“官员读博泛滥成灾”的现象，有好事者想在清华大学图书馆里一探究竟却无功而返。为此，中国政法大学教授杨帆建议公开中国大陆上所有博士论文以辩真伪，立刻被外界认为是剑指习近平 。
  <br/>
  <br/>
  当时被在中国境内广为转发的一则微博中，这位杨帆教授说，“现年60岁左右的博士们，他们的博士论文都没有公布，授予他们博士的单位也不允许其他人去查，不知是根据什么规定？”杨帆建议，中国社会科学院、教育部、中国科学院要承担取消假学位的责任，法院也应该受理这样的诉讼，“要允许公众去自由查阅所有博士论文，依靠群众进行检举”；“这些二三十年前的博士们，许多人早已身居高位，捞足了利益......。” 杨教授指出：“不懂存量，就不能禁绝增量。不公平的事情，在历史、人心里，总是站不住脚的。”
 </p>
 <p>

  <br/>
  杨帆呼吁说：“所有的论文都是公共产品，是获得个人升迁的重要条件，绝对不可以有造假”，“所有博士论文，必须在教育部‘知网’等网站公布。博士学位授予单位，应公开所有博士论文，允许公开查询、复制，不得为他们保密，保密就说明有鬼。”
  <br/>
  <br/>
  杨帆此言一出，北京知名学者荣剑立刻发表评论：“中共现在有两个不敢公开，一个是财产不敢公开，一个是博士论文不敢公开。可以断言，凡是当官期间获得的博士学位，十有八九是找人代笔，那个代笔的十有八九是抄别人的或抄自己的。因此，只要审查博士论文，肯定露馅。杨帆建议审查40年来所有博士论文，打死他们也不敢实行。官员博士论文已成了国家机密。”
  <br/>
  <br/>
  笔者在北京的一位朋友说，他为此专门去了一趟中国国家图书馆学位论文收藏中心，发现李克强的博士论文也查不到了。至于李克强的那篇《论我国经济的三元结构》，因为1991年公开发表的文章，网上自然还能查找到全文。而且笔者也发现，最早这篇说成是李克强博士论文的文章竟然是党的高级“飞盘手”胡锡进。
  <br/>
  <br/>
  2013年5月，胡锡进的《环球时报》刊登了一篇翻译文章，标题是《中国城镇化的推进方向》，其内容中把李克强的《论我国经济的三元结构》说成是他的博士论文，并且用此文与习近平的《中国农村市场化研究》作对比。
  <br/>
  <br/>
  从那以后，无论是海内还是海外，都以讹传讹，越传越错。
  <br/>
  <br/>
  笔者第一次在本专栏里介绍习近平自2007年10月，在进入中央政治局常委会之后就把自己此前简历中，“大学普通班毕业”的表述删除是2012年10月中共十八大开过的当年底。当时笔者注意到，“满进文武”尽是“在职博士”和“在职硕士”的情况下，新任政治局常委兼中纪委书记王歧山居然是“出污泥而不染”，“众人皆醉我独醒”，坚持在自己的简历中介绍自己只是“大学普通班学历”。与在简历中刻意隐去自己“大学普通班毕业”，以突出自己“法学博士”学历的习近平相比，王歧山多了一份坦诚，习近平则少 了一份自信。
  <br/>
  <br/>
  但是，也有中国内地的读者认为，笔者忽略了王歧山简历中突出的“高级经济师”职称的介绍内容。这位读者认为，无论李克强的经济学硕士和博士学位有没有水份，无论他的硕士和博士论文的含金量是多是少，他在担任国务院副总理职务之前的工作和任职单位履历，是没有资格审请从而获取高级经济师职称的。
  <br/>
  <br/>
  笔者为此特别上网查对了中国内地的高级经济师职称考试审请资格，其中学历及资历的要求是：
 </p>
 <p>
  （一）研究生毕业并获得博士学位，取得经济师或相关职（执）业资格满2年，从事经济工作2年以上。
  <br/>
  （二）研究生毕业并获得硕士学位，取得经济师或相关职（执）业资格满4年，从事经济工作5年以上。
  <br/>
  （三）大学本科毕业，取得经济师或相关职（执）业资格满5年，从事经济工作9年以上。
 </p>
 <p>
  <figure class="image-richtext image-inline captioned" style="width:5000px;">

  <br/>
  <br/>
  那么，即使王歧山的“大学普通班毕业”不被认可“相当于”1978年初以后陆续入学的大学本科毕业，也还有如下规定为他放行，那就是：
 </p>
 <p>
  （四）不具备上述规定学历，须通过考试取得经济师资格或相关职（执）业资格满5年，其中大学专科毕业须从事经济工作11年以上，中专毕业须从事经济工作15年以上……。
 </p>
 <p>
  如此说来，王歧山从上个世纪八十年代初至1997年，出任广东省委常委之前的至少15年的工作经历全部都是在直接的经济理论和金融主管部门，资历之深弥补了学历不足。而相比于王歧山，基础学历连“大学普通班”还不如的现任十九届中央政治局常委兼国务院第一副总理韩正的“高级经济师”职称，就要被打上一个大大的问号了。
  <br/>
  <br/>
  1954年4月生人的的韩正在简历中，介绍自己是华东师范大学国际问题研究所国际关系与世界经济专业毕业，在职研究生学历，经济学硕士学位，高级经济师。
  <br/>
  <br/>
  但其简历中按年代排序的内容，对他1975年，也就是21岁之前的生平内容完全没有介绍：从1975年之后，他从一个仓库管理员和供销股办事员干起，一步步爬升至工厂团总支副书记，再到工厂团委负责人和上海市化工局团委书记，期间参加了为期两年的复旦大学大专班的不脱产学习，并凭此“学历”成为一个中等专科学校的党委副书记；而又在华东师范大学开办的“夜大学完成了两年时间的不脱产学习”，并凭此“学历”成为中共上海团市委的负责人……。
  <br/>
  <br/>
  日后在担任上海团市委书记和中共上海市卢湾区委副书记、区长期间，他又是靠所谓“在职学习”，拿到了华东师范大学国际问题研究所国际关系与世界经济专业的“经济学硕士”学位。
  <br/>
  <br/>
  先不说朝韩正这一生中，连半天正规校园生活都没有经历过的经济学硕士的“学历”水分如何，即使他的经济学硕士学位是货真价实，那么他获此学位之后的经历一直都是上海市委的党务和政务工作，就是把他在担任上海市政府副秘书长期间，兼任的市计划委员主任和市证券管理办公室主任的工作内容算成是“从事经济工作”，那满打满算也还不足两年。
  <br/>
  <br/>
  所以，唯一的“合理”解释就是，韩正把他自己担任上海市长的工作经历也算成了“从事经济工作”，不然就满足不了报考高级经济师的资历要求。后续的分析内容，留待下篇文章继续向听众们介绍。
 </p>
