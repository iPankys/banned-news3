 <p>
  我们《夜话中南海》上期节目刊登和播出的《重用和提拔陈全国是习近平的“事业需要”》一文中，已经分析过了明年十月召开的中共二十大上，现任十九届中央政治局委员、目前在位的所有省市自治区党委一把手中的八个六十后之一陈敏尔，进常委是十拿九稳。而除他之外的另外七个省级党委一把手中的六十后，不可能全都进入二十届中央政治局。比如，半个月前刚刚被任命为甘肃省委书记的尹弘。
  <br/>
  <br/>
  这位尹弘是1963年6月生人，本是“土生土长”的上海籍干部，究其仕途轨迹很明显是长期得益于韩正，也是因其韩正主政上海时的上海市委秘书长出身，而被认为“前途无量”。但若拿他与除陈敏尔之外的其他几个六十后省委书记做比，如今仅仅是十九届中央候补委员的尹弘还是资历太浅。比如，1961年11月出生的现任黑龙江省委书记张庆伟，到目前已经是连任四届中央委员（十六，十七，十八和十九届）；再比如1964年8月 出生的现任辽宁省委书记张国清，也已经是一届中央候补委员（十七届）和两届中央委员（十八和十九届）；还有1962年9月出生的现任浙江省委书记袁家军，也已经是一届中央候补委员（十七届）和十九届中央委员。
  <br/>
  <br/>
  除了因为仅仅是十九届中央候补委员，这个尹弘如今没有被从河南省长位置上直接晋升为该省委一把手，接替还有两个月就年满65岁的现任省委书记王国生，而是被安排到一个极不重要的省份甘肃担任一把手。虽说也是升迁，但似乎不是一种为晋升副国级而政治热身的安排方式。
  <br/>
  <br/>
  其实，到目前为止的所有中共六十后省委一把手中，除了尹弘和景俊海只是十九届中央候补委员，其他几位都是十九届中央委员。而如果说，仅仅是十九届中央候补委员的现任省 级一把手中的六十后们，也不是全都没有在明年召开的二十大上“一步登天”的可能的话 ，那景俊海肯定是比尹弘看好。
  <br/>
  <br/>
  去年11月，由吉林省长就地升任省委书记的景俊海在习近平进入中央政治局常委会的2007年，还仅仅是陕西省的一个副地市级干部。习近平以政治局常委和书记处书记身份从曾庆红手中接过中央组织大权之后没几个月，景俊海就被跃升为陕西省副省长。
  <br/>
  <br/>
  习近平接任总书记位置前几个月，景俊海又被安排为陕西省委常委兼宣传部长，从此开始受到外部媒体的关注……。
  <br/>
  <br/>
  去年11月下旬，当景俊海被从吉林省长位置上就地提拔为省委书记的消息被公布后，反应最快的中国青年报记者故意以《三年三次职务调整，中候补景俊海任吉林省委书记》为题，向读者暗示这位景俊海的“与众不同”。报道文章中说：全国31个省份的省级党委书记中，景俊海是唯一一位十九届中央候补委员（当时的尹弘还只是河南省长）。2017年至今，景俊海职务已经三次调整，先后担任了北京市委副书记、吉林省省长和吉林省委书记。
  <br/>
  <br/>
  文章详细介绍，这个景俊海是1992年才从陕西西安一所高校的教师岗位上转轨从政的，此后用了16年时间晋升西安高新区管委会主任，其间曾短暂担任西安市计委主任；2005年成为西安市委常委。2008年1月，不满48岁的景俊海出任陕西省副省长，晋升副部级。
  <br/>
  请读者注意，西安市委常委只不过是一个副厅局级干部，而景俊海2008年从西安市委常委晋升陕西省副省长是被越级提拔，这种情况在最近十来年里是很少见的。
 </p>

  中青报的报道文章中还介绍说：2012年5月，景俊海入列陕西省委常委，任宣传部部长后，多次在央媒发表文章，属于刊文较为频繁的地方省委常委。例如，2014年7月，景俊海在《求是》刊文《始终与党中央保持高度一致》；2015年2月，他在《求是》刊文《弘扬柳青精神扎根人民生活》；2015年5月，他在《光明日报》发文《建设新型主流媒体不断增强舆论引导公信力影响力》；2015年6月，他在《求是》刊文《中华优秀传统文化是实现中国梦的智慧宝库》。
  <br/>
  <br/>
  2015年6月，景俊海任中央宣传部副部长。当时，海内盛传习近平的目的是要让景俊海 以中宣部副部长名义兼任中央电视台台长。但事实上，景俊海在中宣部副部长位置上坐了不足两年，即于2017年4月从中央回到地方，担任北京市委（专职 ）副书记。在北京任职8个月之后，2017年12月底，景俊海调任吉林，任吉林省委副书记，并于次年1月初任吉林代省长，1月底当选省长，正式晋升正部级。
  <br/>
  <br/>
  中青报的文章特别强调：此番任吉林省委书记已经是景俊海自2017年以来，三年多时间内的第三次职务调整。根据《吉林日报》的报道，这次职务调整之前，景俊海刚刚从北京返回吉林，而被景俊海顶替的前任吉林省委书记才刚满65岁。
  <br/>
  <br/>
  景俊海何德何能被中共高层如此器重地频繁调动，一步步重用？总部在北京的多维新闻网在景俊海由北京市委副书记位置上调赴吉林之初，接连发表《 景俊海调任吉林 中共西北系“黑马”突起》和《景俊海出任吉林省省长 引外界关注》两篇评论文章，回答了这个问题。头一篇强调了“陕西是中国西部大省，尽管对于习近平而言颇为‘特殊’，其父习仲勋曾在此起家，又是习本人下乡插队当知青的地方。但中共用人有其成熟规则，除去个体间的部分差异，西北干部之所以集体势起，来自于他们的独特优势”；“ 2012年5月主管陕西省委宣传部后，景俊海曾主导了修建习近平父亲习仲勋在陕西的陵园。”
  <br/>
  <br/>
  后一篇则进一步提醒读者，“从西部提入中央，再被空降到北京，一时间，仕途顺风顺水的景俊海成为外界关注的焦点。有观点认为，景俊海仕途快速升迁获益于其身上的‘陕西’标签——因为他主管陕西省委宣传部期间，曾一手策划并推动中共现任总书记习近平父亲的墓园扩建成陵园……。”
  <br/>
  <br/>
  《环球实报》在其“中南海观察”栏目里，刊登《扩建宣传习仲勋陵园有功 景俊海出掌央视》一文中揭露：景俊海能进入习近平的慧眼，被从西部提入中央出掌重要的喉舌，是因为他从2012年5月主管陕西省委宣传部期间，有“先见之明”，一手策划并推动把习近平父亲的墓园扩建成陵园，不但规模较之前扩大数十倍，而且规格也大大升级，成为“陕西省爱国主义传统教育基地”。每年当局都组织悼念活动，其声势不亚于北京人民大会堂的毛泽东纪念堂。
  <br/>
  <br/>
  海外媒体引陕西当地网民指，习仲勋故居是2005年启动重建的，但直到2012年习近平准备上台时，当地政府才开始大规模扩建工程，至2015年完工。有负责故居防水工程的公司在官网写道：“习大大，您安心治理天下，带领我们的国家走向昌盛繁华，我们许诺定会让习仲勋故居翻修后滴水不漏，再无后顾之忧！”
  <br/>
  <br/>
  而习仲勋陵园被扩建之后的规模有多大，其实也是那家总部在北京的多维新闻网最先报道出来 。
 </p>
 <p>

  <br/>
  <br/>
  2014年中国传统春节期间，多维新闻记者实地探访了习仲勋故里陕西省富平县，“重点拜谒了被定为革命教育基地的习仲勋陵园”。该记者在报道文章中记述：园区内景布局讲究，坐西北面东南的座像，象征着习仲勋一生的革命轨迹。在出高速路进入富平县省道106的不远处，伫立着一个明显的地标牌，上面写着“习仲勋故里”几个大字。地标牌掩映在松柏绿植之间，虽然已是隆冬又临高速，常年来往的车辆呼啸不止，但周围的松柏仍保持着郁郁葱葱。按照当地人的说法，这是预示着习老的永垂不朽。习仲勋陵园坐落在温泉河生态园之内。有资料显示，习仲勋陵园总占地约十余亩，整个生态园的面积大约将近四百亩，工程分两期建成。
  <br/>
  <br/>
  该记者的报道文章语带讽刺地介绍说：游客随身携带的背包不论大小都必须要寄存，这与毛泽东纪念堂的要求保持了一致。此外，游客寄存时使用的是指纹录入对比的存取包保管柜，这也保障了物品的绝对安全。令很多人意外的是，尽管已经登记了身份证、存了包，入园仍然要接受检查金属感应器的扫描。游客随身携带的物品，特别是手机等电子产品都要拿出来放到桌子上，以便不影响金属感应器的检查效果。以至于有人不由感叹：“这比机场安检还严格啊！”
  <br/>
  <br/>
  多维的报道文章中还详细介绍说：从正门进入后，是一个长300多米的宽敞大道，两边挂满了代表新年的红灯笼。春节期间挂红灯笼是中国传统习俗，寓意着红红火火，吉祥如意。但如此大范围地“点缀”，在其他陵园并不多见……。入园后，远远可见习仲勋雕像坐落在青山翠柏中。根据工作人员介绍，陵园内植有侧柏、龙柏、白皮松、火球等。其坐西北、向东南的布局，颇有政治寓意，风水极其讲究，生态园内景观规划和打理都显出用心。据园区管理人员透露，陵园内种植的一草一木，位置在哪里都是很有讲究的，“虽然不是迷信风水，但也基本上没能脱离开风水”。 陵园南有温泉河，北有桥山，占地面积10余亩，这些都跟风水学有关。
  <br/>
  <br/>
  在这类描述的字里行间，该记者还突兀地加进了一句“虽然其子习近平十八大后就任中共总书记，但习仲勋简朴的‘家风’一直延续到了现在”，给人的感觉满满都是讽刺，也令笔者不由得想起习近平常常来用痛斥下属的那句“不信马列信鬼神”。
  <br/>
  <br/>
  按照《环球实报》评论文章的说法：对比中共已故党国领导人故居陵园，只有邓小平在四川广安的故居纪念园面积比习仲勋大。而据网民统计，孙中山的陵园长不过700余米，最宽处不过300余米，大小是习仲勋的陵园九分之一；而明太祖朱元璋的孝陵长不过一公里，最宽处不过500余米，是习陵的四分之一左右。
  <br/>
  <br/>
  习仲勋陵园的扩建工程第二期完成于2015年年中。工程刚刚完成，景俊海就被中组部任命为中央宣传部副部长。
  <br/>
  <br/>
  两年前，对景俊海一直进行追踪报道的多维新闻网即已经在《吉林省长景俊海言论惹争议 仕途一直顺风顺水》一文中断言：从西部提入中央，再被“空降”到北京，一时间，仕途顺风顺水的景俊海成为外界关注的焦点。如今又从北京转岗吉林，成为二把手，未来仕途可佳……。可以大胆推测一下，这位陕西籍干部的未来非常可期。从年龄上来讲，58岁的景俊海尚不满60岁，未来还是很有想象的空间。
  <br/>
  <br/>
  而依笔者判断，未来的中共内宣和大外宣总管，明年二十大上将产生的新一任以中央政治局委员和中央书记处书记身份兼任中宣部部长的那个人，很可能就是景俊海。更详细的分析内容，留待下篇文章继续道来。
