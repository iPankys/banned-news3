<div id="storytext">
 <p>
  我们自由亚洲电台《夜话中南海》节目的上篇文章《温家宝的谢幕哀歌与习近平的文革复辟》中回顾了十年前，习近平以中共中央政治局常委、国家副主席和党的总书记接班人身份，专程到重庆为薄煕来和王立军的“唱红打黑”站台时，欢呼雀跃的“毛左”们纷纷发文称颂，声称在习近平给予重庆的掌声当中，可以预见的是重庆的某些好做法或将作为“样板”在其他部分省市，乃至全国广泛推行。
  <br/>
  <br/>
  现如今，借庆祝建党百年之际，由习近平亲自批准，由中央宣传部组织，中央党史和文献研究院等部门编写，由人民出版社、中共党史出版社联合出版的2021新版《中国共产党简史》刚刚面世。“毛左”媒体民族复兴网立即刊发了同一位“毛左”，即中央民族大学张宏良的一篇文章，标题为《新版党史大突破 于无声处听惊雷》。
  <br/>
  <br/>
  按照张宏良这位中国大陆“毛左”领军人物的说法：这个习近平版《党史》在肯定毛泽东在中国革命中领导作用等一系列问题上，与以往旧版本相比有了质的突破。而最重大的突破是，40年来，第一次全面肯定了从建国到文化大革命结束这27年的伟大成就。
  <br/>
  <br/>
  张宏良认吹捧说：新版《党史》是对中国共产党建党100周年的一份献礼，是中国意识形态领域的一个重大突破，是21世纪中华民族实现伟大崛起的最重要一个历史跳板，其重大意义必将会载入中国崛起的史册。
  <br/>
  <br/>
  张宏良认为：“在过去所接触到的，对中国具有敌意和偏见的所有西方记者中，几乎都把‘文革浩劫’作为攻击中国的火力点；中国国内极右势力更是把‘文革浩劫’当作是反共反社会主义的法宝。如今，新版《党史》彻底砸碎了这个浩劫枷锁，文革不是浩劫，不是动乱，文革期间取得的一系列伟大成就举世瞩目，天地可鉴。”
  <br/>
  <br/>
  张宏良口中的中国国内极右势力，无疑就包括了温家宝在内。我们本专栏上篇文章中已经回顾了，当年正是温家宝在政治局和它的常委会内，坚决不肯随波逐流，拒绝到重庆为薄煕来的文革复辟站台，也因此成为以张宏良为代表的毛左势力们的眼中钉，肉中刺。
 </p>
 <p>

  <br/>
  早在温家宝还是政治局常委和国务院总理，习近平正在等待接班，已经以“副总书记”自居的2010年，这个张宏良即敢于在中国大陆公开发表文章《美国媒体对中国两个领导人截然相反的不同态度》。文中说:“推荐大家关注有关西方国家舆论的这篇文章，不是对刘晓波有兴趣，而是想让大家关注在西方国家引起截然不同反响的两位中国领导人，一位是温家宝总理，一位是习近平副主席。”
  <br/>
  <br/>
  这篇当年出笼即被禁，习近平上台之后又被张宏良重公开发表于2015年7月，中国大陆当前最主要毛左理论宣传阵地《民族复兴网》上。文章中说：“今年（2010年）以来，美国连续四次把温家宝总理评选为‘拯救美国经济十大功臣（唯一外国人）’、‘当今世界十大最重要领导人（居于首位）’、‘当今世界十大最受尊敬领导人’，最近又把温家宝总理选为美国《新闻周刊》封面人物，在不到一年时间里连续四次把温家宝总理推上荣誉巅峰。”
  <br/>
  <br/>
  “可是”，张宏良的文章鲜明比对比说：“对于同样是中央政治局常委的习近平副主席，西方媒体则表现出了截然相反的极端情绪。习近平刚刚被选为中央军委副主席，只是发表了一个让中国人扬眉吐气的纪念抗美援朝讲话，就遭到了西方媒体的恶毒攻击。韩国媒体甚至要求习近平副主席向韩国道歉，理由就是习近平把抗美援朝说成是正义战争。众所周知，韩国和日本完全是美国的小三儿，如果说日本是美国的小妾，韩国就是美国的二奶，没有美国撑腰，就算借给韩国100个胆子，它也不敢向中国挑衅。”
  <br/>
  <br/>
  张宏良在其文章的最后提醒道：“我们暂且不理会韩国挑衅的背后因素，而是应该认真关注和思考美国等西方媒体对中国两位领导人截然不同的极端反映：对温家宝总理极端推崇；对习近平副主席极端贬低。美国等西方国家为什么要这样做？权且作为一道世纪性思考题，请关心祖国命运的人们认真考虑判断。”
  <br/>
  <br/>
  在张宏良这篇文章后面的网友评论中，充斥着“记住毛主席说的话：凡是敌人反对的，我们就要拥护；凡是敌人拥护的，我们就要反对”；“一个是维护中华民族利益的民族主义者！一个是中国历史上最大的汉奸。卖国贼和中国共产党最无耻的叛徒”；“一个是维护中华民族利益的民族主义者！一个是中国历史上最大的卖国贼”之类的捧习骂温内容。
  <br/>
  <br/>
  在中国大陆理论界，尤其左派人物最善于拉大旗做虎皮，喜欢对外标榜自己的“背景特殊”或者“上面有人”，张宏良也不能免俗。十年前笔者就在网上看到过，他张宏良的文章很受习近平重视的所谓“内部人士透露”。虽然笔至今认为，这很有可能是张宏良自己对外“透露”的所谓“内幕”，但也不得不承认很可能是张宏良的文章提醒了习近平，以至习近平在登基次年，温家宝刚刚辞去国务院总理职务才几上月的时间，就迫不及待地在一篇内部讲话中不点名地批判温家宝的“故做开明姿态”，以“赢得海外各种舆论的好评”；同时还要求全党上下都要像他习近平一样，“不怕被污名化”。
 </p>
 <p>

  <br/>
  <br/>
  习近平在他那篇臭名昭著的“八一九”内部讲话中说：“我们不搞无谓争论，但牵涉到大是大非问题，牵涉到政治原则问题，也决不能含糊其辞，更不能退避三舍。‘千呼万唤始出来，犹抱琵琶半遮面’是不行的！领导干部要敢于站在风口浪尖上进行斗争。我曾经说过，领导干部不能搞‘爱惜羽毛’那一套。有些干部对大是大非问题绕着走，态度暧昧，独善其身，怕丢分，怕人家说自己不开明。这是万万要不得的！这是什么羽毛？这是什么形象？故作开明姿态嘛！你在某些人那里形象好了，在广大党员、干部心中的形象就差了，在人民群众心中的形象就差了。战场上没有开明绅士，在大是大非问题上也没有开明绅士，就得斗争。作为党的干部，不要去想博得社会各种人的喝彩、赢得海外各种舆论的好评。只要站在党和人民立场上，坚持原则，就不可能取得这样的结果。为了党和人民事业，我们共产党人连流血牺牲都不怕，还怕损失一点蜗角虚名吗？我们党也不会以这种虚名来评价干部。作为党的干部，不能用‘不争论’、‘不炒热’、‘让说话’为自己的不作为开脱，决不能东西摇摆、左右迎合！在事关党和国家命运的政治斗争中，所有领导干部都不能作旁观者。今后，谁再围攻我们的同志，我们宣传思想部门要发声，党委要发声，各个方面都要发声！要发出统一的明确信号，形成一呼百应的态势，不要怕被污名化。我常常讲干部要敢于担当，这就是一个重要检验。”
  <br/>
  <br/>
  由此可见，习近平与温家宝之间结下梁子，事实上是从温家宝还是国务院总理，而习近平还是以政治局常委身份分管党务，等待接班的那五年，也就是中共十七大至十八大之间的那五年时间里就开始了。
  <br/>
  <br/>
  在温家宝国务院总理的第二个任期的后半段，具体大概是从2011年以后，温家宝即不断在公开场合为政治体制改革，特别是为党和国家领导制度的改革大声疾呼。何以至此？以余杰先生为代表的右派评论界认为，他纯粹是在做秀。但事实上，当时有一个重要的内部政治背景，就是已经分管了整个中共中央的党务工作、以政治局常委身份兼任了中央党校校长的习近平一朝权在手，便把令来行：主持意识形态工作之后的第一个向左转的明确信号，就是对宣传邓小平的“党和国家领导制度改革”设置障碍。
 </p>
 <p>

  笔者在九年前的一篇相关文章中，即已经介绍过：习近平已经是中央政治局常委，而且是分管党务，兼任中央党校校长时期的2010年8月，北京曾有一个小范围的“纪念小平同志《党和国家领导制度的改革》发表三十周年研讨会”，会上即有中央党校传出来的话说，近平同志对这个纪念活动不感兴趣，所以纪念会和研讨会一定不能高调，也别指望中央主要媒体会大力宣传……。
  <br/>
  <br/>
  习近平登基后，政治体制改革逐渐成了宣传“禁词”，邓小平的“党和国家领导制度的改革” 日益成为学习宣传和理论研讨禁区。有内地网友说，他们相信已经有过较高层级的内部传达，要求对邓小平生前讲过的关于政治体制改革的内容因为有讽刺和影射习近平之嫌，而“不能再宣传”。最有代表性的是如下几段：
  <br/>
  <br/>
  “权力不宜过分集中。权力过分集中，妨碍社会主义民主制度和党的民主集中制的实行，妨碍社会主义建设的发展，妨碍集体智慧的发挥，容易造成个人专断，破坏集体领导，也是在新的条件下产生官僚主义的一个重要原因。权力过分集中的现象，就是在加强党的一元化领导的口号下，不适当地、不加分析地把一切权力集中于党委，党委的权力又往往集中于几个书记，特别是集中于第一书记，什么事都要第一书记挂帅、拍板。党的一元化领导，往往因此而变成了个人领导。”
  <br/>
  <br/>
  “要着手解决党政不分、以党代政的问题。中央一部分主要领导同志不兼任政府职务，可以集中精力管党，管路线、方针、政策。这样做，有利于加强和改善中央的统一领导，有利于建立各级政府自上而下的强有力的工作系统，管好政府职权范围的工作。”
  <br/>
  <br/>
  “如果一个党、一个国家把希望寄托在一两个人的威望上，并不很健康。那样，只要这个人一有变动，就会出现不稳定。”
 </p>
 <p>
  <br/>
  “一个国家的命运建立在一两个人的声望上面，是很不健康的，是很危险的，不出事没问题，一出事就不可收拾。”
  <br/>
  <br/>
  “我历来不主张夸大一人的作用，这样是危险的，难以为继的。把一个国家，一个党的稳定建立在一两个人的威望上，是靠不住的，很容易出问题。所以要搞退休制。”
  <br/>
  <br/>
  不难看出，四十年前邓小平的如上针对毛泽东时代政治体制弊端的批判和否定内容，都是如今习近平已经全面复辟甚至变本加厉的内容。而正是因为习近平在他作为总书记接班人而进入中央政治局常委会的第一个五年里，已经在从政治体制改革，特别是“党和国家领导制度的改革”的角度，否定邓小平有所动作，所以当时即已经引起了温家宝的强烈警惕和担忧是毫无疑问的。所以笔者才在上篇文章中断言：温家宝九年前，以国务院总理身份举行的最后一次中外记者会上，关于文革随时可能在中国大地上复辟的示警，表面上针对的是薄煕来和王立军所代表的当时重庆当局，话里话外更暗含了对习近平上台之后，整个中国的政治局势将会全面倒退的强烈担忧。
  <br/>
  <br/>
  有心人应该还记得，九年前即二零一二年的三月份，温家宝在他最后一次中外记者会上借不点名批判薄煕来，再次呼吁政治改革呼吁警惕文革复辟的次日，中共当局即公开宣布了薄煕来去职的消息；继而就是宣布对薄煕来违法行为进行调查；再继而就是《纽约时报》对温家宝“家族贪腐败”的“重磅炸弹”，事实上成了庆祝习近平在中共十八大上登基的礼炮。更进一步的分析内容，留待本专栏的下篇文章继续介绍 。
