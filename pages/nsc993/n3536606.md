<div><p>
 【大纪元2012年03月11日讯】
 <br/>
 擅长博采风头，
 <br/>
 常借镜头搔头。
 <br/>
 两会掉头挠头，
 <br/>
 薄三大触霉头。
 <br/>
 牵出警痞巨头，
 <br/>
 公开内斗开头。
 <br/>
 往下没啥奔头，
 <br/>
 戏倒有点看头。
</p>
<p>
 西来邪灵魔头，
 <br/>
 遇到大法上头，
 <br/>
 鸡血一冲昏头，
 <br/>
 害民欺善压头。
 <br/>
 薄三紧跟后头，
 <br/>
 猛出黑手拳头。
 <br/>
 酷刑草菅人头，
 <br/>
 活摘器官打头。
</p>
<p>
 唱红哄唱晕头，
 <br/>
 打黑黑打断头。
 <br/>
 谋反图当贼头，
 <br/>
 败露内讧砸头。
 <br/>
 无神谬论充头，
 <br/>
 作恶不知回头。
 <br/>
 不甘烂额焦头，
 <br/>
 挣扎想再出头。
</p>
<p>
 天灭中共取头，
 <br/>
 九评剑亮上头。
 <br/>
 善恶必报到头，
 <br/>
 天理应记心头。
 <br/>
 三退保命留头，
 <br/>
 向善向恶哪头？
 <br/>
 好人有好盼头，
 <br/>
 自明真相起头。@
</p>
